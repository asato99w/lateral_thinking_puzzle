# パラダイム発見

使用規則: 空間構築 → A_深度階層型

## Step 1: 緊張構造の把握

F' に内在する主要な緊張関係:

1. **驚き × 異常なし（Fs-91）**: 女性が驚いたのに地下室に異常なものがなかった — パズルの核心的緊張
2. **通報 × 異常なし（Fs-92）**: 警察に駆け込んだのに異常なものがなかった — 通報動機の不明
3. **禁止 × 異常なし（Fs-93）**: 絶対に開けるなと言われていたのに異常なものがなかった — 禁止理由の不明
4. **禁止 × 開けた（Fs-95）**: 禁止されていたのに開けた — 禁止を破った動機の問題

## Step 2: メインパスの発見

### P1: 表層パラダイム — 「禁じられた部屋に恐ろしいものがある」

Ps を字面通りに受け取った解釈。「地下室のドアを絶対に開けてはいけない」という禁止から、地下室には何か恐ろしいもの・危険なもの・秘密があると推測する。女性がドアを開けてそれを目撃し、驚いて警察に通報した、という理解。

- 解消できる緊張: 緊張4（禁止を破ったのは好奇心）
- 解消できない緊張: 緊張1（驚いたのに異常なし）、緊張2（通報したのに異常なし）、緊張3（禁止の理由が異常物ではない）

### P2: 中間パラダイム — 「異常なものは事後に消えた」

P1 の矛盾（驚いたのに異常なし）を解消するために、異常なものが一時的に存在したが事後に消えた・撤去された・隠されたという解釈を導入する。

- 解消できる緊張: 緊張1（異常なものは最初はあったが後で消えた）、緊張2（女性が見た時点では通報に足るものがあった）
- 解消できない緊張: 緊張3（禁止の理由が依然不明瞭）、「消えた」理由の説明が必要

### P3: 中間パラダイム — 「女性の主観と客観が異なる」

P2 の「消えた」説明に無理があることから、女性が驚いたものと客観的事実のずれに着目する。異常なものがなかったのは客観的判断であり、女性の主観では異常に見えた何かがあったという解釈。女性の認知・期待・精神状態が通常と異なっていた可能性。

- 解消できる緊張: 緊張1（主観と客観のずれ）、緊張2（女性の主観的判断で通報）
- 解消できない緊張: 主観がなぜずれていたかの説明が必要。禁止との関係が不明

### P4: 深層パラダイム — 「女性は嘘をつかれていた・操られていた」

女性の主観がずれていた原因として、誰かが女性に嘘をつき、現実と異なる認識を持たせていたという解釈を導入する。禁止者が女性を騙しており、「地下室」や「開けてはいけない」の意味が額面通りではない。

- 解消できる緊張: 緊張3（禁止は嘘の維持のため）、主観のずれの原因（騙されていた）
- 解消できない緊張: 嘘の具体的内容、女性が置かれていた具体的状況

### P5（T）: 最終パラダイム — 「女性は監禁されていた」

女性は地下室に監禁されており、「開けてはいけないドア」は外への出口だった。ドアを開けて見た「地下室の中」は実は外の世界であり、普通の外の世界に驚いた。監禁の事実に気づいて警察に通報した。

- 解消できる緊張: 全ての緊張が解消される

### メインパス

P1 → P2 → P3 → P4 → P5（T）

### 大区分の対応

- **大区分 I**（Fs 内の探索）: P1, P2
- **大区分 II**（Fs と Ft の境界）: P3, P4
- **大区分 III**（Ft 固有領域への到達）: P5（T）

## Step 3: P_pred のスコーピング

各パラダイムの解釈が予測を持つ記述素の範囲を特定する。

### P1 の P_pred スコープ

P1「禁じられた部屋に恐ろしいものがある」の解釈が予測を持つ範囲:

- 地下室・ドア・禁止に関する記述素（Ps-1〜Ps-3, Fs-1〜Fs-4, Fs-23〜Fs-25 等）
- 驚きの原因に関する記述素（Fs-60〜Fs-63: 何か恐ろしいものを見た系の仮説）
- 通報理由に関する記述素（Fs-71, Fs-75: 犯罪現場を発見して通報）
- 禁止理由に関する記述素（Fs-39〜Fs-43: 危険物・犯罪証拠等の仮説）

**残余誤りの確認**: P1 は「地下室に異常なものがある」と予測するが、T では異常なものはない。Fs-60〜Fs-63 等の仮説を P_pred(d)=1 として持つため、T との食い違いがある。✓

### P2 の P_pred スコープ

P2「異常なものは事後に消えた」の解釈が予測を持つ範囲:

P1 のスコープに加え:
- 事後撤去に関する記述素（Fs-87: 事後に撤去された）
- 時間経過に関する記述素
- 確認のタイミングに関する記述素（Fs-82）

**残余誤りの確認**: P2 は「異常なものが一時的に存在した」と予測するが、T ではそもそも存在しない。✓

### P3 の P_pred スコープ

P3「女性の主観と客観が異なる」の解釈が予測を持つ範囲:

P1/P2 のスコープに加え:
- 女性の認知・精神状態に関する記述素（Fs-86, Fs-88, Fs-98, Fs-100）
- 主観と客観のずれに関する記述素（D-4, D-6）
- 女性の期待と現実のギャップに関する記述素（Fs-56）

**残余誤りの確認**: P3 は「女性の精神状態に問題がある」方向に予測を持ちうるが、T では精神状態ではなく監禁が原因。また「地下室の中で」何かを見たという前提をまだ持っており、T の「外に出た」とは食い違う。✓

### P4 の P_pred スコープ

P4「女性は嘘をつかれていた」の解釈が予測を持つ範囲:

P3 のスコープに加え:
- 嘘・欺瞞に関する記述素（M-4, Ft-22, D-2）
- 禁止者の動機に関する記述素（Fs-35, Fs-36, D-7, B-6）
- 世界観の操作に関する記述素（D-1, B-1）
- 女性の自由に関する記述素（M-1, Ft-32）

**残余誤りの確認**: P4 は「嘘をつかれていた」ことまでは到達するが、監禁の具体的内容（地下室 = 監禁部屋、ドアの向こう = 外の世界）の解像度がまだ T に達しない。特に「地下室」の意味の反転（女性のいた場所こそが地下室）は P4 の時点では確定しない。✓

## Step 4: サブパスの発見

### S1: 「女性が犯罪に加担していた」（P1 から分岐）

P1 と同じ段階の緊張に対して、女性が被害者ではなく関係者・加担者である可能性を探る解釈。地下室に自分の秘密があり、発覚を恐れて警察に先手を打ったなど。

- 分岐元: P1
- 合流先: P3（女性の行動の説明が破綻し、主観のずれの問題に戻る）

### S2: 「建物自体に秘密がある」（P2 から分岐）

異常なものが「消えた」のではなく、建物の構造自体に秘密がある（二重構造、隠し部屋等）という解釈。

- 分岐元: P2
- 合流先: P4（建物の構造的秘密から、誰かが意図的に何かを隠していたという方向へ）

## Step 5: 到達性の検証

各パラダイムの T とのアノマリー数概算:

| パラダイム | P_pred 内で T と異なる記述素 | 傾向 |
|-----------|---------------------------|------|
| P1 | 多数（恐ろしいものがある、犯罪現場、禁止理由の全仮説） | 最多 |
| P2 | やや減少（事後撤去仮説は残るが、驚き自体は説明可能に） | 減少 |
| P3 | さらに減少（主観ずれは認識、ただし原因を精神状態に帰属） | 減少 |
| P4 | 少数（嘘の存在は認識、監禁の具体的内容が未到達） | 少数 |
| P5（T） | 0 | 0 |

メインパスに沿ってアノマリー数が単調に減少する傾向を確認。✓
