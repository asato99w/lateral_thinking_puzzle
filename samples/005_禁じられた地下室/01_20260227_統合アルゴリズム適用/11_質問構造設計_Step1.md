# 質問構造設計 Step 1: アノマリー目標設定

使用規則: 質問構造設計 → 質問構造設計アルゴリズム Step 1

## 各パラダイムのアノマリー記述素

Anomaly(O*, P_i) = {d ∈ O* | P_pred(P_i, d) ∈ {0, 1} かつ P_pred(P_i, d) ≠ O*(d)}

**P_pred(P1)=1 だが O*(d)=0（P1 が真と予測するが実際は偽）:**
- Fs-14（住人）
- Fs-27（危険物・秘密で禁止）
- Fs-34（善意の禁止）
- Fs-39（危険物で禁止）
- Fs-40（犯罪証拠で禁止）
- Fs-51（地下室内部を探索）
- Fs-54（異常な光景を見た）

- Fs-61（犯罪証拠を見た）
- Fs-67（犯罪目撃で警察）
- Fs-75（犯罪現場発見→通報）

**P_pred(P1)=0 だが O*(d)=1（P1 が偽と予測するが実際は真）:**
- Pt-1〜5（T の全記述素）

**|Anomaly(O*, P1)| ≈ 55+**

### P2 のアノマリー記述素

P1 との差分:
- Fs-34 が P2=0(O*=0) → 修正済み（アノマリーから脱落）
- Fs-35 が P2=1(O*=1) → 修正済み（アノマリーから脱落）
- Fs-87(事後撤去) P2=1, O*=0 → **新たなアノマリー**
- Fs-89(巧妙に隠蔽) P2=1, O*=0 → **新たなアノマリー**
- Fs-83 が P2=0(O*=1) → アノマリー（P1 から継続）

**P1 からの変化**: Fs-34, Fs-35 のアノマリー解消（-2）、Fs-87, Fs-89 の新規アノマリー（+2）。差引同程度。

P2→P3 で大幅減少。Fs-54, 60, 61, 67, 75 が P3 で正しく =0 に修正されたため。D-4, D-6, Ft-20, M-3 等が P3 で正しく =1 に修正されたため。

**P_pred(P4)=0 だが O*(d)=1:**

**P_pred(P4)=不定 で本来は確定:**

**|Anomaly(O*, P4)| ≈ 1**

P4 はアノマリーが極めて少ない。残る主要アノマリーは Fs-51（地下室内部を探索 vs 外に出た）のみ。ただし Pt 群が不定であるため、resolve の計算では不定の記述素は寄与しない点に注意。

## アノマリー質問の目標設計

### 基本方針

各パラダイム P_i に対し、そのアノマリー記述素を effect に含む質問を設計する。質問の回答がアノマリー記述素を O に追加し、P_i との矛盾が顕在化する。

### P1 のアノマリー質問目標

P1 の主要アノマリーをカテゴリごとに整理する:

**カテゴリ A: 地下室の中身に関する誤り**
- Fs-60(死体), Fs-61(犯罪証拠), Fs-54(異常光景) → 「地下室に恐ろしいものがあった」という P1 の核心が誤り
- 対応するアノマリー質問: 「地下室の中で何か恐ろしいものを見ましたか？」→ いいえ
- effect: Fs-60=0, Fs-61=0, Fs-54=0（P1 では =1 と予測、実際は =0 → アノマリー）

**カテゴリ B: 禁止理由の誤り**
- Fs-27(危険物), Fs-39(危険物禁止), Fs-40(犯罪証拠禁止), Fs-34(善意) → 禁止理由が P1 の想定と異なる
- 対応するアノマリー質問: 「ドアは危険なものから女性を守るために禁止されていましたか？」→ いいえ
- effect: Fs-27=0, Fs-34=0（P1 では =1、実際は =0 → アノマリー）

**カテゴリ C: 通報理由の誤り**
- Fs-67(犯罪目撃), Fs-75(犯罪現場通報) → 通報理由が犯罪目撃ではない
- 対応するアノマリー質問: 「女性は犯罪現場を発見したから警察に行きましたか？」→ いいえ
- effect: Fs-75=0, Fs-67=0（P1 では =1 → アノマリー）

**カテゴリ D: 女性の立場の誤り**
- Fs-14(住人) → 女性は住人ではない
- 対応するアノマリー質問: 「女性はその建物に自由に住んでいましたか？」→ いいえ
- effect: Fs-14=0（P1 では =1 → アノマリー）

**カテゴリ E: 行動の誤り**
- Fs-51(内部探索) → 中に入ったのではなく外に出た
- これは最深層のアノマリー。全パラダイム共通。

**目標数**: P1→P2 シフトに必要な resolve 条件を充足するため、少なくとも 3〜5 のアノマリー質問が必要。

### P2 のアノマリー質問目標

P2 の P1 からの追加アノマリー:
- Fs-87(事後撤去), Fs-89(巧妙に隠蔽) → P2 の核心仮説が誤り

**カテゴリ F: 事後撤去の否定**
- 対応するアノマリー質問: 「異常なものが一時的にあったが後で消えましたか？」→ いいえ
- effect: Fs-87=0（P2 では =1 → アノマリー）

P2→P3 シフトには、P1 のアノマリーに加え Fs-87 等の新規アノマリーが蓄積する必要がある。

### P3 のアノマリー質問目標（修正後）

P3 の主要アノマリー:
- Fs-88(精神的問題) → P3 の核心仮説が誤り
- Fs-51(内部探索) → 全パラダイム共通

**カテゴリ G: 精神的問題の否定**
- 対応するアノマリー質問: 「女性の驚きは精神的な問題が原因ですか？」→ いいえ
- effect: Fs-88=0（P3 では =1 → アノマリー）

**カテゴリ H: T 側の記述素の部分的浮上**
- P3=0 だが O*=1: 多数のFt群 → これらは質問を通じて O に入る必要がある
- Ft-22(虚偽世界観)=0→O*=1、M-4(嘘の人物)=0→O*=1 等は P3→P4 シフトの鍵

### P4 のアノマリー質問目標

P4 のアノマリー:
- Fs-51(内部探索) のみ

**カテゴリ I: 空間の反転**
- 対応するアノマリー質問: 「女性は地下室の中に入りましたか？」→ いいえ（外に出た）
- effect: Fs-51=0（P4 では =1 → アノマリー）

P4→P5 シフトには、Pt群の不定→確定が主要な経路。質問によって Pt-1(監禁) 等が O に入ることが鍵。

## 完全性条件の確認

各パラダイムの全アノマリー記述素が少なくとも一つの質問候補の effect に含まれるべき。

### P1 の完全性

| アノマリー記述素 | カバーする質問カテゴリ |
|---|---|
| Fs-14(住人) | D |
| Fs-27(危険物) | B |
| Fs-34(善意) | B |
| Fs-39(危険物禁止) | B |
| Fs-40(犯罪証拠禁止) | B |
| Fs-51(内部探索) | E |
| Fs-54(異常光景) | A |
| Fs-60(死体) | A |
| Fs-61(犯罪証拠見た) | A |
| Fs-67(犯罪目撃) | C |
| Fs-75(犯罪現場通報) | C |
| Pt-1〜5, Ft群, D群, B群, M群（=0→O*=1） | 各質問を通じて O に追加 |

P_pred=0 だが O*=1 のアノマリー（Pt/Ft/D/B/M群）は、対応する質問が T 方向の回答をもたらすことでカバーされる。これらは後の Phase で具体的質問に紐づける。

### P2/P3/P4 の完全性

各追加アノマリーに対応する質問カテゴリが存在する。 → ✓
