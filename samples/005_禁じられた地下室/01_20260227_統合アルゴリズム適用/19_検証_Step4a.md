# 検証 Step 4a: 完全性検証

使用規則: 補完検証 → Step 1

## 方針

各パラダイム P_i（T を除く）について、RequiredAnomaly(P_i) = Anomaly(P_i) \ UpperShared(P_i) が質問群の effect でカバーされていることを検証する。

- **局所条件**: RequiredAnomaly(P_i) が Q(P_i) でカバーされること
- **大域条件**: 全アノマリー記述素がいずれかの質問でカバーされていること

| 検証 | P1 | P2 | P3 | P4 | 状態 |
|---|---|---|---|---|---|
| 局所条件 | ✓ | **✗** | ✓ | ✓ | **NG** |
| 大域条件 | — | — | — | — | **NG** |

**不足**: Fs-40（犯罪の証拠があるため禁止された、P1=1, P2=1, O*=0）がいずれの質問の effect にも含まれていない。

**対処方針**: Phase 3 への差し戻し。q-02 の ans_no の effect を {Fs-39:0, Fs-27:0} → {Fs-39:0, Fs-27:0, **Fs-40:0**} に拡張する。q-02 は「危険なもの（爆発物や毒物など）があるから禁止されていましたか？」であり、「いいえ」の回答は犯罪証拠による禁止も否定すると解釈できる。
