# 質問構造設計 Step 2: 認知的到達経路の逆算

使用規則: 質問構造設計 → 質問構造設計アルゴリズム Step 2

## 方針

各アノマリー質問カテゴリについて、「この質問を思いつくには何を事前に知る必要があるか」を逆方向に展開する。前提なし（層0）に到達するまで逆算する。

## P1 のアノマリー質問への到達経路

### カテゴリ A: 地下室の中身（Fs-54, 60, 61）

**目標質問 A**: 「地下室の中には恐ろしいものがありましたか？」
- effect: {(Fs-60, 0), (Fs-54, 0)} — 恐ろしいものはない
- P1: Fs-60=1, Fs-54=1 → アノマリー

**逆算**: この質問は層 0 で前提なしに問える。P1 では「地下室に恐ろしいものがある」が核心的仮説であり、プレイヤーは問題文を読んだ直後にこの仮説を持つ。

→ **層 0 質問**（前提なし）

### カテゴリ B: 禁止理由（Fs-27, 34, 39, 40）

**目標質問 B1**: 「ドアは危険なものがあるから禁止されていましたか？」
- effect: {(Fs-27, 0), (Fs-39, 0)} — 危険物ではない
- P1: Fs-27=1, Fs-39=1 → アノマリー

**逆算**: 禁止理由を問うには、禁止の存在（Ps-2）を認識していればよい。Ps-2 は初期観測に含まれる。

→ **層 0 質問**（前提: Ps-2、初期観測に含まれる）

**目標質問 B2**: 「禁止は女性を守るためでしたか？」
- effect: {(Fs-34, 0)} — 善意ではない
- P1: Fs-34=1 → アノマリー

**逆算**: B1 と同様に、禁止の存在から直接問える。

→ **層 0 質問**

### カテゴリ C: 通報理由（Fs-67, 75）

**目標質問 C**: 「女性は犯罪を目撃したから警察に行きましたか？」
- effect: {(Fs-75, 0), (Fs-67, 0)} — 犯罪目撃ではない
- P1: Fs-75=1, Fs-67=1 → アノマリー

**逆算**: 通報理由を問うには、女性が警察に行った事実（Ps-5）を認識していればよい。Ps-5 は初期観測。

→ **層 0 質問**

### カテゴリ D: 女性の立場（Fs-14）

**目標質問 D**: 「女性はその建物に自由に住んでいましたか？」
- effect: {(Fs-14, 0)} — 自由に住んでいない
- P1: Fs-14=1 → アノマリー

**逆算**: 女性の立場を問うには前提なし。問題文の女性について直接問える。

→ **層 0 質問**

ただし、この質問は P1 だけでなく P2, P3 にとっても「住人」の前提を崩す重要な質問。認知的には「なぜ女性はその場にいるのか」を問う基本的質問。

### カテゴリ E: 空間の反転（Fs-51）

**目標質問 E**: 「女性は地下室の中に入りましたか？」
- effect: {(Fs-51, 0), (Ft-21, 1)} — 中に入ったのではなく外に出た
- P1〜P4: Fs-51=1 → アノマリー

**逆算**:
この質問が自然に思いつくには、「中に入った」という問題文の記述に疑念を持つ必要がある。

ステップ 1: 問題文の記述が女性の主観視点であることを知る必要がある
- 前提: Ft-20（S は女性の主観）= 1 を知っている
- Ft-20 を知るための前提: D-6（S 全体が主観で客観とずれ）

ステップ 2: 主観と客観がずれていることを知る必要がある
- 前提: Fs-100（主観と客観の乖離）= 1 を知っている

ステップ 3: 乖離を知るための前提
- 前提: 「異常なものはないのに驚いた」矛盾（Fs-91）を知っている → 初期観測
- かつ: 恐ろしいものがなかった（A の回答）、禁止理由が危険物でない（B の回答）等の蓄積

→ **深層質問**（層 2〜3）。カテゴリ A, B, C の回答蓄積 → 主観客観ずれの認識 → 空間の反転

### カテゴリ F: 事後撤去の否定（Fs-87, 89）

**目標質問 F**: 「異常なものがかつては地下室にあったが後で消えましたか？」
- effect: {(Fs-87, 0), (Fs-89, 0)} — 事後撤去はない
- P2: Fs-87=1 → アノマリー

**逆算**: P2 の解釈に到達している（P1 のアノマリー蓄積後）プレイヤーが自然に問う質問。「恐ろしいものはない」と知った後に、「じゃあ一時的にはあったのでは」と考える。

前提:
- カテゴリ A の回答（恐ろしいものはない）を知っている
- まだ「最初からなかった」とは確信していない

→ **層 1 質問**（前提: A の回答）

### カテゴリ G: 精神的問題の否定（Fs-88）

**目標質問 G**: 「女性の驚きは女性自身の精神的な問題が原因ですか？」
- effect: {(Fs-88, 0)} — 精神的問題ではない
- P3: Fs-88=1 → アノマリー

**逆算**: P3 の解釈に到達しているプレイヤーが問う質問。「主観と客観がずれている」と認識した上で、「女性自身の問題か？」と考える。

前提:
- Fs-100（主観客観の乖離）を知っている
- カテゴリ F の回答（事後撤去でもない）を知っている
- → 「物がないなら女性の問題？」という推論

ステップ 1: 恐ろしいものがない（A の回答）
ステップ 2: 事後撤去でもない（F の回答）
ステップ 3: → 「女性の認知の問題？」と推論

→ **層 1〜2 質問**（前提: A + F の回答）

### カテゴリ H: 嘘・欺瞞の浮上（Ft-22, M-4）

**目標質問 H**: 「誰かが女性に嘘をついていましたか？」
- effect: {(M-4, 1), (Ft-22, 1)} — 嘘をついていた人物がいる
- P3: M-4=0, Ft-22=0 → 回答が O に入り P3 と矛盾

**逆算**: 嘘の存在を問うには、「女性の認知が歪んでいた」ことを知り、かつ「女性自身の問題ではない」（G の回答）ことを知る必要がある。

前提:
- Fs-88=0（精神的問題でない）→ G の回答
- Fs-100=1（主観客観の乖離がある）
- → 「自分の問題でないなら外部に原因？」→ 「嘘をつかれていた？」

→ **層 2 質問**（前提: G + 主観客観乖離の認知）

### カテゴリ I: 監禁の具体化（Pt-1〜5）

**目標質問 I1**: 「女性は自由に外出できる状況にありましたか？」
- effect: {(M-1, 1)} — 外出不能
- P3 以前: M-1=0 → アノマリー

**逆算**: 女性の自由について問うには、嘘の存在を知った上で「どのような制約を受けていたか」を探る。

前提:
- M-4=1（嘘をつかれていた）→ H の回答
- Fs-14=0（住人ではない）→ D の回答
- → 「住人でもなく嘘もつかれていたなら、制約を受けていた？」

→ **層 2〜3 質問**

**目標質問 I2**: 「女性は監禁されていましたか？」
- effect: {(Pt-1, 1)} — 監禁されていた
- 全パラダイム（P1〜P3）: Pt-1=0 → アノマリー

**逆算**: 監禁の概念に到達するには、外出不能 + 嘘の存在 + 禁止の構造 の蓄積が必要。

前提:
- M-1=1（外出不能）→ I1 の回答
- M-4=1（嘘）→ H の回答
- Ft-15 or D-7（禁止は犯罪隠蔽/監禁維持）
- → 「外出できず嘘もつかれていた → 監禁？」

→ **層 3 質問**（深層）

**目標質問 I3**: 「『地下室の中に入った』とは実際には外に出たということですか？」
- effect: {(Fs-51, 0), (Ft-21, 1), (Ft-16, 1), (Ft-26, 1)} — 空間の反転
- P1〜P4: Fs-51=1 → アノマリー

**逆算**: 空間反転に到達するには、監禁の認識 + 問題文が主観視点であることの認識が必要。

前提:
- Pt-1=1（監禁）→ I2 の回答
- Ft-20=1（主観視点）
- → 「監禁されていて主観視点なら、地下室=監禁部屋、中に入った=外に出た？」

→ **層 3〜4 質問**（最深層）

## 到達経路の DAG 構造

```
層 0（前提なし）:
  A: 恐ろしいもの?→いいえ
  B1: 危険物で禁止?→いいえ
  B2: 善意で禁止?→いいえ
  C: 犯罪目撃で通報?→いいえ
  D: 自由に住んでいた?→いいえ
       |
       ↓
層 1（A/B/C の回答を前提）:
  F: 事後撤去?→いいえ
  (追加質問: 女性の期待と現実にずれ?→はい)
       |
       ↓
層 2（F + 乖離の認知を前提）:
  G: 精神的問題?→いいえ
  H: 誰かが嘘?→はい
       |
       ↓
層 3（G/H + D の回答を前提）:
  I1: 外出できた?→いいえ
  I2: 監禁されていた?→はい
       |
       ↓
層 4（I2 + 主観視点を前提）:
  I3: 中に入った=外に出た?→はい
```

## 各パラダイムと層の対応

| 層 | 主要質問 | 主にアノマリーとなるパラダイム | シフト方向 |
|---|---|---|---|
| 層 0 | A, B1, B2, C, D | P1 | P1→P2 |
| 層 1 | F + 補助 | P2 | P2→P3 |
| 層 2 | G, H | P3 | P3→P4 |
| 層 3 | I1, I2 | P4 | P4→P5 |
| 層 4 | I3 | P4（残余） | → T |

## 中間記述素の確認

各逆算ステップで参照される記述素が F' に存在するか:

| ステップ | 参照記述素 | F' に存在 |
|---|---|---|
| A→F | Fs-60=0, Fs-54=0 | ✓ |
| A+B→主観客観認識 | Fs-100 | ✓ |
| F→G | Fs-87=0 | ✓ |
| G→H | Fs-88=0, Fs-100=1 | ✓ |
| H+D→I1 | M-4=1, Fs-14=0 | ✓ |
| I1→I2 | M-1=1, M-4=1 | ✓ |
| I2→I3 | Pt-1=1, Ft-20=1 | ✓ |

→ 中間記述素の不足は検出されない。
