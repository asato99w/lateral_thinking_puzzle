# 入力

## 問題文

砂漠の真ん中で裸の男が死んでいた。手にはマッチ棒を握りしめている。

## Ps（問題文の命題分解）

| ID | 内容 |
|----|------|
| Ps-1 | 砂漠の真ん中で発見された |
| Ps-2 | 男が死んでいた |
| Ps-3 | 男は裸だった |
| Ps-4 | 手にマッチ棒を握りしめていた |

## 真相

男は数人で熱気球に乗っていたが、高度が下がり始めた。荷物を全て捨て、服も脱いで軽量化したが足りず、くじ引き（マッチの長さ）で一人が飛び降りることになった。短いマッチを引いた男が飛び降りて死亡した。

## Pt（真相の命題分解）

| ID | 内容 |
|----|------|
| Pt-1 | 男は数人で熱気球に乗っていた |
| Pt-2 | 熱気球の高度が下がり始めた |
| Pt-3 | 荷物を全て捨てて軽量化した |
| Pt-4 | 服も脱いで軽量化した |
| Pt-5 | 軽量化が足りなかった |
| Pt-6 | くじ引きで一人が飛び降りることになった |
| Pt-7 | マッチの長さでくじ引きをした |
| Pt-8 | 男は短いマッチを引いた |
| Pt-9 | 男は熱気球から飛び降りて死亡した |
