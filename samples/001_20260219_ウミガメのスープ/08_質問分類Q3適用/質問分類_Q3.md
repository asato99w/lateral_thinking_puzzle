| カテゴリ | 初期オープン | 後発 | 全体 |

最大/最小 = 30/10 = **3.0**（閾値 > 3 のちょうど境界）

### バランスの検討

- **A が最小（10問）**: レストランの場面設定に関する質問は限定的。問題文の舞台装置であり、パズルの核心はレストラン外の出来事にある。来店関連の質問がこれ以上存在しないため、他カテゴリから移動できる候補がない
- **D が最大（30問）**: 男の過去の経験（遭難、仲間、生存状況）に関する質問が多数。このパズルの構造上、謎の核心が男の過去にあるため自然な偏り
- **B も大きい（29問）**: スープの材料に関する質問（材料の種類、過去のスープとの比較等）が多い。スープが問題文の中心要素であることから自然

**分割の検討**:
- D の分割: 初期オープン質問（q05:専門家, q75:職業, q79:年齢）は全て「男の属性」であり、自然な下位区分（例:「男の過去」と「男の属性」）を初期オープンから導出できない。分割すると「男の過去」カテゴリの名称がP1外の情報を示唆するリスクがある
- B の分割: 初期オープン質問はスープの性質（味、品質、真偽）に集中しており、「スープの材料」と「スープの味」に分ける自然な根拠は初期オープンからは得られない

**結論**: 比率 3.0 は閾値の境界であり、かつ分割による情報漏洩リスクを考慮し、現行の 5 カテゴリ構成を維持する。

## Step 5: 情報漏洩検証

### カテゴリ名の検証

| カテゴリ | 名称が示唆する内容 | P1外の情報を含むか | 判定 |
|---------|------------------|-------------------|------|
| A | レストラン、来店 | 問題文の場面設定そのもの | ✓ 漏洩なし |
| B | スープ、味 | 問題文の中心要素 | ✓ 漏洩なし |
| C | 男の行動、認識 | 問題文の主人公の振る舞い | ✓ 漏洩なし |
| D | 男の人物像、背景 | 一般的な人物情報 | ✓ 漏洩なし |
| E | 自殺の原因 | 問題文の問い（なぜか？） | ✓ 漏洩なし |

いずれのカテゴリ名も、問題文から直接読み取れる要素のみを反映している。「遭難」「仲間」「人肉」「騙し」等の深いパラダイムの内容は、どのカテゴリ名にも含まれない。

### パラダイム集中の検証

各カテゴリに含まれる後発質問の effect 記述素がどのパラダイムに主に属するかを確認する。

| カテゴリ | 関連するパラダイムの分布 | 特定パラダイムへの過度な集中 |
|---------|----------------------|------------------------|
| A | P1 のみ（後発 2 問とも P1 スコープ） | ✓ A は来店関連で P1 に限定されるのが自然 |
| B | P1, P2, P3, P4, P5 全てにまたがる | ✓ 集中なし |
| C | P1, P2, P3, P5 にまたがる | ✓ 集中なし |
| D | P1(shift), P3, P4 に偏りあり | ※下記検討 |
| E | P1(shift), P3, P4, P5 にまたがる | ✓ 集中なし |

**D のパラダイム偏りについて**:

D には遭難（P3 スコープ）・仲間の行動（P4 スコープ）に関する質問が集中している。しかし:

1. カテゴリ名「男の人物像と背景」は、遭難や仲間を直接示唆しない
2. これらの質問は prerequisites 付きであり、ゲーム開始時には非オープン。プレイヤーがカテゴリ一覧を見ても質問内容は見えない
3. 質問がオープンする時点では、プレイヤーは既にパラダイムシフトを経験しており、「男の背景」カテゴリに遭難関連の質問が現れることは自然

→ 情報漏洩リスクは低い。✓

### カテゴリサイズの動的変化

ゲーム進行に伴う各カテゴリのサイズ変化が、パラダイム遷移のタイミングを示唆しないかを検証する。

- 各パラダイムシフトで追加される質問は複数カテゴリに分散する（B, C, D, E にまたがる）
- 特定のカテゴリが一斉に大幅増加するタイミングはない
- カテゴリサイズの変化からパラダイム遷移の時期を特定することは困難

→ 漏洩リスクなし。✓

## Step 6: 最終検証

### カテゴリの安定性

| 検証項目 | 結果 |
|---------|------|
| 全カテゴリがゲーム開始時に質問を持つか | ✓（A:8, B:7, C:5, D:3, E:5） |
| カテゴリがゲーム中に消滅するか | ✓ 消滅しない（各カテゴリの初期オープン質問は常に存在） |
| 後発質問の割り当ての自然さ | ✓ 概ね自然（下記補足参照） |

### 後発質問の割り当てに関する補足

一部の質問は複数カテゴリにまたがる性質を持つ:

- **q50**（仲間が男にスープを飲ませましたか？）: スープの提供者に関する質問としてBに割り当て。Dへの割り当ても自然だが、「誰がスープを作った/出したか」はスープについての疑問として B が主要
- **q60**（本物の味を知って人肉だったと悟った）: 男の悟りとしてCに割り当て。Bへの割り当ても自然だが、質問の焦点は「男が悟った」という認識行為
- **q73**（仲間の肉をスープにすること）: スープの製造に関する質問としてBに割り当て。Dへの割り当ても自然だが、「スープの材料は何か」という問いの延長線上
- **q66**（命に関わる状況でスープを飲んだ）: 過去の状況としてDに割り当て。Bへの割り当ても自然だが、焦点は「どのような状況だったか」

これらの二重性は、テキストベースの分類の限界であり、情報漏洩には該当しない。
