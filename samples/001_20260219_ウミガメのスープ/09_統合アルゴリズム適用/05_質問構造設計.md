# 質問構造設計（Phase 2）

使用規則: 質問構造設計
使用個別アルゴリズム: 質問構造設計

Phase 1 の出力（記述素空間、パラダイム群、内部構造）を受けて、アノマリーからの逆算により各パラダイムの質問構造を設計する。

## Step 1: アノマリー目標設定

各パラダイム P_i（T を除く）について、P_pred(P_i) のキーのうち T と矛盾する記述素を列挙し、アノマリー質問の目標を設定する。

### P1

**アノマリー記述素（9 個）:**
- Ft-03(D⁻=0, T=1): 生命の危機
- Ft-09(D⁻=0, T=1): 飢餓状態
- Ft-11(D⁻=0, T=1): 極限状態
- Ft-22(D⁻=0, T=1): 材料を隠す意図
- Ft-37(D⁻=0, T=1): 過去のスープと別食材
- Fs-26(D⁻=0, T=1): 過去にスープを飲んだ経験
- Fs-39(D⁻=0, T=1): スープと自殺の因果関係
- Fs-40(D⁻=0, T=1): 耐えられない何かを知った
- Fs-43(D⁻=0, T=1): 生き続けられないほどのもの

**偶然の開き構造の候補:**

P1 の D⁺ 記述素と D⁻ 記述素を組み合わせ、両方に自然に言及できる質問を設計する。

| 開放記述素（D⁺, safe） | 異常記述素（D⁻, anomaly） | 質問の自然さ |
|----------------------|------------------------|------------|
| Fs-17（味を知覚した） | Ft-37（別食材） | ○ 味に関する質問で両方に触れられる |
| Fs-01（食事目的） | Fs-39（スープと自殺の因果） | ○ 来店目的の質問で食事の帰結に触れられる |
| Fs-19（味の確認） | Fs-26（過去の経験） | ○ 確認行為の質問で過去の比較を含められる |
| Fs-24（判別基準） | Fs-26（過去の経験） | ○ 判別能力の質問で経験の有無に触れられる |
| Fs-32（店員は嘘なし） | Ft-22（隠す意図） | △ 嘘の文脈で両方に触れられるが少し遠い |

**目標アノマリー質問数:** P2 への遷移を成立させるため、偶然の開きを持つ質問が少なくとも 4〜5 個必要。

- Ft-09(D⁻=0, T=1): 飢餓状態

**目標アノマリー質問数:** P3 への遷移を成立させるため、偶然の開きを持つ質問が少なくとも 3 個必要。

## Step 2: 認知的到達経路の逆算

各アノマリー質問への認知的到達経路を、目的地から逆方向に設計する。

### P1 のアノマリーへの逆算

#### Ft-37（別食材）への経路

```
層0: Fs-17「味を知覚した」→ Fs-19「味を確認するために飲んだ」
       ↓（同化: 味の確認 → 過去との比較を想起）
層1: Fs-26「過去にスープを飲んだ経験がある」→ Fs-25「真偽確認は過去と関係」
       ↓（同化: 過去の体験 → 味の違いの可能性を想起）
層2: Ft-37「過去のスープと別の食材である」★ anomaly
```

#### Fs-39, Fs-40, Fs-43（自殺の因果系列）への経路

```
層0: Ps-6「自殺した」（初期既知）
       ↓（直接: 自殺の原因は何かを問う）
層1: Fs-39「スープと自殺の因果関係」★ anomaly
       ↓（同化: 因果関係がある → 具体的な内容）
層2: Fs-40「耐えられない何かを知った」★ anomaly
       ↓（同化: 耐えられない → 程度の確認）
層3: Fs-43「生き続けられないほどのもの」★ anomaly
```

#### Ft-03, Ft-09, Ft-11（危機系列）への経路

```
層0: Fs-61「男の過去に重大な出来事があった」（irrelevant 質問だが層0で問える）
       ↓（同化: 重大な出来事 → 具体的な背景）
層1: Ft-50「過去に危機的状況に置かれた」
       ↓（同化: 危機的状況 → 生命の危機）
層2: Ft-03「生命の危機」★ anomaly
       ↓（同化: 生命の危機 → 具体的な状況）
層3: Ft-09「飢餓状態」★ anomaly
       → Ft-11「極限状態」★ anomaly
```

#### Ft-22（隠す意図）への経路

```
層0: Fs-27「飲まされた」（Fs-26 の到達後にオープン）
       ↓（prerequisites: Fs-26）
層1: Ft-22「材料を隠す意図があった」★ anomaly
```

注: Fs-26 → Fs-27 は R(P2) で接続される（w=0.8）。P1 フェーズでは Fs-26 はアノマリーであるため、Fs-27 への到達は P2 遷移後に自然に行われる。

### P2 のアノマリーへの逆算

#### Ft-37（別食材）への経路

```
層0: Fs-26「過去にスープを飲んだ」（P2 の D⁺、同化で H が高い）
       ↓（同化: 過去のスープ → 今回との比較）
層1: Ft-37「過去のスープと別食材」★ anomaly
```

P2 フェーズでは Fs-26 が D⁺ であり H が高いため、Ft-37 への到達は比較的浅い。

#### Ft-03, Ft-09, Ft-11（危機系列）への経路

```
層0: Fs-27「飲まされた」→ Ft-36「記憶を確かめたかった」
       ↓（同化: なぜ飲まされたのか → 背景の探索）
層1: Ft-50「危機的状況」
       ↓
層2: Ft-03「生命の危機」★ anomaly
       → Ft-09「飢餓状態」★ anomaly
       → Ft-11「極限状態」★ anomaly
```

### P3 のアノマリーへの逆算

#### Ft-03, Ft-09（危機系列）への経路

```
層0: Ft-22「材料を隠す意図」（P3 の D⁺）
       ↓（同化: なぜ隠す必要が → 状況の深刻さ）
層1: Ft-03「生命の危機」★ anomaly
       → Ft-09「飢餓状態」★ anomaly
```

P3 フェーズでは嘘の動機を問うことが自然であり、「生命の危機下で隠蔽」への到達は比較的短い。

### P4 のアノマリーへの逆算

#### Ft-47（動物の肉）への経路

```
層0: Ft-28「通常食べないもの」（P4 の D⁺）
       ↓（同化: 何を食べたのか → 具体的な推測）
層1: Ft-47「動物の肉」★ anomaly（T=0, 実際は人肉）
```

P4 フェーズでは「通常食べないもの」から「動物の肉」への推測は自然だが、その否定（人肉）への到達は P5 フェーズを要する。

## Step 4: 素材不足の検出

### 中間記述素の不足

逆算で必要とされる中間ステップの記述素がすべて F' に存在することを確認:

- Ft-50（危機的状況）: F' に存在 ✓
- Fs-61（過去に重大な出来事）: F' に存在 ✓
- Ft-36（記憶を確かめたかった）: F' に存在 ✓

**素材不足なし。**

### ゲート辺の不足

Step 3 で設計されたゲート辺のうち、Phase 1 で構成した R(P) にないもの:

| パラダイム | ソース | ターゲット | 状態 |
|-----------|--------|-----------|------|
| P1 | Ps-6 | Fs-39 | 新規追加が必要 |
| P2 | Fs-27 | Ft-50 | 新規追加が必要 |
| P3 | Ft-22 | Ft-03 | 新規追加が必要 |

これらは Phase 1 の R(P) に追加する（Rule 5 ゲート辺）。

### 経路の疎な領域

- P1: Fs-39 → Fs-40 → Fs-43 の経路は単一だが、Fs-39 への到達が層0 から直接可能（Ps-6 起点）であるため冗長性は許容範囲
- P2: Ft-37 への経路は Fs-26 から直接であり、Fs-26 は P2 の中核記述素のため到達は堅牢
- P3, P4: アノマリー数が少なく（2個、1個）、経路の単一性は構造上不可避

**重大な素材不足なし。Phase 1 への差し戻し不要。**

## Phase 3 への引き継ぎ

以下が Phase 3（質問生成）への入力となる:

1. **アノマリー目標**: 各パラダイムのアノマリー記述素と偶然の開き構造の候補
2. **認知的到達経路**: 各アノマリーへの逆算された認知ステップの連鎖
3. **層配置**: 各パラダイムの質問の層割り当て
4. **ゲート辺**: R(P) に追加されたゲート辺の仕様
5. **素材不足レポート**: 重大な不足なし
