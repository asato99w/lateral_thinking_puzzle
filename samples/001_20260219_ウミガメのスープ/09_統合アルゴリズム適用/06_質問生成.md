# 質問生成（Phase 3）

使用規則: 質問生成
参照: Q1（質問候補生成）、Q2（フェーズ到達・接続性検証）

## 適用概要

Phase 2 の質問構造設計（層配置、認知的到達経路）をもとに、具体的な質問を生成する。

凡例:
- **safe**: P_pred と T が一致する記述素（3a でオープンの根拠となる）
- **anomaly**: P_pred と T が不一致の記述素（O に入るとアノマリーとなる）
- **prereq**: prerequisites（O に含まれている必要がある記述素）
- **paradigms**: 質問の所属パラダイム。Q(P) はこの所属に基づいて導出される

### 所属パラダイム割り当て

各質問はそのフェーズのパラダイムに所属する。Q(P) = {q | P ∈ q.paradigms}。

| 質問 ID | paradigms | 根拠 |
|---------|-----------|------|
| q01〜q24 | [P1] | P1 フェーズの質問（層0初期・探索・否定仮説・層1-2アノマリー） |
| q25〜q29 | [P2] | P2 フェーズの質問（層0探索） |
| q30〜q32 | [P1, P2] | P2 アノマリー質問（P1 でもアノマリーを導入） |
| q33〜q45 | [P3] | P3 フェーズの質問（層0嘘の構造探索・層1アノマリー） |
| q46〜q58 | [P4] | P4 フェーズの質問（層0遭難探索・層1アノマリー） |
| q59〜q65 | [P5] | P5 フェーズの質問（最終確認） |
| q66 | [P1,P2,P3,P4,P5,S1,S2] | 完全解答質問（全パラダイムで常時可能） |
| q67 | [P1,S1] | サブパス分岐質問（P1 初期オープン + S1 のアノマリー検証） |
| q68 | [P1,S2] | サブパス分岐質問（P1 初期オープン + S2 のアノマリー検証） |
