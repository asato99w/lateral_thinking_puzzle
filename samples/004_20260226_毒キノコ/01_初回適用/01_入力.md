# 入力

## 問題文

Aさんは、辺りに毒キノコしかないのを見て安堵した。なぜだろうか。

## Ps（問題文の命題分解）

| ID | 内容 |
|----|------|
| Ps-1 | Aさんがいる場所がある |
| Ps-2 | 辺りに毒キノコしかなかった |
| Ps-3 | Aさんはそれを見た（観察した） |
| Ps-4 | Aさんは安堵した |

## 真相

Aさんはキノコ狩りの最中に山で遭難していた。辺りに毒キノコしか残っていないということは、食用キノコは誰かが既に採った＝この辺りに人が来ているということ。近くに人里や他の採集者がいると分かり、助かる見込みがあると安堵した。

## Pt（真相の命題分解）

| ID | 内容 |
|----|------|
| Pt-1 | Aさんはキノコ狩りをしていた |
| Pt-2 | Aさんは山にいた |
| Pt-3 | Aさんは遭難していた |
| Pt-4 | 食用キノコは誰かが既に採っていた |
| Pt-5 | この辺りに人が来ている |
| Pt-6 | 近くに人里や他の採集者がいる |
| Pt-7 | 助かる見込みがあると判断した |
