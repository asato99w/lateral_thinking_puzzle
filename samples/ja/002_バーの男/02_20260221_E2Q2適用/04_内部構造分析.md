## B/M 記述素の各パラダイムへの所属

### 所属判定の方針

各パラダイムの「解釈」に照らして、B/M 記述素の真偽予測を判定する。

- D⁺: パラダイムがその記述素を真と予測する
- D⁻: パラダイムがその記述素を偽と予測する
- 不定: パラダイムのスコープ外（D(P) に含めない）

### 判定根拠

**P1（日常的やりとり）**: 全 B/M を D⁻ と予測。Fs-09（喉渇き）と Fs-16（脅し）が P1 の中核であり、「身体的事情」「助け」「治療」等の概念はすべて否定される。

**P2（善意）**:
- B-02, B-03, B-04 → D⁺: P2 は銃≠脅し、助け、目的達成を認めるため、これらの架け橋を受容
- B-01, B-06, M-01, M-05 → D⁻: P2 は Fs-09（喉渇き）を継承しており、身体的事情・症状関連は否定
- B-05, M-02, M-03, M-04 → 不定: P2 は具体的な問題・治療方法を知らない

**P3（隠れた問題）**:
- B-01〜B-04, M-01, M-02, M-04 → D⁺: P3 は身体的問題（Fs-10）を中核とし、水対策・援助・治療意図を認める
- M-03 → D⁻: P3 は「隠れた」問題と解釈するため、外部からの気づきを否定
- B-05, B-06, M-05 → 不定: 解決手段の具体的メカニズムを P3 は知らない

**P4（= T）**: 全 B/M を D⁺ と予測。真相としてすべて正しい。
