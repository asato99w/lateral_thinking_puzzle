# Explained 包含連鎖の検証

使用規則: 空間構築 Step 4

## 定義

Explained(P) = {d ∈ O* | P_pred(P, d) ∈ {0, 1} かつ P_pred(P, d) = O*(d)}

O* は完全確定状態。T が各記述素に対して真偽を持つか不定かを決定する。

O*(d) の決定:
- T が真と定めるもの → O*(d) = 1
- T が偽と定めるもの → O*(d) = 0
- T のもとで不定 → O*(d) は定義されない（O* に含まれない）

P5（T）の P_pred がそのまま O* の構造を反映する。ただし O* は「T が真偽を持つ」記述素のみを含む。T のもとで不定の記述素（Fs-5〜8, 10, 18, 26, 28〜30, 38, 47〜48, 68〜70, 80〜82, Ft-4〜8, 11, 18〜19 等）は O* に含まれない。

## 包含連鎖の検証

### P1 ⊂ P2

Explained(P2) = Explained(P1) ∪ {Fs-34, Fs-35}
→ ✓ Explained(P1) ⊂ Explained(P2)

### P2 ⊂ P3

Explained(P3) = Explained(P2) ∪ {Fs-83, 86, 98, 100, D-4, D-6, Ft-20, M-3, Fs-54, 60, 61, 67, 75, 87, 89} \ {Fs-34, Fs-35}

**問題発見**: Fs-34 と Fs-35 が P2 の Explained に含まれていたが P3 で不定になっている。

- Fs-34: P2=0(O*=0)✓ → P3=不定 → 脱落
- Fs-35: P2=1(O*=1)✓ → P3=不定 → 脱落

これは Explained(P2) ⊄ Explained(P3) を意味する可能性がある。

ただし、P3 で追加される 15 要素は P2 にない新規要素であり、脱落は 2 要素のみ。**厳密な包含関係は不成立**。

### 対処

規則「空間構築」Step 4:
> 不成立の場合、P_{i+1} の P_pred に不足している記述素を特定し、Step 3 に戻って P_pred を修正する。浅いパラダイムが正しく予測する表面的事実が、深いパラダイムの P_pred から漏れていることが典型的な原因である。

**不足記述素の特定:**
1. **Fs-34（禁止者は女性を危険から守ろうとしていた）**: P2=0, O*=0（T では善意の禁止ではない）。P3 は禁止者の動機を限定しないため不定としたが、P3 の「主観と客観が異なる」解釈からも善意の禁止を否定することは可能。
2. **Fs-35（禁止者は自分の秘密を守ろうとしていた）**: P2=1, O*=1（T では秘密の隠蔽）。P3 は禁止者の動機を限定しないため不定としたが、P3 の「主観が歪んでいる」解釈から「なぜ禁止者は強く禁止したのか」を考えると、秘密の保持は不定のままでも不自然ではない。

**修正方針:**
P3 の P_pred を以下のように修正する:
- Fs-34 = 0: P3 の解釈「主観と客観がずれている」状況で、禁止者が善意であるかどうかについて、P3 は「善意の禁止では主観客観のずれは生じにくい」と推論できる。善意なら女性に正直に理由を伝えるはずであり、ずれの原因にならない。根拠: Fs-100（主観客観の乖離）→ 善意の禁止は乖離の説明にならない → Fs-34=0
- Fs-35 = 1: P3 の解釈から「禁止者に何らかの隠し事がある」ことは、主観の歪みの間接的な原因として導出可能。女性の主観が歪んでいるのは、禁止者が何かを隠しているからこそ強い禁止をし、女性が禁止の背景について不正確な情報を持っていたため。根拠: Fs-100（乖離）→ 乖離の原因として禁止者の秘密が推定 → Fs-35=1

この修正により:
- Fs-34: P3=0(O*=0)✓ → Explained に残留
- Fs-35: P3=1(O*=1)✓ → Explained に残留

**修正後は Step 1-3 に戻り P3 を更新する必要がある。**

### P3 ⊂ P4 （修正後の P3 を前提）

修正後の P3 の Explained に含まれる全要素が P4 でも含まれるか:
- P3 の =1 は全て P4 で =1 のまま、または P4 で依然 =1 → ✓
- P3 の =0 は全て P4 でも =0 → Fs-43, 15, 21, 34, 54, 60, 61, 67, 75, 87, 89 → P4 でも =0 ✓
→ ✓ Explained(P3) ⊂ Explained(P4)

### P4 ⊂ P5

P4 の Explained の全要素は P5 でも Explained に含まれるか:
- P4 の =1 は全て P5 でも =1（T と整合するものは維持）
- Fs-51: P4=1(O*=0)✗ → 元々 Explained に含まれない
- P4 の =0 は全て P5 でも =0
→ ✓ Explained(P4) ⊂ Explained(P5)
