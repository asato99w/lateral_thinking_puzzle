## 3. 経路の疎な領域

アノマリーへの到達経路が単一（冗長性なし）の箇所:

| 質問 | 到達経路数 | 状態 |
|---|---|---|
| Q-A1 (恐ろしいもの) | 複数（初期オープン、独立） | ✓ |
| Q-F1 (事後撤去) | 1経路（A→F のみ） | **⚠️ 単一経路** |
| Q-G1 (精神的問題) | 1経路（F+Q-13→G のみ） | **⚠️ 単一経路** |
| Q-H1 (嘘) | 1経路（G→H のみ） | **⚠️ 単一経路** |
| Q-I2 (監禁) | 1経路（I1+H→I2） | **⚠️ 単一経路** |
| Q-I3 (空間反転) | 1経路（I2+Q-21→I3） | **⚠️ 単一経路** |

**分析**: 層2以降の深層質問は到達経路が単一になりやすい。これは問題の構造上やむを得ない面がある（深層の洞察に到達するには特定の認知的積み上げが必要）。

**緩和策**:
- 層1に複数の質問を配置し、同じ情報に別角度から到達可能にする
- 例: Q-F1（事後撤去）の代わりに「最初から異常なものはなかったのですか？」という質問でも同等の情報が得られる
- 例: Q-G1（精神的問題）への到達を「女性の認知は正常でしたか？」等の別表現で補強

→ Phase 3 で質問候補を複数生成し、冗長性を確保する

## 素材不足レポートまとめ

| 不足の種類 | 内容 | 対応 |
|---|---|---|
| 中間記述素 | なし | — |
| ゲート辺 | 層間遷移のゲート辺が各 R(P) に未設定 | Phase 1 の R(P) に追加（後述） |
| 経路の疎 | 層2以降の深層質問が単一経路 | Phase 3 で質問の冗長性を確保 |

**Phase 1 への差し戻し不要**。ゲート辺の追加は R(P) の拡張として実施可能（Phase 1 の再実行は不要）。
