# パラダイム発見：深度階層型アルゴリズム A 適用（E2 記述素を含む）

入力:
- 記述素空間（01_初回適用/02_記述素展開.md + 02_E2Q2適用/02_記述素展開.md）
- T（真理論）

## Step 1: 緊張構造の把握

- T1 [Ps-2 × Ps-3] バーで水を頼んだだけなのに銃を向けられた
  - 最大の緊張。「水の注文」と「銃」の因果関係が不明
- T2 [Ps-3 × Ps-4] 銃を向けられたのに感謝した
  - 「銃＝脅威」の前提を覆す。男にとって有益だった示唆
- T3 [Ps-2 × Ps-5] 水を頼んだのに飲まずに帰った
  - 真の目的が水ではないことの示唆

T1 が中核的緊張。T2 が解決の方向を示し、T3 が「水は手段であって目的ではない」ことを裏付ける。

## Step 2: メインパスの発見

### P1: バーでの日常的やりとり

- 説明: 男は普通に水が欲しくてバーに来た。バーテンダーは何らかの理由で怒った。
- 鍵記述素: Fs-01, Fs-09, Fs-14, Fs-16
- 解消する緊張: なし
- 未解消の緊張: T1, T2, T3
- 大区分: I

### P2: バーテンダーの善意

- 説明: バーテンダーは男を助けようとした。銃は攻撃ではなく別の意図があった。
- 鍵記述素: Fs-17, Fs-19, Fs-22, Fs-23, B-02, B-03
- 解消する緊張: T2（感謝の説明がつく）
- 未解消の緊張: T1（なぜ銃で助ける？）, T3（なぜ水を飲まない？）
- 大区分: I → II への橋渡し

### P3: 男の隠れた問題

- 説明: 男は水を飲むこと以外の目的があった。何かの身体的問題を抱えており、それが解決した。
- 鍵記述素: Fs-10, Fs-26, Fs-28, Ft-01, Ft-03, B-01, M-01
- 解消する緊張: T2, T3（目的達成→水不要→感謝して帰った）
- 未解消の緊張: T1（銃と身体的問題の関係が不明）
- 大区分: II

### P4: しゃっくりと驚かし（= T）

- 説明: 男はしゃっくりが止まらず、バーテンダーは驚かせて止めた。水は不要になった。
- 鍵記述素: Ft-05, Ft-10, Ft-13, Ft-15, Ft-19, B-05, B-06, M-05
- 解消する緊張: T1, T2, T3（全緊張解消）
- 大区分: III（= T）

### メインパス

P1 → P2 → P3 → P4

- P1→P2: 「銃＝攻撃」の解釈から「銃＝善意」への転換
- P2→P3: 「バーテンダーの意図」から「男の側の問題」への視点移動
- P3→P4: 「隠れた問題」の正体が「しゃっくり」だと判明

## Step 3: D(P) のスコーピング

### P4 の D(P)（= T）

終端パラダイム。記述素空間の大部分に予測を持つ。

|D(P4)| = 42

### 残余誤りの確認

各非終端パラダイム P_i について、D(P_i) 内に T と異なる予測を持つ記述素があることを確認する。

| パラダイム | 残余誤りの例 | 確認 |
|-----------|------------|------|
| P1 | Fs-09=1(P1) vs 0(T), Fs-16=1 vs 0, Fs-17=0 vs 1, Ft-01=0 vs 1 等 | ✓（多数） |
| P2 | Fs-09=1(P2) vs 0(T), Ft-01=0 vs 1, Ft-05=0 vs 1 等 | ✓ |
| P3 | Ft-07=0(P3) vs 1(T), Ft-08=0 vs 1, Ft-09=0 vs 1, M-03=0 vs 1 | ✓ |

全非終端パラダイムが残余誤りを持つ。P3 は「隠れた問題」と解釈しているため、バーテンダーの観察力（Ft-07）や外部認識可能性（Ft-08）を否定しており、これが T との食い違いとなる。

## Step 4: サブパスの発見

### S1: 強盗・犯罪

- 分岐元: P1
- 説明: 男はバーを襲おうとした、またはバーテンダーが男を脅した犯罪行為
- 鍵記述素: Fs-16, Fs-18, Fs-33
- D(S1): Fs-14, Fs-16, Fs-17, Fs-18, Fs-22, Fs-23, Fs-33, Ft-11 (|D(S1)| = 8)
- T2（感謝）で排除される

### S2: 男とバーテンダーの個人的関係

- 分岐元: P1
- 説明: 二人は知り合いで、何らかの約束・合図・ゲームがあった
- 鍵記述素: Fs-04, Fs-22, Fs-30
- D(S2): Fs-04, Fs-22, Fs-30, Ft-01, Ft-05, Ft-13 (|D(S2)| = 6)
- T3（水を飲まない）との整合性で弱まる

## Step 5: 到達性の検証

### アノマリー数の概算

T を基準に、各パラダイムの D(P) 内で解釈が T と異なる記述素の数を算出する。

| パラダイム | anomalies(O*, P) | 非ゼロ | 減少傾向 |
|-----------|------------------|--------|---------|
| P1 | 36 | ✓ | — |
| P2 | 8 | ✓ | ✓（36 > 8） |
| P3 | 4 | ✓ | ✓（8 > 4） |
| P4 (= T) | 0 | — (終端) | ✓（4 > 0） |

### 検証結果

1. 全非終端パラダイムのアノマリー数が 0 でない → ✓（Step 3 の残余誤り確認と整合）
2. メインパスに沿ってアノマリー数が単調減少 → ✓（36 > 8 > 4 > 0）

## パラダイム群の構造

```
P1（日常的やりとり）
├── → P2（善意）
│     └── → P3（隠れた問題）
│           └── → P4（しゃっくりと驚かし = T）
├── S1（強盗・犯罪）← T2 で排除
└── S2（個人的関係）← T3 で弱化
```

メインパス: P1 → P2 → P3 → P4
