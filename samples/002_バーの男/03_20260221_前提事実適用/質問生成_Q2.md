## Step 6: 前提事実の割り当て

各質問について、その質問が論理的に成立するために確定済み（O に含まれていること）を要求する記述素を特定し、prerequisites として設定する。

判定基準: 「この記述素が確定していなければ、この質問を聞くこと自体が意味をなさない」という関係があるか。質問のテキスト中の前提節や定冠詞的用法が、特定の事実の確定を前提としているかどうかで判断する。

### prerequisites なしの質問

残りの 48 問は prerequisites を空（[]）とする。これらは以下のいずれかに該当する:

- 問題文の情報のみで成立する基本的な質問（q01, q06, q09, q10 等）
- 探索的・仮説的な質問（q04「体調に問題がありましたか」、q18「身体的症状を抱えていましたか」等）
- 否定代替仮説系の質問（q35〜q40）
- S1 で付与された偶然の開き構造の開放記述素を含む質問（q07, q08, q11 等）— 開放記述素の前提節は開放メカニズムの一部であり、内容的前提とは区別する

### 初期オープン質問への影響

init_questions(P1) で開く質問のうち、prerequisites 付きの質問がないことを確認する。

P1 の初期オープン質問は safe 質問（P1 予測と一致する effect のみ）であり、Fs-10 の確定前に開く。Fs-10 を prerequisites に持つ質問は、q04 が回答されるまで開かない。

ただし q04 自体は初期オープンに含まれる（Fs-10 は P1 の D⁻ だが、q04 の effect は (Fs-10, 1) であり P1 の prediction(Fs-10)=0 と矛盾するため、init_questions の has_conflict 条件で除外）。

**修正**: q04 は P1 の D⁻ 記述素（pred=0）を effect に (Fs-10, 1) で含むため、has_conflict=true で init_questions から除外される。q04 が初期オープンに含まれないなら、Fs-10 は初期フェーズで確定しない。

これは anomaly_discovery の発掘連鎖で確認する必要がある。07_補充検証.md の発掘連鎖（Step 13: q56 回答 → Fs-10 観測 → q04, q43 発掘）により、q04 は Step 13 で発掘されて Fs-10 が O に入る。よって Fs-10 を prerequisites に持つ質問群は Step 13 以降で利用可能になる。

ゲーム進行上、これは P1 フェーズの中盤〜後半に相当し、症状に関する詳細な質問がその段階から開くのは自然な進行である。✓
