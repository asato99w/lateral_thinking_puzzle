# 初期質問の決定

## 概要

ゲーム開始時に提示する質問群を決定する。

## 入力

- クイズデータ（Ps, T, 全質問群, 各質問の ans 定義）

## 処理

1. H を初期化する。O = Ps より、Ps に含まれる記述素は確定値（0 or 1）、それ以外は中間値。
2. 初期パラダイム P_init に基づいて、初期質問群を決定する。

**init_questions(P_init) = { q | ∃(d, v) ∈ effect(q) : d ∈ D(P_init) ∧ P_init(d) = v }**

open(H) とは別のロジックとする。初期パラダイムの構造から直接決まるため、H の閾値設定に依存しない。

## 出力

- 初期 H
- 初期質問群（プレイヤーに提示する質問群）
