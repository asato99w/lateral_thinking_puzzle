# POC 仕様

## 画面遷移

1. **クイズ一覧画面**: クイズの一覧を表示。選択するとゲーム画面へ遷移。
2. **ゲーム画面**: 質問の選択を繰り返し、クリアを目指す。

## ゲーム画面の構成

- **オープン質問群**: 現在選択可能な質問の一覧
- **選択済み質問群**: 回答済みの質問とその結果（YES / NO / 関係ない）の一覧

## ゲームフロー

1. ゲーム開始時、初期のオープン質問群が提示される
2. プレイヤーが質問を一つ選択する
3. 回答（YES / NO / 関係ない）が表示される
4. 選択した質問は選択済み質問群に移動する
5. 新たな質問が 0〜複数個オープン質問群に追加される
6. 2〜5 を繰り返す
7. 特定の質問（クリア質問）のいずれかを選択するとクリア
