## 戦術連鎖

### Chain A: 味の違和感（S→根命題 CA3）

source: {S5, S6, S16} → target: P1

```
{S5, S6, S16} →[因果の逆転(5)]→ I1: 男はスープの味がきっかけで店員にスープの正体を確認した
I1 → P1: 男はスープを飲んで違和感を覚えた
```

プレイヤー視点: 一口飲んで確認した → 味が原因 → 味に違和感があった。自然な推論。

### Chain A2: 味の比較（P1→P10）

source: {P1} → target: P10

```
{P1} →[時間軸の拡張(3)]→ I2: 男は以前にもウミガメのスープとして提供されたスープを飲んだことがある
{I2, S15} →[矛盾の指摘(6)]→ I2b: 以前の「ウミガメのスープ」が本物なら今回と同じ味のはずだが、味が違った
I2b → P10: 男は以前「ウミガメのスープ」として飲んだスープと今回のスープの味が違うと感じた
```

プレイヤー視点: 違和感がある → 以前の経験との比較 → 本物と味が違う。自然。

### Chain A3: アノマリー解消（P10→P11）

source: {P10, S15} → target: P11

```
{P10, S15} →[ラベルと実体の乖離(7)]→ I3: 以前「ウミガメのスープ」と言われて飲んだスープは名称と実態が異なっていた
I3 → P11: 男が以前飲んだスープはウミガメのスープではなかった
```

プレイヤー視点: 今回が本物で味が違う → 以前のは偽物だった。論理的に自然。
