# Phase 2: モデル具象命題と分割構造

## Step 1: 各モデルの具象命題

### T: 遭難時の人肉スープ

1. 男は船乗りだった
2. 男はかつて仲間と船で遭難した
3. 遭難中に食料が尽きた
4. 仲間の一人が死んだ
5. 残った仲間が死んだ仲間の肉でスープを作った
6. 仲間は男に「ウミガメのスープだ」と偽って飲ませた
7. 男は当時それをウミガメのスープだと信じた
8. レストランで本物を飲み、過去の味と違うことに気づいた
9. 「本物だ」という回答から、過去のスープがウミガメではなかったと悟った
10. 過去のスープが人肉だったと推理した
11. 真実に絶望して自殺した

### M1: 遭難時の別動物のスープ

1. 男は遭難経験がある
2. 遭難中に仲間が別の動物（犬等）の肉でスープを作った
3. 仲間は「ウミガメのスープだ」と偽って飲ませた
4. レストランで本物の味と違うことに気づき確認
5. 過去のスープが別の動物だったと悟った
6. 忌避すべき動物を食べた罪悪感で自殺

### M2: 亡き家族の偽スープ

1. 男の亡き母が「ウミガメのスープ」と偽って安い肉で作ったスープを食べさせていた
2. レストランで本物の味と違うことに気づき確認
3. 母の嘘と貧しさの中での苦労を悟った
4. 母を失った悲しみと感謝を伝えられない後悔で自殺

### M3: 捕虜時の偽スープ

1. 男は戦争中に捕虜だった
2. 看守が「ウミガメのスープだ」と言って仲間の捕虜の肉を食べさせた
3. レストランで本物の味と違うことに気づき確認
4. 捕虜時代に人肉を食べさせられていたと悟った
5. 真実に絶望して自殺

### M4: 遭難時の正体不明のスープ

1. 男は島で遭難した
2. 自分で捕まえた動物を「ウミガメ」だと思ってスープにした（他者の欺瞞ではなく自己の誤認）
3. レストランで本物の味と全く違うことに気づき確認
4. 過去に食べたものの正体が不明になった
5. 何を食べていたのか分からない恐怖から自殺

### M5: 料理人の詐欺

1. 男は料理人だった
2. 長年「ウミガメのスープ」として別の食材で作ったスープを客に提供していた
3. レストランで本物を飲み、自分の料理と全く味が違うことに気づいた
4. 確認して本物だと知り、自分の料理が偽物だったと確信
5. 長年の詐欺の罪悪感と職業的アイデンティティ崩壊で自殺

### M6: 宗教的禁忌

1. 男はウミガメを神聖視する信仰を持つ文化の出身
2. 禁忌と知りつつ興味本位で注文した
3. 一口飲んで不安になり確認
4. 本物と分かり宗教的禁忌を犯した罪悪感から自殺

### M7: 味覚喪失と病気の末期

1. 男は重病を患っていた
2. ウミガメのスープを飲んだが味が全くしなかった
3. 味がしないため本物か確認
4. 本物なのに味がしない → 味覚完全喪失 → 病気の末期と確信
5. 余命に絶望して自殺

### M8: 幼少期の虐待トラウマ

1. 男は幼少期に虐待者から「ウミガメのスープだ」と何かを飲まされた
2. レストランで本物の味が全く違うことに気づき確認
3. 虐待時に飲まされたものの正体（汚物等）を悟った
4. トラウマの再燃で精神崩壊し自殺

### M9: 保護活動家の絶望

1. 男はウミガメの保護活動家
2. 絶滅危惧種が食用にされている証拠のためレストランに来た
3. 一口飲んで確認し本物と判明
4. 保護活動の無力さに絶望して自殺

### M10: 調査員の別件自殺

1. 男は食品偽装調査のジャーナリスト
2. 偽装疑惑のある店で味を確認し真偽を聞いた
3. 本物だった（調査は空振り）
4. その夜、別件（個人的問題）で自殺

### M11: 計画的自殺の最後の晩餐

1. 男は既に自殺を決意していた
2. 最後の食事としてウミガメのスープを注文
3. 一口飲んで味を堪能し、本物か確認して満足
4. 予定通りその夜自殺

### M12: 飢饉の村の人肉スープ

1. 男はかつて飢饉の村で「ウミガメのスープ」とされるものを村人と食べた
2. 後に人肉だったという噂を聞いたが確信はなかった
3. レストランで本物を飲み味が全く違うことに気づき確認
4. 噂が真実だったと確信
5. 人肉を食べていたという真実に絶望して自殺

## Step 2: モデル横断の共通命題抽出

### 軸 1: 過去の偽スープ体験

具象命題の比較:
- T: 「仲間が人肉を『ウミガメのスープ』と偽って飲ませた」
- M1: 「仲間が別の動物の肉を『ウミガメのスープ』と偽って飲ませた」
- M2: 「母が安い肉を『ウミガメのスープ』と偽って食べさせた」
- M3: 「看守が捕虜の肉を『ウミガメのスープ』と偽って食べさせた」
- M4: 「自分で捕まえた動物をウミガメだと誤認してスープにした」
- M8: 「虐待者が何かを『ウミガメのスープ』と偽って飲ませた」
- M12: 「飢饉の村で『ウミガメのスープ』とされるものを食べた」

共通命題: **男は過去に「ウミガメのスープ」だと信じて実際はウミガメではないものを飲んだ経験がある**

| モデル | 真偽 |
|--------|------|
| T, M1, M2, M3, M4, M8, M12 | 真 |
| M5, M6, M7, M9, M10, M11 | 偽 |

### 軸 2: 遭難体験

具象命題の比較:
- T: 「船で遭難した」
- M1: 「遭難した」
- M4: 「島で遭難した」
- M3: 「戦争中に捕虜」（遭難ではない）
- M2, M8, M12: 日常的/非遭難の文脈

共通命題: **過去の体験は海での遭難に起因する**

| モデル | 真偽 |
|--------|------|
| T, M1, M4 | 真 |
| M2, M3, M5, M6, M7, M8, M9, M10, M11, M12 | 偽 |

### 軸 3: 他者の意図的な欺瞞

具象命題の比較:
- T: 「仲間が『ウミガメのスープだ』と偽って飲ませた」
- M1: 「仲間が偽って飲ませた」
- M4: 「自分で誤認した」（他者の欺瞞ではない）

共通命題: **偽りのスープは他者が意図的に「ウミガメのスープだ」と嘘をついて飲ませたものだった**

| モデル | 真偽 |
|--------|------|
| T, M1, M2, M3, M8 | 真（広域） |
| T, M1 | 真（P2 true 内） |
| M4 | 偽（P2 true 内） |

### 軸 4: 人肉

具象命題の比較:
- T: 「死んだ仲間の肉」
- M1: 「別の動物の肉」

共通命題: **偽りのスープの原材料は人間の肉だった**

| モデル | 真偽 |
|--------|------|
| T | 真（P3 true 内） |
| M1 | 偽（P3 true 内） |

## Step 3: 分割命題と階層構造

### 分割命題一覧

| ID | ラベル | T での真偽 | True モデル | False モデル |
|----|--------|-----------|-------------|-------------|
| P1 | 男は過去に「ウミガメのスープ」だと信じて実際はウミガメではないものを飲んだ経験がある | 真 | T, M1, M2, M3, M4, M8, M12 | M5, M6, M7, M9, M10, M11 |
| P2 | その過去の体験は海での遭難に起因する | 真 | T, M1, M4 | M2, M3, M8, M12（P1 true 内） |
| P3 | 偽りのスープは他者が意図的に嘘をついて飲ませたものだった | 真 | T, M1 | M4（P2 true 内） |
| P4 | 偽りのスープの原材料は人間の肉だった | 真 | T | M1（P3 true 内） |

### 分割の階層構造

```
│  │  │  ├─ P4=真: {T}  ← T を一意に特定
│  │  │  └─ P4=偽: {M1}
│  │  │
│  │  └─ P3=偽: {M4}
│  │

```

## Step 4: T の抽象的位置づけ

P1〜P4 の全てが真であるモデルは T のみ。分割命題によって T は一意に特定される。

**棄却不可能モデル: なし**

全ての競合モデルは P1〜P4 のいずれかが偽であり、対応する分割命題の確認により棄却可能。

T の抽象的位置:
- P1+P2+P3+P4 を全て満たすモデルは T のみ
- 最も T に近いのは M1（P4 のみ偽: 人肉ではなく別の動物）と M3（P2 が偽: 遭難ではなく捕虜）
- M3, M12 は T と「人肉スープの偽り」を共有するが、文脈（遭難 vs 捕虜 vs 飢饉）が異なるため P2 で分離される
