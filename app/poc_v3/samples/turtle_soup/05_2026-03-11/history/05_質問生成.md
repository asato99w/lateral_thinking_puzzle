導入時検証（1次チェック）:
- T文脈での真偽: T では男は遭難時に「ウミガメのスープだ」と言われて飲んだ。「認識していますか」は信念を問うため、はい は T で真 ✓
- 注意: 「飲んだことがありますか？」（客観的事実）であれば T で偽となる。「認識していますか」（主観的認識）とすることで T 整合性を確保

**確認質問 Q1c**: P1 を reveal（いいえパターン）

| 項目 | 値 |
|------|-----|
| ID | Q1c |
| text | 男が過去に飲んだ「ウミガメのスープ」は、本物のウミガメのスープでしたか？ |
| answer | いいえ |
| chain_id | C1 |
| reveals | [P1] |
| recall_conditions | [[I1, S12]] |
| prerequisites | I1 |

導入時検証（1次チェック）:
- T文脈での真偽: 男が過去に飲んだのは人肉スープであり本物のウミガメのスープではない。いいえ は T で真 ✓
- reveals の厳密性: I1（認識している）が前提にある状態で、「本物でしたか？」→「いいえ」→「認識していたものは本物ではなかった」= P1 ✓
- いいえパターン: I1（信じている）+ いいえ（本物ではない）→ P1（偽物を飲んだ経験がある）が自然に導かれる ✓

## Step 6: 探索パスの検証

### パス A: 確認行為からの入口（Q1a → Q1b → Q1c → Q2a → Q2b → Q3a → Q4a → Q4b）

1. 開始: confirmed = {S1-S12}
2. Q1a オープン（fc: {S5,S10,S11} ⊂ confirmed）→ いいえ → 常識的仮説排除
3. Q1b オープン（fc: {S5,S10,S11} ⊂ confirmed）→ はい → I1 confirmed
4. Q1c オープン（fc: {I1,S12} ⊂ confirmed）→ いいえ → P1 confirmed → M5,M6,M7,M9,M10,M11 棄却
5. Q2a オープン（fc: {P1} ⊂ confirmed）→ いいえ → 日常的文脈排除
6. Q2b オープン（fc: {P1} ⊂ confirmed）→ はい → P2 confirmed → M2,M3,M8,M12 棄却
7. Q3a オープン（fc: {P2} ⊂ confirmed）→ はい → P3 confirmed → M4 棄却
8. Q4a オープン（fc: {P3} ⊂ confirmed）→ いいえ → 動物排除
9. Q4b オープン（fc: {P3} ⊂ confirmed）→ はい → P4 confirmed → M1 棄却 → T 到達

想起の合理性: 各質問の開放は自然。確認行為の不自然さ→過去の認識→本物ではない（いいえ）→遭難→欺瞞→正体と段階的に深まる。

### パス B: 確認のみ最短パス

1. 開始: confirmed = {S1-S12}
2. Q1b オープン（fc: {S5,S10,S11} ⊂ confirmed）→ はい → I1 confirmed
3. Q1c オープン（fc: {I1,S12} ⊂ confirmed）→ いいえ → P1 confirmed → 6 モデル棄却
4. Q2b オープン（fc: {P1} ⊂ confirmed）→ はい → P2 confirmed → 4 モデル棄却
5. Q3a オープン（fc: {P2} ⊂ confirmed）→ はい → P3 confirmed → M4 棄却
6. Q4b オープン（fc: {P3} ⊂ confirmed）→ はい → P4 confirmed → T 到達

想起の合理性: 排除質問をスキップした最短パス。Q1c のいいえパターンが自然に機能する：I1（認識している）に対し「本物でしたか？」→「いいえ」。6 質問で T 到達。

### パス C: derived 経由最短パス

1. 開始: confirmed = {S1-S12}, derived: I1（fc:{S5,S10,S11}⊂confirmed）
2. Q1c オープン（fc: {I1,S12}、I1 は derived で known）→ いいえ → P1 confirmed → 6 モデル棄却
3. Q2b オープン（fc: {P1} ⊂ confirmed）→ はい → P2 confirmed → 4 モデル棄却
4. Q3a オープン（fc: {P2} ⊂ confirmed）→ はい → P3 confirmed → M4 棄却
5. Q4b オープン（fc: {P3} ⊂ confirmed）→ はい → P4 confirmed → T 到達

想起の合理性: I1 が derived として自動的に known に入り、Q1b をスキップ可能。4 質問で T 到達。
