# Phase 3: 棄却条件

## Step 1: 各モデルの偽命題の特定

| モデル | P1 | P2 | P3 | P4 | 偽となる命題 |
|--------|----|----|----|----|-------------|
| T | 真 | 真 | 真 | 真 | — |
| M1 | 真 | 真 | 真 | **偽** | P4 |
| M2 | 真 | **偽** | — | — | P2 |
| M3 | 真 | **偽** | — | — | P2 |
| M4 | 真 | 真 | **偽** | — | P3 |
| M5 | **偽** | — | — | — | P1 |
| M6 | **偽** | — | — | — | P1 |
| M7 | **偽** | — | — | — | P1 |
| M8 | 真 | **偽** | — | — | P2 |
| M9 | **偽** | — | — | — | P1 |
| M10 | **偽** | — | — | — | P1 |
| M11 | **偽** | — | — | — | P1 |
| M12 | 真 | **偽** | — | — | P2 |

注: P2〜P4 は P1 が真であるモデルにのみ意味を持つ。P1 が偽のモデルは P1 の確認のみで棄却される。同様に P3, P4 は上位命題が真であるモデルに対して評価される。

## Step 2: 最小棄却条件

| モデル | 最小棄却条件 |
|--------|-------------|
| M1 | P4 |
| M2 | P2 |
| M3 | P2 |
| M4 | P3 |
| M5 | P1 |
| M6 | P1 |
| M7 | P1 |
| M8 | P2 |
| M9 | P1 |
| M10 | P1 |
| M11 | P1 |
| M12 | P2 |

全モデルが単一の分割命題の確認で棄却可能。複数命題の AND 条件が必要なモデルはない。

## Step 3: 棄却力の評価

| 命題 | 棄却対象 | 棄却数 |
|------|---------|--------|
| P1 | M5, M6, M7, M9, M10, M11 | 6 |
| P2 | M2, M3, M8, M12 | 4 |
| P3 | M4 | 1 |
| P4 | M1 | 1 |

P1 の棄却力が最大（6 モデル）。探索の初期段階で P1 を確認すべき。

## Step 4: 棄却不可能モデル

**棄却不可能モデル: なし**

全競合モデルが P1〜P4 のいずれかの確認により棄却可能。T は P1〜P4 の全てが真である唯一のモデル。

## Step 5: 棄却の進行順序

| ステップ | 確認される命題 | 棄却されるモデル | 残存モデル |
|---------|--------------|----------------|-----------|
| 1 | P1 | M5, M6, M7, M9, M10, M11 | T, M1, M2, M3, M4, M8, M12 |
| 2 | P2 | M2, M3, M8, M12 | T, M1, M4 |
| 3 | P3 | M4 | T, M1 |
| 4 | P4 | M1 | T |

4 段階の確認で T に一意到達。進行は線形連鎖（P1→P2→P3→P4）。
