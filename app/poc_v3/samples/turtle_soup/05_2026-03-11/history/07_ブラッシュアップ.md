## Step 2: 解消（入れ子化）

### C1: 動機の探索 > 段階的絞り込み

元: {S5,S10,S11} →[動機の探索]→ I1

入れ子化後:
- {S5,S10,S11} → Q1a「味が不味いから？」→ いいえ → N1a（味の不味さが理由ではない）
- {S5,S10,S11,N1a} → Q1b「以前に飲んだと認識？」→ はい → I1

改善: Q1b の rc に N1a を追加。「味が理由ではない」が既知の状態から「以前の経験との比較か？」は自然に浮かぶ。

### C2: 人物像の探索 > 段階的絞り込み

元: {P1} → Q2a → Q2b → P2

入れ子化後:
- {P1} → Q2a「日常生活の出来事？」→ いいえ → N2a（日常的ではない）
- {P1,N2a} → Q2b「極限状態での出来事？」→ はい → N2b（極限状態）
- {N2b} → Q2c「海で遭難？」→ はい → P2

改善: 旧Q2b（遭難）の rc が P1 から N2b（極限状態）に変更。「極限状態 → 海で遭難？」は「偽物を飲んだ → 海で遭難？」より自然。

### C3: 登場人物の探索 > 段階的絞り込み

元: {P2} → Q3a → P3

入れ子化後:
- {P2} → Q3a「遭難時に仲間がいた？」→ はい → N3a（仲間がいた）
- {N3a} → Q3b「仲間が嘘をついて飲ませた？」→ はい → P3

改善: 旧Q3a（欺瞞）の rc が P2 から N3a（仲間の存在）に変更。「仲間がいた → その仲間が嘘をついた？」は「遭難した → 嘘をついて飲ませた？」より自然。

### C4: 段階的絞り込み（Q4a の意味的修正）
