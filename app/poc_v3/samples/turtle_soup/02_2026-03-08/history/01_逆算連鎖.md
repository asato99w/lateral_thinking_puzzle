# Phase 1: 逆算連鎖

## T

男は船乗りで、かつて仲間と船で遭難し、食料が尽きた。仲間の一人が死に、残った仲間は男に「ウミガメのスープだ」と言ってスープを飲ませた。実際にはそれは死んだ仲間の肉で作ったスープだった。レストランで本物のウミガメのスープを飲んだ男は、味が違うことに気づき、あのとき飲まされたものの正体を悟り、絶望して自殺した。

## Step 1: T の逆算

**なぜ自殺した？**
→ D-10: 男は絶望して自殺した ← D-9

**なぜ絶望した？**
→ D-9: 過去に飲まされたスープの正体（人肉）を悟った ← D-8 + D-7

**なぜ正体を悟った？**
味の違いと、過去のスープの正体の結合:
- D-8: レストランのスープと過去のスープの味が違うことに気づいた
- D-7: 過去に飲んだスープは実際には人肉で作られたものだった

**なぜ味の違いに気づけた？**
ここが T の最も微妙な部分。以下の事実が同時に成立している:
- レストランのスープは本物のウミガメのスープ（S で確認済み）
- 過去のスープは人肉（D-7）
- 男は過去のスープの味を記憶していた

→ T から導出: **D-11: 男はレストラン以前に本物のウミガメのスープを飲んだことがなかった**
  根拠: 過去に飲んだのは人肉スープ。T にはレストラン以前に本物のウミガメのスープを飲んだ記述がない。レストランが本物との初接触であることが T の因果構造の前提。

→ T から導出: **D-12: 男は過去に飲まされたスープをウミガメのスープだと信じていた**
  根拠: D-6「ウミガメのスープだ」と言われて飲んだ。疑う根拠がない状況。

→ T から導出: **D-13: 男の「ウミガメのスープの味」の記憶は実際には人肉スープの味だった**
  根拠: D-11（本物を飲んだことがない）+ D-12（偽物を本物と信じていた）→ 記憶されている味は人肉スープの味。

これらにより、味の違いの構造が明確になる:
- 男の記憶: 人肉スープの味（D-13）を「ウミガメのスープの味」として記憶（D-12）
- レストラン: 本物のウミガメのスープを初めて飲む（D-11）
- → 味が違う（D-8）→ 正体を悟る（D-9）

**なぜ人肉スープを飲んだ？**
→ D-6: 仲間は男に「ウミガメのスープだ」と言ってそのスープを飲ませた
→ D-5: 残った仲間が死んだ仲間の肉でスープを作った

**なぜ肉でスープを作った？**
→ D-4: 遭難中に仲間の一人が死んだ
→ D-3: 遭難中に食料が尽きた

**なぜ遭難？**
→ D-2: 男はかつて仲間と船で遭難した
→ D-1: 男は船乗りだった
