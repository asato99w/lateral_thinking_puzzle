# Phase 2: モデル具象命題と分割構造

## Step 1: 各モデルの具象命題

各モデルの因果構造を命題として記述する。

### T

1. 男は船乗りだった
2. 男は仲間と船で遭難し、食料が尽きた
3. 仲間の一人が死んだ
4. 残った仲間が死んだ仲間の肉でスープを作った
5. 仲間は男に「ウミガメのスープだ」と偽って飲ませた
6. 男はウミガメのスープだと信じて飲んだ
7. 男は助けられた
8. レストランで本物のウミガメのスープを注文した
9. 一口飲んで味が違うことに気づいた
10. 以前飲まされたものは本物のウミガメのスープではなかったと悟った
11. 飲まされたものが人肉だったと理解した
12. 絶望して自殺した

### M1（自己欺瞞）

1. 男は仲間と船で遭難し、食料が尽きた
2. 仲間の一人が死んだ
3. 男自身が生存のために仲間の肉でスープを作った
4. 「これはウミガメだ」と自分に言い聞かせて食べた
5. レストランで本物を飲み、味の違いから自己欺瞞を悟り自殺

### M2（幼少期の貧困）

1. 男は幼少期に困窮した家庭で育った
2. 親が「ウミガメのスープだ」と嘘をつき別の肉のスープを飲ませていた
3. 男はそれをウミガメのスープだと信じて飲んでいた
4. 大人になりレストランで本物を飲み、味の違いから親の嘘を悟った
5. 以前飲んでいたものの正体への衝撃から自殺

### M3（戦時捕虜）

1. 男は戦時中に捕虜となった
2. 看守が嗜虐的に人肉で作ったスープを「ウミガメのスープだ」と偽って飲ませた
3. 男はウミガメのスープだと信じて飲んだ
4. 解放後、レストランで本物を飲み、味の違いから真実を悟り自殺

### M4（無人島漂着）

1. 男は無人島に漂着した
2. 同行者が複数おり、一人が死んだ
3. 別の同行者が死んだ同行者の肉で「ウミガメを捕まえた」と偽ってスープを作った
4. 男はウミガメのスープだと信じて飲んだ
5. 救助後、レストランで本物を飲み、味の違いから真実を悟り自殺

### M5（料理人）

1. 男はかつて料理人だった
2. 業者が「ウミガメ」と称して納入した材料で「ウミガメのスープ」を作っていた
3. 男はそれをウミガメだと信じて調理・味見していた
4. レストランで本物を飲み、味が全く異なることに気づいた
5. 自分が使っていた材料がウミガメではなかったと悟った
6. 材料の正体への恐怖から自殺

### M6（飢饉）

1. 男は飢饉の村の住民だった
2. 村の長老が「ウミガメのスープだ」と言ってスープを配った
3. 実は行方不明者の肉だった
4. 男はウミガメのスープだと信じて飲んだ
5. 後にレストランで本物を飲み、味の違いから真実を悟り自殺

### M7（病院）

1. 男は長期入院していた
2. 病院食として「ウミガメのスープ」が出されていた
3. 男はそれをウミガメのスープだと信じて飲んでいた
4. 退院後、レストランで本物を飲み、味が違うことに気づいた
5. 病院で飲まされていたものの正体に気づき自殺

### M8（妻の欺瞞）

1. 男の妻が「ウミガメのスープを作った」と言ってスープを出した
2. 男はウミガメのスープだと信じて飲んだ
3. 妻はその後失踪した
4. レストランで本物を飲み、味が異なることに気づいた
5. 妻が作ったスープの正体と妻の失踪の関連を悟り自殺

### M9（食品調査員）

1. 男は食品調査員だった
2. 別のレストランの「ウミガメのスープ」が偽物だとの通報を受けていた
3. 調査で当該店のスープを飲んだ
4. 別のレストランで本物を飲み、味を比較した
5. 以前の店のスープが完全に別物であると確認した
6. その偽物の正体が重大な違法行為に関連すると悟り自殺

### M10（宗教的禁忌）

1. 男の宗教ではウミガメは禁忌の食べ物だった
2. 以前、他者に「ウミガメではない」と言われてあるスープを飲んだ
3. レストランで本物のウミガメのスープを注文して味を確認した
4. 以前飲んだスープと同じ味だった
5. 以前飲んだものもウミガメだった → 禁忌を犯していたと悟り自殺

### M11（亡き人との約束）

1. 男の亡き妻が「いつか一緒にウミガメのスープを飲もう」と約束していた
2. 妻は亡くなった
3. 男は一人でレストランに行き注文した
4. 本物であることを確認した
5. 約束を果たせない現実に耐えきれず自殺

### M12（加害者）

1. 男は遭難中に仲間のためにスープを作った
2. スープの材料が何であるか男は知っていた（ウミガメではない）
3. 男は仲間に「ウミガメのスープだ」と偽って食べさせた
4. レストランで本物を飲み、味の違いから、自分が仲間に食べさせたものの正体を改めて直視した
5. 罪悪感から自殺

## Step 2d: 意味の確定性チェック

### 命題 A: "男は以前に「ウミガメのスープ」だと思ってスープを飲んだ経験がある"

全モデルで真偽を検証:
- M10: 「ウミガメではない」と言われて飲んだ → ウミガメだと思っていなかった → **偽** ✓
- M11: 過去に飲んだことがない → **偽** ✓
- M12: 男は偽物だと知っていた → ウミガメだと**思って**はいなかった → **偽** ✓
- 他は全て「ウミガメのスープだと思って飲んだ」→ **真** ✓

→ 確定。修正不要。

### 命題 B (P2候補): "過去のスープ体験は食料の危機的状況に起因する"

- M3（戦時捕虜）: 食料危機ではなく看守の嗜虐的行為 → **偽**
  - M3 のモデル定義を補完: 「看守が嗜虐的に人肉スープを飲ませた」（食料不足が理由ではない）
- M12: 遭難で食料が尽きた → **真**

ただし、M12 は A で偽（男はウミガメだと思っていなかった）なので、B は A の true_set 内での分割として使う場合は M12 は対象外。

A の true_set {T,M1,M2,M3,M4,M5,M6,M7,M8,M9} 内での B:
- true: T, M1, M2, M4, M6 (5)
- false: M3, M5, M7, M8, M9 (5)

→ 確定。修正不要。

### 命題 C (P3候補): "過去の体験は海での遭難に起因する"

### 命題 D' (P4候補): "他者が男に対して意図的に偽って飲ませた"
