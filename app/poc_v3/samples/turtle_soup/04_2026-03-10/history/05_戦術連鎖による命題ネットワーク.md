# 戦術連鎖による命題ネットワーク

## 目的

仮説的要素の葉・中間具象・境界命題の間のつながりを、汎用戦術による連鎖として明示する。質問の導出条件の基盤となる。複数対複数の関係を含む。

## 凡例

- `→[戦術名]→` : 戦術による導出
- `{...}` : 命題（記述素）集合
- `+` : 複数命題の組み合わせ入力
- `⇒` : 複数命題の同時導出

## 命題の定義

各命題は、S を文脈として、単体で意味が定まることを要件とする。定まらない場合は前提となる命題を明記する。

### H1 の命題

| ID | 命題 | 前提 |
|---|---|---|
| H1{体験} | 男は過去にウミガメのスープを飲んだことがある | S |
| H1{記憶} | 男は過去に飲んだウミガメのスープの味を覚えている | H1{体験} |
| H1{認識} | 男は過去に飲んだものをウミガメのスープだと認識していた | H1{体験} |

### H2 の命題

| ID | 命題 | 前提 |
|---|---|---|
| H2{職業} | 男は船乗りだった | S |
| H2{海} | 男の過去の体験は海で起きた | H1{体験} |
| H2{仲間} | 男の過去の体験には他の人物が関わっている | H1{体験} |
| H2{危機} | 男の過去の体験は生命に関わる危機的状況だった | H1{体験} |

備考: H2{職業}はSの男に関する属性であり、H1を前提としない。他のH2葉はH1{体験}（過去の体験の存在）を前提とし、その体験の性質を記述する。

### H3 の命題

| ID | 命題 | 前提 |
|---|---|---|
| H3{正体} | 男が過去に飲んだものはウミガメのスープではなかった | H1{認識} |
| H3{食} | 男が過去に飲んだものはウミガメ以外の食材で作られていた | H3{正体} |
| H3{禁忌} | 男が過去に飲んだものは口にすべきでないとされるものだった | H3{正体} |
| H3{死者} | 男が過去に飲んだものには死んだ人の体が使われていた | H3{食} |

備考: H3{正体}はH1{認識}（ウミガメのスープだと認識していた）を前提とし、その認識が誤りだったことを述べる。H3{食}、H3{禁忌}はH3{正体}を前提とし、実際の内容を記述する。H3{死者}はH3{食}（食材であった）を前提とし、その食材の内容を記述する。

### H4 の命題

| ID | 命題 | 前提 |
|---|---|---|
| H4{欺瞞} | 男が過去に飲んだものの正体について意図的な偽りがあった | H3{正体} |
| H4{関係} | その偽りの主体は男以外の人物だった | H4{欺瞞} |
| H4{動機} | その偽りには特定の事情があった | H4{欺瞞} |
| H4{窮状} | 男の過去の体験の状況において食料が不足していた | H1{体験} |

備考: H4{欺瞞}はH3{正体}（正体が異なっていた）を前提とし、それが意図的であったことを述べる。H4{関係}とH4{動機}はH4{欺瞞}を前提とする。H4{窮状}はH1{体験}を前提とし、過去の状況の記述であり、偽りの存在を前提としない。

### 前提関係の構造

```
S
├── H1{体験}
│   ├── H1{記憶}
│   ├── H1{認識}
│   │   └── H3{正体}
│   │       ├── H3{食}
│   │       │   └── H3{死者}
│   │       ├── H3{禁忌}
│   │       └── H4{欺瞞}
│   │           ├── H4{関係}
│   │           └── H4{動機}
│   ├── H2{海}
│   ├── H2{仲間}
│   ├── H2{危機}
│   └── H4{窮状}
└── H2{職業}
```

この構造から見えること:
- H1{体験}が大部分の命題の根にある
- H3 と H4 は H1{認識} → H3{正体} → H4{欺瞞} という依存連鎖上にある
- H2{職業}のみが H1 から独立
- H4{窮状}は他の H4 葉と異なり、H4{欺瞞}ではなく H1{体験}を前提とする（状況の記述であり、偽りの存在を前提としない）

## 戦術連鎖: S 命題から中間命題を経て H 命題へ

### S3 + S4 からの連鎖

**S3 + S4 →[矛盾の指摘(6)]→ M1**

S3「一口飲んだ」と S4「本物か確認した」の間に、一口で真偽確認に至る飛躍がある。

| ID | 命題 | 前提 |
|---|---|---|
| M1 | 男は一口飲んで味に違和感を覚えた | S3, S4 |

**M1 →[時間軸の拡張(3)]→ H1{体験}**

味に違和感があるには、比較対象となる過去の味覚体験が必要。

**M1 → M2**

| ID | 命題 | 前提 |
|---|---|---|
| M2 | 男は過去に飲んだウミガメのスープの味と、今飲んだものの味が異なると感じた | M1, H1{記憶} |

M2 は M1 と H1{記憶} の両方を前提とする。

### S4 からの連鎖

**S4 →[動機の探索(8)]→ M3**

| ID | 命題 | 前提 |
|---|---|---|
| M3 | 男にとってウミガメのスープの真偽が重要だった | S4 |

### M2 + S5 からの連鎖

**M2 + S5 →[ラベルと実体の乖離(7)]→ H1{認識}, H3{正体}**

今飲んだものは本物（S5）で味が異なる（M2）。ならば過去のものが本物ではなかった。

この導出で H1{認識}（過去に飲んだものをウミガメのスープだと認識していた）と H3{正体}（実際にはウミガメのスープではなかった）が同時に導出される。

### S5 + S6 からの連鎖

**S5 + S6 →[矛盾の指摘(6)]→ M4**

本物だと確認されたこと（通常は安心材料）が自殺につながっている。

| ID | 命題 | 前提 |
|---|---|---|
| M4 | 本物のウミガメのスープだと確認されたことが、男にとって破壊的だった | S5, S6 |

**M4 + H3{正体} →[因果の逆転(5)]→ M5**

| ID | 命題 | 前提 |
|---|---|---|
| M5 | 男が過去に飲んだものの正体を悟ったことが、自殺の原因である | M4, H3{正体} |

### S6 からの連鎖

**S6 →[動機の探索(8)]→ M6**

| ID | 命題 | 前提 |
|---|---|---|
| M6 | 男が自殺するほどの絶望があった | S6 |

**M6 + M5 →[常識的仮説の排除(1)]→ M7**

通常の食材（安価な代替品等）の発覚では自殺に至らない。

| ID | 命題 | 前提 |
|---|---|---|
| M7 | 男が過去に飲んだものの正体は、知ることが破壊的であるような内容だった | M6, M5 |

### S1 からの連鎖

**S1 →[人物像の探索(9)]→ H2{職業}**

男の属性を問う。H1{体験}を前提としない独立した探索。

### H1{体験} からの連鎖

**H1{体験} →[登場人物の探索(4)]→ H2{仲間}**

過去の体験に他の人物が関わっているかを問う。

**H1{体験} →[観測事実の深掘り(2)]→ H2{海}**

過去の体験がどこで起きたかを問う。H2{職業}（船乗り）が既知の場合、海への到達が自然になる。

**H1{体験} →[観測事実の深掘り(2)]→ H2{危機}**

過去の体験がどのような状況だったかを問う。

### H2{職業} + H2{海} からの連鎖

**H2{職業} + H2{海} →[観測事実の深掘り(2)]→ M8**

| ID | 命題 | 前提 |
|---|---|---|
| M8 | 男は船乗りとして航海中に何かを経験した | H2{職業}, H2{海} |

### H3{正体} からの連鎖

**H3{正体} →[因果の逆転(5)]→ H4{欺瞞}**

正体が異なっていた原因として、意図的な偽りがあったことが導出される。

**H3{正体} →[段階的絞り込み(11)]→ H3{食}**

正体がウミガメのスープでなかったなら、実際には何だったのかを問う。

### H3{食} からの連鎖

**H3{食} + M7 →[段階的絞り込み(11)]→ H3{禁忌}**

ウミガメ以外の食材で、かつ知ることが破壊的 → 口にすべきでないとされるもの。

### H3{禁忌} からの連鎖

**H3{禁忌} →[段階的絞り込み(11)]→ H3{死者}**

口にすべきでないものの中で、何か → 死んだ人の体。

### H4{欺瞞} からの連鎖

**H4{欺瞞} →[動機の探索(8)]→ H4{動機}**

なぜ偽りがあったのか → 特定の事情があった。

**H4{動機} →[観測事実の深掘り(2)]→ H4{窮状}**

どのような事情か → 食料が不足する状況。ただし H4{窮状}は H1{体験}を前提とする命題であり、この導出は H4{動機}と H1{体験}の両方を入力とする。

**H4{欺瞞} + H2{仲間} →[登場人物の探索(4)]→ H4{関係}**

偽りがあった + 他の人物が関わっている → 偽りの主体は男以外の人物。

## H 間の複数対複数の連鎖

### H2 + H3 の連動

**H2{仲間} + H2{危機} → 中間命題 → H3{死者}**:
- {他の人物が関わっている} + {生命に関わる危機的状況} →[因果の逆転(5): 危機で関係者に何が起きたか]→ {関わっていた人物に何かが起きた}（中間命題）
- {関わっていた人物に何かが起きた} →[段階的絞り込み(11)]→ {関わっていた人物が死んだ}（中間命題）
- {関わっていた人物が死んだ} + {ウミガメ以外の食材} →[手段の探索(10): 死んだ人の体はどうなったか]→ {死んだ人の体が利用された} = H3{死者}

**H2{仲間,危機} + H3{死者} ⇒ 三要素収束**:
- {他の人物が関わっている, 生命に関わる危機, 死んだ人の体が利用された} →[因果の逆転(5) + ラベルと実体の乖離(7)]⇒ {関わっていた人物が死に、その体がスープの材料になった}（Tの核心にほぼ到達）

### H2 + H4 の連動

**H2{危機} と H4{窮状}: 関連するが独立な命題**
- {生命に関わる危機的状況} と {食料が不足する状況} は T では同一状況の異なる側面だが、命題としては独立
- 一方から他方への導出は可能だが自明ではない:
  - {生命に関わる危機的状況} →[観測事実の深掘り(2): 危機の内容は]→ {食料が不足する状況だった} = H4{窮状}（危機の一つの具体化）
  - {食料が不足する状況} →[因果の逆転(5): なぜ食料が不足したか]→ {生命に関わる危機的状況だった} = H2{危機}（食料不足の原因として）

**H2{仲間} + H4{欺瞞} → H4{関係}**:
- {他の人物が関わっている} + {正体について偽りがあった} →[登場人物の探索(4): 偽りの主体は誰か]→ {偽りの行為の主体は男以外の人物だった} = H4{関係}

### H3 + H4 の連動

**H3{正体} と H4{欺瞞}: 同一事態の異なる記述**
- この二つは演繹連鎖の「正体を悟った」から導出される
- {飲んだものの正体が異なっていた} と {正体について偽りがあった}: 同じ事態を、認識の側面と行為の側面から記述
- ただし命題としては独立: 正体が異なっていた事実と、それが意図的な偽りであった事実は論理的に別

**H3{禁忌} + H4{動機}: 相互強化**
- {口にすべきでないもの} + {正体について偽りがあった} →[動機の探索(8): 禁忌を偽るには相応の事情が要る]→ {偽りの行為には特定の事情があった} = H4{動機}
- {特定の事情があった} + {ウミガメ以外の食材} →[常識的仮説の排除(1)]→ {口にすべきでないとされるものだった} = H3{禁忌} の確信強化

**H3{禁忌,食} + H4{欺瞞,動機} → 中間命題群**:
- {口にすべきでないもの, ウミガメ以外の食材, 正体について偽りがあった, 特定の事情があった} →[因果の逆転(5)]⇒ {特定の事情から、口にすべきでないものを食材とし、その正体を偽った}（中間命題）
- この中間命題は、パターン E3（罪悪モデル）の境界付近に対応する

### H2 + H3 + H4 の三要素連動

**H2{仲間,危機} + H3{死者} + H4{関係} → T の核心**:
- {他の人物が関わっている, 生命に関わる危機, 死んだ人の体が利用された, 偽りの主体は男以外の人物}
- →[因果の逆転(5) + ラベルと実体の乖離(7) + 動機の探索(8)]
- ⇒ {危機的状況で関わっていた人物が死に、別の人物が死者の体でスープを作り、正体を偽って男に飲ませた}
- = T のほぼ完全な復元

**H2{海,仲間,危機,職業} + H3{禁忌} + H4{動機,窮状} → T への急接近**:
- {船乗り, 他の人物, 海での出来事, 生命に関わる危機, 口にすべきでないもの, 特定の事情, 食料不足}
- →[常識的仮説の排除(1): 海で危機に陥り食料が尽きた状況で口にすべきでないものとは]
- ⇒ {人肉食} が強く示唆される
- この地点がパターン E3（H2高変種）の崩壊点

## 中間命題の一覧

戦術連鎖の過程で出現し、葉そのものではないが導出経路上に位置する命題群:

| 中間命題 | 導出元 | 戦術 | 接続先 |
|---|---|---|---|
| 味が記憶と違った | S3+S4 | 矛盾の指摘 | H1{体験,記憶} |
| 過去のスープは本物ではなかった | H1{記憶}+S5 | ラベルと実体の乖離 | H3{正体}, H4{欺瞞} |
| 航海中に何かがあった | H2{職業,海} | 観測事実の深掘り | H2{危機} |
| 関わっていた人物に何かが起きた | H2{仲間,危機} | 因果の逆転 | H3{死者} |
| 関わっていた人物が死んだ | 上記の絞り込み | 段階的絞り込み | H3{死者} |
| 食料が不足する状況だった | H2{危機} | 観測事実の深掘り | H4{窮状} |
| 特定の事情から禁忌を食材とし偽った | H3{禁忌,食}+H4{欺瞞,動機} | 因果の逆転 | パターンE3境界 |
| 死んだ人物の体がスープの材料だった | H2{仲間,危機}+H3{死者}+H4{関係} | 複合戦術 | T の核心 |

## 連鎖構造の観察

### 導出の自然な順序

演繹連鎖に沿った導出は概ね以下の順序で進む:
1. S → H1{体験,記憶} → H3{正体} → H4{欺瞞}（演繹連鎖の直接的帰結）
2. S → H2{職業} → H2{海}（人物像の探索から）
3. S → H2{仲間}（登場人物の探索から）
4. H3{正体} → H3{食} → H3{禁忌} → H3{死者}（段階的絞り込み）
5. H4{欺瞞} → H4{動機} → H4{窮状}（動機の深掘り）

ステップ1は演繹連鎖から比較的自然に導出される。ステップ2,3は独立した探索。ステップ4,5は段階的な具象化。

### 危険な合流点

複数の独立した導出経路が合流する地点が、T への急接近を引き起こす:

- **合流点A**: H2{仲間} + H3{死者} — 「他の人物が関わっている」と「死んだ人の体が利用された」が合流すると「関わっていた人物の体が利用された」に
- **合流点B**: H2{危機} + H4{窮状} — 「生命に関わる危機」と「食料不足」が合流すると遭難的状況に収束
- **合流点C**: H3{禁忌} + H2{海,仲間,危機} — 「口にすべきでないもの」が海での危機・他の人物という文脈と合流すると人肉食の示唆に

### 安全な並行経路

合流しない限り T に接近しにくい経路の組み合わせ:

- H2{職業,海} と H4{欺瞞} — 船乗りの背景と偽りの存在は、合流しても食品偽装モデル等で吸収可能
- H3{正体,食} と H2{職業} — 正体の相違・食材の問題と職業は多様な解釈を許す
- H1（全般）と H2（個別の葉） — H1 の具象化はどのモデルとも整合する

### パターン境界との対応

各代表モデルのパターン境界は、特定の合流が起きる直前の状態に対応する:

- **パターン E3（H2高）の境界**: 合流点C の直前。H2 が全て具象化し H3{禁忌} まで到達しているが、H3{死者}が未到達のため合流点A は発生していない
- **パターン E3'（H2低）の境界**: 合流点A に部分的に到達（H2{仲間}+H3{死者}）しているが、H2{海,危機}が未到達のため合流点C は発生していない
- **パターン E2 の境界**: 合流点B が未発生（H4 が未到達）。H2 と H3 に情報があるが、欺瞞の次元が不在
- **パターン D2 の境界**: 合流点A が未発生（H2{仲間}が未到達）。H3, H4 に情報はあるが「仲間」との接続がない
