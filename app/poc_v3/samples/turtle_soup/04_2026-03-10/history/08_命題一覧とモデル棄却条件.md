## モデル棄却条件の検討

各モデル（03 より）に対して、導入済み命題の中から棄却条件（OR of AND）を検討する。

棄却条件の基準: その命題集合が全て confirmed になった場合に、そのモデルがもはや成立し得ないこと。

### D1: 毒物モデル

「過去に毒物・薬物が混入されたスープを飲まされた」

モデルの核心: 食材の代替ではなく、スープへの異物混入。動機は害意。

| 棄却条件（AND グループ） | 根拠 |
|---|---|
| [H3{食}] | 「ウミガメ以外の食材で作られていた」= 食材自体が異なる。毒物モデルは食材の代替ではなく異物混入であり、食材が異なるという事実と矛盾 |
| [H3{死者}] | 「死んだ人の体が使われていた」→ 毒物とは全く異なるカテゴリ |
| [H4{窮状}] | 「食料が不足していた」→ 毒殺の文脈ではない |
| [M7, H3{食}] | 「知ることが破壊的な内容」+「食材が異なる」→ 毒物の発覚は破壊的だが、食材が異なるという条件と合わせると毒物モデルの枠を超える |
| [M8, H2{危機}] | 「船乗りとして航海中に経験」+「生命に関わる危機」→ 海難・遭難文脈。毒殺の文脈から離脱 |
| [H4{動機}, H4{窮状}] | 「やむを得ない事情」+「食料不足」→ 生存目的。毒殺の動機（害意）と矛盾 |

検討中:
- H3{禁忌} 単体: 「口にすべきでないもの」— 毒物も「口にすべきでないもの」と言えるか？ 毒物は確かに口にすべきでないが、H3{禁忌} の文脈は「食材としての禁忌」（食の文化的・倫理的タブー）を指す。ただし命題のラベル自体はそこまで限定していない。→ 単体では棄却力が弱い
- H2{海} 単体: 海で毒殺されることは不可能ではない → 単体不十分
- H2{職業} 単体: 船乗りが毒殺されることは不可能ではない → 単体不十分

### D2: ペット・愛着動物モデル

「大切にしていた動物の肉をウミガメのスープとして飲まされた」

モデルの核心: 食材は動物の肉。人間ではない。

| 棄却条件（AND グループ） | 根拠 |
|---|---|
| [H3{死者}] | 「死んだ人の体が使われていた」→ 人間であり、動物ではない |
| [M8, H2{危機}] | 「航海中に経験」+「生命に関わる危機」→ 海難文脈。ペットとの日常的関係の文脈から離脱 |
| [H2{海}, H2{仲間}] | 「海で起きた」+「他の人物が関わっている」→ 海での人間関係の出来事。ペット文脈からの離脱 |
| [I2] | 「関わっていた人物が死んだ」→ 人間の死。ペットモデルでは人は死なない |

検討中:
- H2{仲間} 単体: 「他の人物が関わっている」— ペットモデルでも他の人物が料理した可能性がある → 単体不十分
- H4{関係} 単体: 「偽りの主体は男以外の人物」— ペットモデルでも他者が偽ることはありうる → 単体不十分

### D3: 宗教的禁忌モデル

モデルの核心: 食の禁忌は宗教的なもの。状況は日常的。

| 棄却条件（AND グループ） | 根拠 |

| [H4{窮状}] | 「食料が不足していた」→ 宗教的偽装の動機は信仰の無知・軽視であり、生存文脈ではない |
| [M8] | 「船乗りとして航海中に経験」→ 宗教的禁忌の文脈と結びつきにくい |
| [H2{海}, H2{危機}] | 「海」+「危機」→ 海難文脈。宗教的禁忌の日常的文脈から離脱 |
| [H4{動機}, H4{窮状}] | 「やむを得ない事情」+「食料不足」→ 生存目的。宗教的無知による偽装とは異なる動機 |

検討中:
- H2{危機} 単体: 「生命に関わる危機」→ 宗教的禁忌が危機的状況で起きることは不可能ではない（強制改宗等）が、やや不自然 → 弱い棄却力

## 問題点

### 問題 1: 偽命題の混入

T において、男が過去に飲んだものは人肉スープであり、ウミガメのスープではない。以下の命題は「ウミガメのスープ」というラベルを事実として記述しており、T に照らして偽である。

| ID | label | 真偽 | 理由 |
|---|---|---|---|
| H1{体験} | 男は過去にウミガメのスープを飲んだことがある | 偽 | 飲んだのは人肉スープ。ウミガメのスープは一度も飲んでいない |
| H1{記憶} | 男は過去に飲んだウミガメのスープの味を覚えている | 偽 | 覚えているのは人肉スープの味 |
| M2 | 男は過去に飲んだウミガメのスープの味と、今飲んだものの味が異なると感じた | 偽 | 同上 |

H1{認識}（「ウミガメのスープだと**認識していた**」）は真。{体験}と{記憶}は認識ではなく事実として記述しているため偽。

### 問題 2: H 葉命題の抽象度不足

葉命題は仮説木の最外縁であり、最も抽象化された命題であるべきである。複数の競合モデルが共有できる座標軸として機能するためには、T の具体的要素ではなく、抽象的なカテゴリでなければならない。しかし現状の H 葉の多くは T の具象をそのまま置いている。

| ID | label | 問題 |
|---|---|---|
| H1{体験} | 男は過去にウミガメのスープを飲んだことがある | 偽である上に具象的。T の具体的食品名 |
| H1{記憶} | 男は過去に飲んだウミガメのスープの味を覚えている | 同上 |
| H2{職業} | 男は船乗りだった | T の具体的職業。抽象的な葉ではない |
| H3{死者} | 男が過去に飲んだものには死んだ人の体が使われていた | T の核心そのもの。葉として抽象的でない |
| H4{窮状} | 男の過去の体験の状況において食料が不足していた | T の具体的状況 |

対照的に、適切に抽象化されている葉:
- H2{仲間}「他の人物が関わっている」
- H2{危機}「生命に関わる危機的状況だった」
- H4{欺瞞}「意図的な偽りがあった」
- H3{禁忌}「口にすべきでないもの」
