# パターンと代表モデルの構築

## 入力

- Phase 1 の緊張構造分析、仮説的要素 H1-H4、初期競合モデル
- 抽象の葉（02）

## 競合モデルの体系的列挙

### 列挙の方針

競合モデルを二つの方向から列挙する:

1. **Tの演繹連鎖を共有し、仮説的要素（H）を代替するモデル**（D系列）
2. **Sの緊張を異なる演繹連鎖で解消するモデル**（B/C系列）

### Sの緊張構造（再掲）

- 緊張A: 一口で真偽を確認する切迫感（S3→S4）
- 緊張B: 本物の確認が自殺のトリガー（S5→S6）
- 緊張C: スープ体験から自殺への因果の飛躍（S3→S6）
- 緊張D: 確認行為自体の異常性 — 普通はウミガメのスープの真偽を店員に確認しない

### Tの演繹連鎖（参照）

一口飲んだ → 味が記憶と違う → 本物か確認 → 本物だと判明 → 過去に飲んだものは本物ではなかった → 正体を悟った → 破壊的な感情 → 自殺

---

## D系列: Tの連鎖を共有し、Hを代替するモデル

Tと同じ演繹連鎖（味の違い → 確認 → 過去は偽物 → 正体の発覚 → 絶望）を共有し、仮説的要素の内容が異なるモデル。

### D1: 毒物モデル

「過去に飲まされたスープには毒物・薬物が混入されていた。本物の味との違いで、過去のスープが偽物だったと悟り、毒を盛られていたことを確信した」

- H2 代替: 毒を盛る動機のある関係・状況（家庭内、職場等）
- H3 代替: 毒物・薬物
- H4 代替: 害意による偽装
- 緊張の緩和: A✓ B✓ C△（自殺動機: 信頼の裏切りだが、過去の毒なら現在の危機ではない） D✓
- 想起しやすさ: やや高い（毒殺は日常的ではないがフィクションでは一般的）

### D2: ペット・愛着動物モデル

「過去にペットや大切にしていた動物の肉をウミガメのスープとして飲まされた。本物の味で正体に気づいた」

- H2 代替: 動物と暮らしていた状況
- H3 代替: 愛着のある動物の肉
- H4 代替: 知らせずに処分・調理した
- 緊張の緩和: A✓ B✓ C△（自殺動機: 感情的には強いが破壊性にやや欠ける） D✓
- 想起しやすさ: 中程度

### D3: 宗教的禁忌モデル

「過去に宗教上厳格に禁じられた食材をウミガメのスープとして飲まされた。本物の味で正体に気づき、信仰上の取り返しのつかない違反を悟った」

- H2 代替: 信仰の厳格な環境
- H3 代替: 宗教的禁忌の食材
- H4 代替: 信仰を知らない・軽視する者による偽装
- 緊張の緩和: A✓ B✓ C△（自殺動機: 信仰の深さ次第で成立しうる） D✓
- 想起しやすさ: 中程度（文化依存）

### D4: 人体由来モデル（T亜種）

「過去に人体の一部（血液、臓器等）を含むスープをウミガメのスープとして飲まされた。ただし遭難ではなく、別の文脈（犯罪、儀式等）」

- H2 代替: 犯罪・儀式等の文脈
- H3 代替: 人体由来物質（人肉食の亜種）
- H4 代替: 文脈に応じた欺瞞
- 緊張の緩和: A✓ B✓ C✓（自殺動機は強い） D✓
- 想起しやすさ: 低い（Tとほぼ同程度に非日常的）
- 備考: Tに非常に近く、独立した競合モデルとしての意義は薄い

## B系列: 恐れの確認連鎖

演繹連鎖: 疑念を持っていた → スープで確かめた → 確認 → 恐れていたことが確定 → 自殺

Tの連鎖との違い: 「過去のスープが偽物だった」という推論を必ずしも含まない。「確認行為の動機」がTとは異なる。

### B1: 過去の被害確認モデル

「男は過去に飲んだスープに異常があったことを疑っていた。本物の味を確かめるために注文し、味の違いから疑念が確定した。信頼していた人物による加害を悟り、絶望した」

- 緊張の緩和: A✓（比較目的の注文） B✓ C△（自殺動機は裏切りの程度次第） D✓（確認は比較のため）
- 想起しやすさ: 中程度
- 備考: 構造的にTの連鎖にかなり近い（「過去が偽物→正体の発覚」を含む）

### B2: 陰謀確認モデル

「男は自分が何者かに狙われており、食事に何かを混入されていると疑っていた。ウミガメのスープを注文して安全な味を確認しようとしたが、本物だと確認されたことで、逆に自分の疑念の根拠を失い、精神的に追い詰められた（被害妄想の確認）」

- 緊張の緩和: A✓ B△（確認後の絶望の構造が異なる） C△（自殺動機: 精神的崩壊） D✓
- 想起しやすさ: やや低い
- 備考: 自殺の機序がTとかなり異なる

---

## C系列: 禁忌違反連鎖

演繹連鎖: 本物を口にしてはならない理由がある → 模造品を前提に注文 → 一口で違和感 → 本物か確認 → 本物だった → 違反が確定 → 絶望 → 自殺

Tの連鎖との違い: 「過去のスープが偽物だった」ではなく「今のスープが本物であること自体が問題」。確認行為の方向が逆転（Tでは過去を疑うための確認、Cでは現在を恐れての確認）。

### C1: 医療禁忌モデル

「男はウミガメに対する重篤なアレルギーまたは持病があり、本物は禁忌。模造品が一般的であることを前提に注文したが、本物だった」

- 緊張の緩和: A✓（違和感からの確認） B✓（本物=危険） C△（自殺動機: 健康不安だけでは弱い） D✓（確認は恐れからの行為）
- 想起しやすさ: 高い（アレルギーは日常的）
- 備考: 自殺動機が非常に弱い

### C2: 誓約違反モデル

「男は大切な人（故人）との誓いにより本物のウミガメのスープを二度と口にしないと決めていた。模造品を前提に注文したが、本物だった。誓約を破った罪悪感から自殺」

- 緊張の緩和: A✓ B✓（本物=誓約違反） C△（自殺動機: 誓約の重み次第） D✓
- 想起しやすさ: 中程度
- 備考: 「なぜ模造品でも注文するのか」にやや緊張が残る

### C3: 倫理的禁忌モデル

「男はウミガメの保護活動に深く関わっており、本物のウミガメのスープを口にすることは自らの信念を裏切る行為。模造品を前提に注文したが、本物だった」

- 緊張の緩和: A✓ B△ C△（自殺動機: 非常に弱い） D✓
- 想起しやすさ: やや高い
- 備考: 自殺への接続がほとんど成立しない

## E系列: 初期モデル（m0 近傍）

Sを読んだ直後に最も想起しやすいモデル群。緊張の緩和が小さい。

### E1: 食品偽装モデル

「過去にウミガメのスープとして安価な食材を出された。本物との味の違いで偽装に気づいた」

- 緊張の緩和: A✓ B△ C✗（自殺動機が成立しない） D✓
- 想起しやすさ: 非常に高い

### E2: 思い出モデル

「大切な人との思い出にウミガメのスープが関わっていた。味の違いで喪失を実感した」

- 緊張の緩和: A△ B△ C△（弱い） D△（確認行為の説明が弱い）
- 想起しやすさ: 高い

### E3: 罪悪モデル

「過去に罪深いことをした際にウミガメのスープが関わっていた。本物の味で過去を突きつけられた」

- 緊張の緩和: A✓ B✓ C△（罪の内容次第） D✓
- 想起しやすさ: 中〜高
- 備考: 最も広い情報状態で生存可能。「罪の内容」が不定であるために、Hの具象化に対する柔軟性が高い

## 各モデルの H 具象化上限（パターン候補）

各モデルについて、H1-H4 の具象化がどの程度まで進んだ段階でモデルが T に対する競争力を失うかを検討する。上限は葉の組み合わせで表現する。

※ B/C 系列は T とは異なる演繹連鎖を持つため、H（T の仮説的要素）への対応が間接的な場合がある。H の具象化は「プレイヤーが観測を通じて得る情報」として、モデルの連鎖に関わらず共通の座標軸となる。

### D1: 毒物モデル

- H1{体験,記憶,認識}: 過去の体験は毒物モデルにも必要 → **全て許容**
- H2: 毒を盛る文脈は商業・家庭・職場等であり、海や遭難とは結びつかない
  - {仲間}: 仲間による加害として成立 → 許容
  - {海}: 海の出来事は毒物と結びつきにくい → 排除
  - {危機}: 危機的状況からの毒殺はやや不自然 → 排除
  - {職業}: 船乗りは毒物と直接関連しない → 排除
  - → **H2{仲間}**
- H3: 毒物は「正体が異なる」に合致するが、禁忌の食・死者とは合わない
  - {正体}: 核心 → 許容
  - {食}: 特殊食材とは異なる方向 → 排除
  - {禁忌}: 毒は禁忌だが食の禁忌ではない → 排除
  - {死者}: 死者の利用は毒物と無関係 → 排除
  - → **H3{正体}**
- H4: 欺瞞は毒物モデルの核心だが、動機や関係が異なる
  - {欺瞞}: 核心 → 許容
  - {関係}: 仲間が主体 → H2{仲間}と整合するなら許容
  - {動機}: 生存目的は毒殺の動機と異なる → 排除
  - {窮状}: 食料不足は毒殺と無関係 → 排除
  - → **H4{欺瞞, 関係}**

**パターン D1**: H1{体験,記憶,認識}, H2{仲間}, H3{正体}, H4{欺瞞,関係}

### D2: ペット・愛着動物モデル

- H1: **全て許容**
- H2: ペットの文脈に海や遭難は不要
  - {仲間}: ペットを「仲間」と捉えれば成立するが、H2 の仲間は人間を指す → 排除
  - {危機}: 危機的状況でペットを食べざるを得なかった → 弱く許容
  - {海},{職業}: 結びつかない → 排除
  - → **H2{危機}**（弱い）
- H3: ペットの肉は「特殊な食材」「禁忌に近い」
  - {正体}: 許容
  - {食}: 特殊食材として許容
  - {禁忌}: ペットを食べることは社会的禁忌に近い → 弱く許容
  - {死者}: 死んだ「人」ではない → 排除
  - → **H3{正体, 食, 禁忌}**（禁忌は弱い）
- H4: 偽装した動機
  - {欺瞞}: 許容
  - {動機}: 生存目的ならペット文脈とも合う → 許容
  - {関係}: 仲間が主体 → ペット文脈では不自然 → 排除
  - {窮状}: 食料不足 → 危機と連動して許容
  - → **H4{欺瞞, 動機, 窮状}**

**パターン D2**: H1{体験,記憶,認識}, H2{危機}, H3{正体,食,禁忌}, H4{欺瞞,動機,窮状}

ただし H2 が{危機}を超えて{海}や{仲間}に達すると即座に崩壊する。

### D3: 宗教的禁忌モデル

- H1: **全て許容**
- H2: 宗教的文脈に遭難は不要
  - {仲間}: 信仰共同体の仲間として弱く成立 → 弱く許容
  - {海},{危機},{職業}: 宗教的禁忌と結びつかない → 排除
  - → **H2{仲間}**（弱い）
- H3: 宗教的禁忌の食材
  - {禁忌}: 核心 → 許容
  - {正体}: 許容
  - {食}: 特殊食材として許容
  - {死者}: 宗教的禁忌は死者の利用とは異なる → 排除
  - → **H3{禁忌, 正体, 食}**
- H4: 宗教を知らない者による偽装
  - {欺瞞}: 許容
  - {動機}: 生存目的とは異なる動機 → 排除
  - {関係}: 仲間とは限らない → 排除
  - {窮状}: 食料不足は宗教的禁忌と無関係 → 排除
  - → **H4{欺瞞}**

**パターン D3**: H1{体験,記憶,認識}, H2{仲間}, H3{禁忌,正体,食}, H4{欺瞞}

### B1: 過去の被害確認モデル

構造が T の連鎖に非常に近いため、パターンも D1 に類似する。

**パターン B1**: H1{体験,記憶,認識}, H2{仲間}, H3{正体}, H4{欺瞞,関係} ≈ パターン D1

### B2: 陰謀確認モデル

演繹連鎖が T と大きく異なり、H の具象化が進むとすぐに競争力を失う。

- H2: いかなる具象化も陰謀モデルを支持しない → **H2{}**
- H3: 同上 → **H3{}**
- H4: 同上 → **H4{}**

**パターン B2**: H1{体験,記憶,認識}, H2{}, H3{}, H4{}

H1 以外の情報がほぼない初期状態でのみ生存。代表モデルとしての機能は期待できない。

### C1: 医療禁忌モデル

- H2: 医療文脈に遭難は無関係 → **H2{}**
- H3: {正体}のみ弱く許容（本物かどうかの関心） → **H3{正体}**（弱い）
- H4: 医療禁忌に欺瞞は不要 → **H4{}**

**パターン C1**: H1{体験,記憶,認識}, H2{}, H3{正体}, H4{}

パターン B2 に近い。非常に狭い。

### C2: 誓約違反モデル

- H2: 誓約の相手としての仲間 → **H2{仲間}**（弱い）
- H3: 誓約モデルはスープの正体に関心がない → **H3{}**
- H4: 誓約モデルには欺瞞が不要 → **H4{}**

**パターン C2**: H1{体験,記憶,認識}, H2{仲間}, H3{}, H4{}

H3, H4 がほぼ許容されない。非常に狭い。

### C3: 倫理的禁忌モデル

- H2: 環境保護の文脈に遭難は無関係 → **H2{}**
- H3, H4: 倫理的禁忌はスープの正体や欺瞞と無関係 → **H3{}, H4{}**

**パターン C3**: H1{体験,記憶,認識}, H2{}, H3{}, H4{}

パターン B2 と同等。初期状態でのみ生存。

### E1: 食品偽装モデル

- H2: 商業文脈。{職業}は船上の食事としてやや接点 → **H2{職業}**
- H3: {正体}は核心。{食}はやや許容 → **H3{正体, 食}**
- H4: {欺瞞}は核心 → **H4{欺瞞}**

**パターン E1**: H1{体験,記憶,認識}, H2{職業}, H3{正体,食}, H4{欺瞞}

### E2: 思い出モデル

- H2: {海}思い出の場所, {仲間}大切な人, {職業}やや互換 → **H2{海,仲間,職業}**
- H3: {死者}大切な人の死, {食}特別な食事 → **H3{死者,食}**
- H4: 思い出モデルに欺瞞は不要 → **H4{}**

**パターン E2**: H1{体験,記憶,認識}, H2{海,仲間,職業}, H3{死者,食}, H4{}

H4 の具象化に対して非常に脆弱。

### E3: 罪悪モデル

- H2: 罪悪感は多様な状況と結合可能 → **H2{海,仲間,危機,職業}**
- H3: 禁忌・死者は罪悪の文脈に合う。ただし **H2 との相互作用**:
  - H2 高{海,仲間,危機,職業} の場合: H3{禁忌} まで（禁忌+死者+海+仲間 → T がほぼ特定）
  - H2 低{仲間} の場合: H3{禁忌,死者} まで
- H4: {動機,欺瞞,窮状}は罪悪の文脈に合う。{関係}は罪の主体が仲間に移るため方向が異なる → **H4{動機,欺瞞,窮状}**

**パターン E3（H2 高）**: H1{体験,記憶,認識}, H2{海,仲間,危機,職業}, H3{禁忌}, H4{動機,欺瞞,窮状}

**パターン E3'（H2 低）**: H1{体験,記憶,認識}, H2{仲間}, H3{禁忌,死者}, H4{動機,欺瞞,窮状}

## 代表モデルの選定

### 不適格の判断

以下のモデルは代表モデルとして不適格とする:

- **B2 陰謀, C1 医療禁忌, C3 倫理的禁忌**: H1 以外ほぼ許容なし。初期状態でのみ生存し、T との境界を形成しない
- **C2 誓約違反**: H3, H4 が許容されず、非常に狭い
- **B1 過去の被害確認**: D1 とほぼ同一のパターン。独立したモデルとして扱う意義がない
- **D4 人体由来（T亜種）**: T に近すぎ、独立した競合モデルとしての意義がない

B/C 系列は全般的にパターンが狭い。H の具象化が進むこと自体が T の連鎖を支持するため、異なる連鎖のモデルは急速に弱体化する。これらは初期モデル（m0 近傍）として自然に脱落するものであり、代表モデルではない。

### 代表モデルとして採用

| ID | モデル | 根拠 |
|---|---|---|
| E3 | 罪悪モデル | 最も広い具象化空間で生存。H2-H3 の相互作用で複数の変種を持つ。最有力 |
| E2 | 思い出モデル | H2 がやや広い。H4 に脆弱だが、H4 未知の状態で競合可能 |
| D2 | ペットモデル | H3, H4 にやや広い。H2 が狭いが、D系列の中では最も機能的 |
| D3 | 宗教的禁忌モデル | H3（禁忌方向）に強い。H2, H4 が狭いが特定の経路で競合可能 |
| E1 | 食品偽装モデル | 各 H に最小限の足場がある。想起しやすさが非常に高く、広い初期段階で競合 |
| D1 | 毒物モデル | E1 に類似するが仲間の文脈を許容。E1 より狭いが異なる方向性 |

## 各代表モデルの境界（命題表現）

各パターンの上限を、プレイヤーがその時点で把握している情報として命題で記述する。この情報状態において代表モデルはなお T より想起しやすい。

### パターン E3: 罪悪モデル（H2 高変種）

上限: H1{体験,記憶,認識}, H2{海,仲間,危機,職業}, H3{禁忌}, H4{動機,欺瞞,窮状}

境界の命題:
- 男は過去にウミガメのスープを飲んだ記憶があり、味を覚えている

- 極限状態で、やむを得ない事情から何かが偽られた

この情報状態での罪悪モデル:「男は遭難中に何か禁忌に触れる罪深いことに関わり、それが偽られていた。本物のスープの味で過去を突きつけられ、罪の意識から自殺した」

T への接近条件: H3 に{死者}が加わると「遭難中に仲間が死に、禁忌に関わる何かがあった」→ 人肉食がほぼ特定され、罪悪モデルは T に吸収される。
