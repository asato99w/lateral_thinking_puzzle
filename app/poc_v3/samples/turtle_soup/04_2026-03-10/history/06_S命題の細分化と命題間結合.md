# S 命題の細分化と命題間結合

## 目的

S を最小単位の命題に分解し、S 分解要素（または空集合）を起点として、戦術連鎖により中間命題を導入しつつ、全ての葉命題への到達経路を構築する。

要件:
- 全ての葉命題が到達可能であること
- S 要素を全て使い切る必要はない
- 各連結は文脈に依存しない純粋な戦術適用として記述する

---

## S の最小分解

S: ある男がレストランに入り、「ウミガメのスープ」を注文した。一口飲んだ男は店員に「これは本当にウミガメのスープか？」と尋ねた。店員は「はい、間違いなくウミガメのスープです」と答えた。男はその夜、自殺した。

### S1: ある男がレストランに入った

| ID | 命題 |
|---|---|
| s1.1 | ある人物が存在する |
| s1.2 | その人物は男性である |
| s1.3 | その人物は特定されていない |
| s1.4 | その人物はある場所に入った |
| s1.5 | その場所はレストランである |
| s1.6 | レストランは食事を提供する施設である |
| s1.7 | レストランは営業施設である（客として訪れている） |

### S2: 男は「ウミガメのスープ」を注文した

| ID | 命題 |
|---|---|
| s2.1 | 男は料理を注文した |
| s2.2 | 注文は消費の意図を含む |
| s2.3 | 注文した料理には「ウミガメのスープ」という名称がある |
| s2.4 | その料理はメニューに存在した |
| s2.5 | 「ウミガメのスープ」は特定の食材（ウミガメ）に基づく名称である |

### S3: スープを一口飲んだ

| ID | 命題 |
|---|---|
| s3.1 | 男はスープを飲んだ |
| s3.2 | 飲んだ量は一口である |
| s3.3 | 一口は最小限の量である |
| s3.4 | 飲む行為により男は味を知った |
| s3.5 | 男はそれ以上飲まなかった |

### S4: 男は店員に「これは本当にウミガメのスープか？」と尋ねた

| ID | 命題 |
|---|---|
| s4.1 | 男は質問をした |
| s4.2 | 質問の相手は店員である |
| s4.3 | 店員はこの料理について知る立場にある |
| s4.4 | 質問の内容は「これは本当にウミガメのスープか」である |
| s4.5 | 男はこのスープの名称の真正性を疑っている |
| s4.6 | 男には疑う根拠がある |
| s4.7 | 疑いは飲んだ後に生じた |
| s4.8 | 名称の真正性が男にとって重要である |

### S5: 店員は「はい、間違いなくウミガメのスープです」と答えた

| ID | 命題 |
|---|---|
| s5.1 | 店員は質問に回答した |
| s5.2 | 回答は肯定である |
| s5.3 | 肯定は強い（「間違いなく」） |
| s5.4 | このスープはウミガメのスープである（店員の証言） |
| s5.5 | この確認は料理について知る立場の者による |

### S6: 男はその夜、自殺した

| ID | 命題 |
|---|---|
| s6.1 | 男は死んだ |
| s6.2 | 死因は自殺である |
| s6.3 | 自殺は自らの意志による |
| s6.4 | 自殺に至るには強い動機がある |
| s6.5 | 時期は「その夜」である |
| s6.6 | 「その夜」はレストランの出来事と同日である |
| s6.7 | 同日中であることは因果的近接を示唆する |

## 各葉への到達経路

各葉命題について、S分解要素または空集合から戦術連鎖でどのように到達できるかを検討する。各葉を独立に扱い、そこへの到達を考える。

### H1{体験}: 男は過去にウミガメのスープを飲んだことがある

**経路 1**: s4.6（男には疑う根拠がある）+ s3.4（飲んで味を知った）→[時間軸の拡張(3)]→ 味に基づく疑いの根拠は過去の味覚体験 → **H1{体験}**

**経路 2**: s4.8（名称の真正性が重要）→[動機の探索(8)]→ なぜ重要か → 個人的な過去の体験がある → **H1{体験}**

### H1{記憶}: 男は過去に飲んだウミガメのスープの味を覚えている

**経路**: s3.3（一口は最小限）+ s4.7（疑いは飲んだ後）→[矛盾の指摘(6)]→ 最小量で即座に疑える → 比較可能な味の記憶がある → **H1{記憶}**

### H1{認識}: 男は過去に飲んだものをウミガメのスープだと認識していた

**経路**: s4.4（「これは本当にウミガメのスープか」）+ s2.5（ウミガメに基づく名称）→[ラベルと実体の乖離(7)]→ 男が「ウミガメのスープ」というラベルを過去の体験に適用していた → **H1{認識}**

備考: 「本当にウミガメのスープか」という問いは、過去に「ウミガメのスープ」として飲んだものとの比較を前提とする。比較対象にも同じラベルが貼られていた。

### H2{職業}: 男は船乗りだった

**経路**: s1.3（人物は特定されていない）→[人物像の探索(9)]→ この人物の職業は何か → **H2{職業}**

備考: 空集合に近い。「ある人物が存在する」だけで職業を問える。

### H2{海}: 男の過去の体験は海で起きた

**経路 1**: H2{職業}（船乗り）→[観測事実の深掘り(2)]→ 船乗りの活動場所は海 → **H2{海}**

**経路 2**: 空集合 →[段階的絞り込み(11)]→ 過去の体験はどこで起きたか → 屋外か屋内か → 陸か海か → **H2{海}**

### H2{仲間}: 男の過去の体験には他の人物が関わっている

**経路**: 空集合 →[登場人物の探索(4)]→ 体験に他の人物が関わっているか → **H2{仲間}**

備考: 登場人物の探索は空集合から適用可能（戦術定義: 明示されていない人物の存在を問う）。

### H2{危機}: 男の過去の体験は生命に関わる危機的状況だった

**経路 1**: s6.4（自殺に至る強い動機）→[因果の逆転(5)]→ 動機の原因は過去の深刻な体験 →[段階的絞り込み(11)]→ 深刻な体験とは → 生命に関わる危機 → **H2{危機}**

**経路 2**: 空集合 →[段階的絞り込み(11)]→ 過去の体験はどのような状況だったか → 日常か非日常か → 生命に関わるか → **H2{危機}**

### H3{正体}: 男が過去に飲んだものはウミガメのスープではなかった

**経路**: s5.4（このスープはウミガメのスープである）+ s4.6（男には疑う根拠がある）→[ラベルと実体の乖離(7)]→ 今のが本物で味が違う → 過去のは本物ではなかった → **H3{正体}**

### H3{食}: 男が過去に飲んだものはウミガメ以外の食材で作られていた

**経路**: H3{正体}（ウミガメのスープではなかった）→[段階的絞り込み(11)]→ では実際は何で作られていたか → ウミガメ以外の食材 → **H3{食}**

備考: H3{正体}を前提とするため、S から直接は到達しない。

### H3{禁忌}: 男が過去に飲んだものは口にすべきでないとされるものだった

**経路 1**: s6.4（自殺に至る強い動機）+ H3{正体} →[常識的仮説の排除(1)]→ 通常の食材の相違では自殺に至らない → 口にすべきでないもの → **H3{禁忌}**

**経路 2**: H3{食}（ウミガメ以外の食材）→[段階的絞り込み(11)]→ どのような食材か → 通常の食材か → 禁忌に属するものか → **H3{禁忌}**

### H3{死者}: 男が過去に飲んだものには死んだ人の体が使われていた

**経路 1**: H3{禁忌}（口にすべきでないもの）→[段階的絞り込み(11)]→ 何が禁忌か → 人に関わるものか → 死んだ人の体 → **H3{死者}**

**経路 2**: H2{仲間}（他の人物が関わっている）+ H2{危機}（生命に関わる危機）→[因果の逆転(5)]→ 危機で人が死んだ →[手段の探索(10)]→ 死んだ人の体はどうなったか → 食材に使われた → **H3{死者}**

### H4{欺瞞}: 男が過去に飲んだものの正体について意図的な偽りがあった

**経路 1**: H3{正体}（本物ではなかった）→[因果の逆転(5)]→ なぜ正体が異なっていたか → 偶然か意図的か → 意図的な偽り → **H4{欺瞞}**

**経路 2**: s2.3（「ウミガメのスープ」という名称がある）+ H3{正体} →[ラベルと実体の乖離(7)]→ 名称と実体が異なる → 誰かが名称を偽った → **H4{欺瞞}**

### H4{関係}: その偽りの主体は男以外の人物だった

**経路**: H4{欺瞞}（偽りがあった）→[登場人物の探索(4)]→ 偽りの主体は誰か → 男自身か他者か → 男以外の人物 → **H4{関係}**

備考: H2{仲間}が既知の場合、他者の候補が具体化する。

### H4{動機}: その偽りには特定の事情があった

**経路 1**: H4{欺瞞}（偽りがあった）→[動機の探索(8)]→ なぜ偽ったか → 特定の事情がある → **H4{動機}**

**経路 2**: H3{禁忌}（口にすべきでないもの）+ H4{欺瞞} →[動機の探索(8)]→ 禁忌のものを偽る必然性 → 強い事情 → **H4{動機}**

### H4{窮状}: 男の過去の体験の状況において食料が不足していた

**経路 1**: H2{危機}（生命に関わる危機）→[観測事実の深掘り(2)]→ 危機の具体的内容 → 食料不足 → **H4{窮状}**

**経路 2**: H4{動機}（偽りの事情）+ H3{食}（食材の問題）→[因果の逆転(5)]→ 食材を偽る事情 → 正規の食料がない → **H4{窮状}**

**経路 3**: 空集合 →[段階的絞り込み(11)]→ 過去の状況に何か制約があったか → 物資に関する制約 → 食料不足 → **H4{窮状}**

全15葉が到達可能。

### S 要素との接続状況

S分解要素から直接（1ステップで）葉に至るもの:
- **s1.3** → H2{職業}
- **s3.3 + s4.7** → H1{記憶}
- **s3.4 + s4.6** → H1{体験}
- **s4.4 + s2.5** → H1{認識}
- **s5.4 + s4.6** → H3{正体}
- **s6.4** → H2{危機}（間接的）、H3{禁忌}（H3{正体}も必要）

S要素に直接接続しない葉: H2{海}, H2{仲間}, H3{食}, H3{死者}, H4{欺瞞}, H4{関係}, H4{動機}, H4{窮状}

これらは他の葉や空集合からの戦術適用で到達する。S からの距離が遠い葉ほど、プレイヤーの能動的な探索（水平思考）を要する。
