# 仮説要素の関係分析

## 目的

質問生成の基盤として、仮説的要素（H1-H4）の具象化方向の内部関係と、要素間のつながりを分析する。

## 各 H の具象化方向の内部関係

### H1:「過去にウミガメのスープを飲んだという認識がある」

葉: 体験 / 記憶 / 認識

- {体験} と {記憶} は密接: 体験があるから記憶がある
- {認識} は {体験,記憶} を包含する上位概念に近い: 体験の記憶に基づく認識
- 3つの葉は相互に独立性が低く、いずれかが具象化すると他も連動しやすい
- H1 は全体として「一枚岩」に近い — 部分的具象化があまり意味をなさない

### H2:「仲間と海で遭難した」

葉: 海 / 仲間 / 危機 / 職業

- {海} と {職業}: 船乗りという職業が海との接点を与える。{職業} → {海} は自然な推論
- {海} と {危機}: 海での危機 = 遭難。この組み合わせで遭難がほぼ特定される
- {仲間} と {危機}: 仲間との危機的体験。遭難以外にも解釈可能（戦争、事故等）
- {仲間} と {海}: 仲間と海での出来事。遭難以外にも解釈可能だが範囲は狭まる
- {職業} は他の葉と比較的独立: 船乗りという情報だけでは状況が特定されない

具象化の方向:
- {海,危機} → 遭難がほぼ直接的に示唆される（強い収束）
- {仲間,危機} → 困難な共同体験（遭難以外の可能性も残る）
- {職業} 単独 → 背景情報にとどまり、状況を限定しない

### H3:「死んだ仲間の肉で作ったスープだった」

葉: 禁忌 / 死者 / 食 / 正体

- {正体} は最も抽象的: 「正体が異なっていた」だけでは内容が不定
- {食} は正体を食の文脈に限定: 「特殊な食材で作られた」
- {禁忌} は倫理的次元を追加: 「口にすべきでないもの」
- {死者} は人の死を導入: 「死んだ人が関わる」
- {禁忌} + {死者}: この組み合わせは人肉食を強く示唆する
- {食} + {正体}: 食品偽装的な方向（比較的日常的）
- {禁忌} + {食}: 禁忌の食材（宗教的禁忌、ペット等、人肉以外も含む）

具象化の方向:
- {正体} → {食,正体} → {禁忌,食,正体}: 段階的に深刻さが増す
- {死者} の追加はどの段階でも大きな飛躍: 人の死が関わることで一気にTに接近
- {禁忌,死者} の組み合わせが最も危険（人肉食がほぼ特定）

### H4:「食料が尽き、生存のために仲間が偽った」

葉: 動機 / 欺瞞 / 関係 / 窮状

- {欺瞞} は最も抽象的: 「何かが偽られた」だけ
- {関係} は行為の主体: 「仲間が行った」
- {動機} は行為の理由: 「やむを得ない事情」
- {窮状} は状況の深刻さ: 「食料が尽きた極限状態」
- {欺瞞} + {関係}: 仲間による欺瞞（多様な解釈可能）
- {動機} + {窮状}: やむを得ない状況（生存文脈を示唆）
- {窮状} は H2{危機} と強く連動する

具象化の方向:
- {欺瞞} 単独 → 食品偽装的モデルでも成立する広い概念
- {欺瞞,関係} → 仲間が偽った（個人的な関係の中での欺瞞）
- {動機,窮状} → 極限状態でのやむを得ない行為（生存文脈に収束）
- {欺瞞,関係,動機,窮状} → Tにかなり接近

## 仮説要素間のつながり

### H2 — H3 間

- **H2{仲間} ↔ H3{死者}**: 仲間が死んだ → その肉がスープになった。最も直接的な要素間接続。この両方が具象化すると「死んだ仲間の肉」がほぼ特定される
- **H2{危機} ↔ H3{食}**: 危機的状況 → 通常でない食材の使用。間接的だが自然な連想
- **H2{海} ↔ H3{禁忌}**: 直接的なつながりは薄い

### H2 — H4 間

- **H2{仲間} ↔ H4{関係}**: 仲間が欺瞞の主体。この対応は直接的
- **H2{危機} ↔ H4{窮状}**: 危機的状況 = 食料不足の状況。ほぼ同一の事態を異なる角度から記述
- **H2{海} ↔ H4{動機}**: 直接的なつながりは薄い

### H3 — H4 間

- **H3{正体} ↔ H4{欺瞞}**: 正体を偽った。パズルの核心的な対応関係。ただし抽象レベルでは食品偽装等でも成立する
- **H3{禁忌} ↔ H4{動機}**: 禁忌的な内容 → それを偽る強い動機が必要。禁忌が具象化すると動機の深刻さも要求される
- **H3{死者} ↔ H4{関係}**: 死んだ仲間 → 仲間が行った行為。H2{仲間} を介して三者が連動する

### H1 と他の H

- H1 は演繹連鎖の起点（過去の体験 → 味の記憶 → 比較が可能）であり、他の全ての H の前提
- H1 の具象化は他の H に対して中立: H1 が具象化しても特定の H の方向を制約しない
- 逆に、H2-H4 のいずれが具象化しても H1 への影響は小さい

### 要素間の連動パターン

**強い連動（同時に具象化すると T に急接近）**:
- H2{仲間} + H3{死者} → 死んだ仲間の肉
- H3{禁忌,死者} + H4{動機,窮状} → 極限状態で禁忌を犯した
- H2{仲間,危機} + H3{死者} + H4{関係} → 仲間と危機的状況、仲間が死に、仲間が何かをした

**弱い連動（同時に具象化しても T とは距離がある）**:
- H2{海} + H3{食} → 海で特殊な食材（多様な解釈可能）
- H2{職業} + H4{欺瞞} → 船乗りの職場での欺瞞（多様な解釈可能）
- H3{正体} + H4{欺瞞} → 何かの正体を偽った（食品偽装でも成立）

**三要素連動（最も危険）**:
- H2{仲間} + H3{死者} + H4{関係}: 仲間が死に、仲間が行為の主体 → T の核心
