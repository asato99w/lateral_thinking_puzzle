## Step 1: T から分割命題への演繹構造

T から各分割命題への直接演繹:
- T → P1 (偽られた体験), T → P2 (人肉), T → P3 (極限状況), T → P4 (人の死亡), T → P5 (身近な人物)

命題間の演繹関係:
- P2 → P4: 「スープの材料は人間の肉」→「人が死亡していた」（人肉があれば死者がいる）
- P5 → P1: 「身近な人物が偽った」→「他者が偽った」（身近は他者の部分集合）

冗長辺の除去:
- T → P4 は T → P2 → P4 に包含 → 除去
- T → P1 は T → P5 → P1 に包含 → 除去

```

```

```

  →（死体からスープを落とす）→ P3: 極限状況
```

検証:
- 演繹的分岐: P2 と P3 を同時演繹 ✓
