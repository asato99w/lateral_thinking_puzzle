# Phase 3: 導出構造と最終命題

## Step 1: T から分割命題への演繹構造

T → 各分割命題の直接演繹:
- T → P1: 「実際にはそれは死んだ仲間の肉」→ ウミガメのスープではなかった ✓
- T → P2: 「ウミガメのスープだと言って飲ませた」→ 他者が正体を偽った ✓
- T → P3: 「遭難し食料が尽きた」→ 極限状況 ✓
- T → P4: 「死んだ仲間の肉で作ったスープ」→ 人肉 ✓
- T → P5: 「正体を悟り絶望して自殺」→ 過去への絶望 ✓

命題間の演繹関係:
- **P4 → P1**: 素材が人肉（P4）→ ウミガメのスープではない（P1）。人肉はウミガメの肉ではないので、具象→抽象の演繹 ✓
  - 理由: P4 は P1 の具象化（素材を特定）
- **P2 → P1**: 他者が正体を偽った（P2）→ 本物ではなかった（P1）。正体を偽って飲ませた以上、本物ではない ✓
  - 理由: P2 は P1 の具象化（偽物である理由を特定）

冗長辺の除去:
- T → P1 は T → P4 → P1 および T → P2 → P1 に包含される → 除去

整理後の構造:
```
T
├── P2 → P1
├── P3
├── P4 → P1
└── P5
```

## Step 2-4: 共通具象元の発見

### 候補1: P2 と P4 の共通具象元
「他者が人肉スープをウミガメのスープと偽って飲ませた」
- → P2: 「人肉」を落とす → 正体を偽って飲ませた ✓
- → P4: 「偽って」を落とす → 素材は人肉 ✓
- true_models: {G1} のみ
- 非自明分割を満たさない。抽象化を試みる:
  - 「他者が実際の素材を隠してスープを偽って飲ませた」→ これは P2 とほぼ同義（P2 true = {G1, M5, M6, M7, M11, M16}）で P4 を演繹しない
- **不適格**: 分岐を維持しつつ非自明分割を達成する抽象化レベルが見つからない

### 候補2: P3 と P2 の共通具象元
「極限状況下で他者がスープを偽って飲ませた」
- → P2: 「極限状況」を落とす ✓
- → P3: 「偽って」を落とす ✓
- true_models: P2∩P3 = {G1} のみ
- **不適格**: 同上

### 結論
有効な中間命題は見出されなかった。現在のグラフ構造を維持する。

## Step 5: 抽象化による発見

T直下ノード: P2, P3, P4, P5

### P2 と P3 の抽象化
- P2（他者の欺瞞）→ 「過去に異常な状況があった」
- P3（極限状況）→ 「過去に異常な状況があった」
- 共通抽象: 「過去のスープの体験に異常な背景があった」
  - true_models: G1, M3?, M4?, M5, M6, M7, M8?, M9, M10?, M11, M12, M13?, M16
  - 「異常」の定義が曖昧で意味確定性チェックに抵触。**不適格**。

### P4 と P5 の抽象化
- P4（人肉）→ 「過去の体験に深刻な秘密がある」
- P5（過去への絶望）→ 「過去の体験が致命的影響を持つ」
- 共通構造の候補: 「過去の体験は男の道徳観を根本から揺るがすものだった」
  - true_models: G1（人肉摂食）。他モデルでは道徳観の「根本的」動揺と言えるか曖昧。
  - **不適格**: true_models の判定が不安定。

### P3 と P5 の抽象化
- P3（極限状況）→ 「深刻な過去の体験」
- P5（過去への絶望）→ 「深刻な過去の体験の帰結」
- 共通構造: 「過去の体験は男に重大な心理的影響を残した」
  - true_models: 多すぎて（G1, M5, M6, M7, M9, M10, M11, M12, M13）有意な分割にならず。
  - **不適格**: P5 とほぼ同じ分割。

### 結論
Step 5 を実施したが、有効な中間命題は新たに発見されなかった。

## Step 6: 最終命題（FP）の定義

T直下ノード: P2, P3, P4, P5

FP は T の情報から、これら全てを演繹できる単一命題として定義する。

**FP: 男は過去に生命の危機を伴う極限状況下で、他者に人肉のスープを「ウミガメのスープ」と偽って飲まされた**

FP からの演繹:
- FP → P2: 「他者に〜偽って飲まされた」→ 他者がスープの正体を偽った ✓
- FP → P3: 「生命の危機を伴う極限状況下で」→ 極限状況 ✓
- FP → P4: 「人肉のスープを」→ 素材は人肉 ✓
- FP → P5: 人肉の欺瞞体験の発覚 → 過去への絶望 ✓

FP の true_models: {G1}（T とそのバリアントのみ。最終命題に非自明分割は不要）
