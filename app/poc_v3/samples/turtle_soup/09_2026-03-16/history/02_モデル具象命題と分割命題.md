# Phase 2: モデル具象命題と分割命題

## Step 1: 各抽象モデルの具象命題

### G1（受動的欺瞞→禁忌摂食の道徳的壊滅）
1. 男は過去に極限状況（遭難・飢餓等）を経験した
2. 食料が尽き、他者がスープを作って男に提供した
3. 他者は「ウミガメのスープだ」と偽って飲ませた
4. 実際の素材は禁忌的なもの（人肉）だった
5. 男はそのスープの味を極限状況下で記憶した
6. レストランで本物を飲み、味の違いに気づいた
7. 過去のスープの正体を推論し悟った
8. 禁忌摂食の道徳的壊滅で自殺した

### M3（料理コンテスト詐欺師）
1. 男は料理の専門家だった
2. 「ウミガメのスープ」と偽って別素材で料理を作り社会的成功を得た
3. 偽装品の味を職業的反復で熟知していた
4. レストランで本物を飲み味の違いに気づいた
5. 他者が同様に比較すれば詐欺が露見すると確信した
6. 社会的名声の喪失に絶望して自殺した

### M4（闇市場偽装販売者）
1. 男は違法取引に関わっていた
2. 別素材を「ウミガメのスープ」として闇市場で販売していた
3. 商品の味を商業的反復で熟知していた
4. レストランで本物を飲み偽装を確信した
5. 犯罪組織顧客からの暴力的報復を恐れ自殺した

### M5（亡き母の料理の秘密）
1. 母が「ウミガメのスープ」として料理を作り続けていた
2. 男は幼少期からその味を記憶していた
3. 母は既に亡くなっている
4. レストランで本物を飲み味の違いに気づいた
5. 母の料理がウミガメのスープではなかったと悟った
6. 素材への恐怖と母への信頼崩壊で自殺した

### M6（宗教儀式の偽装）
1. 男は宗教団体に所属していた
2. 団体の儀式で「ウミガメのスープ」が振る舞われていた
3. 長年にわたり儀式でスープを飲んでいた
4. レストランで本物を飲み味の違いに気づいた
5. 儀式のスープが偽物だったと悟り教義への信頼が崩壊した
6. 信仰に人生を捧げたアイデンティティが壊滅し自殺した

### M7（友人の死の原因確定）
1. 男は旅行中に他者と共に「ウミガメのスープ」を飲んだ
2. その後、同席した友人が死亡した
3. スープと死の関連を疑っていた
4. レストランで本物を飲み味の違いに気づいた
5. 旅行時のスープが有害物だったと確信した
6. 友人の死への罪悪感で自殺した

### M8（食品会社の共犯者）
1. 男は食品会社の品質管理担当だった
2. 会社は偽の「ウミガメのスープ」を販売していた
3. 男は自社製品の味を職業的に熟知していた
4. レストランで本物を飲み偽装を確認した
5. 告発vs沈黙のジレンマに追い詰められ自殺した

### M9（孤島での自己誤認）
1. 男は孤島で長年暮らしていた
2. 島の生物を「ウミガメ」と信じて捕獲・調理していた
3. 長年の自作スープの味を記憶していた
4. レストランで本物を飲み味の違いに気づいた
5. 島で捕獲していた生物がウミガメではなかったと悟った
6. 絶滅危惧種の乱獲可能性への罪悪感で自殺した

### M10（裁判での誤証言）
1. 男は食品偽装裁判で専門家証人だった
2. 被告のスープを味見し「ウミガメのスープではない」と証言した
3. 証言により被告が有罪になった
4. レストランで本物を飲み、裁判時のスープの記憶と照合した
5. 被告のスープが実は本物だったと気づいた
6. 冤罪を生んだ罪悪感で自殺した

### M11（養子のルーツ喪失）
1. 男は養子として育った
2. 「実母のウミガメのスープのレシピ」が唯一のルーツだった
3. 養父母がそのレシピで作るスープの味を記憶していた
4. レストランで本物を飲み味の違いに気づいた
5. 「実母のレシピ」の話が虚偽だったと悟った
6. アイデンティティの根幹喪失で自殺した

### M12（軍のドラッグ混入）
1. 男は軍人として作戦に参加していた
2. 作戦中に「ウミガメのスープ」として支給されたスープにドラッグが混入されていた
3. ドラッグの影響下で残虐行為を行った
4. レストランで本物を飲み味の違い（ドラッグ由来の味）に気づいた
5. 支給スープへの混入物を確信した
6. 軍に利用された絶望と残虐行為の記憶で自殺した

### M13（料理学校教師の誤り）
1. 男は料理学校の教師だった
2. 「ウミガメのスープ」のレシピを長年教えてきた
3. 自分のレシピの味を教育的反復で熟知していた
4. レストランで本物を飲みレシピの誤りに気づいた
5. 教え子への罪悪感とキャリア否定で自殺した

### M14（食品評論家の味覚衰え）[部分緩和]
1. 男は食品評論家だった
2. 過去に本物のウミガメのスープを飲んだ記憶がある
3. レストランで飲み、記憶と異なる味に気づいた
4. 本物と確認され、味覚の衰えを自覚した

### M15（致命的アレルギー）[部分緩和]
1. 男はウミガメに致命的アレルギーがある
2. 誤ってウミガメのスープを注文した
3. 確認して本物と判明し致死的反応を確信した
4. 死を覚悟し苦痛回避のため自殺を選んだ

### M16（過去の普通の食事体験）[部分緩和]
1. 以前別のレストランでウミガメのスープを飲んだ
2. 大切な記念日の思い出と結びついていた
3. レストランで本物を飲み味の違いに気づいた
4. 以前のレストランが偽物を出していたと判明、思い出が汚された

## Step 2b: グルーピング

- **男→他者（能動的加害者/共犯）**: M3, M4, M8
- **欺瞞なし（誤認/誤り/無関係）**: M9, M10, M13, M14, M15
- **組織→男**: M12（欺瞞だが内容はスープの正体ではなく薬物混入）

- **本物（または本物ベース）**: M10, M12, M14

- **不十分/特殊**: M14, M15, M16

## Step 2c: 抽象命題の生成と Step 2d: 意味確定性チェック

### P1: 男が過去にウミガメのスープとして飲んだスープは実際にはウミガメのスープではなかった
- true_models: G1, M3, M4, M5, M6, M7, M8, M9, M11, M13, M16
- false_models: M10, M12, M14, M15
- 意味確定性: 過去に「ウミガメのスープ」として認識して飲んだ体験がないモデル（M15）ではfalse（偽物の体験自体がない）。M10は過去のスープが実は本物→false。M12はベースがウミガメのスープ→false。M14は本物を飲んだ→false。確定。
- 判例チェック: 真偽明確。手段/目的の混同なし。

### P2: 他者が男にスープの正体を偽ってウミガメのスープとして飲ませた
- true_models: G1, M5, M6, M7, M11, M16
- false_models: M3, M4, M8, M9, M10, M12, M13, M14, M15
- 意味確定性: 「偽って飲ませた」は他者による能動的な偽装提供を指す。M3/M4/M8は男が偽装する側→false。M9/M10/M13は自己誤認→false。M12は薬物混入だがスープ名称自体は偽っていない→false。確定。
- 判例チェック: OK

### P3: 過去のスープの体験は生命の危機を伴う極限状況下のものだった
- true_models: G1, M9, M12
- false_models: M3, M4, M5, M6, M7, M8, M10, M11, M13, M14, M15, M16
- 意味確定性: 「生命の危機」は客観的に判定可能。遭難(G1)、孤島生活(M9)、戦場(M12)は生命危機あり。その他は日常的/職業的/社会的文脈であり生命危機なし。確定。
- 判例チェック: OK

### P4: 過去のスープの実際の素材は人間の肉だった
- true_models: G1
- false_models: M3, M4, M5, M6, M7, M8, M9, M10, M11, M12, M13, M14, M15, M16
- 意味確定性: 客観的に確定。Tは「死んだ仲間の肉」と明言。他モデルは人肉ではない。確定。
- 判例チェック: OK

### P5: 男の自殺の動機は過去の出来事そのものに対する絶望だった
- true_models: G1, M5, M6, M7, M9, M10, M11, M12, M13
- false_models: M3, M4, M8, M14, M15, M16
- 意味確定性: 「過去の出来事そのもの」は既に起きた事実への反応を指す。M3/M4/M8は将来の社会的結果（露見・報復・ジレンマ）が動機→false。M14/M15/M16は自殺に至らないか特殊→false。確定。
- 判例チェック: OK
