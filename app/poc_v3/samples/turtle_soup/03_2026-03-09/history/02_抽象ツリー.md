# 抽象ツリーの構成

## 入力

Phase 1 Step 4 で抽出した主要概念: 遭難、人肉食、欺瞞

## 各概念の抽象ツリー

### 遭難

- 遭難 → 危機的状況 → 非日常体験
- 遭難 → 海での出来事 → 特定の場所での体験
- 遭難 → サバイバル → 極限状態
- 遭難 → 仲間との体験 → 集団体験

### 人肉食

- 人肉食 → 禁忌の食 → 禁忌
- 人肉食 → 極限状態での行為 → 極限行為
- 人肉食 → 死者の利用 → 死への関わり
- 人肉食 → 食の内容 → 食事の実態

### 欺瞞

- 欺瞞 → 偽りのラベル → ラベルと実体の乖離 → 見かけと実体の不一致
- 欺瞞 → 情報の非対称 → 認知の歪み
- 欺瞞 → 仲間による行為 → 人間関係における裏切り
- 欺瞞 → 善意の嘘 → 保護的行為
