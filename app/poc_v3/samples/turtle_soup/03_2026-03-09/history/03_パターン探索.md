# パターン探索

## 入力

抽象ツリー（02_抽象ツリー.md）の各概念。

各概念は複数方向に枝分かれする樹構造を持つ。パターン探索では、各概念のどの枝のどのレベルを選択し、どうグルーピングするかを検討する。

## 各概念の枝と抽象レベル

### 遭難

| 枝 | 具象寄り | → | 抽象寄り |
|---|---------|---|---------|
| 状況 | 遭難 | → 危機的状況 | → 非日常体験 |
| 場所 | 海での遭難 | → 海での出来事 | → 特定の場所での体験 |
| 生存 | 遭難 | → サバイバル | → 極限状態 |
| 社会 | 仲間との遭難 | → 仲間との体験 | → 集団体験 |

### 人肉食

| 枝 | 具象寄り | → | 抽象寄り |
|---|---------|---|---------|
| 禁忌 | 食人 | → 禁忌の食 | → 禁忌 |
| 極限 | 人肉食 | → 極限状態での行為 | → 極限行為 |
| 死 | 人肉食 | → 死者の利用 | → 死への関わり |
| 食 | 人肉食 | → 食の内容 | → 食事の実態 |

### 欺瞞

| 枝 | 具象寄り | → | → | 抽象寄り |
|---|---------|---|---|---------|
| ラベル | 偽りのラベル | → ラベルと実体の乖離 | | → 見かけと実体の不一致 |
| 情報 | 欺瞞 | → 情報の非対称 | | → 認知の歪み |
| 関係 | 仲間の嘘 | → 仲間による行為 | | → 人間関係における裏切り |
| 意図 | 善意の嘘 | | | → 保護的行為 |

## パターン候補

パターン = (各概念の枝選択 + 抽象レベル) × (グルーピング)

目標: 独立性を保ちつつ、できるだけ具象的なパターン。

### 枝間の親和性に注目したグルーピング

概念間で親和性の高い枝の組み合わせ:
- 遭難:極限 × 人肉食:極限 → 「極限状態」で交差
- 遭難:社会 × 欺瞞:関係 → 「仲間」で交差
- 人肉食:食 × 欺瞞:ラベル → 「食のラベル」で交差
- 遭難:生存 × 人肉食:極限 → 「サバイバルと極限行為」で交差

### 候補パターン

**パターンA**: 枝の親和性を活かしたグルーピング
- 遭難:極限「サバイバル」 + 人肉食:極限「極限状態での行為」 → グループ1「極限状況下の行為」
- 欺瞞:ラベル「偽りのラベル」 → グループ2「偽りのラベル」
- 交差による具象: G1「極限状況で禁忌の行為が行われた」/ G2「それが偽りのラベルで隠された」

**パターンB**: 社会的枝を活かしたグルーピング
- 遭難:社会「仲間との体験」 + 欺瞞:関係「仲間による行為」 → グループ1「仲間との出来事」
- 人肉食:禁忌「禁忌の食」 → グループ2「禁忌の食」
- 交差による具象: G1「仲間に何かをされた」/ G2「禁忌の食が関わっている」

**パターンC**: 食の枝を軸にしたグルーピング
- 人肉食:食「食の内容」 + 欺瞞:ラベル「ラベルと実体の乖離」 → グループ1「食の実態が隠されている」
- 遭難:状況「危機的状況」 → グループ2「危機的状況」
- 交差による具象: G1「食べたものの正体が偽られている」/ G2「それは危機的状況で起きた」

**パターンD**: 低抽象を維持したグルーピング
- 遭難:状況「遭難」 + 人肉食:食「食の内容」 → グループ1「遭難中の食事」
- 欺瞞:情報「情報の非対称」 → グループ2「情報の非対称」
- 交差による具象: G1「遭難中に何を食べたか」/ G2「本人が知らない情報がある」

## 独立性の評価（T'生成による）

### パターンA: {極限状況下の行為} / {偽りのラベル}

**G1固定・G2変動**:
- T'a: 極限状況で禁忌行為があったが、記憶喪失で隠された → 自然
- T'b: 極限状況で禁忌行為があったが、記録の改ざんで隠された → 自然

**G2固定・G1変動**:
- T'c: 偽りのラベルで禁忌の食品が提供されたが、日常的な場面で → 自然
- T'd: 偽りのラベルで危険な薬品が投与されたが、医療現場で → 自然

→ ◎

### パターンC: {食の実態が隠されている} / {危機的状況}

**G1固定・G2変動**:
- T'e: 食の実態が隠されていたが、平時の食品工場で → 自然
- T'f: 食の実態が隠されていたが、宗教的行事で → 自然

**G2固定・G1変動**:
- T'g: 危機的状況があったが、食ではなく行動が隠された → 自然
- T'h: 危機的状況があったが、場所が隠された → 自然

→ ◎

### パターンD: {遭難中の食事} / {情報の非対称}

**G1固定・G2変動**:
- T'i: 遭難中の食事があったが、本人は全て知っていた（情報の対称）→ 成立するがパズルとしての緊張が弱い
- T'j: 遭難中の食事があったが、嘘ではなく本人の誤解で → やや不自然

**G2固定・G1変動**:
- T'k: 情報の非対称があったが、戦時中の食事で → 自然
- T'l: 情報の非対称があったが、入院中の食事で → 自然

→ G1→G2方向がやや制約 = ○（遭難と食事を同一グループにすると結合が強い）

## 品質評価

- **パターンA**: 枝の親和性が活きたグルーピング。独立性◎。ただし「極限状況下の行為」はやや抽象的。
- **パターンC**: 食という具体的な軸。独立性◎。グループの意味が明確。
- **パターンB**: 「仲間」で交差。独立性は要検討（仲間が両グループに暗黙に関与する可能性）。
- **パターンD**: 最も具象的だが独立性○。遭難と食事の結合が強すぎる。

パターンAとCが有力候補。
