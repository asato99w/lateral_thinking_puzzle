## Step 2-3: 戦術連鎖

### 導出グラフと連鎖の対応

### C1: S → P1 (味を記憶していた)
{S4, S5} →[観測事実の深掘り]→ I1(男にはスープの味を判断する基準があった)
{I1} →[時間軸の拡張]→ P1(過去にウミガメのスープだとして飲んだ味を記憶していた)
- 中間命題: I1
- プレイヤー視点: 「一口飲んで真偽を確認した」→「味の基準がある」→「過去に飲んだ記憶がある」。自然。

### C2: S → P2 (欺瞞があった)
{S11, S12, S7} →[因果の逆転]→ I3(本物であることの確認が男にとって否定的な意味を持った)
{I3} →[ラベルと実体の乖離]→ I4(過去に「ウミガメのスープ」として飲んだものは実際にはウミガメのスープではなかった)
{I4} →[段階的絞り込み]→ P2(過去のスープには他者が偽りの説明をした欺瞞があった)
- 中間命題: I3, I4
- プレイヤー視点: 「本物→自殺」→「本物であることが悪い意味」→「過去のは偽物だった」→「誰かが嘘をついた」。自然。

### C3: S → P3 (極限状況)
{S1, S7} →[人物像の探索]→ I5(男の自殺には日常的でない深刻な背景事情がある)
{I5} →[時間軸の拡張]→ I6(男は過去に危機的な体験をしたことがある)
{I6} →[段階的絞り込み]→ P3(過去のスープを飲んだ状況は生命の危機を伴う極限的なものだった)
- 中間命題: I5, I6
- プレイヤー視点: 「自殺するほどの男」→「深刻な背景がある」→「過去に危機的な体験」→「生命の危機を伴う極限状況」。自然。

### C4: S → P4 (通常食さないもの)
{S7, S11} →[動機の探索]→ I7(男が知った真実は自殺に至るほど衝撃的だった)
{I7} →[段階的絞り込み]→ I8(衝撃的な真実はスープの正体に関するものだった)
{I8} →[段階的絞り込み]→ P4(過去のスープの正体は人が通常食するものではなかった)
- 中間命題: I7, I8
- プレイヤー視点: 「自殺するほど→衝撃的な真実」→「スープの正体について」→「通常食さないもの」。自然。

### C5: S → P5 (人の死が関わる)
{I7} →[段階的絞り込み]→ I9(スープに関する真実は生死に関わる深刻さだった)
{I9} →[段階的絞り込み]→ P5(男が悟った真実には人の死が直接的に関わっていた)
- 起点由来: I7 は C4 と共有（S7, S11 から）
- 中間命題: I9
- プレイヤー視点: 「衝撃的な真実」→「生死に関わる深刻さ」→「人の死が関わっていた」。自然。

### C6: P2 → P6 (親密な人物の欺瞞) [toward_concrete]
{P2} →[段階的絞り込み]→ I10(欺瞞を行った人物は男と直接的な関わりがあった)
{I10} →[段階的絞り込み]→ P6(欺瞞者は男と個人的に親密な信頼関係にあった)
- fc_source: P2 は C2 の reveals
- 中間命題: I10

### C7: P3 + P6 → P7 (極限＆仲間の欺瞞) [toward_concrete]
{P3, P6} →[登場人物の探索]→ I11(極限状況を男と共に経験した仲間がいた)
{I11} →[段階的絞り込み]→ P7(男は極限状況で親密な仲間からスープの正体を偽られて飲んだ)
- fc_source: P3 は C3、P6 は C6 の reveals
- 中間命題: I11

### C8: P1 + P4 + P5 + P7 → FP [toward_concrete]
{P7, P4} →[段階的絞り込み]→ I12(極限状況で偽られて飲んだスープの正体は通常食さないものだった)

- fc_source: P1 は C1、P4 は C4、P5 は C5、P7 は C7 の reveals
- 中間命題: I12, I13
