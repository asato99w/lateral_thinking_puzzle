# Phase 1: 緊張構造分析とモデル列挙

## 緊張構造

### 表層の緊張

- **α**: 水の要求に対する銃の反応 — 「水をください」という無害な頼みにバーテンダーが銃を向ける
  - M0 の予測: 水を出す、あるいは断る
  - 説明不能: なぜバーテンダーは銃という暴力的手段に訴えたのか

- **β**: 銃に対する感謝 — 銃を向けられた男が恐怖ではなく「ありがとう」と感謝
  - M0 の予測: 恐怖・怒り・逃走
  - 説明不能: なぜ男は銃を向けられたことを有難いと感じたのか

- **γ**: 目的未達での帰宅 — 水を求めて来たのに水を受け取らず帰った
  - M0 の予測: 水を飲んでから帰る
  - 説明不能: なぜ男は水なしで満足して帰ったのか

### 分解

**α → α1 + α2**:
- α1: バーテンダーが「水をください」から男の隠れた状態・情報を読み取った
- α2: 読み取った情報に基づき銃を向けるという手段を選択した
- 独立検証: α1 のみ緩和（何かに気づいたが銃の理由は別）✓ / α2 のみ緩和（銃は合理的だが水の要求とは無関係）✓

**β → β1 + β2**:
- β1: 男は銃を向けられたことを脅威と感じなかった
- β2: 男は銃を向けられたことで利益を得て積極的に感謝した
- 独立検証: β1 のみ緩和（おもちゃで脅威なしだが感謝理由なし）✓ / β2 のみ緩和（利益はあったが瞬間は恐怖）✓

**γ → γ1 + γ2**:
- γ1: 男にとって水は最終目的ではなく手段だった
- γ2: バーでの出来事により水の必要がなくなった
- 独立検証: γ1 のみ緩和（水は合図で銃とは無関係）✓ / γ2 のみ緩和（水は飲みたかったが銃で事情が変わった）✓

## T の緩和

| 緊張 | T の緩和メカニズム |
|------|-------------------|
| α1 | しゃっくりの声や動作をバーテンダーが認識した |
| α2 | 驚かせるとしゃっくりが止まるという民間療法の知識に基づく |
| β1 | しゃっくりが止まった直後に銃の意図（治療）を理解した |
| β2 | しゃっくりが止まった実利 |
| γ1 | 水はしゃっくり緩和の手段であり飲むこと自体が目的ではない |
| γ2 | 驚きでしゃっくりが止まり水による緩和が不要になった |
