## Step 2d + 導入時検証（判例参照付き）

判例: `precedents/命題の適切さの判例.md`
- パターン1: 日常的意味と厳密な意味の乖離（「動物の肉」判例）
- パターン2: 手段と目的の混同による排他的表現（「水を飲むこと以外」判例）

### P1「男は何らかの身体的・心理的な不調を抱えていた」

- **T 文脈**: しゃっくりは身体的不調 → **true** ✓
- **判例照合**: 「不調」の範囲は曖昧か？ → しゃっくり(G1)・自殺願望(M7)・泥酔(M8)・恐怖症(M9) はいずれも「不調」の日常的意味に含まれる。泥酔は自己起因だが状態としては不調。パターン1（日常的/厳密意味の乖離）に該当しない
- **意味確定性**: 「不調」は身体的・心理的と限定されており一意 ✓
- **前回からの変更**: 「症状」→「不調」。泥酔を「症状」と呼ぶのは医学用語としては妥当だが日常語感からの乖離があるため、より広い「不調」に変更

### P2「男にとって水を飲むことは最終目的ではなく手段だった」

- **T 文脈**: 「水を飲めば治ると思い」→ 水は手段、最終目的はしゃっくりを止めること → **true** ✓
- **判例照合**: パターン2（手段/目的混同）に直接該当するケース。前回「水を飲むこと以外にあった」は排他的表現で手段の存在を否定するように読めた。修正後は「最終目的ではなく手段だった」とし、水を飲む意図の存在を否定せず階層関係を明示
- **意味確定性**: 「最終目的」と「手段」の関係が明確 ✓
- **モデル別検証**:
  - G2(暗号): 水は合図であり飲料としての手段でもない → 「最終目的ではない」は true。ただし「手段」というより「合図」。命題後半の「手段だった」は G2 では正確ではないが、合図も広義の手段と解釈可能。厳密にはやや緩い → **許容**（合図を「目的達成の手段」と見なせる）
  - M7(自殺): 水は自殺計画の一環 → 飲むこと自体が最終目的ではない → true ✓

### P3「バーテンダーはその場で男の異変の兆候を察知して行動した」

- **T 文脈**: しゃっくりの音や動作をその場で察知 → **true** ✓
- **判例照合**: パターン1に該当しないか確認。前回「外見的な異常」はしゃっくりの聴覚的兆候を含むか曖昧だった。「異変の兆候」に変更し、視覚・聴覚を問わず察知可能な兆候を指す表現に
- **意味確定性**: 「その場で」が事前知識との区別を明確化。「異変の兆候」は一意 ✓

### P4「バーテンダーが銃を向けた目的は男にショック・驚きを与えることだった」

- **T 文脈**: 「驚かせて止めてやろうと」→ 直接目的はショック・驚き → **true** ✓
- **判例照合**: 既存パターンに該当しない ✓
- **意味確定性**: 「目的」は内面だが T が明示的に述べている。直接目的（驚かせる）と最終目的（しゃっくりを止める）の混同はないか → 命題は「ショック・驚きを与えること」に限定しており直接目的のみ。最終目的については言及しない。✓

### P5「バーテンダーは男を助ける意図で銃を向けた」

- **T 文脈**: 「止めてやろうと」→ 善意 → **true** ✓
- **判例照合**: 既存パターンに該当しない ✓
- **客観性の検証**: 「意図」は内面的状態だが、T が「止めてやろう」と明示しており、パズル世界の事実として検証可能。terms.md の客観性要件（「主観的な判断や程度の評価を含まない」）に対して: 「助ける意図があった」は事実の記述であり、「良い行動だった」のような価値判断ではない → **許容**
- **意味確定性**: 「助ける」が曖昧にならないか → 各モデルで「男にとって有益な結果をもたらす意図」と一貫して判定可能 ✓

### P6「男が抱えていた不調は身体的なものだった」

- **T 文脈**: しゃっくり → 身体的 → **true** ✓
- **判例照合**: P1 と同じく「不調」の範囲を確認。M8(泥酔) → 身体的(アルコールの影響) → true。M7(自殺願望) → 心理的 → false。区分は明確 ✓
- **意味確定性**: P1 の存在を前提とする（P1 が偽なら P6 は自動的に偽）。前提関係は明確 ✓

### P7「男とバーテンダーの間に事前の面識や取り決めはなかった」

- **T 文脈**: バーテンダーはその場でしゃっくりに気づいた → 事前関係なし → **true** ✓
- **判例照合**: 既存パターンに該当しない ✓
- **意味確定性**: 「面識」「取り決め」は具体的で一意 ✓
- **モデル別検証**: G3(呪い/毒) → バーテンダーは水の危険を知っていたが男との面識はない → true ✓。区別が成立
