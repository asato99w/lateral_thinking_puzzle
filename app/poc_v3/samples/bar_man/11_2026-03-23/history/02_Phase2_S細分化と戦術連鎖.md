## 戦術連鎖

### 依存関係

```
S → TA1_kr（独立）  S → TB1_kr（独立）
TA1_kr → TA2_kr     TB1_kr → TB2_kr
TA2_kr → TA3_kr     TB2_kr → TB3_kr
                    TB3_kr + TA3_kr → TB4_kr（要素間交差）
TA1_kr+TA2_kr+TA3_kr → FPC1
TB1_kr+TB2_kr+TB3_kr+TB4_kr → FPC2
FPC1 + FPC2 → FP
```

### Chain A1: S → TA1_kr「男に異変があった」

{S2, S3, SD7} →[矛盾の指摘]→ I_A1「水の要求に対して銃を向けるのは通常の反応ではない」
{I_A1, S3} →[因果の逆転]→ I_A2「バーテンダーが銃を向けたのは水の要求とは別の理由による」
{I_A2, S1, SD3} →[段階的絞り込み]→ I_A3「銃を向けた理由は男自身の状態に関係している」
{I_A3} →[段階的絞り込み]→ TA1_kr

プレイヤー視点: 水を頼んだら銃を向けられた → 異常 → 水以外の理由 → 男に関すること → 男に異変があった ✓

### Chain A2: → TA2_kr「バーテンダーがそれに気づいた」

{TA1_kr, SD3, S3} →[因果の逆転]→ I_A4「バーテンダーの銃を向ける行為は男の異変と因果関係がある」
{I_A4} →[観測事実の深掘り]→ TA2_kr

プレイヤー視点: 男に異変 + バーテンダーが銃 → 因果関係 → バーテンダーが気づいた ✓

### Chain A3: → TA3_kr「銃を向けることが対処だった」

{TA2_kr, S3} →[動機の探索]→ I_A5「バーテンダーは男の異変に対して意図的に銃を向けた」
{I_A5, S4, SD8} →[矛盾の指摘]→ I_A6「銃を向ける行為は男にとって有益だった」
{I_A6} →[段階的絞り込み]→ TA3_kr

プレイヤー視点: 気づいて意図的に銃 → でも「ありがとう」と言った → 有益だった → 対処だった ✓

### Chain B1: S → TB1_kr「男に問題があった」

{S2, SD4} →[因果の逆転]→ I_B1「男が水を求めたのにはそれを必要とする事情がある」
{I_B1, S1, SD5} →[時間軸の拡張]→ I_B2「男はバーに来る前から水を必要としていた」
{I_B2} →[段階的絞り込み]→ TB1_kr

プレイヤー視点: 水を求めた → 理由がある → バーに来る前から → 問題があった ✓

### Chain B2: → TB2_kr「水は対処として求めた」

{TB1_kr, S2} →[因果の逆転]→ I_B3「男の水の要求は問題への対処に関連している」
{I_B3, SD4} →[観測事実の深掘り]→ TB2_kr

プレイヤー視点: 問題がある + 水を求めた → 問題への対処 → 水は対処として求めた ✓

### Chain B3: → TB3_kr「問題が解消した」

{TB2_kr, S4, S5} →[矛盾の指摘]→ I_B4「水による対処が不要になった」
{I_B4, S3, SD8} →[因果の逆転]→ I_B5「銃を向けられた出来事と水が不要になったことの間に因果関係がある」
{I_B5} →[段階的絞り込み]→ TB3_kr

プレイヤー視点: 水が対処なのに帰った → 水不要 → 銃の後に不要に → 銃が原因 → 問題解消 ✓

### Chain B4: → TB4_kr「意図を理解した」（要素間交差）

{TB3_kr, S4} →[因果の逆転]→ I_B6「男の「ありがとう」は問題が解消したことへの感謝である」
{I_B6, TA3_kr} →[観測事実の深掘り]→ TB4_kr

プレイヤー視点: 問題解消 + ありがとう → 感謝 → 銃が対処と知っている → 意図を理解した ✓

### Chain C1: → FPC1

{TA1_kr, TA2_kr} →[観測事実の深掘り]→ I_C1「バーテンダーは男の状態を問題として認識した」
{I_C1, TA3_kr} →[段階的絞り込み]→ FPC1

### Chain C2: → FPC2

{TB1_kr, TB2_kr, TB3_kr} →[段階的絞り込み]→ I_C2「男は問題を水で対処しようとしたが別の方法で問題が解消された」
{I_C2, TB4_kr} →[段階的絞り込み]→ FPC2

### Chain C3: → FP

{FPC1, FPC2} →[矛盾の指摘]→ I_C3「バーテンダーの対処（銃）と男の対処（水）は同じ問題に対するものだった」
{I_C3} →[段階的絞り込み]→ FP
