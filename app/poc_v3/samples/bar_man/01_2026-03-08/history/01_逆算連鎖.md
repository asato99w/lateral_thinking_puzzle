# Phase 1: 逆算連鎖

## T

男はしゃっくりが止まらなかった。男は水を飲めばしゃっくりが治ると考えていた。バーテンダーは男のしゃっくりに気づいた。バーテンダーは驚かせればしゃっくりが止まると考えた。バーテンダーは男に銃を向けて驚かせた。男は驚いた。男のしゃっくりが止まった。男は感謝した。

## Step 1: T の逆算

1. 男は感謝した ← なぜ？
2. 男のしゃっくりが止まった ← なぜ止まった？
3. 男は驚いた ← なぜ驚いた？
4. バーテンダーは男に銃を向けて驚かせた ← なぜそうした？
5. バーテンダーは驚かせればしゃっくりが止まると考えた ← なぜしゃっくりを止めようとした？
6. バーテンダーは男のしゃっくりに気づいた ← なぜしゃっくりしていた？
7. 男はしゃっくりが止まらなかった（前提）
8. 男は水を飲めばしゃっくりが治ると考えていた（前提、水を頼んだ理由）

## Step 3: 因果充足性の検証と T 導出命題の抽出

### 検証: 「感謝した ← しゃっくりが止まった」

- **充足性テスト**: 銃を向けられた男がなぜ怒らず感謝したのか？ しゃっくりが止まっただけでは説明不十分。男がバーテンダーの意図を理解したことが必要
- **候補**: 男はバーテンダーが自分のしゃっくりを止めるために銃を向けたと理解した

検証:
- **含意テスト**: D-1（しゃっくり）+ D-5（銃を向けた）+ D-7（止まった）→ 男がしゃっくりの消失と銃の因果関係を認識。T が「感謝した」と述べている以上、この理解は含意される
- **必要性テスト**: 否定すると（理解しなかった）、銃を向けられた相手に感謝する因果が破綻する → 必要
- **内部性テスト**: T の世界内の認知的事実。外部知識不要 → OK

→ **D-9**: 男はバーテンダーが自分のしゃっくりを止めるために銃を向けたと理解した（導出元: D-1, D-5, D-7）

### 検証: その他のリンク

- 「水を頼んだ ← D-1 + D-2」: 充足 ✓
- 「銃を向けた ← D-3 + D-4」: 充足 ✓
- 「驚いた ← D-5」: 充足 ✓
- 「しゃっくりが止まった ← D-6」: 充足 ✓
- 「しゃっくりに気づいた ← D-1」: 充足 ✓
