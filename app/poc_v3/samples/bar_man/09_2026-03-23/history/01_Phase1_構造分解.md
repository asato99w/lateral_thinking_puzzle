# Phase 1: 構造分解（2要素分解）

## 適用規則
- `rules/2要素分解.md`

## Step 1: 要素の特定（補足から）
- 要素A: 水を求めた男にバーテンダーが銃を向けた
- 要素B: 銃を向けられた男が水を受け取らず「ありがとう」と言って帰った

## Step 2: 各要素を導出するTの原子命題

### 要素A
| ID | 原子命題 |
|---|---|
| TA1 | 男はしゃっくりが止まらなくて困っていた |
| TA2 | バーテンダーは男のしゃっくりに気づいた |
| TA3 | バーテンダーは驚かせてしゃっくりを止めようと銃を向けた |

### 要素B
| ID | 原子命題 |
|---|---|
| TB1 | 男はしゃっくりが止まらなくて困っていた（= TA1） |
| TB2 | 男は水を飲めばしゃっくりが治ると思っていた |
| TB3 | 銃で驚かされたことでしゃっくりが止まった |
| TB4 | 男はバーテンダーがしゃっくりを止めてくれたと理解した |

## Step 3: 鍵の特定
鍵: **しゃっくり**（両要素に共通する核心的事実）

## Step 4: FPCの構成

### FPC1（要素A）
「バーテンダーは男の身体的異変に気づき、その対処として銃を向けた」

導出検証:
- → 男に身体的な異変があった (TA1') ✓
- → バーテンダーがそれに気づいた (TA2') ✓
- → 銃を向けることが対処だった (TA3') ✓

### FPC2（要素B）
「男は身体的問題の対処として水を求めていたが、バーテンダーの行為で問題が解消し、男はその意図を理解した」

導出検証:
- → 男に問題があった (TB1') ✓
- → 水は問題の対処として求めた (TB2') ✓
- → バーテンダーの行為で問題が解消した (TB3') ✓
- → 意図を理解した (TB4') ✓

## Step 5: FPの構成
FP: 「男にはある身体的問題があり、水はその対処法として求めたものだった。バーテンダーはその問題に気づき、銃で驚かせるという別の方法で対処して問題を解消させた」

- FPC1 への演繹: FP → バーテンダーが問題に気づき対処として銃を向けた ✓
- FPC2 への演繹: FP → 水は対処+問題解消+意図理解 ✓

## Step 6: 競合モデルの体系的列挙

### 要素A（バーテンダーが銃を向けた理由）
| ID | モデル | 概要 |
|---|---|---|
| MA1 | 対処（T） | 男のしゃっくりに気づき、驚かせて止めようとした |
| MA2 | 強盗 | 男から金品を脅し取ろうとした |
| MA3 | 自衛 | 男が危険人物に見え、身を守ろうとした |
| MA4 | 排除 | 不審・迷惑な客を追い出そうとした |
| MA5 | 冗談 | 悪ふざけや度胸試しとして |
| MA6 | 人違い | 男を指名手配犯と間違えた |

### 要素B（男が感謝して水なしで帰った理由）
| ID | モデル | 概要 |
|---|---|---|
| MB1 | 問題解消（T） | しゃっくりが止まり水が不要になった。対処への感謝 |
| MB2 | 恐怖逃走 | 命の危険を感じ、刺激しないよう「ありがとう」と言って逃げた |
| MB3 | 暗号取引 | 「水」は合言葉、銃は応答の合図。目的達成で帰った |
| MB4 | 覚醒 | 銃の衝撃で我に返り、水を求めた本来の理由が解消した |
| MB5 | テスト完了 | バーテンダーの対応力テスト。結果確認で帰った |

## Step 8-9: 共通抽象の導出と評価

### 要素A — 全て除外
| 共通抽象候補 | 客観性 | 確定性 | 非トートロジー | 弁別力 | 判定 |
|---|---|---|---|---|---|
| 男にBTが認識可能な外見的特徴があった | ○ | △ | △ | 弱 | **除外** |
| BTは男について判断を下した | ○ | × | △ | 弱 | **除外** |
| BTは銃を向けることで目的を達成しようとした | ○ | △ | ○ | ×（全モデル共通） | **除外** |

結果: 要素Aの全T原子命題はT固有。

### 要素B — TB2のみ通過
| T命題 | 共通抽象候補 | 判定 |
|---|---|---|
| TB1（= TA1） | 要素Aと同様 | **除外**（T固有） |
| TB2 | **男が水を求めた目的は、喉の渇きを癒すためではなかった** | **通過** |
| TB3 | BTの行為が男の状態を変化させた | **除外**（非トートロジー性不足） |
| TB4 | 男はBTの行為に感謝した | **除外**（Sの言い換え） |

TB2の通過評価:
- 客観性: ○（事実的に判定可能）
- 確定性: ○（「渇きではない」は具体的な否定）
- 非トートロジー: ○（水を求める最も自然な理由を否定する情報量あり）
- 弁別力: ○（MB2を排除。T/MB3/MB4/MB5に共通）

