# Phase 3: パターンと代表モデル（やり直し: 緊張緩和起点）

## 入力

Phase 1 の緊張構造（α, β, γ）および Phase 2 の葉命題:

| ID | 葉命題 | 真偽 | H 出自 |
|----|--------|------|--------|
| D-L1 | 男に身体的な不調があった | 真 | H1 |
| D-L2 | 男は自分の困りごとへの対処法を持っていた | 真 | H1 |
| D-L3 | 男の表面的な行動と真の目的が異なっていた | 真 | H1 |
| D-L4 | バーテンダーは男の真の状態を認識していた | 真 | H2 |
| D-L5 | バーテンダーは善意で行動した | 真 | H2 |
| D-L6 | バーテンダーの行動は見かけと意図が異なっていた | 真 | H2 |
| D-L7 | 男は心理的衝撃を受けた | 真 | H3 |
| D-L8 | 当初の手段とは別の方法で目的が達成された | 真 | H3 |
| D-L9 | 間接的な作用で男の問題が解消された | 真 | H3 |

## Step 1: 各緊張の代替解消方向

### 緊張 α（水→銃）の代替解消

- a1: 男に明らかな問題があり、バーテンダーが水ではなく別の対処を選んだ
- a2: 「水」が合図・暗号であり、銃はそれへの応答
- a3: 男が危険人物で、バーテンダーが防衛した
- a4: 男とバーテンダーに事前の取り決めがあった

### 緊張 β（銃→感謝）の代替解消

- b1: 銃を向けられたことで男が実際に助けられた
- b2: 銃を向けることが事前に合意された行為だった
- b3: 感謝は皮肉・その場しのぎだった
- b4: 銃によって男が何かに気づいた・目覚めた

### 緊張 γ（水不要で帰宅）の代替解消

- c1: 水の必要性がなくなった（問題が別の方法で解決された）
- c2: 水は最初から本来の目的ではなかった（合図・口実）
- c3: 銃を向けられて水を受け取る状況ではなくなった

## Step 2: 整合的な組み合わせ → 競合モデル

### モデル群 A: 身体的不調の治療（a1 + b1 + c1）

男に身体的症状がある → バーテンダーが認識して驚かせる → 症状が治る → 水不要 → 感謝

- T: しゃっくり
- M1: 喉詰まり
- M2: 酔い覚まし

### モデル群 B: 精神的危機への介入（a1 + b4 + c1）

男が精神的に追い詰められている → バーテンダーが察知して衝撃を与える → 危機から脱する → 水不要 → 感謝

- M3: パニック発作
- M4: 自殺防止

### モデル群 C: 合図・暗号（a2 + b2 + c2）

「水」は合言葉 → 銃は応答信号 → プロトコル完了 → 水は最初から不要 → 確認の感謝

- M5: 暗号モデル

### モデル群 D: 事前合意のテスト（a4 + b2 + c2）

男がバーテンダーに依頼 → 銃を向けてもらう → 完了 → 水は口実 → 感謝

- M6: 度胸試しモデル

### モデル群 E: 防衛（a3 + b3 + c3）

男が危険人物 → バーテンダーが防衛 → 男が退散 → 「ありがとう」は皮肉

- β の解消が不自然（皮肉としての「ありがとう」は S の素直な読みと合わない）。整合性が弱く、競合モデルとしての採用は見送る

## Step 5: 代表モデルの選定

| パターン | 代表モデル | 選定根拠 |
|---------|-----------|---------|
| P-T | M1（喉詰まり） | 身体的不調＋驚きで治るという構造が T と同型。最も自然な類似モデル |
| P-A | M3（パニック発作） | 非身体的だが対処法を持つモデルとして最も想起しやすい |
| P-B | M4（自殺防止） | 対処法を持たない深刻な状態への介入として想起しやすい |
| P-C | M5（暗号） | このパターンの唯一のモデル |
| P-D | M6（度胸試し） | このパターンの唯一のモデル |

## Step 6: パターン境界とT との距離

| パターン | T との差分（偽の葉） | 距離 |
|---------|-------------------|------|
| P-T | なし | 0（葉レベルで区別不可能） |
| P-A | L1 | 1 |
| P-B | L1, L2 | 2 |
| P-C | L1, L2, L5, L7, L9 | 5 |
| P-D | L1, L2, L9 | 3 |

### 注記

P-T（M1 喉詰まり、M2 酔い覚まし）は T と全く同じ葉パターンを持つ。探索フェーズの葉レベルでは T と区別できないため、T 導出フェーズで中間命題（具象レベル）を用いた棄却が必要になる。

L3（表面と真の目的が異なる）、L4（バーテンダーの認識）、L6（見かけと意図の乖離）は全モデルで真であり、このパズルの基本構造を成す。
