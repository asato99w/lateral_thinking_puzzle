## Step 1: 競合モデルの列挙

### T: しゃっくりモデル（正解）

しゃっくりが止まらない → 水で治そうとバーへ → バーテンダーが気づき驚かせるために銃を向けた → しゃっくり停止 → 水不要 → 感謝して帰った

パターン: 真真真真真真真真真

### M1: 喉詰まりモデル

食べ物が喉に詰まった → 水で流そうとバーへ → バーテンダーが気づき驚かせた → 咳き込んで吐き出した → 水不要 → 感謝

パターン: 真真真真真真真真真

### M2: 酔い覚ましモデル

泥酔状態 → 水で覚まそうとバーへ → バーテンダーが気づき驚かせた → ショックで覚醒 → 水不要 → 感謝

パターン: 真真真真真真真真真

### M3: パニック発作モデル

パニック発作 → 水で落ち着こうとバーへ → バーテンダーが気づきショック療法 → パニック停止 → 水不要 → 感謝

パターン: 偽真真真真真真真真（L1 偽: パニック発作は身体的不調ではなく精神的症状）

### M4: 自殺防止モデル

自殺願望 → 最後の水を求めてバーへ → バーテンダーが察し介入 → 銃を向けて目を覚まさせた → 翻意 → 感謝

パターン: 偽偽真真真真真真真（L1, L2 偽: 身体的不調ではなく、水は対処法ではない）

### M5: 暗号モデル

「水」は合言葉 → バーテンダーが認識 → 銃は応答信号 → プロトコル完了 → 確認の感謝

パターン: 偽偽真真偽真偽真偽（L1, L2, L5, L7, L9 偽）

### M6: 度胸試しモデル

度胸試しを依頼（「水」が合図） → 銃を向けてもらう → 耐えた → 感謝

パターン: 偽偽真真真真真真偽（L1, L2, L9 偽）

## Step 3: 代表モデルの選定

| パターン | 代表モデル | 選定根拠 |
|---------|-----------|---------|
| P-T | M1（喉詰まり） | 身体的不調＋驚きで治るという構造が T と同型。最も自然な類似モデル |
| P-A | M3（パニック発作） | 非身体的だが対処法を持つモデルとして最も想起しやすい |
| P-B | M4（自殺防止） | 対処法を持たない深刻な状態への介入として想起しやすい |
| P-C | M5（暗号） | このパターンの唯一のモデル |
| P-D | M6（度胸試し） | このパターンの唯一のモデル |

## Step 4: パターン境界とT との距離

| パターン | T との差分（偽の葉） | 距離 |
|---------|-------------------|------|
| P-T | なし | 0（葉レベルで区別不可能） |
| P-A | L1 | 1 |
| P-B | L1, L2 | 2 |
| P-C | L1, L2, L5, L7, L9 | 5 |
| P-D | L1, L2, L9 | 3 |

### 注記

P-T（M1 喉詰まり、M2 酔い覚まし）は T と全く同じ葉パターンを持つ。探索フェーズの葉レベルでは T と区別できないため、T 導出フェーズで中間命題（具象レベル）を用いた棄却が必要になる。

L3（表面と真の目的が異なる）、L4（バーテンダーの認識）、L6（見かけと意図の乖離）は全モデルで真であり、このパズルの基本構造を成す。
