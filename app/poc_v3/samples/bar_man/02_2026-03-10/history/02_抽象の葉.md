# Phase 2: 抽象の葉（第2回やり直し）

## 入力

Phase 1 の仮説的要素（具象命題群）:
- H1: 男はしゃっくりが止まらず困っていて、水を飲めば治ると思いバーに立ち寄った（D-1, D-2, D-3）
- H2: バーテンダーは男のしゃっくりに気づき、驚かせて止めようと銃を向けた（D-4, D-5, D-6）
- H3: 男は銃を向けられて驚き、しゃっくりが止まったため水が不要になり、感謝して帰った（D-7, D-8, D-9, D-10）

## Step 1: 具象 H の原子命題への分解

### H1 から

- 男はしゃっくりが止まらなかった → [身体的症状があった] + [症状が持続していた]
- 男は水を飲めばしゃっくりが治ると考えた → [対処法を知っていた] + [水が対処手段だった]
- 男はしゃっくりを止めるためにバーに立ち寄った → [来店の目的が水を飲むこと自体ではなかった]
- 含意: 水は手段であり目的ではない → [表面的な行動と真の目的が異なる]

### H2 から

- バーテンダーはしゃっくりに気づいた → [男の真の状態を認識した]
- 驚かせて止めようとした → [善意で行動した] + [男の問題を解消しようとした]
- 銃を向けた=驚かせる手段 → [銃を本来の用途と異なる目的で使った] + [見かけの行動と意図が異なっていた]

### H3 から

- 銃を向けられて驚いた → [心理的衝撃を受けた]
- しゃっくりが止まった → [問題が解消された]
- 水が不要になった → [当初の手段が不要になった] + [別の手段で目的が達成された]
- 含意: 驚き→停止 → [間接的な作用で問題が解消された]

## Step 2: 原子命題の抽象化 → 葉

| ID | 葉命題 | 真偽 | H 出自 | 抽象化元 |
|----|--------|------|--------|---------|
| D-L1 | 男に身体的な不調があった | 真 | H1 | しゃっくり→身体的症状 |
| D-L2 | 男は自分の困りごとへの対処法を持っていた | 真 | H1 | 水を飲めば治ると考えた→対処法 |
| D-L3 | 男の表面的な行動と真の目的が異なっていた | 真 | H1 | 水を頼む≠最終目的 |
| D-L4 | バーテンダーは男の真の状態を認識していた | 真 | H2 | しゃっくりに気づいた→真の状態の認識 |
| D-L5 | バーテンダーは善意で行動した | 真 | H2 | 助けようとした→善意 |
| D-L6 | バーテンダーの行動は見かけと意図が異なっていた | 真 | H2 | 銃=脅威に見えるが実際は援助 |
| D-L7 | 男は心理的衝撃を受けた | 真 | H3 | 驚いた→心理的衝撃 |
| D-L8 | 当初の手段とは別の方法で目的が達成された | 真 | H3 | 水ではなく驚きで解消 |
| D-L9 | 間接的な作用で男の問題が解消された | 真 | H3 | 驚き→しゃっくり停止（直接的治療ではない） |

## Step 3: 抽象性の検証

- D-L1: 「しゃっくり」→「身体的不調」。喉詰まり・過呼吸・発作等と共有可能。非身体的モデルでは偽
- D-L2: 「水で治す」→「対処法を持っていた」。自力で解決を試みるモデルと共有可能。対処法なし（偶然の来店）モデルでは偽
- D-L3: 「水を頼む目的がしゃっくり治療」→「表面と真の目的が異なる」。暗号・陽動・間接的手段等のモデルと共有可能
- D-L4: 「しゃっくりに気づいた」→「真の状態を認識」。バーテンダーが何らかの問題を察知するモデルと共有可能。気づかず行動したモデルでは偽
- D-L5: 「助けようとした」→「善意」。援助・防衛・治療等のモデルと共有可能。敵対モデルでは偽
- D-L6: 「銃=驚かせる道具」→「見かけと意図が異なる」。銃が信号・道具・防衛手段等のモデルと共有可能
- D-L7: 「驚いた」→「心理的衝撃」。恐怖・驚愕・ショック等のモデルと共有可能。心理的衝撃を受けないモデルでは偽
- D-L8: 「水ではなく驚きで解消」→「別の方法で目的達成」。計画外の解決手段モデルと共有可能
- D-L9: 「驚き→しゃっくり停止」→「間接的作用で解消」。直接的治療ではないモデルと共有可能

### 備考

D-L3 と D-L6 は「見かけと実態の乖離」という共通構造を持つ（主体が異なる: 男の行動 vs バーテンダーの行動）。原子命題への分解は重複を許容する。異なる具象命題から同じ抽象構造が抽出されることは、それが重要な座標軸であることの証左でもある。
