# Phase 3: S 細分化と戦術連鎖

## S 原子命題（7個）
S1: ある男がいた / S2: 男はバーに入った / S3: バーテンダーがいた / S4: 男は「水をください」と頼んだ / S5: バーテンダーは銃を向けた / S6: 男は「ありがとう」と言った / S7: 男は帰った

## S 導出命題（7個）
S8: バーがあった(←S2) / S9: 男はバーの外から来た(←S2) / S10: 男は水を求めた(←S4) / S11: バーテンダーは銃を持っていた(←S5) / S12: 水の要求→銃(←S4,S5) / S13: 銃→ありがとう(←S5,S6) / S14: ありがとう→帰った(←S6,S7)

## 戦術連鎖

### 上位連鎖（S → 分割命題）
- **C1**: {S5,S6,S13} →[矛盾の指摘]→ P1（銃に感謝 → 望ましい結果）
- **C2**: {S10} →[ラベルと実体の乖離]→ P2（「水」は文字通りか）
- **C3**: {S4} →[動機の探索]→ P3（水を求めた動機 → 困りごと）

### 下位連鎖（上位 → 下位分割命題）
- **C4**: {P1,S5} →[動機の探索]→ P5（望ましい結果の銃の動機 → 状態察知）← P1 from C1
- **C5**: {P3,S10} →[段階的絞り込み]→ P4（困りごと+水 → 身体的症状）← P3 from C3

## 依存関係
- C1 before C4（P1 → P5）
- C3 before C5（P3 → P4）
- C1, C2, C3 は相互に独立（全て S から直接到達）
