# Phase 2: モデル具象命題と分割構造

## 軸マッピング

7軸（来訪動機, BTの認識, 銃の目的, 男の認識, 感謝の理由, 水不要化の理由, 関係性）で8抽象モデルをマッピング。

## グルーピングと分割命題の導出

主要な分割軸:
- **結果の方向性**: 有益(6) vs 不利益(2) → P1
- **水の性質**: 文字通り(5) vs 合図/口実(3) → P2
- **来訪の動機**: 困りごとあり(2) vs なし(6) → P3
- **困りごとの性質**: 身体的(1) vs その他(7) → P4
- **BTの察知**: 男の状態を察知(2) vs その他(6) → P5

## 分割階層

```
P1(有益な結果) → 棄却: M9, M10
  └ P2(文字通りの水) → 棄却: G2, M8
    └ P3(困りごとあり) → 棄却: M11, M13
      └ P4(身体的症状) → 棄却: M7
        → 残存: G1のみ
```

## T の抽象的位置づけ

P4到達後、G1内部（T, M1, M2, M3）は分割命題で区別不可能。症状の種類（しゃっくり）がユニーク要素。
