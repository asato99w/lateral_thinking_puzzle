# Phase 2: モデル具象命題と分割命題

## Step 1: 各抽象モデルの具象命題

### G1 (不随意の身体反応の驚愕中断)
- 男は不随意の身体的反応（反射・痙攣）に苦しんでいた
- 男は水を飲むことでその反応を抑えようとした
- 男はその目的でバーに入った
- バーテンダーは男の発声・態度の異常から身体症状に気づいた
- バーテンダーは驚かせることで症状を止めようとした
- バーテンダーは銃を向けて男を驚かせた
- 驚きにより不随意反応が中断された
- 症状が解消され水が不要になった
- 男はバーテンダーに感謝した

### G2 (意識レベル低下の驚愕覚醒)
- 男は意識が正常でない状態にあった（夢遊病・低血糖等）
- 男は意識朦朧の中で反射的に水を求めた（意図的な行動ではない）
- バーテンダーは男の意識異常（ふらつき・焦点の合わない目等）に気づいた
- バーテンダーは覚醒させるために銃を向けた
- 恐怖刺激により男が覚醒した
- 覚醒した男は状況を理解し水が不要になった
- 男はバーテンダーに感謝した

### G3 (過剰感情の恐怖上書き)
- 男は制御できない感情状態（極度の緊張・怒り）にあった
- 男は水を飲んで感情を鎮めようとした
- バーテンダーは男の感情異常（震え・興奮等）に気づいた
- バーテンダーは恐怖で感情を上書きしようと銃を向けた
- 恐怖により感情が正常化した
- 感情が落ち着き水が不要になった
- 男はバーテンダーに感謝した

### G4 (暗号プロトコル応答)
- 男とバーテンダーは事前に取り決めたプロトコルを共有していた
- 「水をください」は暗号（合言葉・合図）だった
- 男は実際には水を飲みたかったわけではない
- バーテンダーは暗号を理解し、銃を向けることでプロトコル応答を行った
- プロトコルの完了が確認された
- 男はプロトコル完了に感謝して帰った

### M10 (度胸試し)
- 男とバーテンダーは度胸試しのゲームに参加する合意があった
- 「水をください」はゲーム開始の合図だった
- 男は実際には水を飲みたかったわけではない
- バーテンダーは銃を向けるという合意された行為を実行した
- ゲームが完了した
- 男はゲーム完了に感謝して帰った

### M11 (催眠解除)
- 男は催眠状態にあった（本人に自覚なし）
- 催眠下で「水をください」と言うことは計画された行動だった
- 男は自覚的に水を飲みたかったわけではない
- バーテンダーは催眠術師で、銃を向けることで催眠を解除した
- 催眠が解けた男は状況を理解した
- 男は催眠解除に感謝して帰った

### M12 (自殺志願)
- 男は自殺を決意していた
- 水は自殺前の最後の行為として求めた（水を飲む意図はあるが手段ではない）
- バーテンダーは男の雰囲気・態度から自殺意図を察知した
- バーテンダーは銃を向けて脅すことで自殺を思いとどまらせた
- 男は生きる決意を固めた
- 男は命を救ってくれたことに感謝して帰った

### M13 (挑発への制裁)
- 男は単に水が欲しかった
- バーで酒を頼まず水を頼むことがマナー違反だった
- バーテンダーは侮辱と受け取り怒った
- バーテンダーは銃を向けて怒りを表した
- 男はマナー違反に気づいた
- 男はマナーを教えてくれたことに感謝して帰った

### M14 (強盗の誤認)
- 男は単に水が欲しかった
- 男の外見（帽子・サングラス等）が強盗に見えた
- バーテンダーは防衛のため銃を向けた
- 男は自分の外見の問題に気づいた
- 男は気づかせてくれたことに感謝して帰った

## Step 2d: 意味の確定性チェック

### c1「男は何らかの困りごとを抱えてバーに来た」
- T 文脈: しゃっくりに困っていた → 真 ✓
- 判例照合: 手段/目的の混同なし、日常的意味と厳密な意味の乖離なし ✓
- 意味確定: 「困りごと」は T で「困っていた」と明記 → 明確 ✓

### c2「バーテンダーは男を助ける意図で銃を向けた」
- T 文脈: 「驚かせて止めてやろうと」→ 助ける意図 → 真 ✓
- 判例照合: 特に問題なし ✓
- 意味確定: 明確 ✓

### c3「男とバーテンダーの間に事前の関係・取り決めはなかった」
- T 文脈: 「バーに立ち寄った」→ 偶発的な来店。バーテンダーが「気づき」→ その場での察知 → 真 ✓
- 判例照合: 特に問題なし ✓
- 意味確定: 明確 ✓
- 注意: T に明示的な「初対面」の記述はないが、「立ち寄った」「気づき」から強く含意される

### c4「銃を向けた驚き・恐怖が男の異常な状態を即座に解消した」
- T 文脈: 驚きでしゃっくりが止まった → 真 ✓
- 判例照合: 特に問題なし ✓
- 意味確定: 「異常な状態」= しゃっくりという身体的異常。「即座に」= その場で。明確 ✓

### c5「男の困りごとは不随意の身体的反応に関するものだった」
- T 文脈: しゃっくり = 横隔膜の不随意な痙攣 → 真 ✓
- 判例照合: 特に問題なし ✓
- 意味確定: 「不随意の身体的反応」は明確（意志で制御できない身体反応）✓

### c6「男はバーに来た時点で水を飲みたいという意図を持っていた」
- T 文脈: 「水を飲めば治ると思いバーに立ち寄った」→ 水を飲みたい意図あり → 真 ✓
- 判例照合: **bar_man 07 の判例を確認** — 「水を飲むこと以外にあった」は手段と目的の混同で NG。本命題は「水を飲みたい意図を持っていた」であり、目的の階層構造を否定せず、手段としての水の意図の存在を述べるだけ → 安全 ✓
- 意味確定: 明確 ✓
