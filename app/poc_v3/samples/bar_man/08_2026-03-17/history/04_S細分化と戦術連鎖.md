## Step 2-3: 戦術連鎖と中間命題

### S_to_root 連鎖

**C1: S → P1 (男は何らかの困りごとを抱えてバーに来た)**
- source: [S10]
- S10(水を求めた) →[動機の探索]→ I1(男がバーで水を求めた背後には、水を飲むこと自体を超えた目的があった) →[段階的絞り込み]→ P1
- I1 検証: T「水を飲めば治ると思い」→ 水以上の目的（しゃっくり治療）あり → 真 ✓

**C2: S → P2 (バーテンダーは男を助ける意図で銃を向けた)**
- source: [S5, S6]
- {S5(銃を向けた), S6(ありがとう)} →[矛盾の指摘]→ I2(銃を向けた行為は男にとって有害ではなく、むしろ有益だった) →[動機の探索]→ P2
- I2 検証: T 銃→しゃっくり止まった→感謝 → 真 ✓

**C3: S → P3 (事前の関係なし)**
- source: [S2]
- S2(バーに入った) →[観測事実の深掘り]→ I3(男はバーの通常の客として入店した) →[段階的絞り込み]→ P3
- I3 検証: T「バーに立ち寄った」→ 通りがかりの客 → 真 ✓

**C4: S → P6 (水を飲みたい意図)**
- source: [S4]
- S4(水をくださいと頼んだ) →[観測事実の深掘り]→ I4(男は水を飲むためにバーに立ち寄った) → P6
- I4 検証: T「水を飲めば治ると思いバーに立ち寄った」→ 真 ✓

### toward_concrete 連鎖

**C5: P1 → P5 (不随意の身体的反応)**
- source: [P1] (C1 で reveals)
- P1(困りごとがあった) →[段階的絞り込み]→ I5(男の困りごとは身体の機能に関するものだった) →[段階的絞り込み]→ P5
- I5 検証: T しゃっくり = 身体の機能 → 真 ✓
- fc_source: P1 は C1 の reveals

**C6: P1 → P4 (驚きで異常解消)**
- source: [P1] (C1 で reveals)
- P1(困りごとがあった) →[因果の逆転]→ I6(銃を向けた行為が男の困りごとの解消につながった) →[手段の探索]→ P4
- I6 検証: T 銃で驚かせて止まった → 真 ✓
- fc_source: P1 は C1 の reveals

**C7: {P5, P6} → IP1**
- source: [P5, P6] (C5, C4 で reveals)
- {P5(身体反応), P6(水の意図)} →[因果の逆転]→ I7(男は不随意の身体的反応に対して水が効くと考えた) → IP1
- I7 検証: T「水を飲めば治ると思い」→ 真 ✓
- fc_source: P5 は C5 の reveals, P6 は C4 の reveals

**C8: {P2, P4} → IP2**
- source: [P2, P4] (C2, C6 で reveals)
- {P2(助ける意図), P4(驚きで解消)} →[因果の逆転]→ I8(バーテンダーは男の異常を察知した上で介入した) → IP2
- I8 検証: T「気づき、驚かせて止めてやろうと」→ 真 ✓
- fc_source: P2 は C2 の reveals, P4 は C6 の reveals

**C9: {IP1, IP2, P3} → FP**
- source: [IP1, IP2, P3] (C7, C8, C3 で reveals)
- {IP1, IP2, P3} →[段階的絞り込み]→ I9(初対面のバーテンダーが男の身体的な異常に気づき、驚かせて治した) → FP
- I9 検証: T の内容と一致 → 真 ✓
- fc_source: IP1 は C7 の reveals, IP2 は C8 の reveals, P3 は C3 の reveals
