# Phase 1: 緊張構造分析とモデル列挙

## 緊張構造分析

### 特定した緊張（表層）

- **α: バーテンダーの異常な反応** — 水を求めた客に銃を向けた
- **β: 男の感謝** — 銃を向けられたのに「ありがとう」と言った
- **γ: 水の不受領** — 水を求めて来たのに水を受け取らずに帰った

### 緊張の分解

**α → α1, α2（程度の分離）**:
- α1: 水を求めた客に対して通常の応対（水を出す）をせず異なる行動を取った
- α2: その行動が「銃を向ける」という生命を脅かす極端な手段だった
- 独立検証: α1のみ緩和（水不足で断った等）✓ / α2のみ緩和（銃は水の依頼と無関係の理由）✓ → 通過

**β: 分解なし**
- 検討: β1=恐怖しない / β2=感謝する → β2→β1（感謝するなら恐怖していない）の含意あり。β2のみ緩和モデルは必ずβ1も緩和する → 独立検証不通過

**γ: 分解なし**
- 検討: γ1=水以外の目的 / γ2=目的がバーで達成 → γ2→γ1 の含意あり → 独立検証不通過

### T による緩和

| 緊張 | T の緩和メカニズム |
|------|------------------|
| α1 | しゃっくりに気づき、水を出す代わりに驚かせて止める対処を選択 |
| α2 | 驚かせるために銃を向けるという手段が有効 |
| β | しゃっくりが止まったことを即座に認知し、行為に感謝 |
| γ | しゃっくりが止まり水が不要になったため帰った |

## モデル列挙

13モデル（T含む）を列挙。全モデルが全緊張（α1, α2, β, γ）を緩和する。

### 緩和パターン分析とバリアント識別

**バリアント群 G1: 身体的症状の察知と驚愕刺激による即座の解消**
- メンバー: T（しゃっくり）, M6（パニック発作）, M9（泥酔）, M13（催眠術）
- 代表: T
- 共通パターン: 身体的症状察知 → 驚愕刺激 → 即座の症状解消 → 感謝 + 水不要化
- 構造的同一性: 全て「察知→ショック療法→反射的解消→不要化」のメカニズム

**バリアント群 G2: 精神的異常状態の察知と脅威による心理的介入**
- メンバー: M1（自殺願望）, M7（薬物中毒）
- 代表: M1
- 共通パターン: 精神的異常察知 → 心理的ショック/脅迫 → 精神改善 → 感謝 + 水不要化
- G1との構造的差異: G1は反射的解消（自動的身体反応）、G2は心理的介入（意志・認知の変化）

**バリアント群 G3: 事前合意に基づく非実質的パフォーマンス**
- メンバー: M11（罰ゲーム）, M12（映画撮影）
- 代表: M11
- 共通パターン: 事前合意/台本 → 演出としての銃 → 協力完了感謝 → 水は演出で不要

**単独モデル**: M2（強盗偽装）, M3（友人の悪ふざけ）, M4（ロシアンルーレット）, M10（バー内強盗進行中）, M15（暗号体系）
