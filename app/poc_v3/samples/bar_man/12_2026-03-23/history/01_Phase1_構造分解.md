# Phase 1: 構造分解（2要素分解）

## 適用規則
- `rules/2要素分解.md`

## Step 1: 要素の特定
- 要素A: 水を求めた男にバーテンダーが銃を向けた
- 要素B: 銃を向けられた男が水を受け取らず「ありがとう」と言って帰った
- 2要素間のギャップ: 「水を求めた→銃を向けた」と「銃を向けられた→ありがとう」の繋がりの不在

## Step 2: T の原子命題
### 要素A
- TA1: 男はしゃっくりが止まらなかった
- TA2: バーテンダーはしゃっくりに気づいた
- TA3: バーテンダーは驚かせて止めようとして銃を向けた

### 要素B
- TB1: 男はしゃっくりが止まらなくて困っていた
- TB2: 水を飲めば治ると思っていた
- TB3: 銃を向けられた驚きでしゃっくりが止まった
- TB4: 男はバーテンダーの意図を理解した

## Step 3: 鍵の特定
しゃっくり — 要素A（バーテンダーの対処動機）と要素B（男の問題の解消）の両方に共通して関与する核心的事実。

## Step 4: FPC の構成
- FPC1: 「バーテンダーは男の異変に気づき、その対処として銃を向けた」
  - → 男に異変があった ✓
  - → バーテンダーは男の異変に気づいた ✓
  - → バーテンダーは異変への対処として銃を向けた ✓
- FPC2: 「男は身体的な問題の対処として水を求めていたが、バーテンダーの行為で問題が解消し、男はその意図を理解した」
  - → 男は身体的な問題を抱えていた ✓
  - → 男は問題の対処として水を求めた ✓
  - → バーテンダーの行為で問題が解消した ✓
  - → 男はバーテンダーの行為の意図を理解した ✓

## Step 5: FP の構成
FP: 「男はしゃっくりを止めたくて水を求めたが、バーテンダーがしゃっくりに気づき銃で驚かせて止めてやり、しゃっくりが止まった男は水が不要になり礼を言って帰った」
- FPC1 演繹可能 ✓（しゃっくり→異変、気づき→銃で対処）
- FPC2 演繹可能 ✓（しゃっくり→問題、水→対処、銃→解消、礼→意図理解）

## Step 6-7: 競合モデルと原子命題
### 要素A（5モデル）
| モデル | 原子命題 |
|---|---|
| 対処（T） | しゃっくりが止まらなかった / 気づいた / 驚かせて止めようとした |
| 強盗 | 金品を奪おうとした |
| 自衛 | 男が脅威的行動をした / 身を守ろうとした |
| 排除 | 男が迷惑行為をしていた / 追い出そうとした |
| 冗談 | 冗談として銃を向けた |

### 要素B（5モデル）
| モデル | 原子命題 |
|---|---|
| しゃっくり解消（T） | 困っていた / 水は対処 / 止まった / 意図理解 |
| 脅迫退散 | 恐怖を感じた / 退散した / 皮肉で「ありがとう」 |
| 別目的達成 | 真の目的は水以外 / 目的達成 / 水は不要 |
| 命の恩人 | 危険な状況 / 銃で守った / 感謝 |
| 悟り | 衝撃を受けた / 重要な気づき / 水が不要 |

## Step 8-9: 共通抽象の導出と評価
### 要素A
全共通抽象候補が評価基準（確定性・弁別力）を満たさず除外。しゃっくりという鍵の特異性により、競合モデルとの意味のある共通抽象が得られなかった。

### 要素B
TB2「水を飲めば治ると思っていた」× MB2「男の本当の目的は水を得ることではなかった」
→ 共通抽象: 「男が水を求めた動機は喉の渇きではなかった」 — **通過**
（客観性○、確定性○、非トートロジー○、弁別力: 中）

その他の候補は全て除外。
