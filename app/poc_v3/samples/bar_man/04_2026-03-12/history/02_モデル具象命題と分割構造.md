# Phase 2: モデル具象命題と分割構造

## Step 1: 各モデルの具象命題

### T（しゃっくり）
1. 男はしゃっくりが止まらなくて困っていた
2. 男は水を飲めばしゃっくりが治ると思った
3. 男はしゃっくりの対処のためにバーに立ち寄った
4. バーテンダーは男のしゃっくりに気づいた
5. バーテンダーは驚かせてしゃっくりを止めようとした
6. バーテンダーは銃を向けて男を驚かせた
7. 男は驚いてしゃっくりが止まった
8. しゃっくりが止まったので水が不要になった
9. 男はバーテンダーに感謝して帰った

### M1（喉の異物）
1. 男は喉に何かが詰まって困っていた
2. 男は水を飲んで流そうとした
3. バーテンダーは男の喉の詰まりに気づいた
4. バーテンダーは銃で驚かせて咳き込ませ異物を排出させようとした
5. 男は驚いて咳き込み異物が取れた
6. 異物が取れたので水が不要になった
7. 男はバーテンダーに感謝して帰った

### M2（激しい咳）
1. 男は咳が止まらなくて困っていた
2. 男は水を飲めば咳が治まると思った
3. バーテンダーは男の咳に気づいた
4. バーテンダーは銃で驚かせて咳を止めようとした
5. 男は驚いて咳が止まった
6. 咳が止まったので水が不要になった
7. 男はバーテンダーに感謝して帰った

### M3（泥酔）
1. 男は泥酔していた
2. 男は水で酔いを覚まそうとした
3. バーテンダーは男の泥酔に気づいた
4. バーテンダーは銃の衝撃で酔いを覚まさせようとした
5. 男は驚いて酔いが覚めた
6. 酔いが覚めたので水が不要になった
7. 男はバーテンダーに感謝して帰った

### M4（パニック発作）
1. 男はパニック発作を起こしていた
2. 男は水を飲んで落ち着こうとした
3. バーテンダーは男のパニック状態に気づいた
4. バーテンダーは銃の衝撃で我に返らせようとした
5. 男は驚いて我に返った
6. パニックが収まったので水が不要になった
7. 男はバーテンダーに感謝して帰った

### M5（催眠状態）
1. 男は催眠・夢遊状態にあった
2. 男は無意識にバーに入り水を求めた（対処ではなく催眠の衝動）
3. バーテンダーは男の異常な状態に気づいた
4. バーテンダーは銃の衝撃で覚醒させようとした
5. 男は驚いて覚醒した
6. 正気に戻ったので水が不要になった
7. 男はバーテンダーに感謝して帰った

### M6（自殺企図）
1. 男は自殺するつもりだった
2. 男は毒を持っており水で飲もうとした（水は対処法ではなく手段）
3. バーテンダーは男の意図を察した
4. バーテンダーは銃を向けて男を制止した
5. 男は思い留まった
6. 男は止めてくれたことに感謝して帰った

### M7（裏取引）
1. 「水」は裏社会の隠語だった
2. 男は取引のためにバーに来た
3. バーテンダーは取引相手だった
4. 銃を出したのは取引手順の一部だった
5. 取引が成立した
6. 男は取引成立に感謝して帰った

### M8（映画撮影）
1. バーで映画やドラマの撮影が行われていた
2. 男とバーテンダーは俳優だった
3. 水の注文→銃→感謝は台本通りの演技だった
4. 撮影が完了した
5. 「ありがとう」は撮影終了の挨拶だった

### M9（賭け）
1. 男は誰かと賭けをしていた
2. 賭けの内容は「バーテンダーに水を頼んだら銃を向けられるか」だった
3. 男は水を頼む口実でバーに入った
4. バーテンダーが銃を向けた
5. 男は賭けに勝った
6. 「ありがとう」は賭けに勝てたことへの感謝だった

### M10（強盗と勘違い）
1. バーテンダーは男を強盗と勘違いした
2. バーテンダーは自衛のため銃を向けた
3. 男は誤解に気づき立ち去った
4. 「ありがとう」は皮肉だった

### M11（出入り禁止）
1. 男は以前バーで問題を起こし出入り禁止だった
2. バーテンダーは男を認識した
3. バーテンダーは銃で退去を求めた
4. 男は諦めて帰った
5. 「ありがとう」は皮肉だった

### M12（旧知の冗談）
1. 男とバーテンダーは旧知の仲だった
2. 水を頼むと銃を向けるのは二人のお決まりの冗談だった
3. バーテンダーは冗談として銃を向けた
4. 男は冗談を楽しんだ
5. 「ありがとう」は冗談への感謝だった

### Step 2b: グルーピング

**軸: 男の困りごと**
- G1a 身体的症状: T, M1, M2, M3
- G1b 心理的/意識的問題: M4, M5
- G1c 自殺衝動: M6
- G1d 困りごとなし: M7, M8, M9, M10, M11, M12

**軸: 水の役割**

- G2c 有害行為の手段: M6
- G2d 暗号/口実/台詞: M7, M8, M9
- G2e 本当に水が欲しい: M10, M11, M12

**軸: 銃の意図**

- G3c 手順/演技: M7, M8
- G3d 特になし/賭けの結果: M9
- G3e 防衛/威嚇: M10, M11
- G3f 冗談: M12

- G4c 賭け/冗談の結果: M9, M12

### Step 2c: 抽象命題の生成

**候補1: 「男は何らかの困りごとを抱えていた」**
- true_set: {T, M1, M2, M3, M4, M5, M6}
- false_set: {M7, M8, M9, M10, M11, M12}
- 根拠: G1a + G1b + G1c vs G1d

**候補2: 「水は男の困りごとへの対処法だった」**
- true_set: {T, M1, M2, M3, M4}
- false_set: {M5, M6, M7, M8, M9, M10, M11, M12}
- 根拠: G2a vs G2b〜G2e。M5 は対処ではなく無意識の衝動、M6 は手段であり対処ではない

**候補3: 「男の困りごとは身体的な症状だった」**
- true_set: {T, M1, M2, M3}
- false_set: {M4, M5, M6, M7, M8, M9, M10, M11, M12}
- 根拠: G1a vs G1b以降

**候補4: 「男の症状は不随意的な反復反応だった」**
- true_set: {T, M2}
- false_set: {M1, M3, M4, M5, M6, M7, M8, M9, M10, M11, M12}
- 根拠: しゃっくり（T）と咳（M2）は不随意的な反復反応。喉の異物（M1）は物理的障害、泥酔（M3）は中毒状態

**候補5: 「バーテンダーは男を助ける意図で銃を向けた」**
- true_set: {T, M1, M2, M3, M4, M5, M6}
- false_set: {M7, M8, M9, M10, M11, M12}
- 根拠: G3a + G3b vs G3c〜G3f
- **候補1と同一の分割** → 候補1を代表とする

**候補6: 「男の感謝は本心だった（皮肉でも演技でもない）」**
- true_set: {T, M1, M2, M3, M4, M5, M6, M7, M9, M12}
- false_set: {M8, M10, M11}
- 根拠: G4a + G4c vs G4d。M7 は取引成立への真の感謝、M8 は演技

### Step 2d: 意味の確定性チェック

| 候補 | 各モデルでの真偽 | 判定 |
|------|----------------|------|
| 候補1 | 全モデルで一意に定まる | PASS |
| 候補2 | M5: 無意識の衝動=対処法ではない→false、M6: 毒の手段=対処法ではない→false。明確 | PASS |
| 候補3 | M4(パニック)=心理的→false、M5(催眠)=意識的→false。明確 | PASS |
| 候補4 | M1(異物)=障害で反復ではない→false、M3(泥酔)=状態で反復ではない→false。明確 | PASS |
| 候補5 | 候補1と同一分割。不採用（代表: 候補1） | — |
| 候補6 | M7: 取引成立への感謝は本心→true。M8: 演技→false。判定に迷いなし | PASS |

候補6 は PASS だが false_set が {M8, M10, M11} と小さく、P1 の false_set {M7-M12} の部分集合に含まれる。P1 で棄却されるモデルを重複して棄却するため、分割命題としての価値は限定的。Phase 5 の棄却側分析で扱う。

## Step 3: 分割構造の構成

### 分割命題

| ID | ラベル | T での真偽 | true_models | false_models |
|----|--------|-----------|-------------|--------------|
| P1 | 男は何らかの困りごとを抱えていた | true | T, M1, M2, M3, M4, M5, M6 | M7, M8, M9, M10, M11, M12 |
| P2 | 水は男の困りごとへの対処法だった | true | T, M1, M2, M3, M4 | M5, M6, M7, M8, M9, M10, M11, M12 |
| P3 | 男の困りごとは身体的な症状だった | true | T, M1, M2, M3 | M4, M5, M6, M7, M8, M9, M10, M11, M12 |
| P4 | 男の症状は不随意的な反復反応だった | true | T, M2 | M1, M3, M4, M5, M6, M7, M8, M9, M10, M11, M12 |

### 分割の階層

```
P1 (困りごと): true={T,M1,M2,M3,M4,M5,M6}, false={M7,M8,M9,M10,M11,M12}
└── P2 (水が対処法): true={T,M1,M2,M3,M4}, false={M5,M6}
    └── P3 (身体的症状): true={T,M1,M2,M3}, false={M4}
        └── P4 (反復反応): true={T,M2}, false={M1,M3}
```

階層の方向: 上位（P1: 抽象的、true_set=7、棄却力小）→ 下位（P4: 具体的、true_set=2、棄却力大）

## Step 4: T の抽象的位置づけ

P4 到達時点で T と M2（咳）が残る。両者は分割命題レベルでは同一だが、具象的な症状（しゃっくり vs 咳）が異なる。

- 棄却不可能モデル: なし（全モデルが分割命題 + ユニーク要素で棄却可能）
- M2 はユニーク要素「しゃっくり」の特定により棄却される
- T と M2 の差異はパズルの核心的驚きではなく症状の具体名のみ
