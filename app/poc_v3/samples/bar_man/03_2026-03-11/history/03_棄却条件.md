# Phase 3: 棄却条件

## 分割命題（全て T において真）

- **P1**: 男は何らかの症状・状態に困っていた
- **P2**: 水はその症状への対処法だった
- **P3**: バーテンダーは男の状態をその場で察知した
- **P4**: バーテンダーは男を助けるために銃を向けた
- **P5**: 銃を向けられた衝撃が男の問題を解消した
- **P6**: 問題が解消したので水が不要になった
- **P7**: 驚きで身体反射が直接停止した

## 各モデルの棄却条件

いずれか1つの確認で棄却される。

| モデル | 偽となる命題 | 最小棄却条件 |
|---|---|---|
| M1 喉詰まり | （P1-P7 全て真） | 共通命題では棄却不可 |
| M2 酔い覚まし | P7 | P7 |
| M3 パニック発作 | （P1-P7 全て真） | 共通命題では棄却不可 |
| M4 自殺防止 | P2, P6 | P2 |
| M5 暗号 | P1, P3, P4 | P1 |
| M6 度胸試し | P1, P3, P4 | P1 |
| M7 過呼吸 | （P1-P7 全て真） | 共通命題では棄却不可 |
| M8 毒 | P7 | P7 |
| M9 護身具 | P1, P3 | P1 |
| M10 銃の鑑定 | P1, P3 | P1 |
| M11 追い出し | P1, P4, P5 | P1 |
| M12 気づき | P1, P3 | P1 |
| M13 強盗誤認 | P1, P4 | P1 |
| M14 不審な状況 | P1, P4 | P1 |
| M15 銃マニア | P1, P4 | P1 |

## 棄却力の整理

- **P1** で {M5, M6, M9, M10, M11, M12, M13, M14, M15} を棄却（9モデル）
- **P2** で M4 を追加棄却
- **P7** で {M2, M8} を追加棄却
- **{M1, M3, M7}** は共通命題では棄却できない → T との区別には具象レベルの命題が必要

## 棄却後のモデル集合の遷移

```
全モデル {T, M1-M15}
  ↓ P1 確認
{T, M1, M2, M3, M4, M7, M8}
  ↓ P2 確認
{T, M1, M2, M3, M7, M8}
  ↓ P7 確認
{T, M1, M3, M7}
  ↓ 具象命題（パズルとしての重要度は低い）
{T}
```
