# Phase 1: 逆算連鎖

## T

男はキノコ狩りの最中に山で遭難していた。辺りに毒キノコしか残っていないということは、食用キノコは誰かが既に採った＝この辺りに人が来ているということ。近くに人里や他の採集者がいると分かり、助かる見込みがあると安堵した。

## Step 1: T の逆算

1. 男は安堵した ← なぜ？
2. 助かる見込みがあると分かった ← なぜ助かると思った？
3. 近くに人里や他の採集者がいると分かった ← なぜそう分かった？
4. 食用キノコは誰かが既に採っていた ← なぜそう判断した？
5. 辺りに毒キノコしか残っていなかった ← なぜ毒キノコしかなかった？
6. 男はキノコ狩りの最中に山で遭難していた（前提）

## Step 2: T の直接命題の抽出

| ID | label |
|----|-------|
| D-1 | 男はキノコ狩りの最中に山で遭難していた |
| D-2 | 辺りに毒キノコしか残っていなかった |
| D-3 | 食用キノコは誰かが既に採っていた |
| D-4 | この辺りに人が来ている |
| D-5 | 近くに人里や他の採集者がいると分かった |
| D-6 | 助かる見込みがあると分かった |
| D-7 | 男は安堵した |

## Step 3: 因果充足性の検証と T 導出命題の抽出

### 検証: 「食用キノコは採られていた ← 毒キノコしか残っていなかった」(D-3 ← D-2)

- **充足性テスト**: 「毒キノコしかない」から「食用キノコが採られた」を導くには、「元々は食用キノコもあったはず」という前提が必要。D-2 単体では不十分
- **候補**: この山には通常、食用キノコも生えている

検証:
- **含意テスト**: D-1「キノコ狩りの最中」→ キノコ狩りに来た場所であるから、食用キノコが生える場所であることが含意される
- **必要性テスト**: 否定すると（この山に食用キノコは元々生えない）、毒キノコしかないことから「誰かが採った」という推論が成立しない → 必要
- **内部性テスト**: T の世界内の環境的事実。外部知識不要 → OK

→ **D-8**: この山には通常、食用キノコも生えている（導出元: D-1）

### 検証: その他のリンク

- 「助かる見込みがある ← 近くに人がいる + 遭難中」(D-6 ← D-5, D-1): 充足 ✓
- 「近くに人がいる ← 食用キノコが採られていた」(D-5 ← D-3): D-3 → D-4 → D-5 の流れで充足 ✓
- 「安堵した ← 助かる見込み」(D-7 ← D-6): 充足 ✓
- 「毒キノコしかない ← 食用キノコが採られた + 元々あった」(D-2 ← D-3, D-8): 充足 ✓
