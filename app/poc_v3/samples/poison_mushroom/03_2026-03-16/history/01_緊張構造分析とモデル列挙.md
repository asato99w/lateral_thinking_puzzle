# Phase 1: 緊張構造分析とモデル列挙

## 緊張構造

### Step 1: S の緊張構造の特定

S: 男は、辺りに毒キノコしかないのを見て安堵した。

**α（表層的緊張）: 毒キノコしかないのを見て安堵した**
- 常識(M0): 毒キノコは食べられない/危険。「毒キノコしかない」= 食料がない or 危険な状況 → 落胆・警戒が自然
- M0 予測: 不安・落胆
- M0 前提不備: 男がキノコを食料として求めていることを前提。また「毒キノコしかない」を食料の有無としてのみ解読している

**β: 男が安堵に至る状況がSに伏せられている**
- 安堵は不安の解消であり、男が何に不安を感じていたかが隠されている
- 男がキノコのある場所（山・森）にいる理由も不明

### Step 1.5: 緊張の分解

α に分解原則を適用:

**α1: 毒キノコ（有害物）の存在が安堵を生んでいる**
- 常識: 毒キノコは有害・危険。発見は通常ネガティブ
- 緊張: 毒キノコの存在がポジティブ情報として機能している

**α2: 食用キノコの不在（「しかない」）が安堵を生んでいる**
- 常識: 食用キノコがない = 食料不足 = ネガティブ
- 緊張: 「ないこと」がポジティブ情報として機能している

独立検証(基準3):
- α1のみ緩和: 毒キノコに薬効成分があり男の怪我を治せる → 毒キノコの存在自体が有用（α1✓）。食用キノコの不在は関係ない（α2✗）
- α2のみ緩和: 食用キノコの不在が「人が採った→人がいる」の推論材料 → 不在に意味がある（α2✓）。毒キノコの存在自体は安堵の理由ではない（α1✗）
→ ✓ 通過

### Step 2: T の演繹連鎖

T による各緊張の緩和:

- α1: 毒キノコの存在は「キノコ生育環境で選択的に食用が採集された残余」の証拠。毒キノコは食用ではないため採集されずに残った。毒キノコの残存が「選択的採集の痕跡」を示す
- α2: 食用キノコの不在 = 誰かが既に採集した = この辺りに人が来ている。食料としての不在が「人の存在」という間接証拠に変換される
- β: 男はキノコ狩り中に山で遭難していた。助けが必要な状況であり、人の存在の証拠を見つけたことで助かる見込みがあると安堵した

演繹連鎖:
1. 男は山でキノコ狩り中に遭難 → 不安の原因（β）
2. 辺りに毒キノコしか残っていない → 食用キノコは採られた（α2）
3. 毒キノコが残っている = キノコ生育地で選択的に食用のみ除去された（α1）
4. 食用キノコが採られた → 人が来ている
5. 人が来ている + 遭難中 → 助かる見込み → 安堵

### Step 3: T の命題群

- 男はキノコ狩りの最中だった（真）
- 男は山で遭難していた（真）
- 食用キノコは誰かが既に採っていた（真）
- この辺りに人が来ている（真）
- 近くに人里や他の採集者がいる（真）
- 助かる見込みがある（真）

### バリアント識別

**G1: {M3, M7}**
- 共通パターン: α1=目的物としての直接的有用性, α2=他者不在の推論, β=目的遂行の不安
- 構造的同一性: 毒キノコが自分の目的に直接有用 + 食用不在が他者の不在を意味 → 目的遂行に好都合
- 差異: 犯罪(M3) vs 研究(M7) は舞台設定の違い

**G2: {M9, M10}**
- 共通パターン: α1=忌避効果による他者排除, α2=動機源の不在, β=外部脅威への不安
- 構造的同一性: 毒キノコ/食用不在が脅威源の接近動機を排除
- 差異: 人間(M9) vs 動物(M10) は舞台設定の違い

**G3: {M2, M8}**
- 共通パターン: α1=視覚的警告による行動抑止, α2=誘因の不在, β=誤食回避の不安
- 構造的同一性: 毒キノコは識別可能→食べない + 食用不在→誘惑なし
- 差異: 子供(M2) vs 自分(M8) は保護対象の違い
