# Phase 4: S 細分化と戦術連鎖

## S 細分化

S: 男は、辺りに毒キノコしかないのを見て安堵した。

### 原子命題
- S1: ある男がいた
- S2: 辺りにキノコが生えていた
- S3: そのキノコは全て毒キノコだった
- S4: 男はその状況を目で見て確認した
- S5: 男は安堵した

### 導出命題
- S6: 辺りに食用キノコはなかった (from S3: 全て毒キノコ→食用なし)
- S7: 男は安堵する前に何らかの不安があった (from S5: 安堵=不安の解消)
- S8: 男は辺りのキノコが毒キノコだと認識した (from S3+S4)
- S9: 男はキノコのある場所にいた (from S2)

initial_confirmed = [S1, S2, S3, S4, S5, S6, S7, S8, S9]

## 戦術連鎖

### 根命題への連鎖（S_to_root）

**C1: S → P1 (安堵は生存に関わる)**
{S5, S7} →[動機の探索: 不安の内容を問う]→ I1("男の安堵は重大な問題の解消に関するものだった")
{I1} →[段階的絞り込み: 重大な問題を具体化]→ P1("安堵は自身の生存に関わるものだった")

**C2: S → P2 (キノコ活動)**
{S2, S9} →[人物像の探索: 男の目的を問う]→ I2("男はある目的のためにその場所にいた")
{I2, S2} →[段階的絞り込み: 目的を具体化]→ P2("キノコに関連する活動のためにいた")

**C3: S → P3 (不在理由推論)**
{S2, S6} →[矛盾の指摘: キノコ生育地なのに食用のみ不在]→ I3("キノコの生育環境にありながら食用キノコだけが不在である")
{I3} →[因果の逆転: なぜ食用のみ不在か]→ P3("食用キノコがない理由について推論した")

**C4: S → P5 (物質的性質無関係)**
{S5, S3} →[常識的仮説の排除: 毒キノコの性質で安堵？]→ I4("毒キノコを直接利用して安堵したわけではない")
{I4} →[段階的絞り込み: 物質的性質全般を排除]→ P5("安堵は毒キノコの毒性や忌避効果など物質的性質に由来しない")

### 具象方向への連鎖（toward_concrete）

**C5: P1 → P4 (遭難)**
{P1, S9} →[段階的絞り込み: 生存の危機を具体化]→ I5("男は助けが必要な状況にいた")
{I5} →[人物像の探索: どのような状況か]→ P4("男は山で遭難していた")
fc_source: P1 は連鎖 C1 の reveals

**C6: P2+P3 → P7 (キノコ活動+不在推論)**
{P2, P3} →[因果の逆転: 活動と不在推論の統合]→ I6("男の活動目的と食用キノコの不在に関連があった")
{I6} →[段階的絞り込み]→ P7("キノコに関連する活動中に食用キノコがない理由について推論した")
fc_source: P2 は C2 の reveals、P3 は C3 の reveals

**C7: P4+P5 → P6 (遭難+食用不在推論)**
{P4, P5} →[段階的絞り込み: 遭難+物質無関係→食用不在の推論]→ I7("遭難中の安堵は食用キノコがないことに関連する推論だった")
{I7} →[観測事実の深掘り]→ P6("男は遭難しており、安堵の理由は食用キノコがないという事実から導いた推論だった")
fc_source: P4 は C5 の reveals、P5 は C4 の reveals

**C8: P7+P6 → FP (最終命題)**
{P7, P6} →[観測事実の深掘り: 両推論を統合]→ I8("キノコ狩り中に遭難した男は、食用キノコの不在の原因から助けの手がかりを得た")
{I8} →[段階的絞り込み: 具体的推論]→ FP
fc_source: P7 は C6 の reveals、P6 は C7 の reveals
