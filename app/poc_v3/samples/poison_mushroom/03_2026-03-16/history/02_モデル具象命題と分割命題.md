# Phase 2: モデル具象命題と分割命題

## バリアント群の修正

Phase 1 で G2={M9,M10} と G3={M2,M8} をバリアントとしたが、分割命題の真偽判定で不整合が発生した:
- G3: P1(生存)が M2=false, M8=true
- G2: P1(生存)が M9=false, M10=true

これらを解体し、M2, M8, M9, M10 を個別の抽象モデルとして扱う。G1={M3,M7} は全命題で整合するため維持。

## Step 1: 各モデルの具象命題

**T**: キノコ狩り中に山で遭難 → 毒キノコしか残っていない → 食用キノコは人が採った → 人がいる → 助かる

**G1 (rep M3)**: 毒キノコを利用する目的 → 毒キノコが目的の遂行手段 → 食用不在で他者が来ない → 目的遂行可能

**M1**: 毒キノコ中毒の患者治療 → 原因キノコ特定のため現場に来た → 毒キノコしかないことで特定容易 → 治療方針

**M2**: 子連れハイキング → 子供が先行 → 毒キノコは見た目で危険 → 食用に見えるキノコなし → 子供が誤食しない

**M4**: キノコ栽培業者 → 競合栽培地の偵察 → 毒キノコのみ＝栽培失敗 → 競合不成功

**M5**: 食用キノコアレルギー(蕁麻疹等) → 野外活動中 → 毒キノコは非アレルゲン → 食用不在でアレルギー反応なし

**M6**: 隠蔽活動中 → 毒キノコの場所は人が避ける → 食用不在で人が来ない → 発覚しない

**M8**: 遭難中で空腹 → 毒キノコは識別可能 → 食用キノコなく誘惑なし → 誤食回避

**M9**: 自宅庭のキノコ問題 → 食用キノコ泥棒に悩む → 毒キノコしかないので侵入動機なし

**M10**: 熊から逃走中 → 毒キノコは動物が食べない → 食用不在で餌なし → 動物が来ない

**M12**: 食品安全検査 → 食用キノコの安全問題 → 毒キノコしかないのでこの地域は問題なし

**M13**: 蛇に噛まれた → 毒キノコ成分で解毒可能 → 治療手段の確保

**M14**: 遭難中で救助待ち → 食用キノコがあると無理に動く → 食用不在で動かずに待てる

## Step 2d: 意味の確定性チェック

**P1**: 「自身の生存」は具体的。M5 のアレルギーの程度について補完（蕁麻疹等、生命に関わらない）。全モデルで一意に判定可能 → PASS

**P2**: 「キノコに関連する活動」の範囲は明確（キノコ狩り、調査、栽培偵察等を含み、ハイキング等は含まない） → PASS

**P3**: 「食用キノコがない理由について推論した」は行為の有無。T と M4 のみが「なぜないか」を考え、他は事実を所与とする → PASS

**P4**: 「山で遭難していた」は事実命題。M13（蛇毒）は遭難とは限らない → PASS

**P5**: 「物質的性質（毒性、忌避効果、成分等）」は具体的に列挙されており曖昧さがない。T では毒キノコの残存パターン（何が残り何が欠けるか）からの推論であり、毒性自体は安堵の理由ではない → PASS

## 導入時検証（判例参照）

`precedents/命題の適切さの判例.md` の2つの判例パターンと照合:

**P1**: 「生存に関わる」— 手段と目的の混同はない。安堵の対象（生存）が明確。生物学的定義の曖昧さも特になし → OK

**P2**: 「キノコに関連する活動」— 「キノコに関連する」の範囲に曖昧さはないか？ M8(遭難中にキノコを見る)は「キノコに関連する活動のため」にいたのではなく、遭難でたまたまいた → false は正しい → OK

**P3**: 「食用キノコがない理由について推論した」— 推論の有無は行為の事実であり、真偽が揺れない → OK

**P4**: 事実命題。曖昧さなし → OK

**P5**: 「物質的性質に由来するものではなかった」— 否定形だが、T では明確に物質的性質（毒性等）ではなく残存パターンからの推論が理由 → 真偽は明確。ただし注意点: T において毒キノコの「存在」は推論の出発点（キノコ生育地の証拠）ではあるが、「存在すること」と「物質的性質（毒性・成分等）」は別概念 → OK
