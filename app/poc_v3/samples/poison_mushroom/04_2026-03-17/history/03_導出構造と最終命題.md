# Phase 3: 導出構造と最終命題

## 導出グラフ

```
FP (遭難中に食用キノコ不在→他者の採集→人がいる→安堵)
├── P3 (他者が食用キノコを採った) → P2 (人間の出入りの推論) → P1 (安堵は毒キノコ自体ではない)
└── P4 (遭難して助けが必要)
```

独立軸: P3（観察・推論の軸）, P4（状況の軸）
根命題: P1, P4

## 共通具象元の探索

- P3 と P4 は概念的に独立（観察 vs 状況）
- FP がこれらを統合する頂点
- 中間命題となる共通具象元は見出せず
