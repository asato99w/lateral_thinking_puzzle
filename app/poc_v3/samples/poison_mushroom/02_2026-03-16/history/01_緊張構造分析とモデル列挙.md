# Phase 1: 緊張構造分析とモデル列挙

## 入力

- S: 男は、辺りに毒キノコしかないのを見て安堵した。なぜだろうか。
- T: 男はキノコ狩りの最中に山で遭難していた。辺りに毒キノコしか残っていないということは、食用キノコは誰かが既に採った＝この辺りに人が来ているということ。近くに人里や他の採集者がいると分かり、助かる見込みがあると安堵した。

## Step 1: 緊張構造の特定

### α: 毒キノコしかないことに対する安堵のパラドックス

- 内容: 「毒キノコしかない」は通常ネガティブな状況（食用キノコが手に入らない、危険なキノコのみ）であるにも関わらず、男は安堵した
- M0 の予測: 毒キノコしかない → 困る・残念に思う・不安になる
- M0 の前提不備: M0 は「男がキノコを食料として求めている」という前提に立つが、S はそれを明言していない。また M0 は毒キノコの存在を単なる食料の観点からのみ評価しているが、他の情報的意味がありうる
- M0 では説明不能: なぜ毒キノコしかないという状況がポジティブな感情（安堵）を引き起こすのか

## Step 1.5: 緊張の分解

### α → α1, α2

#### α1: 毒キノコしかないことの情報的意味

「毒キノコしかない」という事実が、「食用キノコがない」以外のどのような情報を持つか。男にとってこの事実がポジティブな情報に転換されるメカニズムは何か。

#### α2: 安堵が生じる状況的文脈

安堵が生じるためには先行する不安・懸念が必要。男はどのような状況にいて、何を不安に感じていたのか。毒キノコしかないことから得られる情報がなぜその不安を解消するのか。

#### 独立検証

- α1 を緩和するが α2 を緩和しない: 「毒キノコしかない → 食用キノコが採られた → 人がいる」（情報の意味は説明されるが、なぜ人がいることが安堵になるかの文脈は不明）
- α2 を緩和するが α1 を緩和しない: 「男は山で遭難中で不安」（不安の文脈は説明されるが、毒キノコがなぜ安堵のトリガーになるかのメカニズムは不明）

→ 独立に緩和可能。分解は有効。

## Step 2: T の演繹連鎖

- α1 の緩和: 毒キノコしか「残っていない」→ 食用キノコは先に採られた → この辺りに人が来ている証拠（不在からの逆推論による情報転換）
- α2 の緩和: 男はキノコ狩り中に山で遭難 → 助けが必要な状況 → 人がいると判明 → 助かる見込み → 安堵（遭難→救助希望という文脈）

## Step 3: T の命題群

- T1: 男はキノコ狩りをしていた（真）
- T2: 男は山で遭難していた（真）
- T3: 毒キノコしか残っていないのは食用キノコが既に採られたからである（真）
- T4: 食用キノコが採られている → この辺りに人が来ている（真）
- T5: 人がいると分かり、助かる見込みがあると安堵した（真）

## モデル列挙

### 全緊張（α1 + α2）を緩和するモデル

| ID | モデル名 | 概要 |
|----|---------|------|
| T | 遭難者の人跡推論 | キノコ狩り中に山で遭難。毒キノコしかない→食用が採られた→人がいる→助かる |
| M1 | 毒キノコ研究者 | 毒キノコの研究者で特定種を探索中。毒キノコしかない→目的素材が豊富→安堵 |
| M2 | 同行者の誤食防止 | キノコに詳しくない同行者の誤食を懸念。毒キノコのみ→明らかに毒なので誤食リスク低→安堵 |
| M3 | 毒素利用者 | 害虫駆除や薬品製造のため毒キノコを採取する目的。毒キノコしかない→素材が豊富→安堵 |
| M4 | 秘密スポットの守護者 | 秘密のキノコスポットを他人に知られたくない。毒キノコしかない→食用がなく人が来ない→秘密が守られる→安堵 |
| M5 | サバイバル教官 | 野外訓練で食用・毒キノコの識別を教える教官。毒キノコのみ→「食べてはいけない」と明確に判断できる教材→安堵 |
| M6 | キノコアレルギー持ち | 食用キノコのアレルギー。同行者に食べることを勧められるのが苦痛。毒キノコしかない→食べる圧力がない→安堵 |
| M7 | 管理区域の巡回者 | 立入制限区域の森林管理者。毒キノコしかない→食用キノコがなく不法侵入者が来ない→安堵 |
| M8 | キノコ狩り競技者 | キノコ狩り大会でライバルのエリアを偵察。毒キノコしかない→ライバルが不利→自分の優位→安堵 |
| M9 | 中毒患者の治療医 | 毒キノコ中毒患者の治療中、毒の種類特定のため現場調査。毒キノコしかない→種類が限定→治療方針が絞れる→安堵 |
| M10 | 毒殺事件の捜査官 | 毒キノコによる殺人事件の捜査。毒キノコしかない場所の発見→犯人の毒入手経路の特定→捜査前進→安堵 |
| M11 | ペットの安全確保 | キノコを食べる癖のある犬と散歩中。毒キノコしかない→犬が嫌って食べない→安堵 |
| M12 | 菌類生態系の研究者 | 特定の毒キノコ群落の生態研究。毒キノコしかない→食用キノコに圧迫されず生態系が健全→安堵 |

### バリアント群の識別

#### G2: {M1, M3} — 毒キノコの直接的利用価値

共通メカニズム:
- α1: 毒キノコ自体が目的の素材として直接的価値を持つ
- α2: 特定目的の素材確保の必要性

M1（研究者）と M3（害虫駆除等）は舞台設定の違いのみでメカニズムの本質は同一。

#### G3: {M2, M11} — 誤食リスクの回避

共通メカニズム:
- α1: 毒キノコしかない→誤食リスクが低い（安全指標）
- α2: 同行する存在（人間/ペット）の安全懸念

M2（同行者）と M11（ペット）は保護対象の違いのみでメカニズムの本質は同一。

#### G4: {M4, M7} — 食用不在→人の不在確認

共通メカニズム:
- α1: 食用キノコが不在→人が来ていない/来ない
- α2: 保護対象が人に荒らされていないことの確認

M4（秘密スポット）と M7（管理区域）は保護対象の違いのみでメカニズムの本質は同一。
