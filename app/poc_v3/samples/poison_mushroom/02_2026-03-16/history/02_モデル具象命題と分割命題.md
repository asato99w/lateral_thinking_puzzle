# Phase 2: モデル具象命題と分割命題

## Step 1: 各モデルの具象命題

### G1 (T): 食用不在→人の存在を逆推論→遭難からの救助
- 男はキノコ狩りのために山に来ていた
- 男は山中で遭難し、自力では下山できない状態だった
- 辺りを観察したところ毒キノコしか残っていなかった
- 食用キノコが残っていない → 誰かが先に採った → 人がこの辺りに来ている
- 人が近くにいるなら助けてもらえる → 助かる見込み → 安堵

### G2 (M1, M3): 毒キノコの直接的利用価値による素材確保
- 男は毒キノコ自体を目的として野外に来ていた（研究用素材・薬剤原料等）
- 毒キノコが豊富にある場所を探していた
- 辺りに毒キノコしかない → 目的の素材が豊富に手に入る → 安堵

### G3 (M2, M11): 毒キノコのみ→誤食リスク回避
- 男は同行する存在（人間やペット）がキノコを誤食することを懸念していた
- 食用キノコと毒キノコが混在していると識別が難しく誤食のリスクがある
- 毒キノコしかない → 食用キノコと紛らわしいものがない → 誤食リスクが低い → 安堵

### G4 (M4, M7): 食用不在→人の不在確認→保護対象の安全
- 男は特定の区域に人が来ないことを望んでいた（秘密スポット・管理区域）
- 食用キノコがあると人を引き付けるが、毒キノコしかなければ人が来る動機がない
- 毒キノコしかない → 食用キノコがない → 人が来ていない → 保護対象が安全 → 安堵

### M5: 毒キノコのみ→教育的に明確な判断材料
- 男は野外教育の教官で、キノコの識別訓練を行っていた
- 食用と毒の混在は識別訓練に適するが、誤食リスクもある
- 毒キノコのみ → 「全て食べてはいけない」と明確に教えられる → 安全な教材 → 安堵

### M6: 食用不在→食べる機会なし→アレルギー等の回避
- 男は食用キノコにアレルギーがあり、同行者にキノコを食べるよう勧められるのが苦痛
- 毒キノコしかない → 食用キノコがない → 食べる圧力がかからない → 安堵

### M8: ライバルエリアの不利→競争的優位
- 男はキノコ狩り大会に参加し、ライバルのエリアを偵察していた
- 毒キノコしかない → ライバルのエリアに食用キノコがない → 自分が有利 → 安堵

### M9: 毒種の限定→医療的情報の確定
- 男は医者で、毒キノコ中毒の患者を治療中
- 毒の種類特定のため、患者がキノコを採った現場を調査
- 毒キノコしかない → 毒の種類が限定される → 治療方針を絞れる → 安堵

### M10: 犯行証拠の特定→捜査の進展
- 男は捜査官で、毒キノコによる殺人事件を捜査中
- 犯人の毒入手経路を特定するため現場を調査
- 毒キノコしかない場所を発見 → 犯行の証拠 → 捜査が前進 → 安堵

### M12: 毒キノコのみ→生態系の健全さ
- 男は菌類学者で、特定の毒キノコ群落の生態を研究中
- 食用キノコの侵入は毒キノコの生育環境を圧迫する
- 毒キノコしかない → 食用キノコに圧迫されず生態系が健全 → 研究対象が保全 → 安堵

## Step 2c: 抽象命題の生成

### 候補命題

**C1: 男にとって、毒キノコしかないことの重要性は、食用キノコの不在にあった**
- true: {G1, G4, M6, M8}
- false: {G2, G3, M5, M9, M10, M12}
- 検討: T では食用キノコが採られた→人がいるという推論の起点。G4 は食用不在→人が来ない。M6 は食用不在→食べなくて済む。M8 は食用不在→ライバル不利。

**C2: 男はキノコの状態から、その場所への人の出入りについて推論した**
- true: {G1, G4}
- false: {G2, G3, M5, M6, M8, M9, M10, M12}
- 検討: G1 は「人が来た」と推論、G4 は「人が来ない」と推論。両者とも人の出入りに関する推論。

**C3: 男の安堵は、生命や安全に関わるものだった**
- true: {G1, G3}
- false: {G2, G4, M5, M6, M8, M9, M10, M12}
- 検討: G1 は遭難→自分の命、G3 は同行者の誤食→安全。

**C4: 男はキノコ狩り（食用キノコの採取）のためにその場所に来ていた**
- true: {G1, M8}
- false: {G2, G3, G4, M5, M6, M9, M10, M12}
- 検討: G1 はキノコ狩り中に遭難。M8 はキノコ狩り大会。

**C5: 毒キノコしかないことが、その場所に人がいる（来た）証拠となった**
- true: {G1}
- false: {G2, G3, G4, M5, M6, M8, M9, M10, M12}
- 検討: G4 は逆に「人がいない」証拠。G1 のみ。
