# Phase 2: モデル具象命題と分割構造

## Step 1: 各モデルの具象命題

### T (熱気球からの転落)
- 男は数人で熱気球に乗っていた
- 熱気球の高度が下がり始めた
- 軽量化のため荷物を全て捨てた
- 荷物を捨てても不十分で服も脱いだ
- 軽量化でも高度が回復しなかった
- くじ引きで一人が飛び降りることになった
- くじ引きはマッチの長さの比較で行った
- 男は短いマッチを引いた
- 男は熱気球から飛び降りた
- 男は落下により死亡した

### M1 (飛行機の不時着)
- 男は小型飛行機に乗っていた
- 飛行機が砂漠に墜落した
- 墜落時の衝撃と火災で衣服が失われた
- 男は生き残ったが負傷した
- 救難信号のためマッチで火を起こそうとした
- マッチが折れて火を起こせなかった
- 救助を待つうちに死亡した

### M2 (集団遭難とくじ引き)
- 男は数人で砂漠を車で横断していた
- 車が故障し遭難した
- 残りの水が少なくなった
- 暑さで服を脱いだ
- 残りの水を誰が得るかマッチのくじ引きで決めた
- 男は短いマッチを引いた
- 男は水を譲り脱水で死亡した

### M3 (強盗被害)
- 男は砂漠を旅行中だった
- 強盗に襲われた
- 身ぐるみ剥がされ裸にされた
- 砂漠に放置された
- 残されたマッチで火を起こそうとした
- マッチが折れて火を起こせなかった
- 救助が来ず死亡した

### M4 (サバイバル挑戦)
- 男はサバイバル企画に参加した
- 裸で砂漠を渡るルールだった
- 唯一許された道具がマッチ1本だった
- 火を起こそうとしたがマッチが折れた
- 火を起こせず脱水で死亡した

### M5 (砂漠での自殺)
- 男は自殺を決意した
- 砂漠で死ぬことを選んだ
- 持ち物を全て捨て服も脱いだ
- マッチで遺書を燃やした
- 折れたマッチだけが手に残った
- 脱水で死亡した

### M6 (宗教的苦行)
- 男は宗教的修行者だった
- 砂漠での裸の苦行に挑んだ
- 聖火をつけるためにマッチを持参した
- マッチが折れて聖火をつけられなかった
- 脱水で死亡した

### M7 (処刑・制裁)
- 男は犯罪組織に関わっていた
- 裏切り者として制裁を受けた
- 裸にされ砂漠に放置された
- マッチ1本だけ与えられたが意図的に折られていた
- 救助の手段なく死亡した

### M8 (車の故障と信号)
- 男は砂漠を車で走行中だった
- 車が故障した
- 暑さで服を脱いだ
- マッチでガソリンに火をつけて煙の信号を送ろうとした
- マッチが折れて火をつけられなかった
- 救助が来ず死亡した

### M9 (映画撮影事故)
- 男は映画の撮影に参加していた
- 砂漠ロケで裸のシーンを撮影していた
- マッチは小道具として握っていた
- 撮影中に事故が発生した
- 事故により死亡した

### M10 (脱水による錯乱)
- 男は砂漠を歩いていた
- 脱水症状に陥った
- 精神錯乱状態で服を脱いだ
- マッチは元々の所持品だった
- 脱水により死亡した

### M11 (パラシュート事故)
- 男はパラシュートで降下した
- 降下中に服が引き裂かれた
- 砂漠に着地した
- マッチは携行品だった
- 着地の衝撃で死亡した

### M12 (砂漠施設の火災)
- 男は砂漠の施設にいた
- 火災が発生した
- 衣服が燃えて裸になった
- マッチが出火原因だった
- マッチを握ったまま施設から逃げた
- 力尽きて死亡した

## Step 2b: グルーピング

### 到達方法
- 上空から落下: T, M1, M11
- 地上の乗り物: M2, M8
- 徒歩/地上: M3, M10
- 目的を持って来た: M4, M5, M6
- 連れてこられた: M7
- 現地にいた: M9, M12

### 裸の理由
- 合理的判断で脱いだ: T(軽量化), M2(暑さ), M5(覚悟), M8(暑さ)
- ルール/儀式: M4, M6
- 他者に剥がされた: M3, M7
- 物理的に失った: M1(火災), M11(引裂), M12(火災)
- 撮影/演技: M9
- 精神錯乱: M10

### マッチの役割
- くじ引き(運命の決定): T, M2
- 火起こし(サバイバル): M1, M3, M4, M6, M8
- 燃焼行為: M5, M12
- 関係なし(携行品/小道具): M9, M10, M11
- 偽の希望: M7

### 他者の関与
- 仲間と共に危機: T, M2
- 敵対者: M3, M7
- 関係者(非危機): M4, M9
- 一人: M1, M5, M6, M8, M10, M11, M12

## Step 2c-2d: 抽象命題の生成と意味確定

| ID | ラベル | T | true_models | false_models |
|----|--------|---|-------------|--------------|
| P1 | 男が砂漠にいるのは予期せぬ事態の結果だった | true | T,M1,M2,M3,M7,M8,M10,M11,M12 | M4,M5,M6,M9 |
| P2 | 男は乗り物のトラブルにより砂漠に取り残された | true | T,M1,M2,M8,M11 | M3,M4,M5,M6,M7,M9,M10,M12 |
| P3 | マッチは火をつけるための道具として使われたのではない | true | T,M2,M7,M9,M10,M11 | M1,M3,M4,M5,M6,M8,M12 |
| P4 | 男は空中から砂漠に到達した | true | T,M1,M11 | M2,M3,M4,M5,M6,M7,M8,M9,M10,M12 |
| P5 | 男には仲間がいて共に危機的状況にあった | true | T,M2 | M1,M3,M4,M5,M6,M7,M8,M9,M10,M11,M12 |
| P6 | マッチはくじ引きの道具として使われた | true | T,M2 | M1,M3,M4,M5,M6,M7,M8,M9,M10,M11,M12 |
| P7 | 裸になったのは乗り物を軽くするためだった | true | T | M1,M2,M3,M4,M5,M6,M7,M8,M9,M10,M11,M12 |

意味確定チェック:
- P1: 「予期せぬ事態」= 本人が意図しない結果として砂漠にいること。T(気球トラブル)は明確に真。M4(企画)は自ら望んで来たので偽。明確。
- P2: 「乗り物のトラブル」= 乗り物の故障・事故。T(気球高度低下)は真。M3(旅行中の強盗)は乗り物トラブルではなく偽。明確。
- P3: 火を付ける目的でないこと。T(くじ引き)は真。M1(火起こし)は偽。明確。
- P4: 空中= 地表より上の高度にいたこと。T(熱気球)は真。M2(車)は偽。明確。
- P5: 「仲間」= 敵対者でなく共同行動する者。「危機的状況」= 生命の危機を共有。T(気球乗員)は真。M7(組織の制裁)は「敵対者」なので偽。明確。
- P6: くじ引き= ランダムに結果を決定する行為。T(マッチの長さで決定)は真。明確。
- P7: 乗り物を軽くする= 重量削減目的。T(気球の軽量化)は真。M2(暑さ)は偽。明確。
