# Phase 1: 緊張構造分析とモデル列挙

## 入力

- S: 砂漠の真ん中で裸の男が死んでいた。手には折れたマッチ棒を握りしめている。
- T: 男は数人で熱気球に乗っていたが、高度が下がり始めた。荷物を全て捨て、服も脱いで軽量化したが足りず、くじ引き（マッチの長さ）で一人が飛び降りることになった。短いマッチを引いた男が飛び降りて死亡した。

## 緊張構造

### α: 砂漠の真ん中にいること

- 因果の欠落: どうやって砂漠の真ん中に来たのかが不明
- M0 の予測: 車や徒歩で来たはず
- M0 の前提不備: 車や装備の痕跡が何も示されていない
- M0 では説明不能な点: 通常の移動手段では裸で折れたマッチを持つ状態にならない

### β: 裸であること

- 期待との不一致: 砂漠は日中灼熱・夜間冷却で、衣服は生存に必要
- M0 の予測: 砂漠にいる人は身を守るため衣服を着ている
- M0 の前提不備: 自発的に裸になる合理的理由がない

### γ: 折れたマッチ棒を握りしめていること

- 因果の欠落: マッチ棒の用途が示されていない
- 期待との不一致: マッチは火を付ける道具だが折れていて使用不能
- M0 では説明不能な点: 死の間際まで折れたマッチを握りしめる動機がない

## T の演繹連鎖

- α の緩和: 熱気球で上空にいた → 飛び降りて落下 → 地上の移動手段不要
- β の緩和: 高度低下 → 軽量化 → 荷物→衣服の順で捨てた
- γ の緩和: マッチはくじ引きの道具 → 短いマッチ＝死の判定 → 握りしめる動機がある

## T の命題群

- 男は熱気球に乗っていた (true)
- 男は数人の仲間と一緒だった (true)
- 熱気球の高度が下がり始めた (true)
- 乗員は荷物を全て捨てた (true)
- 乗員は服も脱いで軽量化した (true)
- 軽量化しても高度が十分に回復しなかった (true)
- くじ引きで一人が飛び降りることになった (true)
- くじ引きの方法はマッチの長さの比較だった (true)
- 男は短いマッチを引いた (true)
- 男は熱気球から飛び降りた (true)
- 男は落下により死亡した (true)

## モデル列挙

### {α, β, γ} 全緊張を緩和

| ID | ラベル | 概要 |
|----|--------|------|
| T | 熱気球からの転落 | 熱気球の高度低下で軽量化のため荷物と服を捨て、マッチのくじ引きで短い方を引いた男が飛び降りて死亡 |
| M1 | 飛行機の不時着 | 小型飛行機が砂漠に墜落。衝撃と火災で衣服を失い、救難信号のためマッチで火を起こそうとしたが折れた |
| M2 | 集団遭難とくじ引き | 数人で砂漠横断中に車が故障し遭難。残りの水を巡りマッチのくじ引き。暑さで服を脱ぎ、短いマッチを引いた男が水を譲って脱水死 |
| M3 | 強盗被害 | 砂漠で強盗に襲われ身ぐるみ剥がされ放置。残されたマッチで火を起こし救助を求めようとしたが折れた |
| M4 | サバイバル挑戦 | 裸で砂漠を渡るサバイバル企画に参加。唯一許された道具がマッチ1本だったが折れ、火を起こせず死亡 |
| M5 | 砂漠での自殺 | 砂漠で死ぬことを選び、持ち物を捨て服も脱いだ。マッチで遺書を燃やし、最後の折れたマッチだけが手に残った |
| M6 | 宗教的苦行 | 砂漠で裸の苦行に挑む修行者。聖火をつけるためのマッチが折れ、儀式を完遂できず脱水死 |
| M7 | 処刑・制裁 | 犯罪組織に裸で砂漠に放置された。マッチ1本だけ与えられたが意図的に折られていた |
| M8 | 車の故障と信号 | 砂漠で車が故障。暑さで服を脱ぎ、マッチでガソリンに火をつけ煙の信号を送ろうとしたが折れた |
| M9 | 映画撮影事故 | 砂漠ロケで裸のシーンを撮影中に事故死。マッチは小道具として握っていた |

### {α, β} を緩和

| ID | ラベル | 概要 |
|----|--------|------|
| M10 | 脱水による錯乱 | 砂漠で迷い脱水。精神錯乱で服を脱いだ。マッチは偶然の所持品 |
| M11 | パラシュート事故 | 上空から降下し砂漠に着地。降下中に服が引き裂かれた。マッチは携行品 |

### {β, γ} を緩和

| ID | ラベル | 概要 |
|----|--------|------|
| M12 | 砂漠施設の火災 | 砂漠の施設で火災が発生。衣服が燃えて裸に。マッチが出火原因で握ったまま逃げ力尽きた |

## 構造的重なり

- T, M2: 複数人＋マッチによるくじ引きの共通構造
- T, M1, M11: 上空からの落下で砂漠に到達
- M1, M3, M4, M8: マッチを火起こし道具として使用する構造
- M3, M7: 犯罪被害による裸
