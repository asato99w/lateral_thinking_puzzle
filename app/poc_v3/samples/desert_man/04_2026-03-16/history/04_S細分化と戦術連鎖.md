## Step 2: 根命題への戦術連鎖

導出グラフ:

根命題（最も抽象的）: P1, P2
独立軸: P5, P6（FP の直接子で、P7/P8 とは独立した次元）

### C1: S → P1（他者関与）

S2(砂漠の真ん中) + S4(死んでいた) →[登場人物の探索(4)]→ **I1**(男は他の人物と一緒にいた) →[段階的絞り込み(11)]→ **P1**(男の死に至る経緯に他者が関わっていた)

- S2+S4: 砂漠の真ん中で死んでいるという異常な状況 → 他者の存在を疑う方向
- I1: 男は単独ではなく、他の人物が存在していた
- P1: 他の人物が存在しただけでなく、死に至る経緯に関わっていた

### C2: S → P2（脱衣は自発的）

S3(裸) + S2(砂漠) →[常識的仮説の排除(1)]→ **I2**(男が裸なのは他者に服を奪われたからではない) →[動機の探索(8)]→ **P2**(男の脱衣は外部から強制されたものではなかった)

- S3+S2: 砂漠で裸 → 常識的には「襲われて奪われた」を疑う
- I2: 「奪われた」仮説を排除
- P2: 外部からの強制全般を否定（奪われたのでも脱がされたのでもない = 自発的）

### C7: S → P5（空中にいた）

S2(砂漠の真ん中) + S9(人里から離れた場所) →[手段の探索(10)]→ **I7**(男は何らかの乗り物で移動していた) →[段階的絞り込み(11)]→ **P5**(男は地上ではなく空中にいた)

- S2+S9: 砂漠の真ん中の人里離れた場所 → 「どうやってここに来た?」
- I7: 何らかの乗り物で移動（徒歩では砂漠の真ん中は困難）
- P5: 地上の乗り物ではなく空中の乗り物

### C8: S → P6（非環境死）

S4(死んでいた) + S5(マッチ棒あり) + S6(折れていた) →[常識的仮説の排除(1)]→ **I8**(男の死因は単純な環境暴露（脱水・熱中症）だけでは説明できない) →[段階的絞り込み(11)]→ **P6**(男の死因は環境（脱水・熱中症・衰弱）ではなかった)

- S4+S5+S6: 死体が折れたマッチ棒を持っている → 単純な脱水死なら不自然
- I8: 環境死だけでは説明がつかない（マッチ棒の存在が別の経緯を示唆）
- P6: 死因は環境ではなかった（落下等の別の原因）

## Step 3: 具象方向への戦術連鎖

### C3: P1 → P3（マッチで人選び）

P1(他者関与) + S5(マッチ棒) + S6(折れている) →[観測事実の深掘り(2)]→ **I3**(マッチ棒は男と他者の間で特定の目的に使われた道具だった) →[段階的絞り込み(11)]→ **P3**(マッチ棒は複数人の間で人を選ぶ道具として使われた)

- P1+S5+S6: 他者が関わっていて、折れたマッチ棒がある → マッチの役割を深掘り
- I3: マッチ棒は単なる所持品ではなく、他者との間で目的を持って使われた
- P3: 具体的には人を選ぶ（くじ引き）道具として使われた

fc_source: P1 は C1 の reveals

### C4: P1 → P4（共に生存の危機）

P1(他者関与) + S2(砂漠の真ん中) + S4(死んでいた) →[段階的絞り込み(11)]→ **I4**(男と他者は共通の困難に直面していた) →[段階的絞り込み(11)]→ **P4**(男を含む複数人が共に生存の危機にあった)

- P1+S2+S4: 他者が関わっていて砂漠で死んだ → どんな状況だったか
- I4: 男と他者は共通の困難に直面（ただの同行ではなく困難を共有）
- P4: 困難のレベルは生存の危機（命に関わる状況）

fc_source: P1 は C1 の reveals

### C5: P2 + P4 → P7（脱衣は危機対処）

P2(自発的脱衣) + P4(共に生存の危機) →[因果の逆転(5)]→ **I5**(男の自発的な脱衣は共に直面した危機的状況と関連していた) →[段階的絞り込み(11)]→ **P7**(脱衣は複数人が共に直面した危機への対処の一環だった)

- P2+P4: 自発的に脱いだ + 生存の危機にあった → 脱衣の原因は何か
- I5: 脱衣は危機的状況と関連（偶然ではなく因果関係がある）
- P7: 危機への対処の一環（軽量化等）

fc_source: P2 は C2 の reveals、P4 は C4 の reveals

### C6: P3 + P4 → P8（マッチくじ in 危機）

P3(マッチで人選び) + P4(共に生存の危機) →[因果の逆転(5)]→ **I6**(マッチによる人の選定は生存の危機への対応として行われた) →[段階的絞り込み(11)]→ **P8**(共に生存の危機にあった複数人の間でマッチのくじ引きが行われた)

- P3+P4: マッチで人を選んだ + 生存の危機 → なぜ人を選ぶ必要があった？
- I6: 人の選定は危機への対応（緊急時の決断）
- P8: 危機の中での正式なくじ引き

fc_source: P3 は C3 の reveals、P4 は C4 の reveals

### C9: P5 + P7 + P8 + P6 → FP

P5(空中) + P8(マッチくじ in 危機) + P7(脱衣は危機対処) →[段階的絞り込み(11)]→ **I9**(空中の乗り物内で軽量化（脱衣含む）とマッチくじ引きが行われた) →[手段の探索(10)]→ **FP**(…くじに選ばれた男が飛び降りて落下死した)

- P5+P8+P7: 空中 + マッチくじで危機対処 + 脱衣も危機対処 → 全体像の構築
- I9: 空中の乗り物で軽量化（脱衣）とマッチくじが行われた
- FP: P6（非環境死）の情報を加え、くじで選ばれた男が飛び降りて落下死（全体像の完成）

fc_source: P5 は C7 の reveals、P7 は C5 の reveals、P8 は C6 の reveals、P6 は C8 の reveals
