# Phase 3: 導出構造と最終命題

## Step 1: T から分割命題への演繹構造

### 直接演繹

- T → P1: 「数人で熱気球に乗っていた」「くじ引きで飛び降り」→ 他者が関わっていた
- T → P2: 「服も脱いで軽量化した」→ 自発的に脱いだ（強制ではない）
- T → P3: 「くじ引き（マッチの長さ）で一人が飛び降りることになった」→ マッチは人を選ぶ道具
- T → P4: 「高度が下がり始めた」「荷物を全て捨て」→ 複数人が共に生存の危機
- T → P5: 「熱気球に乗っていた」→ 空中にいた
- T → P6: 「飛び降りて死亡した」→ 環境死ではなかった

### 命題間の演繹関係

- **P3 → P1**: 「マッチ棒は複数人の間で人を選ぶ道具」→ 「複数人の間で」があるため「他者が関わっていた」が論理的に導かれる。
- **P4 → P1**: 「複数人が共に生存の危機にあった」→ 「複数人が共に」から「他者が関わっていた」が論理的に導かれる。

### 冗長辺の除去

- T → P1 は T → P3 → P1 および T → P4 → P1 で到達可能 → 冗長、除去

### 結果

```
T
├── P2 (直接)
├── P3 → P1
├── P4 → P1
├── P5 (直接)
└── P6 (直接)
```

T 直接ノード: {P2, P3, P4, P5, P6}

## Step 2: 共通具象元の発見

### 候補 1: P2 と P4 の共通具象元

T の事実:「全員で軽量化（荷物投棄・脱衣）を行い」+「高度が下がり始めた」

これは P2（脱衣は自発的）と P4（共に生存の危機）の両方にまたがる事実。

**「脱衣は複数人が共に直面した危機への対処の一環だった」**
- → P2: 「対処の一環」→ 自発的行為（外部強制ではない）。危機対処として自ら行ったのだから強制ではない ✓
- → P4: 「複数人が共に直面した危機」→ 複数人が共に生存の危機にあった ✓
- 異なる情報の削減: P2 へは「危機」「複数人」を落とす。P4 へは「脱衣」「対処」を落とす。

### 候補 2: P3 と P4 の共通具象元

T の事実:「くじ引き（マッチの長さ）」+「高度が下がり始めた」「足りず」

**「共に生存の危機にあった複数人の間でマッチのくじ引きが行われた」**
- → P3: 「マッチのくじ引き」→ マッチは人を選ぶ道具 ✓（「危機」を落とす）
- → P4: 「共に生存の危機にあった複数人」→ 複数人が共に生存の危機 ✓（「マッチ」を落とす）
- 異なる情報の削減: P3 へは「危機」「生存」を落とす。P4 へは「マッチ」「くじ引き」を落とす。

## Step 3: 中間命題の検証

### P7: 脱衣は複数人が共に直面した危機への対処の一環だった

**演繹的分岐**: P2 と P4 の両方を演繹 ✓

**モデル非自明分割**:
- G1: TRUE（軽量化=高度低下という危機への対処としての脱衣）
- M3: TRUE（暑さ=砂漠遭難の環境に対する対処としての脱衣。グループ全員が水不足の危機に直面）
- M4: FALSE（脱衣は罰。賭けは「危機」ではなく自発参加の遊び）
- M7: FALSE（単独、危機ではなく自殺の覚悟）
- M8: FALSE（単独遭難。複数人ではない）
- Others: FALSE
- true_models = {G1, M3}。{T}のみではない ✓

**意味確定性**: 「危機への対処の一環」= 危機に対処する行動群の中に脱衣が含まれていたこと。明確 ✓

### P8: 共に生存の危機にあった複数人の間でマッチのくじ引きが行われた

**演繹的分岐**: P3 と P4 の両方を演繹 ✓

**モデル非自明分割**:
- G1: TRUE（高度低下の危機の中でマッチくじ引き）
- M3: TRUE（水不足の危機の中でマッチくじ引き）
- M4: FALSE（「共に生存の危機」ではない。賭けへの自発参加）
- Others: FALSE
- true_models = {G1, M3}。{T}のみではない ✓

**分割の同一性**: P7 と P8 の true_models は共に {G1, M3} で、P4 と同一。分割の同一性は許容（導出グラフに深さを与える）。

**意味確定性**: 明確 ✓

## Step 5: 抽象化による発見

T 直接ノード: {P7, P8, P5, P6}

### 検討

P5（空中にいた）と P6（非環境死）に共通構造がないか:
- P5 →「通常と異なる場所」/ P6 →「通常と異なる死因」
- 「空中にいたため落下して死んだ」は因果関係であり、演繹（情報の削減）ではない
- 「空中にいた」から「環境死ではない」は論理的に導かれない（空中でも環境死は起こりうる）
- → 共通具象元なし

P7（脱衣=危機対処）と P5（空中）:
- 「空中の乗り物内で軽量化として脱衣した」→ true_models = {G1} のみ → 非自明分割を満たさない
- → 不適格

P8（マッチくじ in 危機）と P6（非環境死）:
- 「マッチくじで選ばれた者が環境死以外で死んだ」→ true_models = {G1} のみ → 不適格
- → 不適格

**結論**: Step 5 で新たな中間命題の発見なし。
