# Phase 1: 緊張構造分析とモデル列挙

## Step 1: S の緊張構造の特定

S: 砂漠の真ん中で裸の男が死んでいた。手には折れたマッチ棒を握りしめている。

### 緊張 α: 砂漠の真ん中にいること

- 内容: 人が砂漠の真ん中にいること自体が異常。交通手段や生活基盤がない場所に人がいる。
- M0 の予測: 砂漠の真ん中に人はいない。いるとすれば車両での移動中か、遊牧民のような生活者。
- M0 の前提不備: 車両や交通手段が一切言及されていない。砂漠の「真ん中」は移動経路上ではなく、隔絶された地点を示唆。
- M0 では説明不能な点: 裸・マッチ棒と組み合わせた場合、通常の砂漠移動者とは全く異なるプロファイル。

### 緊張 β: 裸であること

- 内容: 砂漠で裸でいることは致命的。日中は強烈な日射、夜間は急激な冷え込み。衣服は生存に不可欠。
- M0 の予測: 砂漠にいる人は必ず衣服を着ている。
- M0 の前提不備: 裸であるということは、衣服を自発的に脱いだか、他者に剥がれたか、最初から着ていなかったかのいずれか。
- M0 では説明不能な点: 砂漠で自発的に裸になる合理的理由が常識的には見当たらない。

### 緊張 γ: 折れたマッチ棒を握りしめていること

- 内容: 死体が折れたマッチ棒を握りしめているという極めて特異な状況。「握りしめている」は意図的な保持を示唆。
- M0 の予測: マッチ棒は火を起こすための道具。しかし砂漠の真ん中で裸の状態では、火を起こす対象（燃料）がない。
- M0 の前提不備: 通常の用途（着火）ではマッチ棒が折れている理由も、死に際に握りしめている理由も説明できない。
- M0 では説明不能な点: マッチ棒が「折れている」こと、そして死の間際に「握りしめている」ことが、単なる着火道具では説明できない。

### 緊張 δ: 死因が不明

- 内容: 砂漠の真ん中で裸の男が死んでいるが、直接的な死因が示されていない。状況要素（場所・裸・マッチ棒）と死の因果関係が不明。
- M0 の予測: 砂漠で裸なら脱水・熱中症が想定されるが、マッチ棒との関連が全くない。
- M0 の前提不備: 脱水死であればマッチ棒は不要。マッチ棒の存在が、死因が単純な環境死ではない可能性を示唆。
- M0 では説明不能な点: 全ての要素（砂漠・裸・折れたマッチ棒・死）を統一的に説明する因果関係が見えない。

## Step 1.5: 緊張の分解

### α の分解

α を以下の 2 つに分解する:

- **α1: 砂漠の真ん中に到達した手段が不明** — 砂漠の中央は徒歩で到達困難。何らかの交通手段があったはずだが、痕跡がない。
- **α2: 砂漠の真ん中にいる理由が不明** — 意図的に来たのか、偶発的に至ったのか。目的が見えない。

独立検証:
- α1 緩和・α2 未緩和: 車で砂漠に来た（手段はわかる）が、なぜ砂漠の真ん中に来たかは不明 → 想像可能
- α2 緩和・α1 未緩和: 砂漠を横断する目的はあった（理由はわかる）が、車もないのにどうやって真ん中まで来たか不明 → 想像可能
→ 独立。分解実施。

### β の分解検討

「服を脱いだ」(メカニズム) と「全裸である」(程度) に分解を試みる。

独立検証:
- メカニズム緩和・程度未緩和: 暑さで上着を脱いだ（部分脱衣の理由はわかる）が全裸は説明できない → 想像可能
- 程度緩和・メカニズム未緩和: 持ち物を全部捨てた結果全裸になった（全裸の説明）だが、なぜ捨てるかの動機は不明 → 「全部捨てた」が動機なしに程度だけ説明する状況が不自然。全部捨てる理由がわかれば脱衣動機もわかる傾向がある。
→ 独立性が弱い。分解不実施。β は単一のまま。

### γ の分解

γ を以下の 3 つに分解する:

- **γ1: マッチ棒を持っている理由が不明** — 砂漠で裸の状態でマッチ棒を持つ理由。用途が不明。
- **γ2: マッチ棒が折れていることの意味が不明** — 「折れた」は使用失敗か、意図的な破壊か、別の意味があるか。
- **γ3: マッチ棒を握りしめていること** — 「握りしめている」は死の間際まで手放さなかった強い意志を示す。マッチ棒に単なる道具以上の意味がある。

独立検証:
- γ1 vs γ2: 火をつけるためにマッチを持っていた(γ1緩和)が折れている理由は不明(γ2未緩和) → 想像可能。逆に、折れているのはくじの印(γ2緩和)だがなぜマッチを使うのか(γ1未緩和) → 想像可能。→ 独立。
- γ1 vs γ3: マッチの用途はわかる(γ1緩和)が死に際に握りしめる理由がわからない(γ3未緩和) → 想像可能。大切なものだから握りしめていた(γ3緩和)がなぜマッチなのか(γ1未緩和) → 想像可能。→ 独立。
- γ2 vs γ3: 折れているのは使用結果(γ2緩和)だが握りしめる理由は別(γ3未緩和) → 想像可能。→ 独立。
→ 3 分解全て独立。分解実施。

### δ の分解検討

δ1（直接死因）と δ2（状況要素と死因の因果関係）への分解を検討。

独立検証:
- δ1 緩和・δ2 未緩和: 落下で死んだ（直接死因）、でもなぜ落下したか・マッチとの関係は不明 → 想像可能
- δ2 緩和・δ1 未緩和: 因果の流れはわかる（くじ引き→飛び降り）が具体的にどう死んだかは不明 → 因果がわかれば死因もほぼ確定する
→ δ2 → δ1 は含意関係が強い。独立性が弱い。分解不実施。

## Step 2: T の演繹連鎖

T: 男は数人で熱気球に乗っていたが、高度が下がり始めた。荷物を全て捨て、服も脱いで軽量化したが足りず、くじ引き（マッチの長さ）で一人が飛び降りることになった。短いマッチを引いた男が飛び降りて死亡した。

- α1 の緩和: 熱気球で空中にいた → 飛び降りて砂漠に落下。交通手段は熱気球（空中からの落下であり地上の交通手段痕跡はない）
- α2 の緩和: 熱気球の高度低下による緊急事態。意図的に砂漠に来たのではなく、空中から落下した結果。
- β の緩和: 軽量化のため荷物を全て捨て、服も脱いだ。生存のための合理的行動（重量削減）。
- γ1 の緩和: マッチ棒はくじ引きの道具。誰が飛び降りるかを決める公平な手段。
- γ2 の緩和: 「折れたマッチ棒」= 短いマッチ。くじ引きでマッチの長さを比較し、短い（折れた/折られた）マッチが「負け」の印。
- γ3 の緩和: 自分の運命を決定したくじの証拠。飛び降りの覚悟と結果への執着。
- δ の緩和: 熱気球からの飛び降り（高所からの落下死）。

因果の骨格:
熱気球の高度低下 → 軽量化の必要 → 荷物投棄・脱衣(β緩和) → 軽量化不足 → 犠牲者選定の必要 → マッチくじ(γ1, γ2緩和) → 短いマッチの男が飛び降り(γ3, δ緩和) → 砂漠に落下(α1, α2緩和)

## Step 4: モデル列挙

### 全緊張を緩和するモデル

**T（熱気球）**: 数人で熱気球に乗っていたが高度低下。軽量化で荷物・服を捨てたが足りず、マッチの長さのくじ引きで飛び降りる者を決めた。短いマッチの男が落下死。

**M1（小型飛行機）**: 小型飛行機がエンジン故障で高度低下。軽量化で荷物・服を投棄。パラシュートが足りず、マッチくじで一人が機外に出ることに。

**M2（飛行船）**: 飛行船の浮力低下。乗員が軽量化で荷物・服を捨て、マッチくじで犠牲者を決定。

**M3（砂漠横断遭難グループ）**: グループで砂漠を横断中に遭難。水が尽き、最後の水を誰が飲むかマッチのくじ引き。暑さで服を脱いでいた。負けた男が脱水死。

**M4（賭け/度胸試しの罰）**: 数人で度胸試し・賭けをし、負けた者が砂漠に裸で放置される罰。マッチくじで敗者を決定。脱水死。

**M6（誘拐・砂漠放置）**: 犯人に誘拐され砂漠に連れ出された。裸にされ放置。犯人が皮肉でマッチを渡したが、意図的に折って使用不能にしていた。脱水死。

**M7（計画的自殺）**: 砂漠で自殺を図った。持ち物と遺書をマッチで燃やし、服も脱いで全てを捨てた。マッチは使用後に折れた状態で握りしめていた。

**M8（車故障による単独遭難）**: 砂漠で車が故障。救助を求めて徒歩で移動。暑さで服を脱ぎ、マッチで信号火を起こそうとしたが折れて使えなかった。脱水死。

**M9（宗教的修行）**: 砂漠での苦行・修行の一環で裸になった。マッチで聖火を灯す儀式を行い、マッチを儀式的に折った。修行中の衰弱死。

**M10（脱走犯の逃亡）**: 収容施設から脱走し砂漠を横断。追跡を避けるため服をマッチで燃やした。最後のマッチが折れた。脱水死。

### 一部の緊張を緩和するモデル

**M5（密入国・難民の遺棄）**: 砂漠を渡って密入国を試みた。ブローカーに金と持ち物を奪われ裸で放置。マッチは奪われず残った最後の所持品。脱水死。
→ γ2（折れている意味）が未緩和。

**M11（泥酔パーティー放置）**: 砂漠でのパーティーで泥酔し服を脱いだ。仲間に置き去りにされた。マッチは喫煙用の残り。熱中症死。
→ γ2（折れている意味）, γ3（握りしめている理由）が未緩和。

**M12（保険金殺人）**: 保険金目的で砂漠に連れ出され殺害。犯人が証拠隠滅で服を剥いだ。マッチは格闘中に犯人から奪い取ったもの。
→ γ1（マッチの用途）, γ2（折れている意味）が未緩和。

**M13（サバイバルチャレンジ）**: 極限サバイバルチャレンジに参加。最小限の装備で砂漠横断。暑さで服を脱ぎ、支給されたマッチで火を起こそうとして折れた。熱中症死。

## Step 6: バリアントの識別

全緊張のメカニズムが構造的に同一のモデルの組を識別する。

**G1: {T, M1, M2}** — 空中乗り物の事故＋軽量化脱衣＋マッチくじ引き＋落下死

- 全7緊張で同一メカニズム: 空からの落下 / 乗り物事故 / 軽量化脱衣 / くじ引き道具 / くじ符号 / 運命証拠 / 高所落下死
- 舞台設定の違いのみ（熱気球 / 小型飛行機 / 飛行船）
- 代表: T

他のモデル間でバリアントはなし:
- M3 vs M4: β が異なる（高温自発 vs 罰強制）、α2 が異なる（偶発遭難 vs 自発参加の賭け）
- M7 vs M10: β が異なる（覚悟 vs 証拠隠滅）、α2 が異なる（自殺 vs 脱走）
- M8 vs M13: α1 が異なる（車故障 vs 運営投入）、α2 が異なる（偶発事故 vs 自発参加）
