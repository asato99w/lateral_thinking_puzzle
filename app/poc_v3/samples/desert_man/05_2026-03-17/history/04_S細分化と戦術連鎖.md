## Step 2-3: 戦術連鎖の構成

### C1: {S2,S4} → P1（登場人物の探索）
砂漠の真ん中で死んでいた → 男の死には他者が関わっているのでは？ → 他の人と一緒にいた
- I1: 男の死には他の人間が関わっている [T: 仲間と共に気球に乗りくじ引きで決定 ✓]

### C2: {S2,S4} → P3（手段の探索 → 段階的絞り込み）
砂漠の真ん中にいた → どうやって来た？ → 地上ではない → 上空から
- I2: 男は何らかの手段で砂漠の真ん中に運ばれた [T: 気球で ✓]
- I2b: 男が砂漠に到達した手段は地上の移動ではなかった [T: 空中（気球） ✓]

### C3: {S3} → P4（動機の探索 → 段階的絞り込み）
裸だった → なぜ裸？ → 目的があった → 自分で脱いだ
- I3: 男が裸なのは何かの目的があったからだ [T: 軽量化が目的 ✓]
- I3b: 衣服を脱いだのは他者の強制ではなく男自身の行動だった [T: 自ら脱いだ ✓]

### C4: {P1,S5,S6} → P2（観測事実の深掘り → 段階的絞り込み）
他の人と一緒 + マッチ棒 → マッチの用途は？ → 集団での決め事
- I4: マッチ棒は何かの目的に使われた [T: くじ引きに使用 ✓]
- I4b: マッチ棒は集団の中で何かを決めるために使われた [T: 犠牲者選定のくじ引き ✓]

### C5: {P1,S4} → P5（因果の逆転 → 段階的絞り込み）
他の人と一緒 + 死んでいた → グループの状況が死因？ → グループの決定
- I5: 男の死はグループの置かれた状況に起因する [T: 気球の高度低下 ✓]
- I5b: 男の死はグループ内での決定の結果だった [T: くじ引きで選ばれた ✓]

### C6: {P1,P3} → P6（段階的絞り込み）
他の人と一緒 + 上空から → 同じ乗り物で空中に
- I6: 男と他の人々は同じ乗り物で空中にいた [T: 同じ気球に乗っていた ✓]

### C7: {P2,P5} → P7（段階的絞り込み）
くじ引き + 犠牲 → くじ引きが死を決定した
- I7: くじ引きの結果が男の死を直接決定した [T: 短いマッチ→飛び降り ✓]

### C8: {P6,P4,P7} → FP（因果の逆転）
空飛ぶ乗り物+自発的脱衣+くじ引き犠牲 → 高度低下が原因
- I8: 乗り物の高度低下が衣服投棄とくじ引きによる犠牲者選定の原因だった [T: 気球の高度低下 ✓]
