# Phase 2: モデル具象命題と分割命題

## Step 2a: 軸ごとのモデルマッピング

- 到達手段: 空中(G1,M12) / 地上自力(M5,M7,M9,M15,M20,M22,M17,M18) / 第三者配置(G2,M10,M14)
- 脱衣の原因: 自発的(G1,M7,M9,M12,M15,M22) / 強制(G2,M5,M10,M14) / 環境(M17,M20,M18)

## Step 2b: グルーピング

主要なグルーピングパターン:
1. 空中モデル群 {G1, M12}: 上空から降下 + 落下死
2. くじ引きモデル群 {G1, M12, M14, M15, M20}: マッチがくじの道具
3. 自発脱衣モデル群 {G1, M7, M9, M12, M15, M22}: 自分の判断で脱衣

### 同一分割の候補と選択

- 「男は上空から砂漠に到達した」と「男は高所からの落下で死亡した」は同一の true_set {G1, M12}。より根源的な「上空から到達」を選択（到達手段は死因の前提）
- 「マッチ棒はくじ引きに使われた」と「マッチ棒は元々短かった（くじの結果）」は同一の true_set。より抽象的な「くじ引きに使われた」を選択

## Step 2d: 意味の確定性チェック

全 5 命題について検証:

### P1「男は他の人々と一緒にいた」
- 意味: 事件当時、男には同行者がいた
- 全モデルで真偽一意 ✓

### P2「マッチ棒は集団内でのくじ引き（意思決定）に使われた」
- 意味: マッチ棒の用途がランダム選定の道具だった
- 全モデルで真偽一意 ✓

### P3「男は上空から砂漠に到達した」
- 意味: 男が砂漠の地面に着いたのは空中からの降下による
- 全モデルで真偽一意 ✓

### P4「脱衣は男自身の状況判断に基づく自発的行為だった」
- 意味: 衣服を脱いだのは男の意識的な自己判断（他者による強制や環境的喪失ではない）
- 境界ケース:
  - G2（儀式的要求）: 要求に従って脱いだが、自分の状況判断ではなく外部の指示 → False
  - M18（熱中症の錯乱）: 意識的判断ではなく病的行動 → False
- 全モデルで真偽一意 ✓

### P5「男の死は集団の生存のためのやむを得ない犠牲だった」
- 意味: 男は他の人々が生き延びるために死ぬ側に選ばれた
- 境界ケース:
  - M12（スカイダイビング事故）: パラシュート不開は事故であり「集団の生存のための犠牲」ではない → False
  - M14（密輸の裏切り）: 裏切りであり「やむを得ない」犠牲ではない → False
- 全モデルで真偽一意 ✓

## 導入時検証（判例照合）

### 判例 turtle_soup 05 パターン（日常的意味と厳密意味の乖離）
- P1-P5 のいずれにも該当なし。各命題の語句は日常的意味と厳密意味が一致

### 判例 bar_man 07 パターン（手段と目的の混同）
- P4「自発的行為だった」: 脱衣は軽量化の手段だが、命題は行為の自発性（agency）を問うており目的には言及していない。混同なし
- P5「やむを得ない犠牲だった」: 犠牲は手段（集団救命）の結果だが、命題は犠牲の性質を問うている。混同なし
