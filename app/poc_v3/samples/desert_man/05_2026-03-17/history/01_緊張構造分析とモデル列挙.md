# Phase 1: 緊張構造分析とモデル列挙

## Step 1: S の緊張構造の特定

S: 砂漠の真ん中で裸の男が死んでいた。手には折れたマッチ棒を握りしめている。

### α: 砂漠の真ん中での死亡
- 内容: 砂漠の真ん中で男が死んでいる
- M0 予測: 砂漠を横断中の脱水・熱射病、砂嵐などの自然要因
- M0 前提不備: 砂漠の「真ん中」への到達手段が説明されない。通常砂漠を横断するには車両等が必要だがそれが見当たらない
- M0 説明不能: 死因が明示されていない（外傷？脱水？）。裸でマッチ棒を持っているという状態と自然死は整合しにくい

### β: 裸であること
- 内容: 男が裸の状態である
- M0 予測: 暑さで脱いだ、しかし砂漠では衣服は日差しから身を守るため必要
- M0 前提不備: 砂漠で衣服を脱ぐことは自殺行為に近い。暑さ対策としても衣服は必要
- M0 説明不能: 完全な裸は単なる暑さ対策を超える。何か衣服を脱ぐことよりも重要な理由があったことを示唆

### γ: 折れたマッチ棒を握りしめていること
- 内容: 手に折れたマッチ棒を持ち、それを握りしめている
- M0 予測: 火をつけるためのマッチ、使用済みか破損
- M0 前提不備: 砂漠でマッチを使う合理的理由が不明。タバコ用なら裸の男が他に何も持っていないのと矛盾
- M0 説明不能: 「折れた」「握りしめている」という状態が単なる火起こし道具の失敗を超える意味を持つ

## Step 1.5: 緊張の分解

### α の分解
- α1: 砂漠の真ん中にいること自体の不自然さ（到達手段の不在）
- α2: 死因の不明さ（何が男を殺したのか）
- 程度の分離: 「砂漠にいる」というメカニズムと「死亡に至った」という結果は独立
- 独立検証:
  - α1 のみ緩和: 「砂漠にいる理由は車両故障」→ 死因は説明しない ✓
  - α2 のみ緩和: 「毒で死んだ」→ 砂漠にいる理由は説明しない ✓
- → 分解する

### β の分解検討
- β_a: 衣服を失った/脱いだ事実
- β_b: 砂漠で裸であることの合理性
- 独立検証:
  - β_a のみ緩和: 衣服がない理由を説明 → 砂漠での合理性も通常含意される
  - β_b のみ緩和: 砂漠で裸が合理的 → 衣服喪失の具体的経緯も含意される
- → 独立に緩和不能。分解しない

### γ の分解
- γ1: マッチ棒を所持している理由
- γ2: マッチ棒が折れている理由
- γ3: 死の直前まで握りしめている行為
- 独立検証:
  - γ1 vs γ2: 「キャンプ用マッチ」(γ1 緩和) vs 「使おうとして折れた」(γ2 緩和) ✓
  - γ1 vs γ3: 「タバコ用」(γ1 緩和) vs 「死の恐怖で握った」(γ3 緩和) ✓
  - γ2 vs γ3: 「力んで折れた」(γ2 緩和) vs 「大事なものだった」(γ3 緩和) ✓
- → 3 つに分解する

## Step 2: T の緩和メカニズム

T: 男は数人で熱気球に乗っていたが、高度が下がり始めた。荷物を全て捨て、服も脱いで軽量化したが足りず、くじ引き（マッチの長さ）で一人が飛び降りることになった。短いマッチを引いた男が飛び降りて死亡した。

| 緊張 | T の緩和メカニズム |
|------|-------------------|
| α1 | 熱気球から落下 → 砂漠上空を飛行していたため砂漠の真ん中に着地 |
| α2 | 高所（熱気球）からの落下死 |
| β | 高度を維持するための軽量化として荷物と衣服を全て投棄した |
| γ1 | くじ引きの道具としてマッチ棒を使用した（マッチの長さで当選者を決定） |
| γ2 | 短いマッチ＝「はずれ」のマッチ。長さの異なるマッチを用意したため短い方は折れたように見える |
| γ3 | 自分の運命を決定したマッチを握ったまま飛び降りた（恐怖・無念で握りしめたまま） |
