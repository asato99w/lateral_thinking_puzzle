# Phase 9: 質問分類

## 概要

全33問（真側20問 + 棄却13問）を4つのトピックカテゴリに分類した。

## カテゴリ定義

| カテゴリ | 説明 |
|---|---|
| 状況の確認 | 状況・手段・段階的絞り込みに関する質問 |
| 原因と動機 | 動機・因果関係に関する質問 |
| 人物と背景 | 登場人物・背景事情に関する質問 |
| 前提の確認 | 常識的仮説の排除に関する質問 |

## 分類結果

- 状況の確認: 11問（Q1b, Q2b, Q5a, Q5b, Q6a, Q7b, Q8b, Q9a, Q10b, QF6a, QF9a）
- 原因と動機: 9問（Q3b, Q6b, Q7a, Q8a, Q9b, Q10a, QF2b, QF7a, QF7b）
- 人物と背景: 7問（Q4a, Q4b, QF1b, QF1c, QF3c, QF4b, QF7c）
- 前提の確認: 6問（Q1a, Q2a, Q3a, QF3b, QF8a, QF9b）

## 分類基準

各質問の戦術（tactic）に基づいてカテゴリを割り当てた：
- 手段の探索・段階的絞り込み・観測事実の深掘り → 状況の確認
- 動機の探索・因果の逆転 → 原因と動機
- 登場人物の探索 → 人物と背景
- 常識的仮説の排除 → 前提の確認

棄却質問は、対応する戦術の棄却版として同様の基準で分類した。
