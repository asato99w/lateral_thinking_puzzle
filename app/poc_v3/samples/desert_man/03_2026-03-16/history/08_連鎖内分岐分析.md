# Phase 8: 連鎖内分岐分析

## 概要

各戦術連鎖のステップにおいて、推論者が「別の方向に進んだ場合」に生じる分岐仮説（HF*）とその否定命題（NF*）を洗い出す。これにより偽モデルの棄却を支援する「いいえ」質問（QF*）を体系的に生成する。

## 分析方針

- 各連鎖の各ステップ（中間命題 I* および到達命題 P*）について、「もしここで別の結論に至ったら？」を考える
- 既存の「いいえ」質問（Q*a 系）で既にカバーされている分岐は参照のみ
- 新規の分岐仮説は NF*/HF* ペアとして生成し、QF* 質問を割り当てる

## 連鎖別分岐分析

### C1: {S3,S5} → I1 → P1

**ステップ1: I1（歩いて来たのではない）**
- 既存カバー: Q1a「歩いて来ましたか？」→ いいえ → I1 確認済み
- 分岐なし（既存でカバー）

**ステップ2: P1（乗り物に乗っていた）**
- 分岐b: 「他者に連れてこられた（乗り物ではなく強制連行）」→ HF1b
  - NF1b: 「男は他者に砂漠に連れてこられたのではない」
  - 棄却対象: M3, M13
- 分岐c: 「精神的な問題で砂漠に来た」→ HF1c
  - NF1c: 「男は精神的な問題で砂漠に来たのではない」
  - 棄却対象: M9

### C2: {S5,S3} → I3 → P7

**ステップ1: I3（脱水ではない）**
- 既存カバー: Q2a「脱水や遭難で死にましたか？」→ いいえ → I3 確認済み
- 分岐なし（既存でカバー）

**ステップ2: P7（切迫した危険）**
- 分岐b: 「緩やかな衰弱で死んだ」→ HF2b
  - NF2b: 「男は緩やかな衰弱で死んだのではない」
  - 棄却対象: M5（サバイバル賭けで衰弱死）

### C3: {S4} → I4 → P4

**ステップ1: I4（元々裸ではなく服を失った）**
- 既存カバー: Q3a「元々服を着ていましたか？」→ はい → I4 確認済み
- 分岐なし（既存でカバー）

**ステップ2: P4（自発的脱衣）**

  - NF3b: 「外力や事故で服を失ったのではない」
  - 棄却対象: G1, M11, M14
- 分岐c: 「他者に服を脱がされた」→ HF3c
  - NF3c: 「他者に服を脱がされたのではない」
  - 棄却対象: M3, M13

### C4: {S5,S4,S3} → I6 → P3

**ステップ1: I6（他者の関与）**
- 既存カバー: Q4a「男一人で起きたことですか？」→ いいえ → I6 確認済み
- 分岐なし（既存でカバー）

**ステップ2: P3（他者がいた）**
- 分岐b: 「近くの人物は敵対的だった」→ HF4b
  - NF4b: 「男の近くの人物は敵対的ではなかった」
  - 棄却対象: M3, M13

### C5: {P1,S3} → I8 → P2

**ステップ1: I8（地上ではない）**
- 既存カバー: Q5a「地上の乗り物に乗っていましたか？」→ いいえ → I8 確認済み
- 分岐なし（既存でカバー）

**ステップ2: P2（空中）**
- P2 は I8 から直接導出。P2 の分岐は C6 のソースとして扱う。
- 追加分岐なし

### C6: {P2,P7} → I9 → P6

**ステップ1: I9（空中から落下）**
- I9 は P2+P7 の直接的帰結。追加分岐なし。

**ステップ2: P6（落下死）**
- 分岐a: 「乗り物は衝突・墜落した（乗ったまま）」→ HF6a
  - NF6a: 「乗り物は衝突・墜落したのではない」
  - 棄却対象: G1, M14

### C7: {P4} → I10 → P5

**ステップ1: I10（実用的理由）**
- 分岐a: 「儀式的・宗教的理由」→ HF7a
  - NF7a: 「脱衣は儀式的・宗教的な理由ではない」
  - 棄却対象: M6
- 分岐b: 「ルールや条件による」→ HF7b
  - NF7b: 「脱衣はルールや条件によるものではない」
  - 棄却対象: M5
- 分岐c: 「脱獄のため」→ HF7c
  - NF7c: 「男は脱獄を試みたのではない」
  - 棄却対象: M10

**ステップ2: P5（物理的問題）**
- 分岐a: 「暑さのため」→ HF8a
  - NF8a: 「服を脱いだのは暑さのためではない」
  - 棄却対象: G2

### C8: {P5,P7} → I11 → P9

- I11, P9 は P5+P7 の統合。分岐は C7 と C2 で既にカバー。
- 追加分岐なし

### C9: {P3,S6,S7,S8} → I12 → P8

**ステップ1: I12（他者から渡された）**
- 分岐a: 「マッチ棒は男の個人的な持ち物だった」→ HF9a
  - NF9a: 「マッチ棒は男の個人的な持ち物ではない」
  - 棄却対象: G1, G2, M11, M14
- 分岐b: 「マッチ棒は偶然手に入った」→ HF9b
  - NF9b: 「マッチ棒は偶然手に入ったのではない」
  - 棄却対象: M3

### C10: {P6,P8,P9} → I13 → FP

- I13, FP は最終統合。分岐は上流で全てカバー済み。
- 追加分岐なし
