# Phase 1: 緊張構造分析とモデル列挙

## 緊張構造の特定

S: 砂漠の真ん中で裸の男が死んでいた。手には折れたマッチ棒を握りしめている。

### 表層の緊張

- **α**: 砂漠の真ん中で死んでいること
- **β**: 砂漠で裸であること
- **γ**: 折れたマッチ棒を握りしめていること

### 分解の適用

**α → α1 + α2（程度の分離）**:
- α1: 砂漠の真ん中という到達困難な場所にいる（到達手段・理由が不明）
- α2: 死亡している（死因が不明）
- 独立検証: α1のみ緩和するモデル（到達手段は判明するが死因不明）、α2のみ緩和するモデル（死因は判明するが到達方法不明）がそれぞれ想像可能 → ✓

**β → 分解なし**:
- 程度の分離: S が「裸」と明言しており部分的脱衣は S と矛盾
- 前提の抽出: 脱衣の動機を説明すれば裸の状態も説明される（独立検証不通過）

**γ → γ1 + γ2（前提の抽出）**:
- γ1: マッチ棒が折れている（通常の使用法＝火をつけるとは異なる状態）
- γ2: マッチ棒を握りしめている（死の直前まで手放さなかった重要性を示唆）
- 独立検証: γ1のみ緩和（折れた理由は判明するが握る理由不明）、γ2のみ緩和（握る理由は判明するが折れた理由不明）→ ✓

## T の演繹連鎖

- α1 ← 熱気球で空から砂漠上空に到達し、飛び降りた地点が砂漠の真ん中
- α2 ← 飛び降りによる落下死
- β ← 高度低下の緊急事態で軽量化のため服を脱いだ
- γ1 ← くじ引き（マッチの長さ）で一人が飛び降りることに → マッチを折って長さを比べた
- γ2 ← 短いマッチを引いた男 → 自分の運命を決めたマッチを握りしめたまま死亡

## モデル列挙

13モデルを列挙（T + M1〜M12）。全緊張を緩和するモデル12、部分緩和1。

### 緩和パターンの主要な軸

**γ の緩和パターンによる大分類**:
- くじ引き型（γ1=長さ比較で折った、γ2=運命の象徴）: T, M1, M5, M7, M8, M9
- 火起こし型（γ1=火をつけようとして折れた、γ2=最後の希望）: M2, M3, M6, M10, M11
- 象徴型（γ1=象徴的に折った、γ2=精神的支柱）: M4
- 未緩和: M12

### バリアント識別

M2（犯罪組織による放置）と M10（軍事捕虜の放置）: 全緊張で緩和メカニズムが同一（他者による強制的連行・脱衣 + 火起こし失敗）→ バリアント群 G1

### 抽象モデル

12 の抽象モデルを導出（11 単独 + 1 バリアント群）。Phase 2 への入力として使用する。
