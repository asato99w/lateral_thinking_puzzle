## Step 3: 導出関係の分析

### 導出グラフ

```
P6 ──→ P1 ←── P2
P5 ──→ P4 ←── P3
```

- P2 → P1: 空中にいた → 乗り物に乗っていた
- P6 → P1: 重量削減 → 乗り物に乗っていた
- P3 → P4: くじ引き → グループにいた
- P5 → P4: 命を落とす決定 → グループにいた

### 独立軸と根命題

- **独立軸**: P2, P3, P5, P6（他の分割命題から導出されない）
- **根命題**: P1, P4（最も抽象的、S から直接到達可能な候補）

### fc の仮置き

- P2.fc = [[P1]]（乗り物 → 空中を仮説）
- P3.fc = [[P4]]（グループ → くじ引きを仮説）
- P5.fc = [[P4]]（グループ → 命の決定を仮説）
- P6.fc = [[P1]]（乗り物 → 重量削減を仮説）
- P1, P4: Phase 3 で S からの戦術連鎖で確定
