### Step 2d: 意味の確定性チェック

全命題について全モデルで真偽が一意に定まることを確認。

- **P1**: G3 は「準備は完了した」が「意図した問題（試験合格等）に対して有効に機能した」とは言えない（目標が達成されていない）→ false ✓
- **P2**: G2 の密閉は「密閉の過剰」ではなく「密閉空間に汚染源がある」という別の因果 → false ✓
- **P3**: 明確な事実命題。全モデルで一意 ✓
- **P4**: 「感覚刺激を直接遮断」は G1（音/光の遮断）のみ。G2 は物理的因子の排除であり感覚遮断ではない ✓
- **P5**: 「時間的な損失」= タイミングを逃したことによる絶望。G2 は健康被害、M7 は物的損壊 → false ✓
- **P6**: M4（不眠）は生理的問題であり「環境的要因」ではない → false ✓

## 導入時検証（1次チェック）

### 判例照合

**P1**: 「有効に機能した」は対策の直接目的の達成を問う。最終結果の成否（絶望したかどうか）とは区別されている。手段と目的の混同なし。✓

**P2**: 「意図した以上に効果を発揮した」は量的過剰（M4:薬効過多, M7:加湿過多, M13:冷却過多）と範囲的過剰（G1:遮断範囲超過）の両方を含む。G2 は密閉自体が過剰なのではなく、密閉環境に汚染源があるという別の因果。M2 は鍵の故障であり施錠の「過剰」ではない。→ 真偽の揺れなし ✓

**P3**: 純粋な事実命題。T では寝坊 → true。判例の「日常的意味と厳密な意味の乖離」なし。✓

**P4**: 「感覚刺激を直接遮断する」は、耳栓（聴覚）や遮光カーテン（視覚）など五感への物理的入力を阻止するデバイスを指す。M13（エアコン）は温度を変えるが「感覚刺激の遮断」ではなく「環境の変更」。→ 真偽の揺れなし ✓

**P5**: 「時間的な損失」は明確。寝坊=時間の喪失。M2の閉じ込め=行動可能な時間の喪失。G2の健康被害は時間的損失ではなく身体的損害。→ 真偽の揺れなし ✓

**P6**: 「睡眠を妨げる環境的要因」: 騒音・寒さ・虫・花粉・乾燥・暑さは環境的。不眠（M4）は生理的。防犯不安（M2）は心理的脅威であり環境的要因とは異なる。→ 真偽の揺れなし ✓
