# Phase 4: S 細分化と戦術連鎖

## Step 1: S 細分化

### 原子命題
- S1: ある男がいた
- S2: 男は対策をした
- S3: 男は対策を万全だと評価した
- S4: 男は対策に満足した
- S5: 男は安心して眠りについた
- S6: 翌朝が来た
- S7: 男は絶望した

### 導出命題
- S8: 対策は眠りにつく前に完了した（S2, S5 → 時間的順序）
- S9: 対策と絶望の間に一晩が経過した（S5, S6 → 眠り→翌朝）

### initial_confirmed
[S1, S2, S3, S4, S5, S6, S7, S8, S9]

## Step 2-3: 戦術連鎖

### S → root 連鎖

**C1: S → P1（対策は有効に機能した）**
S2+S3 →[動機の探索]→ I1(男は解決したい問題を抱えていた) →[段階的絞り込み]→ P1
- I1: 「万全の対策」(S2+S3) → なぜ対策をしたか？ → 問題があった

**C2: S → P5（時間的損失）**
S7 →[因果の逆転]→ I2(男の絶望には具体的な原因があった) →[段階的絞り込み]→ P5
- I2: 絶望(S7) → なぜ絶望した？ → 具体的な原因

**C3: S → P6（睡眠環境的要因）**
S2+S5 →[観測事実の深掘り]→ I3(対策は男が快適に眠るために講じたものだった) →[段階的絞り込み]→ P6
- I3: 対策(S2)+安心して眠り(S5) → 何のための対策？ → 快適な眠りのため

**C4: S → P4（感覚刺激の遮断）**
S2+S5 →[段階的絞り込み]→ I5(対策は外部からの何かを遮断・排除するものだった) →[段階的絞り込み]→ P4
- I5: 対策(S2)+眠り(S5) → どんな種類の対策？ → 外部を遮断/排除

### toward_concrete 連鎖

**C5: P1 → P2（過剰効果→問題）**
P1+S7 →[矛盾の指摘]→ I6(対策は成功したにもかかわらず問題が生じた) →[段階的絞り込み]→ P2
- I6: 有効に機能した(P1)なのに絶望(S7) → 矛盾 → 成功と問題の共存

**C6: P5 → P3（起きられなかった）**
P5+S6 →[因果の逆転]→ I7(時間の損失は翌朝の予定に支障をきたしたことによる) →[段階的絞り込み]→ P3
- I7: 時間的損失(P5)+翌朝(S6) → なぜ時間を失った？ → 予定に支障

**C7: P4+P6 → P7（感覚刺激遮断の対策）**
P4+P6 →[観測事実の深掘り]→ I8(男は就寝時に感覚的に不快な環境に置かれていた) →[手段の探索]→ P7
- I8: 感覚遮断(P4)+環境的要因(P6) → どんな環境？ → 感覚的に不快

**C8: P2+P3 → P8（過剰効果で起きられず）**
P2+P3 →[因果の逆転]→ I9(対策の効果が男の覚醒を妨げた) →[直接同定]→ P8
- I9: 過剰効果(P2)+起きられなかった(P3) → 因果の接続

**C9: P7+P8 → FP**
P7+P8 →[矛盾の指摘]→ I10(対策が遮断した感覚刺激の中に覚醒に必要な信号が含まれていた) →[直接同定]→ FP
- I10: 感覚遮断(P7)+覚醒妨害(P8) → 遮断対象に覚醒信号が含まれていた
