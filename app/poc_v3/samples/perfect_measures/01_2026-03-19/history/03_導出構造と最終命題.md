# Phase 3: 導出構造と最終命題

## Step 1: T から分割命題への演繹構造

T → 各分割命題への直接演繹を確認後、命題間の演繹関係を特定:

- **P2 → P1**: 「意図以上に効果を発揮した」→「有効に機能した」（過剰は機能を含意）
- **P3 → P5**: 「起きられなかった」→「時間的損失」（寝坊は時間的損失の具象形）

冗長辺の除去:
- T → P1 を除去（T → P2 → P1 で到達可能）
- T → P5 を除去（T → P3 → P5 で到達可能）

T直接ノード: P2, P3, P4, P6

## Step 2-3: 共通具象元の発見と検証

### P7: P4 と P6 の共通具象元

T の事実「騒音（睡眠を妨げる感覚刺激）を遮断するために耳栓を使った」は P4 と P6 の両方に跨る。

**P7: 男は睡眠を妨げる感覚刺激を遮断するための対策を講じた**
- P7 → P4: 「睡眠を妨げる」を落とす → 「感覚刺激を遮断する目的」
- P7 → P6: 「感覚刺激を遮断する」を落とし「感覚刺激」→「環境的要因」に抽象化 → 「睡眠を妨げる環境的要因」

検証:
- 演繹的分岐: P4 と P6 を同時に演繹 ✓
- モデル非自明分割: true_models = {G1}（G1 のみだが {T} 単独ではない。M3 含む）✓
- 意味確定性: 「睡眠を妨げる感覚刺激を遮断する対策」は一意に確定 ✓

### P8: P2 と P3 の共通具象元

T の事実「耳栓が目覚ましも遮断（過剰効果）→寝坊（起きられなかった）」は P2 と P3 の両方に跨る。

**P8: 対策の過剰な効果により男は翌朝起きられなかった**
- P8 → P2: 「起きられなかった」を落とし抽象化 → 「過剰効果が問題を引き起こした」
- P8 → P3: 「対策の過剰な効果」（原因）を落とす → 「起きられなかった」

検証:
- 演繹的分岐: P2 と P3 を同時に演繹 ✓
- モデル非自明分割: true_models = {G1, M4}（P3 と同一だが演繹的分岐があるので許容）✓
- 意味確定性: 一意に確定 ✓

## Step 4: 再帰的適用

P7 と P8 を追加後、T直接ノードは P7 と P8 のみ（P2,P3,P4,P6 は P7 or P8 経由で到達）。

P7 と P8 間の共通具象元の探索:
- P7 は対策の性質（何をしたか）、P8 は失敗の結果（何が起きたか）。独立した次元であり、両方を統合するとほぼ FP そのものになる。→ 追加の中間命題なし。

## Step 5: 抽象化による発見

T直接ノード: P7, P8

- P7 → 抽象化 → 「対策の性質に関する知識」
- P8 → 抽象化 → 「失敗の因果メカニズムに関する知識」

これらは根本的に異なる次元（what vs why）であり、共通構造は FP 自体以外に見出せない。→ 新規命題なし。

## Step 6: 最終命題（FP）

**FP: 対策は睡眠を妨げる感覚刺激の遮断であり、遮断効果が覚醒信号にも及んで翌朝起きられなかった**

- FP → P7: 失敗部分を落とす → 「睡眠を妨げる感覚刺激を遮断する対策」
- FP → P8: 対策の性質を抽象化 → 「対策の過剰効果で起きられなかった」
- true_models = {G1}

## Step 7: クリア適切性検証

| 緊張 | 解消メカニズム | 解消 |
|---|---|---|
| α1 | FP: 遮断効果が覚醒信号にも及んだ | ✓ |
| α2 | P3+P5: 起きられず時間的損失 | ✓ |
| β1 | FP: 覚醒信号は翌朝まで発生しない | ✓ |
| β2 | FP: 覚醒信号は翌朝の事象 | ✓ |
| γ1 | P6+P4: 睡眠を妨げる環境的感覚刺激 | ✓ |
| γ2 | P7: 感覚刺激を遮断する対策 | ✓ |

全緊張解消 → clear_conditions = [["FP"]]

    │        └─→ P6
