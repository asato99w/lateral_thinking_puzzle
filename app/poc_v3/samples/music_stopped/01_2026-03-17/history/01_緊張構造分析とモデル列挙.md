# Phase 1: 緊張構造分析とモデル列挙

## 緊張構造分析

### Step 1: S の緊張構造の特定

S: 「音楽が止まった。彼女は死んだ。」

2文の並置が因果関係を暗示する。常識的には音楽の停止が人の死を引き起こすことはない。

**α（親）: 音楽の停止と死の関連**
- 常識的説明(M0): 音楽は娯楽・芸術であり、停止が致命的結果に至ることはない
- M0の前提: 音楽は背景的・装飾的なもの、生命維持に無関係
- M0では説明不能: なぜ音楽の停止が彼女の死に直結したのか

**β（親）: 彼女の状況の非自明さ**
- 常識的説明(M0): 彼女は普通の人で、日常的な場面にいた
- M0の前提: 通常の環境、通常の身体状態
- M0では説明不能: 音楽の停止が致命的になるような特殊な状況が隠されている

### Step 1.5: 緊張の分解

**α → α1, α2**
- α1: 音楽が通常の娯楽とは異なる特殊な機能を果たしていた
- α2: 音楽の停止から死に至る因果メカニズムが存在する

独立検証:
- α1を緩和しα2を緩和しない: 「音楽は位置ガイドだった」→死への経路は不明 → 可能
- α2を緩和しα1を緩和しない: 「高所から転落した」→音楽との関係は不明 → 可能
→ 分解成立 ✓

**β → β1, β2**
- β1: 彼女が生命の危機を伴う活動・状況にいた
- β2: 彼女が通常より脆弱・制限された状態にあった

独立検証:
- β1を緩和しβ2を緩和しない: 「綱渡り中だった」→目隠しなしなら音楽なしでも視覚で対応可能 → 可能
- β2を緩和しβ1を緩和しない: 「目隠しをしていた」→安全な場所なら死なない → 可能
→ 分解成立 ✓

### Step 2: T の演繹連鎖

T: 彼女は目隠しをした綱渡り師で、音楽が演技の位置ガイドだった。音楽が止まったことで位置を見失い、綱から落ちて死亡した。

1. 彼女は綱渡り師(T1) + 目隠し(T2) → 視覚なしで綱渡り → 音楽に依存する必然性
2. 音楽=位置ガイド(T3) + 音楽停止(S) → 位置を見失う(T4)
3. 位置喪失(T4) + 高所(T1の含意) → 転落(T5)
4. 転落(T5) + 高所 → 死亡(T6)

緊張の解消:
- α1 → T3 (音楽=位置ガイド)
- α2 → T3→T4→T5→T6 (停止→位置喪失→転落→死)
- β1 → T1 (綱渡り=高所の危険なパフォーマンス)
- β2 → T2 (目隠し=視覚制限)

### Step 3: T の命題群

- T1: 彼女は綱渡り師だった（真）
- T2: 彼女は目隠しをしていた（真）
- T3: 音楽が演技の位置ガイドだった（真）
- T4: 音楽が止まったことで位置を見失った（真）
- T5: 綱から落ちた（真）
- T6: 彼女は死亡した（真）

## モデル列挙

### 列挙結果

19個のモデル（T含む）を列挙。全緊張を緩和するもの18個、部分緩和1個（M18: α1, α2のみ）。

### 緩和パターンの分析

主な緩和メカニズムの類型:
- 空間ガイド型: T, M14（空間情報の提供→喪失→環境危険→死）
- 時間ガイド型: M1, M5（タイミング情報の提供→喪失→事故死）
- 動物制御型: M2, M13（音楽による動物制御→制御喪失→襲撃死）
- 信号型: M3, M9（音が生命状態の信号・装置音）
- ゲーム/信号型: M4, M8, M17（行動トリガー・抑止信号・タイマー）
- 因果逆転型: M6, M10（死が原因で音楽が停止）
- 意識維持型: M7, M16（催眠・睡眠の維持→解除→事故）
- 比喩型: M11, M15（「音楽」「死」が比喩的用法）
- 高度ガイド型: M12（空間情報だが離散的判断に使用）
- 偶然型: M18（共通原因による同時発生）

### バリアント識別

**G1: {T, M14}** — 空間ナビゲーションガイド + 視覚制限 + 環境危険への遭遇
- 共通メカニズム: 音楽が空間情報を提供し、喪失で環境の物理的危険に遭遇して死亡
- T: 綱渡り→転落、M14: 地雷原→爆死（舞台設定の違いのみ）

**G2: {M1, M5}** — 時間タイミングガイド + 視覚制限 + パフォーマンス中事故
- 共通メカニズム: 音楽が時間タイミングを提供し、喪失で致命的タイミングミス
- M1: 空中ブランコ→落下、M5: 水中→溺死（舞台設定の違いのみ）

### 抽象モデル

17個の抽象モデルを導出（2バリアント群 + 15単独モデル）。
