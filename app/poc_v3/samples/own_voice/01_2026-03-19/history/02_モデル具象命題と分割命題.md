# Phase 2: モデル具象命題と分割命題

## Step 1: 各モデルの具象命題

**T:**
1. 女はストーカーに付きまとわれていた
2. ストーカーは繰り返し無言電話をかけていた
3. 女は無言電話に悩まされていた
4. ある夜、ストーカーは女の部屋に侵入しベッドの下に潜んだ
5. ストーカーはベッドの下から女に無言電話をかけた
6. 女が電話に出て声を発した
7. 女の声がストーカーの電話の背景音としてリアルタイムで聞こえた
8. 自分の声が電話越しに聞こえたことで、ストーカーが至近距離にいると気づいた
9. 差し迫った身体的脅威を認識し恐怖に震えた

**G1 (環境型秘密監視デバイス):**
1. 何者かが女の生活空間に盗聴/監視デバイスを設置した
2. デバイスが女の声をリアルタイムで外部に中継/記録していた
3. 女は偶然、デバイス経由の自分の声の出力を聞いた
4. 秘密裏に監視されていたことに気づいた
5. プライバシー・安全の侵害に恐怖した

**M2 (AI合成音声):**
1. 何者かが女の声のサンプルを取得した
2. AIを使って女の声を合成した
3. 女は合成された自分の声を聞いた（自分が言っていない内容）
4. 声の無断複製・悪用に気づいた
5. なりすまし犯罪への恐怖を感じた

**M3 (超自然):**
1. 女は物理的に声が存在しえない状況にいた
2. 自分の声が物理法則に反して聞こえた
3. 超自然的現象だと認識した
4. 超自然的恐怖を感じた

**G2 (録音の悪用):**
1. 何者かが過去に女の声を密かに録音した
2. 録音された声を女に対して悪意ある目的で再生した
3. 女は自分の録音声を聞いた
4. 声が悪用されていることに気づいた
5. 録音者の悪意と意図に恐怖した

**M5 (音響異常/侵入者):**
1. 女は一人のはずの部屋にいた
2. 声を出したところ通常と異なる反響パターンがあった
3. 音響異常から室内に誰かが隠れていると推定した
4. 侵入者の存在に恐怖した

**M6 (通信傍受):**
1. 女は通話をしていた
2. 通話中に自分の声の異常な遅延エコーが聞こえた
3. 通話が第三者に傍受されていると推定した
4. 通信セキュリティ侵害に恐怖した

**M7 (記憶にない録音):**
1. 女は録音デバイスに記憶にない自分の声のメッセージを発見した
2. メッセージの内容は自分が残した覚えのないものだった
3. 記憶にない自己行動の証拠に直面した
4. 意識障害・記憶喪失の可能性に恐怖した

**M8 (声質変化/病気):**
1. 女が声を発した
2. 自分の声が通常と劇的に異なって聞こえた
3. 声質変化が重篤な病気の兆候だと気づいた
4. 病気への恐怖を感じた

**M10 (音漏れ):**
1. 壁や換気口を通じて女の声が隣接空間に漏れていた
2. 女は自分の声が他者に筒抜けだったことに気づいた
3. プライベートな会話内容が漏洩していた可能性に気づいた
4. プライバシー喪失に恐怖した

**M11 (模倣者):**
1. 近くに女の声を完璧に模倣する人物がいた
2. 女は自分の声が聞こえたが、実は模倣者の声だった
3. 模倣者の存在に気づいた
4. 模倣者の目的や意図に恐怖した

**M12 (意図しない発声):**
1. 女が意図せず声を発していた
2. 自分の意図しない発声を自覚した
3. 自己の身体コントロール喪失に気づいた
4. 精神疾患の兆候に恐怖した

## Step 2c: 抽象命題の生成

**候補1: 「恐怖の原因は女自身の内部ではなく外部に存在する」**
- true_set: {T, G1, M2, M3, G2, M5, M6, M10, M11}
- false_set: {M7, M8, M12}
- 分割: 外的脅威 vs 内的問題

**候補2: 「女に対して意図的に害を与えようとする第三者が存在する」**
- true_set: {T, G1, M2, G2, M6, M11}
- false_set: {M3, M5, M7, M8, M10, M12}
- 分割: 悪意ある第三者 vs その他

**候補3: 「女の声が通信デバイスを介して聞こえた」**
- true_set: {T, M6}
- false_set: {G1, M2, M3, G2, M5, M7, M8, M10, M11, M12}
- 分割: 通信チャネル利用 vs 非通信

**候補4: 「脅威の原因となる存在が女と同じ部屋にいた」**
- true_set: {T, M5}
- false_set: {G1, M2, M3, G2, M6, M7, M8, M10, M11, M12}
- 分割: 至近距離の脅威 vs 非至近距離

- 分割: 技術媒介 vs 非技術

## Step 2d: 意味の確定性チェック

**P1「恐怖の原因は女自身の内部ではなく外部に存在する」**
- 全モデルで真偽一意: M7(意識障害=内部), M8(病気=内部), M12(精神疾患=内部) → 明確に False
- M3(超自然): 外部の超自然的現象 → True
- M10(音漏れ): 環境の問題 → True
- ✅ 意味確定

**P2「女に対して意図的に害を与えようとする第三者が存在する」**
- M5(侵入者): 侵入は犯罪だが女を「意図的に害を与えようとしている」かは不明。侵入自体は女を直接狙ったものとは限らない → False
- M3(超自然): 第三者は存在しない → False
- M10(音漏れ): 意図的行為ではない → False
- ✅ 意味確定

**P3「女の声が通信デバイスを介して聞こえた」**
- G1(盗聴器): 盗聴器は通信デバイスか？ 盗聴器は音声の収集デバイスだが、女が聞く経路としては「盗聴器の出力」を偶然聞いた。女の声が「通信デバイスを介して聞こえた」ではなく、盗聴器が通信の中継をしている。→ 厳密には True に見えるが、女にとっての「聞こえ方」として通信デバイスを意識したかは別問題
- → G1 について再検討: P3 の「通信デバイスを介して聞こえた」は「女が通信デバイス（電話等）を使用中に聞こえた」と解釈すべき。G1 では女は通信デバイスを使用していない → False
- ✅ 意味確定

**P4「脅威の原因となる存在が女と同じ部屋にいた」**
- M11(模倣者): 模倣者は「近くにいる」が同じ部屋かは不明 → False（同じ部屋であることは要求されていない）
- ✅ 意味確定

**P5「女の声は技術的手段（電子機器等）を通じて聞こえた」**
- M7: 録音デバイスの再生 = 技術的手段 → True
- M10: 壁越しの物理的音漏れ = 技術ではなく物理的伝搬 → False
- M11: 模倣者の肉声 = 技術ではない → False
- ✅ 意味確定
