# Phase 4: S 細分化と戦術連鎖

## S 細分化

S: 「女は自らの声を聞き、恐怖に震えた。」

### 原子命題
- S1: 女がいた
- S2: 女は声を聞いた
- S3: 聞いた声は女自身の声だった
- S4: 女は恐怖に震えた

### 導出命題
- S5: 女は恐怖を感じた（S4 から含意。「震えた」は恐怖の身体反応）
- S6: 声を聞いたことが恐怖に時間的に先行した（S2, S4 の文構造「聞き、震えた」から）

### initial_confirmed
[S1, S2, S3, S4, S5, S6]

## 戦術連鎖

### C1: S → P1 (S_to_root)
根命題 P1「恐怖の原因は外部に存在する」への連鎖

{S2, S3, S4} →[矛盾の指摘]→ I1(声の聞こえ方が通常と異なっていた)
{I1} →[段階的絞り込み]→ P1(恐怖の原因は女自身の内部ではなく外部に存在する)

矛盾: 自分の声は馴染みある音であり恐怖を引き起こさない。恐怖が生じたのは声の聞こえ方が通常と異なっていたため。

I1 の T 検証: 電話越しの反射は通常と異なる聞こえ方 → True ✓

### C2: S → P5 (S_to_root)
根命題 P5「技術的手段で聞こえた」への連鎖

{S2, S3} →[観測事実の深掘り]→ I2(自分の声が別の経路で再生・中継されていた)
{I2} →[段階的絞り込み]→ P5(女の声は技術的手段を通じて聞こえた)

I2 の T 検証: 電話経由で声が中継されていた → True ✓

### C3: P1 → P2 (toward_concrete)
{P1} →[段階的絞り込み]→ I3(外部の脅威は人為的なものである)
{I3} →[段階的絞り込み]→ P2(女に対して意図的に害を与えようとする第三者が存在する)

fc_source: P1 は C1 の reveals

### C4: P1 → P4 (toward_concrete)
{P1} →[観測事実の深掘り]→ I4(脅威源は女に物理的に近い場所にあった)
{I4} →[段階的絞り込み]→ P4(脅威の原因となる存在が女と同じ部屋にいた)

fc_source: P1 は C1 の reveals

I4 の T 検証: ストーカーはベッドの下（至近距離）→ True ✓

### C5: P2+P5 → P6 (toward_concrete)
