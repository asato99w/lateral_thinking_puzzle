# Phase 1: 逆算連鎖

## T

女性は地下室に監禁されていた。「開けてはいけないドア」とは監禁部屋から外に出るドアのこと。ドアを開けたら普通の外の世界が見えた。「何も異常なものはなかった」のは外の世界が普通だから。彼女は自分の監禁を通報するために警察に行った。

## Step 1: T の逆算

1. 女性は警察に行った ← なぜ？
2. 自分の監禁を通報するため ← なぜ監禁を通報する必要があった？
3. 女性は自分が監禁されていたことに気づいた ← なぜ気づいた？
4. ドアを開けたら普通の外の世界が見えた ← なぜそれで気づく？
5. 「開けてはいけないドア」は監禁部屋から外に出るドアだった ← なぜ開けてはいけないと言われていた？
6. 女性は地下室に監禁されていた（前提）

## Step 2: T の直接命題の抽出

| ID | label |
|----|-------|
| D-1 | 女性は地下室に監禁されていた |
| D-2 | 「開けてはいけないドア」とは監禁部屋から外に出るドアのことだった |
| D-3 | ドアを開けたら普通の外の世界が見えた |
| D-4 | 女性は自分が監禁されていたことに気づいた |

## Step 3: 因果充足性の検証と T 導出命題の抽出

### 検証: 「監禁に気づいた ← 普通の外の世界が見えた」(D-4 ← D-3)

- **充足性テスト**: 普通の外の世界を見ただけでは、なぜ「監禁されていた」と気づくのか不十分。女性がそれまで自分の状況を知らなかったことが前提として必要
- **候補**: 女性は自分が監禁されていることを知らなかった

検証:
- **含意テスト**: D-1（監禁されていた）+ D-2（開けるなと言われた）+ T の「驚いた」→ 禁止に従っていた＝状況を知らなかった。外を見て初めて驚いた構造が含意
- **必要性テスト**: 否定すると（知っていた）、ドアを開けて驚く理由がなく、それまで脱出しなかった理由も説明不能 → 必要
- **内部性テスト**: T の世界内の認知的事実 → OK

→ **D-6**: 女性は自分が監禁されていることを知らなかった（導出元: D-1, D-2, D-3）

### 検証: 「驚いた ← 普通の外の世界が見えた」(S の「驚いた」 ← D-3)

- **充足性テスト**: なぜ普通の世界を見て驚くのか？ 女性にとって地下室が「普通」であり外の世界を知らなかったから
- **候補**: 女性は地下室の外の世界を知らなかった

検証:
- **含意テスト**: D-6（監禁を知らなかった）+ D-1（監禁されていた）→ 長期間地下室で過ごし外を知らない → 含意される
- **必要性テスト**: 否定すると（外を知っていた）、外を見て驚く理由がない → 必要
- **内部性テスト**: T の世界内の認知的事実 → OK

→ **D-7**: 女性は地下室の外の世界を知らなかった（導出元: D-1, D-6）

### 検証: その他のリンク

- 「警察に行った ← 監禁を通報」(D-5 ← D-4): 充足 ✓
- 「外の世界が見えた ← ドアが外に通じていた」(D-3 ← D-2): 充足 ✓
- 「ドアが外に通じていた ← 監禁されていた」(D-2 ← D-1): 充足 ✓
