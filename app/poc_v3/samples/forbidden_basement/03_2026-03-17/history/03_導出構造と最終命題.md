# Phase 3: 導出構造と最終命題

## Step 1: T から分割命題への演繹構造

T → P1: 「監禁されていた」→「地下室の中にいた」。監禁の犯罪性と理由を落とし位置のみ残す。
T → P2: 「監禁されていた」→「犯罪の被害者」。監禁の場所と方法を落とし被害者性のみ残す。
T → P3: 「外の世界が見えた」→「地下室内に異常な対象ではなかった」。外界の具体性を落とし否定のみ残す。
T → P4: 「外の世界は普通（恒常的に正常）」→「状態が変わらなかった」。正常性の理由を落とし時間的不変性のみ残す。
T → P5: 「外の世界を視覚的に確認した」→「客観的に検証可能な現象」。対象の具体性を落とし検証可能性のみ残す。

### 命題間の演繹関係の検討

- P1→P2: 「中にいた」から「犯罪被害者」は導けない（自発的に中にいる可能性がある）。不成立。
- P1→P3: 「中にいた」から「驚きが地下室内の異常でない」は導けない（中にいても中のものに驚く可能性がある）。不成立。
- P3→P4: 「異常対象でない」から「状態不変」は導けない（M13は異常な状況だが状態不変）。不成立。
- P3→P5: 「異常対象でない」から「客観的」は導けない（M7は異常対象でないが客観的でない）。不成立。
- P4→P3: 「状態不変」から「異常対象でない」は導けない（M13は状態不変だが異常状況がある）。不成立。

結論: P1〜P5 間に直接的な演繹関係はない。全て T から独立して演繹される。

## Step 2-3: 共通具象元の発見と検証

- P6 → P1: 「監禁」の犯罪性を落とす →「中にいた」✓
- P6 → P2: 「監禁」の場所を落とす →「犯罪の被害者」✓

二重要件:
- 演繹的分岐: P1 と P2 を同時に演繹 ✓
- モデル非自明分割: true_models = {G1}, false_models = {G2, M7-M15}。G1は {T,M1,M2,M3} の複数モデルを含み、非自明。P1 と同じ分割だが許容 ✓

導入時検証:
- T文脈: 「監禁されていた」は T の核心。真。一意に確定 ✓
- 意味確定性: 「監禁」は明確な法的・事実的概念 ✓

### 候補 P7: 「女性がドアを開けた先は普通の外の世界だった」

T の事実「ドアを開けたら普通の外の世界が見えた」は P3, P4, P5 の共通具象元。

- P7 → P4: 「普通の外の世界」は恒常的に正常 →「状態が変わらなかった」✓
- P7 → P5: 「外の世界を見た」は客観的に確認可能 →「客観的に検証可能な現象」✓

二重要件:
- 演繹的分岐: P3, P4, P5 を同時に演繹 ✓
- モデル非自明分割: true_models = {G1}。P1 と同じ分割だが許容 ✓

導入時検証:
- T文脈: 「ドアを開けたら普通の外の世界が見えた」は T に明記。真 ✓
- 意味確定性: 「普通の外の世界」は明確 ✓

- T: 犯罪者による監禁
