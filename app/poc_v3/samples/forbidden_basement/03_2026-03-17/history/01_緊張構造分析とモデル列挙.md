# Phase 1: 緊張構造分析とモデル列挙

## Step 1: S の緊張構造の特定

S = 「絶対に開けてはいけないと言われていた地下室のドアを開けた女性。中に入って見たものに驚き、すぐに警察に駆け込んだ。しかし地下室の中には何も異常なものはなかった。」

### 緊張 α: 驚きと「異常なし」の矛盾

- S: 「中に入って見たものに驚き」→ 驚くべき何かが存在する
- S: 「地下室の中には何も異常なものはなかった」→ 異常なものは存在しない
- M0 の予測: 驚いたのだから何か異常なもの（死体、犯罪の証拠等）があったはず
- M0 の前提不備: 「驚くもの ＝ 異常なもの」という等式が崩れる。驚きの対象が「異常なもの」に分類されない可能性がある

### 緊張 β: 警察通報と「異常なし」の矛盾

- S: 「すぐに警察に駆け込んだ」→ 犯罪等の緊急通報対象がある
- S: 「地下室の中には何も異常なものはなかった」→ 通報すべき異常はない
- M0 の予測: 警察に駆け込むほどの事態（犯罪の発見等）があったはず
- M0 の前提不備: 警察への通報理由が地下室の中の物的異常とは限らない
- M0 では説明不能な点: 何も異常がない場所について警察に何を通報したのか

### 緊張 γ: 禁止の文脈

- S: 「絶対に開けてはいけないと言われていた地下室のドア」
- M0 の予測: 家主が危険物等を理由に禁止した（青ひげ型）
- M0 の前提不備: 禁止の理由が不明。「誰が」「なぜ」言ったかが欠落している
- M0 では説明不能な点: 「絶対に」という強い禁止と、結果として「何も異常なものはなかった」の組み合わせ

## Step 1.5: 緊張の分解

### α → α1, α2

- α1: **驚きの対象の正体** — 何に驚いたのか（因果の欠落）
- α2: **驚きと「異常なし」の両立メカニズム** — 驚くべきものがあるのに「異常なものはない」が成立する仕組み（期待との不一致）

独立検証:
- α1のみ緩和: 驚きの対象はわかる（例: 死体を見た）が、「異常なものはなかった」との矛盾は未解消 → 可能
- α2のみ緩和: 「異常なし」の理由は説明できる（例: 時間差で消えた）が、そもそも何に驚いたかは不明 → 可能
- → 独立 ✓

### β → β1, β2

- β1: **通報の動機** — 何を通報しに行ったのか（因果の欠落）
- β2: **通報と「異常なし」の両立メカニズム** — 通報すべき事態があるのに「異常なものはない」が成立する仕組み

独立検証:
- β1のみ緩和: 通報対象はわかる（例: 犯罪の目撃）が、「異常なし」との矛盾は未解消 → 可能
- β2のみ緩和: 「異常なし」の理由は説明できるが、何を通報したかは不明 → 可能
- → 独立 ✓

### γ → γ1, γ2

- γ1: **禁止の発信者と目的** — 誰がなぜ「開けてはいけない」と言ったのか（情報の不足）
- γ2: **禁止を破った動機** — なぜ女性はドアを開けたのか（因果の欠落）

独立検証:
- γ1のみ緩和: 禁止者と理由はわかるが、女性が開けた動機は不明 → 可能
- γ2のみ緩和: 開けた動機はわかる（好奇心等）が、禁止者は不明 → 可能
- → 独立 ✓

## Step 2: T の演繹連鎖

T = 「女性は地下室に監禁されていた。「開けてはいけないドア」とは監禁部屋から外に出るドアのこと。ドアを開けたら普通の外の世界が見えた。「何も異常なものはなかった」のは外の世界が普通だから。彼女は自分の監禁を通報するために警察に行った。」

### 緊張の解消

- α1 → 女性は長期監禁後に初めて外の世界を目にした。監禁されていた人間にとって普通の外界が驚きの対象となる
- α2 → 外の世界は文字通り何も異常ではない。「地下室の中」という表現は、外から見た地下室を指すのではなく、ドアの向こう（外の世界）が普通であることを意味する
- β1 → 自分が監禁されていたことを通報するために警察に行った（自身の被害通報）
- β2 → 通報対象は地下室の中の物的異常ではなく、自身の被害（監禁の事実）。地下室に異常なものがないことと矛盾しない
- γ1 → 監禁犯が女性に「開けてはいけない」と言った。脱出を防止するための禁止
- γ2 → 脱出を試みてドアを開けた

### 因果の骨格

1. 女性は地下室に監禁されていた
2. 監禁犯が脱出防止のため「開けてはいけない」と言っていた → γ1, γ2 解消
3. 女性が脱出のためドアを開けた → γ2 解消
4. ドアの向こうは普通の外の世界だった → α1, α2 解消
5. 女性は自分の監禁を通報するために警察に行った → β1, β2 解消

## Step 3: T の命題群

- 女性は地下室に監禁されていた（真）
- 監禁犯が「開けてはいけない」と言った（真）
- 「開けてはいけないドア」は監禁部屋から外に出るドアだった（真）
- 女性は脱出のためにドアを開けた（真）
- ドアの向こうは普通の外の世界だった（真）
- 女性は外の世界を長期間見ていなかった（真）
- 女性は自分の監禁を通報するために警察に行った（真）
