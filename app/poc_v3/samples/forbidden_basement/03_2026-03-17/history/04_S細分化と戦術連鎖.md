## Step 2-3: 戦術連鎖

### 導出グラフ（参照）

```
FP(監禁+外の世界)
├── P6(監禁) → P1(中にいた), P2(犯罪被害者)
└── P7(外の世界) → P3(異常対象でない), P4(状態不変), P5(客観的体験)
```

根命題（最も抽象的）: P1, P2, P3, P4, P5
独立軸: P6, P7

### S→root 連鎖

**C1: S → P1（女性はドアを開ける前から地下室の中にいた）**

source: [S4, S10]
{S4, S10} →[矛盾の指摘]→ I1（「開けてはいけない」の理由は地下室の中に異常なものがあるからではなかった）
{I1} →[ラベルと実体の乖離]→ P1（女性はドアを開ける前から地下室の中にいた）

解説: 「開けてはいけない」(S4) と「異常なし」(S10) の矛盾から、禁止理由は中身の異常ではないと判明(I1)。ならば「開けてはいけない」は何を意味するのか →「中に入ること」の禁止ではなく「中から出ること」の禁止だった → 女性は最初から中にいた(P1)。

intermediates: [I1]

**C2: S → P2（女性は犯罪の被害者だった）**

source: [S9]
{S9} →[動機の探索]→ I2（女性は通報すべき被害を受けていた）
{I2} →[段階的絞り込み]→ P2（女性は犯罪の被害者だった）

解説: 警察に「駆け込んだ」(S9) → なぜ？ → 被害を受けていた(I2)。どんな被害？ → 犯罪の被害者(P2)。

intermediates: [I2]

**C3: S → P3（女性が驚いた原因は地下室内に実在する異常な対象ではなかった）**

source: [S8, S10]
{S8, S10} →[矛盾の指摘]→ I3（女性の驚きは異常なものを発見した驚きではなかった）
{I3} →[段階的絞り込み]→ P3（女性が驚いた原因は地下室内に実在する異常な対象ではなかった）

解説: 驚いた(S8) のに異常なし(S10)。矛盾 → 驚きの種類が「異常なものの発見」ではない(I3)。ではどういう驚きか → 地下室内の異常ではない何かに対する驚き(P3)。

intermediates: [I3]

**C4: S → P4（地下室の状態は女性が見た時点と警察が確認した時点で変わらなかった）**

source: [S10]
{S10} →[観測事実の深掘り]→ I4（地下室は終始正常な状態だった）
{I4} →[段階的絞り込み]→ P4（地下室の状態は女性が見た時点と警察が確認した時点で変わらなかった）

解説: 「異常なものはなかった」(S10) → 地下室はいつから正常だったか？ → 終始正常(I4)。具体的には → 女性が見た時も警察確認時も同じ(P4)。

intermediates: [I4]

**C5: S → P5（女性が驚いた体験は客観的に検証可能な現象だった）**

source: [S7, S8]
{S7, S8} →[観測事実の深掘り]→ I5（女性の驚きは実在する何かに基づいていた）
{I5} →[段階的絞り込み]→ P5（女性が驚いた体験は客観的に検証可能な現象だった）

解説: 何かを見て(S7)驚いた(S8) → 驚きの基盤は何か？ → 実在する何か(I5)。その体験は？ → 客観的に検証可能(P5)。

intermediates: [I5]

### toward_concrete 連鎖

**C6: P1 + P2 → P6（女性は地下室に監禁されていた）**

source: [P1, P2]（P1は連鎖C1、P2は連鎖C2のreveals）
{P1, P2} →[段階的絞り込み]→ I6（女性は自らの意思ではなく地下室の中にいた）
{I6} →[段階的絞り込み]→ P6（女性は地下室に監禁されていた）

解説: 中にいた(P1) + 犯罪被害者(P2) → 自分の意思ではない(I6)。自分の意思でないのに中にいた → 監禁(P6)。

intermediates: [I6]

**C7: P3 + P5 → P7（女性がドアを開けた先は普通の外の世界だった）**

source: [P3, P5]（P3は連鎖C3、P5は連鎖C5のreveals）
{P3, P5} →[段階的絞り込み]→ I7（女性が見て驚いたものは実在する正常なものだった）
{I7} →[段階的絞り込み]→ P7（女性がドアを開けた先は普通の外の世界だった）

解説: 異常対象でない(P3) + 客観的(P5) → 実在する正常なもの(I7)。正常な実在物で驚くドアの先 → 普通の外の世界(P7)。

intermediates: [I7]

**C8: P6 + P7 → FP（女性は地下室に監禁されており、ドアを開けて初めて普通の外の世界を目にした）**

source: [P6, P7]（P6は連鎖C6、P7は連鎖C7のreveals）
{P6, P7} →[段階的絞り込み]→ I8（女性は監禁された地下室から自力で外の世界に出た）
{I8} →[直接同定]→ FP

解説: 監禁(P6) + 外の世界(P7) → 監禁から自力で外に出た(I8) → 核心：監禁されており初めて外を見た(FP)。

intermediates: [I8]
