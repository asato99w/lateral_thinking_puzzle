# Phase 4: S 細分化と戦術連鎖

## S 細分化

### 原子命題
- S1: ある男がいた
- S2: 男は死んでいた
- S3: 男は自室にいた
- S4: 男は銃で頭を撃たれた状態だった
- S5: 男のそばにカセットテープがあった
- S6: テープを再生した
- S7: テープに「もう生きていけない…」という内容が録音されていた
- S8: 録音は遺書であった
- S9: 警察が殺人と断定した

### 導出命題
- S10: テープは再生可能な状態にあった (S6)
- S11: 遺書が存在した (S7+S8)

## 戦術連鎖

### C1: S → P3（非遺体分析）
{S9,S4,S8} →[常識的仮説の排除]→ I1 →[段階的絞り込み]→ P3

- I1: 外見上の自殺証拠（銃創・遺書）以外に殺人を示す証拠が存在する
- Player: 銃創+遺書=自殺に見える → 警察が殺人と断定 → 別の証拠がある → どの領域の証拠か → 遺体分析以外

### C2: S → P2（銃創が死因）
{S2,S4} →[観測事実の深掘り]→ I2 →[段階的絞り込み]→ P2

- I2: 男が銃で撃たれた状況が死因に直結している
- Player: 男は銃で撃たれて死んだ → 銃創が本当の死因か → はい

### C3: P3 → P1（テープ関連証拠）
{P3,S5} →[段階的絞り込み]→ I3 →[段階的絞り込み]→ P1

- I3: 殺人の証拠は現場にある物品や媒体の分析に基づく
- Player: 遺体分析以外 → 現場の物品 → テープに絞り込み

### C4: P1 → P4（テープ改変）
{P1} →[観測事実の深掘り]→ I4 →[因果の逆転]→ P4

- I4: テープの分析から殺人の物的証拠が得られた
- Player: テープ関連の証拠 → テープに物的証拠 → 犯人がテープを改変したから

### C5: P3 → P5（物品の仕組み）
{P3} →[動機の探索]→ I5 →[段階的絞り込み]→ P5

- I5: 殺人の証拠は論理的推論によって導かれるものである
- Player: 遺体分析以外 → 証拠の性質は何か → 論理的推論 → 物品の仕組みからの矛盾

### C6: {P4,P5} → FP
{P4,P5} →[矛盾の指摘]→ I6 →{I6,P2}→[因果の逆転]→ FP

- I6: テープの物理的状態にテープの動作原理に矛盾する異常がある
- Player: テープ改変+物品の仕組み矛盾 → テープの動作原理上の矛盾 → 巻き戻しが証拠+銃殺
