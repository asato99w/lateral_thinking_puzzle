## 導入時検証

### 判例照合
- P1: 「カセットテープに関連する」→ テープの状態・内容・時刻を広く含む。曖昧な包含関係なし ✓
- P2: 「死因は銃創」→ 法医学的に明確な概念。手段と目的の混同なし ✓
- P3: 「遺体の法医学的分析以外」→ 分析対象が遺体か否かで明確に分類可能 ✓
- P4: 「テープの内容や物理的状態を改変」→ T では巻き戻し（物理的状態の改変）✓。G3 では内容の偽造 ✓。M7, M8 は改変していない ✓
- P5: 「物品の仕組みから生じる論理的矛盾」→ T はテープの動作原理、M14 は施錠の仕組み ✓

### 導出構造の予測
- P4 → P1 → P3（テープ改変 → テープ関連 → 非遺体分析）
- P5 → P3（物品の仕組み → 非遺体分析）
- P2 は独立（死因の軸）
