# Phase 1: 緊張構造分析とモデル列挙

## 緊張構造分析

### S の事実
- ある男が自室で銃で頭を撃って死んでいた
- そばのカセットテープを再生すると「もう生きていけない…」という遺書が録音されていた
- 警察はこれを殺人と断定した

### 表層緊張

**α: 自殺に見える状況で殺人と断定された**
- 遺書テープ + 自室で銃で頭を撃って死亡 → 自殺のパターン
- にもかかわらず警察は殺人と断定

**β: 殺人を「断定」できるほど決定的な証拠があった**
- 推測や疑いではなく「断定」→ 論理的に自殺を排除する証拠が必要

### 緊張の分解

**α → α1, α2（程度の分離 + 独立検証）**
- α1: 遺書テープが存在するのに自殺ではないとされた（遺書の証拠力が否定された）
- α2: 自室で銃で頭を撃つという自殺パターンに合致する死因・現場状況なのに殺人とされた
- 独立検証: α1のみ緩和（遺書は偽物だが死因分析は自殺と矛盾しない）✓、α2のみ緩和（銃創角度が不自然だが遺書自体は別問題）✓ → 通過

**β → β1, β2（程度の分離 + 独立検証）**
- β1: 現場に自殺では説明できない物理的・論理的矛盾が存在した
- β2: その矛盾は殺人を断定できるほど論理的に決定的であった
- 独立検証: β1のみ緩和（矛盾はあるが断定には至らない）✓、β2のみ緩和（断定的に見えるが実は誤解）✓ → 通過

### T による緩和

| 緊張 | 緩和メカニズム |
|------|--------------|
| α1 | テープの巻き戻し状態が第三者の介在を証明 → 遺書テープは偽装工作の一部 |
| α2 | 第三者の介在が証明 → 自殺状況は偽装 |
| β1 | テープが巻き戻されている = 自殺では起こりえない物理的矛盾 |
| β2 | テープの動作原理上、録音後に自動巻き戻しは不可能 → 論理的に第三者が必須 |
