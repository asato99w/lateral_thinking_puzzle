# Phase 3: 導出構造と最終命題

## Step 1: T から分割命題への演繹構造

T直接演繹:
- T → P4: テープが巻き戻された = テープの物理的状態が改変された
- T → P5: テープの動作原理上の矛盾 = 物品の仕組みからの論理的矛盾
- T → P2: 男は銃で殺害された = 死因は銃創

命題間の演繹:
- P4 → P1: テープの改変 → テープ関連証拠（改変を落とし関連に抽象化）
- P1 → P3: テープ関連 → 非遺体分析（テープを落とし非遺体に抽象化）
- P5 → P3: 物品の仕組み → 非遺体分析（仕組みを落とし非遺体に抽象化）

冗長辺除去: T→P1（via P4）、T→P3（via P4→P1, P5）

```
T
├── P4 → P1 → P3
├── P5 ─────→ P3
└── P2
```

## Step 2-4: 共通具象元

P4∩P5の共通具象元「テープの動作原理上の矛盾として犯人の操作を証明」→ true_models={T}のみ → モデル非自明分割を満たさず不適格。新規中間命題なし。

## Step 5: 抽象化による発見

T直接ノード: P4, P5, P2
- P4×P5: 共通抽象は「現場の物品に関する証拠」だが演繹的分岐を持たない
- P4×P2, P5×P2: 異なる領域で共通構造なし

新規命題なし。

## Step 6: FP

FP: 「テープが巻き戻されて再生可能な状態にあったことが、録音後に第三者がテープを操作した証拠であり、男は銃で殺害された」

- FP → P4: 第三者のテープ操作 → テープの改変
- FP → P5: テープの巻き戻しの動作原理矛盾 → 物品の仕組みからの矛盾
- FP → P2: 銃で殺害 → 死因は銃創
- true_models: {T}

## Step 7: クリア適切性検証

| 緊張 | 解消 |
|------|------|
| α1 | テープの巻き戻し = 第三者介在 → 遺書は偽装 ✓ |
| α2 | 第三者証明 → 自殺状況は偽装 ✓ |
| β1 | テープの巻き戻し = 自殺では起こりえない矛盾 ✓ |
| β2 | 動作原理上の不可能 → 論理的に断定可能 ✓ |

全緊張解消: clear_conditions = [["FP"]]

## Step 9: fc仮置き

- FP.fc = [[P4, P5, P2]]
- P4.fc = [[P1]]
- P5.fc = [[P3]]
- P1.fc = [[P3]]
- P3, P2: 根命題（S命題から形成）

- M14: P4=F で棄却（動作原理矛盾はあるがテープ改変ではない）
