# Phase 12: 分岐点の特定

プレイヤーがゲーム中に蓄積する事実群（S の事実 + 正解連鎖上で獲得した事実）の部分集合のうち、対立仮説方向への推論を駆動しうるものを分岐点として特定する。

## ゲーム進行に沿った事実の蓄積

- 開始時: {D-1, D-2, D-3, D-4, D-5}（S の confirmed）
- 序盤: D-16, D-22 が加わる（D-5, D-3 からの直接的な問い）
- 中盤前半: D-15, D-17, D-18, D-21 が加わる（ターミナル記述素）
- 中盤後半: D-6, D-7, D-8, D-9 が加わる（P-shipwreck 内部）
- 終盤: D-19, D-20, D-10, D-11, D-12, D-13, D-14 が加わる

## 分岐点一覧

### BP-1: 確認行為の動機

事実群: {D-2, D-3} または {D-3} — 開始時点で利用可能

「注文して確認した」という事実の組み合わせから「なぜ確認した？」が自然に生じる。正解では D-3 →[観測事実の深掘り]→ D-22（過去にも飲んだ）に進むが、別の方向に向かいうる。

対立方向:
- → H'_α_1（味への不満）
- → H'_α_2（見た目の違い）
- → H'_α_3（アレルギー）
- → H'_γ_2（好奇心）
- → H'_γ_3（過去の偽物経験）

### BP-2: 自殺の原因

事実群: {D-5} または {D-1, D-5} — 開始時点で利用可能

「帰宅後自殺した」は S の最も異常な要素。「なぜ自殺した？」が直ちに問われる。正解では D-5 →[因果の逆転]→ D-16（過去の危機的体験）に進むが、原因をレストラン内や人物の属性に求めうる。

対立方向:
- → H'_β_1（レストランで衝撃的出来事）
- → H'_β_2（精神的問題）
- → H'_β_3（別の問題）

### BP-3: 「本物」の意味とウミガメの特殊性

事実群: {D-3, D-4} または {D-2, D-4} または {D-2, D-3, D-4} — 開始時点で利用可能

確認行為と「間違いなく本物」の回答の組み合わせ。正解では「本物であること」が後に「過去のスープが偽物だった証拠」に反転するが、この時点では「本物であること自体」に意味を見出す方向がありうる。

対立方向:
- → H'_γ_1（偽物のスープ）: 店員が嘘 → 実は偽物
- → H'_δ_1（絶滅危惧種）: 本物のウミガメ → 種の特殊性
- → H'_δ_2（宗教的禁忌）: 本物のウミガメ → 信仰上の禁忌
- → H'_δ_3（約束違反）: 本物のウミガメ → 食べない約束を破った

### BP-4: 漂着後の展開

事実群: {D-7} または {D-6, D-7} または {D-7, D-17} — D-7 確認後に利用可能

漂着が判明した後、「漂着先で何が起きたか」が自然に問われる。正解では D-7 →[観測事実の深掘り]→ D-8（食糧枯渇）に進むが、別の展開を想定しうる。{D-7, D-17} の組み合わせは仲間の関与する方向を示す。

対立方向:
- → H'_ε_1（すぐ救助）
- → H'_ε_2（人間関係トラブル）: {D-7, D-17} の組み合わせから
- → H'_ε_3（遭難中の事故）

## まとめ

| 分岐点 | 事実群 | 利用可能時期 | 対立方向 |
|--------|--------|-------------|---------|
| BP-1 | {D-2, D-3} 等 | 開始時 | H'_α_1, H'_α_2, H'_α_3, H'_γ_2, H'_γ_3 |
| BP-2 | {D-5} 等 | 開始時 | H'_β_1, H'_β_2, H'_β_3 |
| BP-3 | {D-3, D-4} 等 | 開始時 | H'_γ_1, H'_δ_1, H'_δ_2, H'_δ_3 |
| BP-4 | {D-7} 等 | 中盤 | H'_ε_1, H'_ε_2, H'_ε_3 |

全 15 対立仮説が 4 つの分岐点のいずれかに紐づく。BP-1〜BP-3 は開始時点から利用可能で、ゲーム序盤にプレイヤーが持ちやすい仮説に対応する。BP-4 は正解連鎖の進行に伴い中盤で出現する。
