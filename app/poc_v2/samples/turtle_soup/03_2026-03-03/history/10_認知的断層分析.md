# Phase 10: 認知的断層分析

## 入力

- S: "ある男がレストランに入り、「ウミガメのスープ」を注文した。一口食べた男は店員を呼び、「これは本当にウミガメのスープですか？」と尋ねた。店員は「はい、間違いなくウミガメのスープです」と答えた。男は会計を済ませ、帰宅後、自殺した。"
- T: "男はかつて乗っていた船が遭難し、仲間と無人島に漂着した。食糧が尽き、仲間が一人死んだ。残った者たちはその肉でスープを作り、男には「ウミガメのスープだ」と偽って食べさせた。レストランで本物のウミガメのスープを飲んだ男は味が記憶と全く違うことに気づき、あの時のスープが仲間の人肉だったと悟り、絶望して自殺した。"

## Step 1: 認知的断層の特定

S と T を突き合わせ、以下の 5 つの認知的断層を特定した。

### 断層 α: 確認行為の意味の転換

- **S の誘導**: 「本当にウミガメのスープか？」→ 品質への不満、注文と違うものが出た
- **T の実際**: 過去の記憶との味の比較。本物であることが過去の偽装を証明するトリガー
- **乖離の種類**: 因果の錯誤（確認行為の動機が異なる）

### 断層 β: 過去の体験の存在

- **S の誘導**: S には過去の情報が一切なく、レストランでの出来事で完結する印象
- **T の実際**: 真の因果は S の時間軸の外（過去の遭難体験）にある
- **乖離の種類**: 時間軸の隠蔽

### 断層 ε: 生存の極限

- **S の誘導**: 過去の体験があったとしても、通常の範囲の体験を想定する
- **T の実際**: 食糧が尽き、仲間が死に、人肉食という禁忌に踏み込んだ
- **乖離の種類**: 因果の錯誤（遭難の帰結が常識の範囲を超える）

### 断層 γ: 「本物である」ことの反転

- **S の誘導**: 「間違いなくウミガメのスープです」→ 本物であることは安心材料
- **T の実際**: 本物であることが「過去のスープは偽物だった」という証拠になる
- **乖離の種類**: 言葉の二重性（「ウミガメのスープ」が本物と偽物の二重の意味を持つ）

### 断層 δ: 自殺動機の深度

- **S の誘導**: 食事で自殺？ → 表面的には不釣り合いな結末
- **T の実際**: 自分が仲間の人肉を食べていたと悟った絶望
- **乖離の種類**: 因果の錯誤（動機の深さが桁違い）

## Step 2: 主要仮説集合 H の定義

各断層を工程1-2 の記述素で構成する。一つの記述素が複数の断層に属してよい。

| 断層 | H の構成記述素 |
|------|---------------|
| α | D-12(味が違った), D-22(過去にもスープを飲んだ) |
| β | D-16(危機的体験), D-15(船), D-6(遭難), D-7(漂着) |
| ε | D-8(食糧尽きた), D-9(仲間死んだ), D-10(肉でスープ), D-17(仲間がいた) |
| γ | D-18(飲まされた), D-19(偽りの名前), D-11(偽って食べさせた), D-21(欺かれた), D-20(悪意なし) |
| δ | D-13(人肉と悟った), D-14(絶望して自殺) |

### 3 断層にまたがる結節点

- D-10（肉でスープを作った）: ε + γ + δ
- D-12（味が違った）: γ + α + δ
- D-17（仲間がいた）: β + ε + γ

## Step 3: 断層の「問い」の定式化

| 断層 | 問い |
|------|------|
| α | なぜ「本当にウミガメのスープか」と確認したのか？ |
| β | 自殺の原因はどこにあるのか？ |
| ε | 過去に何が起きたのか？ |
| γ | 「本物である」ことは何を意味するのか？ |
| δ | なぜ自殺するほどの絶望に至ったのか？ |
