# Phase 1: 逆算連鎖

## S（問題文）

ある男がレストランに入り、「ウミガメのスープ」を注文した。一口食べた男は店員を呼び、「これは本当にウミガメのスープですか？」と尋ねた。店員は「はい、間違いなくウミガメのスープです」と答えた。男は会計を済ませ、帰宅後、自殺した。

## T（真相）

男はかつて乗っていた船が遭難し、仲間と無人島に漂着した。食糧が尽き、仲間が一人死んだ。残った者たちはその肉でスープを作り、男には「ウミガメのスープだ」と偽って食べさせた。レストランで本物のウミガメのスープを飲んだ男は味が記憶と全く違うことに気づき、あの時のスープが仲間の人肉だったと悟り、絶望して自殺した。

## T の逆算連鎖

```
自殺した
← 絶望した
← スープの正体が仲間の人肉だったと悟った
← 本物のウミガメのスープと味が全く違った
← かつて飲んだスープは実際にはウミガメではなかった
← 仲間の肉でスープを作り「ウミガメのスープだ」と偽った
← 食糧が尽き仲間が死に、生き延びるためだった
← 船が遭難し無人島に漂着した
```

## S の記述素

- D-1: 男がレストランに入った
- D-2: ウミガメのスープを注文した
- D-3: 一口食べて店員に「本当にウミガメのスープか」と尋ねた
- D-4: 店員は「間違いなくウミガメのスープです」と答えた
- D-5: 男は帰宅後、自殺した

## T の記述素

- D-6: 男はかつて船が遭難した
- D-7: 仲間と無人島に漂着した
- D-8: 食糧が尽きた
- D-9: 仲間が一人死んだ
- D-10: 残った者たちがその肉でスープを作った
- D-11: 男には「ウミガメのスープだ」と偽って食べさせた
- D-12: 本物のウミガメのスープと味が全く違った
- D-13: あの時のスープが仲間の人肉だったと悟った
- D-14: 絶望して自殺した
