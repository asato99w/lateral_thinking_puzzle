# Phase 2: ピース構成

## 設計方針

- ピースの members は Phase 1 の記述素の機械的グループ化ではなく、ピースの目的に合わせた記述素として再構成する
- クリア条件の記述素はキーピースに集約する（キーピース制約）
- トリガーは記述素の族（OR of AND）。ピース内の全記述素がトリガー族のいずれかの要素に接続する
- 依存はトリガーから導出される: トリガー族に他ピースの記述素が含まれていれば依存
- **S/T 分離原則**: T の構造のみを扱う。S は参照しない

## ピース構成

### P-shipwreck（遭難）— 独立ピース

members:
- D-6: 男はかつて船が遭難した
- D-7: 仲間と無人島に漂着した
- D-8: 食糧が尽きた
- D-9: 仲間が一人死んだ

トリガー族: [[D-15], [D-16], [D-17]]
- D-15: 男は船乗りだった
- D-16: 過去に危機的な体験があった
- D-17: 仲間がいた

独立: トリガー族に他ピースの記述素を含まない。

各記述素との接続:
- D-15 → D-6, D-7
- D-16 → D-6, D-7, D-8
- D-17 → D-7, D-9

### P-deception（偽装）— 独立ピース

members:
- D-18: 男は過去にスープを飲まされたことがある
- D-19: そのスープは偽りの名前だった
- D-20: 偽装に悪意はなかった（生き延びるためだった）

トリガー族: [[D-21], [D-22]]
- D-21: 男は過去に欺かれた
- D-22: 男は過去にもスープを飲んだことがある

独立: トリガー族に他ピースの記述素を含まない。

各記述素との接続:
- D-21 → D-18, D-19
- D-22 → D-18, D-20

### P-realization（発覚と結末）— キーピース

members:
- D-10: 残った者たちがその肉でスープを作った
- D-11: 男には「ウミガメのスープだ」と偽って食べさせた
- D-12: 本物のウミガメのスープと味が全く違った
- D-13: あの時のスープが仲間の人肉だったと悟った
- D-14: 絶望して自殺した

トリガー族: [[D-6, D-8, D-9, D-18, D-19], [D-7, D-9, D-18, D-19, D-20]]
- AND グループ 1: 遭難 + 食糧不足 + 仲間の死 + スープを飲まされた + 偽りの名前
- AND グループ 2: 漂着 + 仲間の死 + スープを飲まされた + 偽りの名前 + 悪意なし

依存（トリガーから導出）: 全ての AND グループに P-shipwreck と P-deception の記述素が十分に含まれるため、両方に依存。遭難と偽装の探索が十分に進んだ上で初めて成立する。

各記述素との接続:
- D-6, D-9 → D-10（遭難し仲間が死んだ文脈から肉のスープへ）
- D-19 → D-11（偽りの名前の具体内容へ）
- D-18 → D-12（過去のスープ体験から味の違いへ）
- D-10, D-11, D-12 → D-13（正体の悟りへ）
- D-13 → D-14（絶望と自殺へ）

キーピース制約: clear_conditions の記述素（D-13, D-14）はこのピースに属する。

## 依存関係

```
P-shipwreck（独立）  ─┐
                       ├→ P-realization（キーピース、両方に依存）
P-deception（独立）  ─┘
```

依存関係はトリガー族の構成から導出:
- P-realization のトリガー族の全 AND グループに P-shipwreck と P-deception の記述素が含まれる → 両方に依存
