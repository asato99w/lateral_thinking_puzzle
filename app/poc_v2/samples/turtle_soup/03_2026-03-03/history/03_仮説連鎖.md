- D-16 (過去に危機的な体験があった): terminal — 抽象的だが他から自動導出不可
- D-17 (仲間がいた): terminal — 基礎的事実

### P-deception triggers: D-21, D-22

- D-21 (男は過去に欺かれた): terminal — 基礎的事実

## Step 2: ピースメンバーの導出関係

### P-shipwreck (D-6, D-7, D-8, D-9)

各事実は独立した出来事であり、一つから他が論理必然的に導出されるほど因果が強くない。全て terminal。

### P-deception (D-18, D-19, D-20)

- D-18 (過去にスープを飲まされた): terminal
- D-19 (そのスープは偽りの名前だった): formation_conditions = [[D-18, D-21]]
  - スープを飲まされた + 欺かれた → スープの名前が偽りだった
  - 最小十分: 両方揃って初めて「名前の偽り」が成立
- D-20 (偽装に悪意はなかった): terminal — 意図の判断は自動導出困難

### P-realization (D-10, D-11, D-12, D-13, D-14) — キーピース

導出の中核。トリガーが満たされると連鎖的に全メンバーが導出される。

- D-10 (残った者がその肉でスープを作った): formation_conditions = [[D-8, D-9, D-18]]
  - 食糧が尽き(D-8) + 仲間が死に(D-9) + スープを飲まされた(D-18) → 肉でスープを作った
  - 最小十分: 3要素揃って初めて成立
- D-11 (「ウミガメのスープだ」と偽って食べさせた): formation_conditions = [[D-10, D-19]]
  - 肉のスープ(D-10) + 偽りの名前(D-19) → 「ウミガメのスープ」と偽った
- D-12 (本物のウミガメのスープと味が全く違った): formation_conditions = [[D-10, D-2]]
  - 肉のスープ(D-10) + 本物を注文(D-2, initial_confirmed) → 味が全く違う
- D-13 (仲間の人肉だったと悟った): formation_conditions = [[D-10, D-12]]
  - 肉のスープ(D-10) + 味が違った(D-12) → 正体を悟った
- D-14 (絶望して自殺した): formation_conditions = [[D-13]]
  - 悟り → 絶望・自殺
