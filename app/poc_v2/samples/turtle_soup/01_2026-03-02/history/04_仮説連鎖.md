# 04 仮説連鎖の展開

トリガーに含まれる仮説について、形成条件を再帰的に展開する。S の事実に到達した時点で末端とする。

## トリガーに登場する仮説

Phase 3 のトリガー定義から、以下の仮説が必要:
- h_past_soup（ピース a のトリガー）
- h_not_daily（ピース b のトリガー）
- h_past_relevant（ピース b のトリガー）
- h_deception（ピース c のトリガー）
- h_dark_truth（ピース e のトリガー）

## 展開

### h_past_relevant（第 1 層）

- 形成条件: {Ps-4}
- 展開: Ps-4 は S の事実 → **末端**
- 根拠: 男が「本当にウミガメのスープか？」と確認した行為は、何らかの過去の経験を示唆する

### h_soup_ok（第 1 層）

- 形成条件: {Ps-5}
- 展開: Ps-5 は S の事実 → **末端**
- 根拠: 店員が「間違いなくウミガメのスープです」と肯定した → スープ自体に問題はない
- 注: トリガーに直接は登場しないが、h_not_daily の形成に必要

### h_not_daily（第 2 層）

- 形成条件: {h_soup_ok, Ps-6}
- 展開:
  - h_soup_ok → {Ps-5}（第 1 層、末端）
  - Ps-6 は S の事実 → 末端
- **完全展開**: {Ps-5} → h_soup_ok → {h_soup_ok, Ps-6} → h_not_daily
- 根拠: スープに問題がないのに自殺した → 原因は日常的なものではない
- 適用戦略: 常識的仮説の排除

### h_past_soup（第 2 層）

- 形成条件: {h_past_relevant, Ps-2}
- 展開:
  - h_past_relevant → {Ps-4}（第 1 層、末端）
  - Ps-2 は S の事実 → 末端
- **完全展開**: {Ps-4} → h_past_relevant → {h_past_relevant, Ps-2} → h_past_soup
- 根拠: 過去の経験が関係する + スープを注文した → 過去のスープ体験が関係する
- 適用戦略: 観測事実の深掘り、時間軸の拡張

### h_deception（第 3 層 — T の事実が必要）

- 形成条件: {Pt-5, Pt-8}
- 展開:
  - Pt-5 は T の事実（ゲーム中に発見） → 末端
  - Pt-8 は T の事実（ゲーム中に発見） → 末端
- 根拠: 「ウミガメのスープだ」と言われた + 味が違う → ラベルと実体が乖離している
- 適用戦略: 矛盾の指摘、ラベルと実体の乖離
- 注: S の事実のみからは形成不可。Pt-5 と Pt-8 の両方が必要。

### h_survival（第 3 層 — T の事実が必要）

- 形成条件: {Pt-2, Pt-3}
- 展開:
  - Pt-2 は T の事実 → 末端
  - Pt-3 は T の事実 → 末端
- 根拠: 食料が尽きた + 仲間が死んだ → 生存のために極限的な行為があった
- 適用戦略: 因果の逆転
- 注: トリガーに直接は登場しないが、h_dark_truth の形成経路の一つ。

### h_taste_key（第 3 層 — T の事実が必要）

- 形成条件: {Pt-8, Ps-4}
- 展開:
  - Pt-8 は T の事実 → 末端
  - Ps-4 は S の事実 → 末端
- 根拠: 味が違った + わざわざ確認した → 味の違いが事件の核心
- 注: 質問の想起条件には使わないが、プレイヤーの思考を表現する仮説。

### h_dark_truth（第 4 層）

- 形成条件: {h_deception, Pt-3} OR {h_survival, Pt-5}
- 展開:
  - 経路 A: h_deception → {Pt-5, Pt-8}、Pt-3 は T の事実
  - 経路 B: h_survival → {Pt-2, Pt-3}、Pt-5 は T の事実
- 根拠:
  - 経路 A: 偽りがあった + 仲間が死んだ → スープの正体は恐ろしいもの
  - 経路 B: 極限状況での行為 + 偽ラベル → 何を食べさせた？
- 適用戦略: 因果の逆転、登場人物の探索

## 仮説連鎖の木（全体像）

```
第 1 層（S の事実のみ）
  h_soup_ok ← {Ps-5}
  h_past_relevant ← {Ps-4}

第 2 層（第 1 層 + S の事実）
  h_not_daily ← {h_soup_ok, Ps-6}
  h_past_soup ← {h_past_relevant, Ps-2}

第 3 層（T の事実が必要）
  h_taste_key ← {Pt-8, Ps-4}
  h_deception ← {Pt-5, Pt-8}
  h_survival ← {Pt-2, Pt-3}

第 4 層（第 3 層仮説 + T の事実）
  h_dark_truth ← {h_deception, Pt-3} OR {h_survival, Pt-5}
```
