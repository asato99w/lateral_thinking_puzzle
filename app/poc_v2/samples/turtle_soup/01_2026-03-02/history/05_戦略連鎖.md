# 05 戦略連鎖の構成

仮説連鎖の末端要素に対して汎用戦略を適用し、S の事実への接続を構成する。

## 末端要素の確認

仮説連鎖の末端（S の事実で構成される要素）:
- h_soup_ok の末端: Ps-5
- h_past_relevant の末端: Ps-4
- h_not_daily の末端: Ps-5（h_soup_ok 経由）, Ps-6
- h_past_soup の末端: Ps-4（h_past_relevant 経由）, Ps-2

T の事実を含む末端（ゲーム中に到達が必要）:
- h_taste_key の末端: Pt-8, Ps-4
- h_deception の末端: Pt-5, Pt-8
- h_survival の末端: Pt-2, Pt-3
- h_dark_truth の末端: h_deception 経由または h_survival 経由

## 戦略の適用

### Ps-5 → h_soup_ok

- 戦略: **観測事実の深掘り**
- 連鎖: 店員が本物だと肯定した（Ps-5）→ スープ自体に問題はない（h_soup_ok）
- プレイヤーの思考: 店員が嘘をついていない → スープに問題はない

### Ps-4 → h_past_relevant

- 戦略: **動機の探索**
- 連鎖: わざわざ確認した（Ps-4）→ 確認する動機がある → 過去の経験が関係する（h_past_relevant）
- プレイヤーの思考: なぜ確認した？→ 何かを知っている/経験しているから

### {h_soup_ok, Ps-6} → h_not_daily

- 戦略: **常識的仮説の排除**
- 連鎖: スープ問題なし（h_soup_ok）+ 自殺（Ps-6）→ 日常的原因の排除 → 非日常的原因（h_not_daily）
- プレイヤーの思考: 毒でも病気でもない + 自殺 → 何か特殊な原因がある

### {h_past_relevant, Ps-2} → h_past_soup

- 戦略: **観測事実の深掘り** × **時間軸の拡張**
- 連鎖: 過去が関係（h_past_relevant）+ スープ注文（Ps-2）→ 過去のスープ体験（h_past_soup）
- プレイヤーの思考: 過去が関係 + スープが鍵 → 過去にもスープを飲んだ？

### {Pt-8, Ps-4} → h_taste_key

- 戦略: **因果の逆転**
- 連鎖: 味が違った（Pt-8）+ 確認行為（Ps-4）→ 味の違いが確認の動機だった（h_taste_key）
- プレイヤーの思考: 味が違ったから確認した。味の違いが事件の核心。

### {Pt-5, Pt-8} → h_deception

- 戦略: **矛盾の指摘** × **ラベルと実体の乖離**
- 連鎖: 「ウミガメのスープだ」と言われた（Pt-5）+ 味が違う（Pt-8）→ ラベルと実体の不一致 → 偽りがあった（h_deception）
- プレイヤーの思考: ウミガメのスープだと聞いたが味が違う → 本当はウミガメのスープではなかった

### {Pt-2, Pt-3} → h_survival

- 戦略: **因果の逆転**
- 連鎖: 食料が尽きた（Pt-2）+ 仲間が死んだ（Pt-3）→ 極限状況での行為（h_survival）
- プレイヤーの思考: 食料がなく仲間が死んだ → 生き延びるために何かした？

### {h_deception, Pt-3} → h_dark_truth（経路 A）

- 戦略: **登場人物の探索** × **因果の逆転**
- 連鎖: 偽りがあった（h_deception）+ 仲間が死んだ（Pt-3）→ 正体は恐ろしいもの（h_dark_truth）
- プレイヤーの思考: 偽物のスープ + 仲間の死 → まさか仲間の…

### {h_survival, Pt-5} → h_dark_truth（経路 B）

- 戦略: **ラベルと実体の乖離** × **因果の逆転**
- 連鎖: 極限行為（h_survival）+ 偽ラベル（Pt-5）→ 正体は恐ろしいもの（h_dark_truth）
- プレイヤーの思考: 生き延びるために何かした + 「ウミガメのスープ」と偽った → 何を食べさせた？

## 使用された汎用戦略

| 戦略 | 適用箇所 |
|------|---------|
| 常識的仮説の排除 | h_not_daily の形成 |
| 観測事実の深掘り | h_soup_ok, h_past_soup の形成 |
| 時間軸の拡張 | h_past_soup の形成 |
| 因果の逆転 | h_taste_key, h_survival, h_dark_truth の形成 |
| 矛盾の指摘 | h_deception の形成 |
| ラベルと実体の乖離 | h_deception, h_dark_truth（経路 B）の形成 |
| 登場人物の探索 | h_dark_truth（経路 A）の形成 |
| 動機の探索 | h_past_relevant の形成 |

8 戦略中 7 戦略を使用（「登場人物の探索」以外の「動機の探索」含め全てカバー）。未使用: なし（全 8 戦略が適用された）。
