# 01 逆算連鎖

## T の逆算

```
T: 絶望して自殺した
  ← なぜ？: 人肉を食べていたことを悟った
    ← なぜ悟った？: レストランのスープと記憶のスープの味が違った
      ← なぜ味が違った？: 記憶のスープは本物のウミガメのスープではなかった
        ← なぜ偽物だった？: 「ウミガメのスープだ」と偽って仲間の肉のスープを飲ませた
          ← なぜそうした？: 遭難中に食料が尽き、仲間が死に、生存のため
            ← 前提: 男はかつて仲間と船で遭難した
```

## 事実の抽出

### S の事実（問題文から直接読み取れる）

| ID | 事実 |
|----|------|
| Ps-1 | 男がレストランに入った |
| Ps-2 | ウミガメのスープを注文した |
| Ps-3 | 一口飲んだ |
| Ps-4 | 店員に「これは本当にウミガメのスープか？」と尋ねた |
| Ps-5 | 店員は「はい、間違いなくウミガメのスープです」と答えた |
| Ps-6 | その夜、自殺した |

### T の事実（逆算連鎖の各ステップ）

| ID | 事実 | 逆算位置 |
|----|------|----------|
| Pt-1 | 男はかつて仲間と船で遭難した | 最深部（前提） |
| Pt-2 | 遭難中に食料が尽きた | 深部（極限状態） |
| Pt-3 | 仲間の一人が死んだ | 深部（極限状態） |
| Pt-4 | 残った仲間がスープを作り男に飲ませた | 中間（行為） |
| Pt-5 | そのスープを「ウミガメのスープだ」と偽った | 中間（偽装） |
| Pt-6 | そのスープの正体は死んだ仲間の肉だった | 中間（真実） |
| Pt-7 | 男は過去にも「ウミガメのスープ」を飲んだ経験がある | 浅部（S との接点） |
| Pt-8 | レストランのスープと過去のスープの味が違った | 浅部（認知） |
| Pt-9 | 過去のスープは本物のウミガメのスープではなかった | 浅部（推論） |
| Pt-10 | 絶望して自殺した | 表層（帰結） |

## 備考

- Pt-7 は Pt-4 + Pt-5 の帰結だが、S からの探索で最初に到達する事実として独立に定義。プレイヤーは「過去にも飲んだ」をまず知り、後に具体的状況（遭難中の偽装）を知る段階構成。
- Pt-9 は Pt-8 + Pt-5 からの推論結果。アノマリー検出の着地点。
