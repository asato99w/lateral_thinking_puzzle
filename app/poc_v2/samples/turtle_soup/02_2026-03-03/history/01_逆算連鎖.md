# Phase 1: 逆算連鎖

## T の逆算連鎖

自殺した
← 絶望した
← あの時のスープの正体を悟った
← 本物の味と記憶の味が違った
← レストランで本物のウミガメのスープを飲んだ / 遭難中に「ウミガメのスープ」と偽って飲まされた
← 実際は死んだ仲間の肉だった
← 遭難して食料が尽き、仲間の一人が死んだ
← 船で遭難した

## Ps（S の事実）

- Ps-1: 男がレストランに入った
- Ps-2: ウミガメのスープを注文した
- Ps-3: 一口飲んだ
- Ps-4: 店員に「これは本当にウミガメのスープか？」と尋ねた
- Ps-5: 店員は「はい、間違いなくウミガメのスープです」と答えた
- Ps-6: その夜、自殺した

## Pt（T の事実）

- Pt-1: 男はかつて仲間と船で遭難した
- Pt-2: 遭難中に食料が尽きた
- Pt-3: 仲間の一人が死んだ
- Pt-4: 残った仲間がスープを作り男に飲ませた
- Pt-5: そのスープを「ウミガメのスープだ」と偽った
- Pt-6: そのスープの正体は死んだ仲間の肉だった
- Pt-7: レストランのスープの味が記憶と違った
- Pt-8: あの時のスープの正体を悟った
- Pt-9: 絶望して自殺を選んだ
