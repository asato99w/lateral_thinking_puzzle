# Phase 1: T の因果分解

## T
男は数人で熱気球に乗っていたが、高度が下がり始めた。荷物を全て捨て、服も脱いで軽量化したが足りず、くじ引き（マッチの長さ）で一人が飛び降りることになった。短いマッチを引いた男が飛び降りて死亡した。

## 逆算連鎖

```
男が死亡した ← なぜ？
  男が熱気球から飛び降りた ← なぜ？
    くじ引きで男が選ばれた ← なぜくじ引き？
      一人が飛び降りる必要があった ← なぜ？
        軽量化が不十分だった ← なぜ軽量化していた？
          高度が下がり始めた ← 前提
    くじ引きの方法はマッチの長さだった
    男は短いマッチを引いた
  男たちは熱気球に乗っていた ← 前提
  男は数人で乗っていた ← 前提
軽量化の内容:
  荷物を全て捨てた
  服も脱いだ
```
