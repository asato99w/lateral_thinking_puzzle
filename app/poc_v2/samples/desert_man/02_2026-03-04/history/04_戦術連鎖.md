# Phase 4: 戦術連鎖

## 1. 末端記述素への戦術連鎖

### D-16「男は地上からこの場所に来たのではない」

起点: S の記述素集合（P-balloon は独立ピース）

```
{D-1, D-3} →[常識的仮説の排除]→ {D-16}
```

砂漠の真ん中(D-1)で死んでいた(D-3)。常識的には遭難死だが、それを排除する方向に「地上から来たのではない」を問う。

### D-17「裸になったのには理由があった」

起点: S の記述素集合

```
{D-2} →[観測事実の深掘り]→ {D-17}
```

男は裸だった(D-2)。裸であることの背景・理由を深掘りする。
