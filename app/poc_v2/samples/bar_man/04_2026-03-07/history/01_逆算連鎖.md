# Phase 1: 逆算連鎖

## T の逆算

T: 男はしゃっくりが止まらず困っていた。水を飲めばしゃっくりが治ると思い、バーに入って水を頼んだ。バーテンダーは男のしゃっくりに気づき、驚かせて止めようと銃を向けた。驚いた男はしゃっくりが止まり、水が不要になったので「ありがとう」と言って帰った。

```
「ありがとう」と言って帰った
← なぜ？: しゃっくりが止まり、水が不要になった
  ← なぜしゃっくりが止まった？: 驚いた（銃を向けられて）
    ← なぜ銃を向けた？: 驚かせてしゃっくりを止めようとした
      ← なぜそう考えた？: バーテンダーが男のしゃっくりに気づいた
← なぜ水を頼んだ？: 水を飲めばしゃっくりが治ると思った
  ← なぜバーに？: しゃっくりが止まらなかった
```

## T の記述素

| ID | label |
|----|-------|
| D-1 | 男はしゃっくりが止まらなかった |
| D-2 | 男は水を飲めばしゃっくりが治ると考えていた |
| D-3 | バーテンダーは男のしゃっくりに気づいた |
| D-4 | バーテンダーは驚かせればしゃっくりが止まると考えた |
| D-5 | 男は驚いた |
| D-6 | 男のしゃっくりが止まった |

各記述素は独立（他の記述素との論理演算を前提としない）かつ客観的に検証可能。
