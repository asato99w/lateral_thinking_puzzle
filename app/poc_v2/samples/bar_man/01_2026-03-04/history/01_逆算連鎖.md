# Phase 1: 逆算連鎖

## 入力

- S: ある男がバーに入り、バーテンダーに「水をください」と頼んだ。バーテンダーは男に銃を向けた。男は「ありがとう」と言って帰った。
- T: 男はしゃっくりが止まらなくて困っていた。水を飲めば治ると思いバーに立ち寄ったが、バーテンダーはしゃっくりに気づき、驚かせて止めてやろうと銃を向けた。しゃっくりが止まった男は水が不要になり、「ありがとう」と言って帰った。

## Step 1: T の逆算

```
結論: 男は「ありがとう」と言って帰った
← なぜ「ありがとう」？ → しゃっくりを止めてくれたことへの感謝
← なぜ帰った？ → 水が不要になったから
← なぜ水が不要？ → しゃっくりが止まったから
← なぜしゃっくりが止まった？ → 銃を向けられて驚いたから
← なぜ銃を向けた？ → 驚かせてしゃっくりを止めようとしたから
← なぜ驚かせようとした？ → バーテンダーが男のしゃっくりに気づいたから
（別の因果線）
← なぜ男は水を頼んだ？ → 水でしゃっくりが治ると思ったから
← なぜしゃっくりを治したかった？ → しゃっくりが止まらなくて困っていたから
```

## Step 2: S の記述素の抽出

| ID | label |
|----|-------|
| D-1 | 男がバーに入った |
| D-2 | バーテンダーに「水をください」と頼んだ |
| D-3 | バーテンダーは男に銃を向けた |
| D-4 | 男は「ありがとう」と言った |
| D-5 | 男は帰った |

## Step 3: T の記述素の抽出

| ID | label | 逆算ステップ |
|----|-------|-------------|
| D-6 | 男はしゃっくりが止まらなくて困っていた | 最深部（前提） |
| D-7 | 水を飲めばしゃっくりが治ると思った | D-6 → D-7（治そうとした） |
| D-8 | バーテンダーは男のしゃっくりに気づいた | 別の因果線の起点 |
| D-9 | バーテンダーは驚かせてしゃっくりを止めようとした | D-8 → D-9 |
| D-10 | 銃を向けたのは驚かせる手段だった | D-9 → D-10（手段の選択） |
| D-11 | しゃっくりが止まった | D-10 → D-11（結果） |
| D-12 | 水が不要になった | D-11 → D-12 |
| D-13 | 「ありがとう」はしゃっくりを止めてくれたことへの感謝 | D-11 → D-13 |
