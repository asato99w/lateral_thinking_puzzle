# Phase 1: 逆算連鎖

## 入力

- S: ある男がバーに入り、バーテンダーに「水をください」と頼んだ。バーテンダーは男に銃を向けた。男は「ありがとう」と言って帰った。
- T: 男はしゃっくりが止まらなくて困っていた。水を飲めば治ると思いバーに立ち寄ったが、バーテンダーはしゃっくりに気づき、驚かせて止めてやろうと銃を向けた。しゃっくりが止まった男は水が不要になり、「ありがとう」と言って帰った。

## Step 1: T の逆算

```
なぜ男は「ありがとう」と言って帰った？
  → しゃっくりが止まり水が不要になった。止めてくれた感謝
    なぜしゃっくりが止まった？
      → バーテンダーが銃を向けて驚かせたから
        なぜバーテンダーは銃を向けた？
          → しゃっくりに気づき、驚かせて止めてやろうとした
            なぜしゃっくりに気づいた？
              → 男がしゃっくりをしていた
なぜ男はバーに来た？
  → しゃっくりが止まらず困っていて、水を飲めば治ると思ったから
```

## Step 2: S の記述素

| ID | label |
|----|-------|
| D-1 | 男がバーに入った |
| D-2 | 男はバーテンダーに「水をください」と頼んだ |
| D-3 | バーテンダーは男に銃を向けた |
| D-4 | 男は「ありがとう」と言った |
| D-5 | 男は帰った |

## Step 3: T の記述素

| ID | label |
|----|-------|
| D-6 | 男はしゃっくりが止まらなくて困っていた |
| D-7 | 水を飲めばしゃっくりが治ると思った |
| D-8 | バーテンダーは男のしゃっくりに気づいた |
| D-9 | バーテンダーは驚かせてしゃっくりを止めようとした |
| D-10 | 男は驚いてしゃっくりが止まった |
| D-11 | しゃっくりが止まったので水が不要になった |
| D-12 | 「ありがとう」はしゃっくりを止めてくれたことへの感謝だった |
