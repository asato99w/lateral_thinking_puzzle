# Phase 1: 逆算連鎖

## T の逆算

```
男は「ありがとう」と言って帰った（S と共通）
← なぜ感謝した？ しゃっくりが止まった（D-6）
  ← なぜ止まった？ 男は驚いた（D-5）
    ← なぜ驚いた？ バーテンダーが銃を向けた（S と共通）
      ← なぜ銃を向けた？ 驚かせれば止まると考えた（D-4）
        ← バーテンダーはしゃっくりに気づいた（D-3）
          ← 男はしゃっくりが止まらなかった（D-1）

水を頼んだ（S と共通）
← なぜ？ 水で治ると考えていた（D-2）+ しゃっくりが止まらなかった（D-1）
```

## T の記述素

| ID | ラベル |
|----|--------|
| D-1 | 男はしゃっくりが止まらなかった |
| D-2 | 男は水を飲めばしゃっくりが治ると考えていた |
| D-3 | バーテンダーは男のしゃっくりに気づいた |
| D-4 | バーテンダーは驚かせればしゃっくりが止まると考えた |
| D-5 | 男は驚いた |
| D-6 | 男のしゃっくりが止まった |

## 設計判断

- S と共通する観測事実（バーに入った、水を頼んだ、銃を向けた、ありがとうと帰った）は T 記述素から除外。Phase 3 で S 記述素として扱う
- 「困っていた」は主観的評価のため除外。客観的事実 D-1「しゃっくりが止まらなかった」のみ抽出
- 「水が不要になった」は D-6 + D-2 から導出可能なため独立記述素としない
