# Phase 1: 逆算連鎖

S: 男がバーに入ってきて、バーテンダーに水を一杯頼んだ。するとバーテンダーは男に銃を向けた。男は「ありがとう」と言って帰っていった。

T: 男はしゃっくりが止まらなかった。男は水を飲めばしゃっくりが治ると考えていた。バーテンダーは男のしゃっくりに気づいた。バーテンダーは驚かせればしゃっくりが止まると考えた。バーテンダーは男に銃を向けて驚かせた。男は驚いた。男のしゃっくりが止まった。男は感謝した。

## T の記述素

| ID | label |
|----|-------|
| D-1 | 男はしゃっくりが止まらなかった |
| D-2 | 男は水を飲めばしゃっくりが治ると考えていた |
| D-3 | バーテンダーは男のしゃっくりに気づいた |
| D-4 | バーテンダーは驚かせればしゃっくりが止まると考えた |
| D-5 | 男は驚いた |
| D-6 | 男のしゃっくりが止まった |

## 因果構造

```
D-1（しゃっくり）→ D-2（水で治ると考えた）→ 水を頼んだ（S）
D-1 → D-3（バーテンダーが気づいた）→ D-4（驚かせて止めようと考えた）→ 銃を向けた（S）
銃を向けた → D-5（驚いた）→ D-6（しゃっくりが止まった）→ 感謝（S）
```
