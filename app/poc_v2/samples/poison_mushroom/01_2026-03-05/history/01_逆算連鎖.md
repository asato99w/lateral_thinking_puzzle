# Phase 1: 逆算連鎖

## S の記述素
- D-1: 辺りに毒キノコしかない
- D-2: 男は安堵した

## T の逆算
```
男は安堵した (= D-2, S)
  ← 助かる見込みがある (D-7)
    ← 近くに人里・採集者がいると分かった (D-6)
      ← 食用キノコは誰かが既に採った (D-5)
        ← 辺りに毒キノコしかない (D-1, S) + キノコ狩りをしていた (D-3)
背景: 男は遭難していた (D-4) + キノコ狩りをしていた (D-3)
```

## T の記述素
- D-3: 男はキノコ狩りをしていた
- D-4: 男は遭難していた
- D-5: 食用キノコは誰かが既に採った
- D-6: この辺りに人が来ている
- D-7: 男は助かる見込みがある
