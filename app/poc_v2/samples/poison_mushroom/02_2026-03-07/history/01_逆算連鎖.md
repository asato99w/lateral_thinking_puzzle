# Phase 1: 逆算連鎖

S: 男は、辺りに毒キノコしかないのを見て安堵した。なぜだろうか。

T: 男はキノコ狩りの最中に山で遭難していた。辺りに毒キノコしか残っていないということは、食用キノコは誰かが既に採った＝この辺りに人が来ているということ。近くに人里や他の採集者がいると分かり、助かる見込みがあると安堵した。

## T の記述素

| ID | label |
|----|-------|
| D-1 | 男はキノコ狩りをしていた |
| D-2 | 男は山で遭難していた |
| D-3 | 食用キノコは他の人が既に採っていた |
| D-4 | 近くに人里や他の採集者がいた |
| D-5 | 男は助かる見込みがあると判断した |

## 因果構造

```
D-1（キノコ狩り）→ 山にいる理由
D-2（遭難）→ 助けが必要な状況
D-3（食用キノコが採られていた）→ D-4（人が近くにいる）
D-2 + D-4 → D-5（助かる見込み）→ 安堵（S）
```

## 独立性の確認

- D-1: 自己完結 ✓
- D-2: 自己完結 ✓
- D-3: 自己完結（「他の人が」により主体が明確）✓
- D-4: 自己完結 ✓
- D-5: 自己完結 ✓

## 客観性の確認

- D-5: 「判断した」は主観的か？ → 「助かる見込みがある」という認知状態の記述。T に「助かる見込みがあると…安堵した」とあり、T の情報の分解として妥当 ✓
