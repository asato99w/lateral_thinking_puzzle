# 技術要件: プラットフォーム・環境

## 対応プラットフォーム

| 項目 | 要件 |
|------|------|
| OS | iOS |
| 最低対応バージョン | iOS 17.0 |
| 対応デバイス | iPhone |
| iPad対応 | 初期リリースでは対象外 |

## 開発環境

| 項目 | 要件 |
|------|------|
| IDE | Xcode 16.x |
| 言語 | Swift 6.x |
| UIフレームワーク | SwiftUI |
| プロジェクト形式 | Xcode Project（.xcodeproj） |
| パッケージ管理 | Swift Package Manager |

## 依存ライブラリ

| ライブラリ | 用途 | 必須/任意 |
|-----------|------|----------|
| （なし） | - | - |

※ 初期段階では外部依存なし。Apple標準フレームワークのみで構成する方針。

## ビルド・実行要件

| 項目 | 要件 |
|------|------|
| ビルドターゲット | arm64（Apple Silicon / A-series） |
| Simulator対応 | 開発・テスト用にサポート |
| ネットワーク接続 | コンテンツダウンロード時に必要。ダウンロード済みパズルはオフラインでプレイ可 |
| 必要な権限・Capability | なし（将来的に In-App Purchase Capability を追加） |

## データ形式

| 項目 | 要件 |
|------|------|
| パズルデータ形式 | JSON |
| データ読み込み | Foundation の JSONDecoder（Codable） |
| 永続化 | 初期リリースでは不要 |

## コンテンツ配信

| 項目 | 要件 |
|------|------|
| 配信方式 | On-Demand Resources (ODR) |
| ホスティング | App Store（自前サーバ不要） |
| 初期同梱パズル | アプリバンドルに含める（initial install tags） |
| 追加パズル | ODR タグ単位でオンデマンドダウンロード（prefetch / on-demand tags） |
| 利用フレームワーク | NSBundleResourceRequest |

## アプリ内課金（将来対応）

| 項目 | 要件 |
|------|------|
| 課金基盤 | StoreKit 2 |
| 導入時期 | 初期リリース後 |
| 初期リリースでの考慮事項 | パズルデータに課金制御用の属性（無料/有料区分等）を持たせておく |

## テスト環境

| 項目 | 要件 |
|------|------|
| テストフレームワーク | XCTest / Swift Testing |
| UIテスト | XCUITest（必要に応じて） |
