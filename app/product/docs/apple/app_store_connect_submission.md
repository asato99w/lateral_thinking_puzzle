## 6. 年齢制限

App Store Connect の年齢制限質問票への回答。

| 質問 | 回答 |
|---|---|
| まれ/軽度なアニメまたはファンタジーの暴力 | `なし` |
| 頻繁/過激なアニメまたはファンタジーの暴力 | `なし` |
| まれ/軽度なリアルな暴力 | **`まれ/軽度`** ※1 |
| 頻繁/過激なリアルな暴力 | `なし` |
| まれ/軽度な性的コンテンツやヌード | `なし` |
| 頻繁/過激な性的コンテンツやヌード | `なし` |
| まれ/軽度な下品なユーモアや冒涜的表現 | `なし` |
| 頻繁/過激な下品なユーモアや冒涜的表現 | `なし` |
| まれ/軽度な薬物・アルコール・タバコの使用 | `なし` |
| 頻繁/過激な薬物・アルコール・タバコの使用 | `なし` |
| まれ/軽度なホラー/恐怖をテーマにしたコンテンツ | **`まれ/軽度`** ※2 |
| 頻繁/過激なホラー/恐怖をテーマにしたコンテンツ | `なし` |
| まれ/軽度な成人向けコンテンツ | **`まれ/軽度`** ※3 |
| 頻繁/過激な成人向けコンテンツ | `なし` |
| ギャンブルのシミュレーション | `なし` |
| 制限なしのWebアクセス | `なし` |

→ 判定結果: **12+**

> ※1 パズルの題材に死亡・自殺・殺人などの暴力的な状況がテキストで登場する。
> 画像や効果音による直接的な暴力表現はないため「まれ/軽度」。
>
> ※2 死亡や自殺をテーマとする不穏なシナリオが含まれる。
> テキストのみの表現で視覚的な恐怖演出はないため「まれ/軽度」。
>
> ※3 自殺などの重いテーマを扱うパズルが含まれる。
> テキストベースで直接的・扇情的な描写はないため「まれ/軽度」。
>
> **判定根拠**: 死亡・自殺がパズルのテーマとして登場するため、
> 「リアルな暴力」「ホラー/恐怖」「成人向けコンテンツ」の各項目で
> 「まれ/軽度」を選択。いずれもテキストのみの間接的表現であり
> 「頻繁/過激」には該当しない。結果として 12+ が適切。
