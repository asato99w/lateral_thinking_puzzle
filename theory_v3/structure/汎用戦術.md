# 汎用戦術

任意のパズルに適用可能な探索の戦術。入力・出力はいずれも命題集合（空集合を含む）。

## 戦術一覧

### 0. 直接同定

- 入力: 命題集合
- 出力: 入力と同一の命題集合
- 説明: 事実がそのまま仮説に該当する場合に適用する。変換を伴わない恒等的な戦術。

### 1. 常識的仮説の排除

- 入力: 命題集合
- 出力: 常識的な説明が否定された命題集合
- 説明: 「ありがちな説明」を否定することで、非日常的な方向へ探索を進める。
- 適用例: {自殺した} → {日常的原因ではない}

### 2. 観測事実の深掘り

- 入力: 命題集合
- 出力: 背景や詳細が明らかになった命題集合
- 説明: 事実の一つに注目し、その背景や詳細を問う。
- 適用例: {店員に尋ねた} → {過去に飲んだことがある}

### 3. 時間軸の拡張

- 入力: 命題集合（現在の出来事）
- 出力: 過去や未来に関する命題集合
- 説明: 「今」の出来事に対して「過去」を問う。S が現在の断面であることを利用する。
- 適用例: {スープを飲んだ} → {以前にも飲んだことがある}

### 4. 登場人物の探索

- 入力: 命題集合
- 出力: 新たな人物に関する命題集合
- 説明: 明示されていない人物の存在を問う。
- 適用例: {男が一人で行動している} → {仲間がいた}

### 5. 因果の逆転

- 入力: 命題集合（結果）
- 出力: 原因に関する命題集合
- 説明: 出来事の因果関係を逆に辿る。結果から原因を問う。
- 適用例: {自殺した} → {スープの味が違った}

### 6. 矛盾の指摘

- 入力: 命題集合（不整合を含む）
- 出力: 不整合が解消された命題集合
- 説明: 事実間、または事実と常識の間の不整合を問う。
- 適用例: {本物のスープ, 確認した} → {記憶と違う}

### 7. ラベルと実体の乖離

- 入力: 命題集合（ラベル付き）
- 出力: 実体に関する命題集合
- 説明: 名前や説明と実際の中身が異なる可能性を問う。
- 適用例: {ウミガメのスープと言われた} → {実際はウミガメではなかった}

### 8. 動機の探索

- 入力: 命題集合（行動）
- 出力: 動機に関する命題集合
- 説明: 行動の理由・動機を問う。特に説明されていない行動に対して。
- 適用例: {自殺した} → {正体を悟った絶望}

### 9. 人物像の探索

- 入力: 命題集合（空集合を含む）
- 出力: 人物の属性・背景に関する命題集合
- 説明: 登場人物の職業、性格、経歴などの属性を問う。
- 適用例: {男がレストランにいる} → {男は船乗りだった}

### 10. 手段の探索

- 入力: 命題集合（状態・結果）
- 出力: 手段・方法に関する命題集合
- 説明: 状態や結果が「どのように」実現されたかを問う。因果の逆転が「なぜ」を問うのに対し、手段の探索は「どうやって」を問う。同じ事実に対しても探索方向と到達する命題が異なる。
- 適用例: {砂漠の真ん中にいた} → {乗り物で移動していた}（「どうやってここに来たか」）
- 適用例: {死んでいた} → {転落死だった}（「どうやって死んだか」）

### 11. 段階的絞り込み

- 入力: 命題集合
- 出力: より具体的な命題集合
- 説明: 目標の具体的命題に直接到達すると跳躍が大きい場合に、まず広いカテゴリの仮説を立て、そこから段階的に絞り込んで具体化する。Yes/No 質問ゲームにおける基本的な探索戦略。広い仮説を置くこと自体がこの戦術に含まれる。
- 適用例: {男がレストランにいる} → {過去に危機的な体験がある} → {船に乗っていた}
- 適用例: {何かが起きた} → {生物が関係する} → {動物が関係する} → {猫だった}

## 適用上の注意

### 集合への適用

各戦術の入力は命題集合である。複数の命題を組み合わせることで、単一命題からは見えない方向が見えることがある。

- 適用例: {砂漠の真ん中で発見された, 死んでいた} →[常識的仮説の排除]→ {歩いてこの場所に来たのではない}
  - D-1 単体では到着方法を問う動機は弱いが、{D-1, D-2} の組み合わせは「なぜこんな場所で死んでいるのか → どうやって来たのか」という方向を自然に指し示す

### 戦術の組み合わせ

戦術は入れ子に組み合わせて使える。上位方針として一つの戦術を取り、その過程でサブ戦術を使う構造が可能。

- 適用例: 常識的仮説の排除の中で段階的絞り込みを使う
  - {砂漠の真ん中, 死んでいた} →[常識的仮説の排除 > 段階的絞り込み]→ 歩いて来たか → 乗り物で来たか → 地上の乗り物か
