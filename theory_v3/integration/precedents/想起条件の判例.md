# 想起条件の判例

想起条件の適切さを判断する際の判例集。

## 原則

想起条件は、reveals される命題の fc（formation_conditions）に記載された命題群が known のとき、**推論なしに質問が頭に浮かぶか**で評価する。fc の命題群から質問へ至るまでに複雑な推論が介在する場合、fc が不十分である。

「1段階推論として妥当」と「自然に想起される」は別の基準。推論として成立しても、推論を経ないと想起できないなら fc が不十分。

**評価は fc に記載された命題のみに基づいて行う。** プレイヤーがその時点で蓄積している他の知識を暗黙に参照してはならない。fc に含まれていない命題が想起に必要なら、それを fc に追加するか、跳躍と判定する。

## 参照元: turtle_soup 05（ウミガメのスープ）

### 問題文（S）

ある男がレストランに入り、「ウミガメのスープ」を注文した。一口飲んだ男は店員に「これは本当にウミガメのスープか？」と尋ねた。店員は「はい、間違いなくウミガメのスープです」と答えた。男はその夜、自殺した。なぜだろう？

### 真相（T）

男は船乗りで、かつて仲間と船で遭難し、食料が尽きた。仲間の一人が死に、残った仲間は男に「ウミガメのスープだ」と言ってスープを飲ませた。実際にはそれは死んだ仲間の肉で作ったスープだった。レストランで本物のウミガメのスープを飲んだ男は、味が違うことに気づき、あのとき飲まされたものの正体を悟り、絶望して自殺した。

### S 命題（initial_confirmed）

| ID | 内容 |
|----|------|
| S1 | ある男がいた |
| S2 | 男はレストランに入った |
| S3 | 男は「ウミガメのスープ」を注文した |
| S4 | 男はスープを一口飲んだ |
| S5 | 男は店員に「これは本当にウミガメのスープか？」と尋ねた |
| S6 | 店員は「はい、間違いなくウミガメのスープです」と答えた |
| S7 | 男はその夜自殺した |
| S8 | レストランがあった |
| S9 | 店員がいた |
| S10 | 男はスープの真偽を疑った |
| S11 | 男は飲んでから確認した |
| S12 | スープは本物のウミガメのスープだった |

### 中間命題・分割命題

| ID | 内容 | T での真偽 |
|----|------|-----------|
| I1 | 男は以前にも「ウミガメのスープ」を飲んだことがある（と信じている） | 真 |
| P1 | 男は過去に「ウミガメのスープ」だと信じて実際はウミガメではないものを飲んだ経験がある | 真 |
| P2 | その過去の体験は海での遭難に起因する | 真 |
| P3 | 偽りのスープは他者が意図的に「ウミガメのスープだ」と嘘をついて飲ませたものだった | 真 |
| N3b | 遭難中に仲間の一人が死んだ | 真 |
| P4 | 偽りのスープの原材料は人間の肉だった | 真 |

## 跳躍あり

### Q1b「過去に飲んだことがあると認識していますか？」（ブラッシュアップ前）

- **fc**: I1 の fc = [[S5, S10, S11]]（ブラッシュアップ前。修正後は [[S5, S10, S11, N1a]]）
- **判定**: 跳躍
- **理由**: 「疑った＋飲んでから確認した」から「過去に飲んだ認識があるか」に至るには、比較基準の推論が介在する。反射的に浮かぶ質問ではない。
- **解消**: N1a（味の不味さが理由ではない）を fc に追加。「味が理由でない → 比較基準がある → 以前飲んだ？」が自然になる。

### Q2b「海で遭難しましたか？」（ブラッシュアップ前）

- **fc**: P2 の fc = [[P1]]（ブラッシュアップ前。修正後は [[N2b, S3]]）
- **判定**: 跳躍
- **理由**: 「偽物のウミガメのスープを飲んだ経験がある」から「海で遭難」への連想は自然に成立しない。偽物を飲む状況は多様であり、遭難は特殊な一例にすぎない。
- **解消**: 段階的絞り込み（N2a: 日常でない → N2b: 極限状態）を経て、S3（ウミガメのスープ = 海の文脈）を fc に追加。「極限状態＋ウミガメのスープ → 海で遭難？」が自然になる。

### Q3a「嘘をついて飲ませましたか？」（ブラッシュアップ前）

- **fc**: P3 の fc = [[P2]]（ブラッシュアップ前。修正後は [[N3a, P1]]）
- **判定**: 跳躍（境界）
- **理由**: 遭難文脈が与えられた状態での連想として許容可能な面もあるが、「嘘をついて飲ませた」という具体的な行為への到達には推論が介在する。遭難→食料問題→代替食料→欺瞞、という複数段の連想が必要。
- **解消**: N3a（仲間がいた）と P1（偽物を飲んだ）を fc に追加。「仲間がいた＋偽物を飲まされた → 仲間が騙した？」が自然になる。

## 問題なし

### Q1a「味が不味いからですか？」

- **fc**: N1a の fc = [[S5, S10, S11]]
- **判定**: OK
- **理由**: 「スープの真偽を確認した」という事実から、「味がおかしかったのか？」はほぼ反射的に浮かぶ質問。推論を要しない。

### Q1c「本物でしたか？」

- **fc**: P1 の fc = [[I1, S12]]
- **判定**: OK
- **理由**: 「過去に飲んだと認識している」＋「今のは本物」が既知なら、「では過去のは本物だったのか？」は直接的な対比として自然に浮かぶ。

### Q2a「日常生活の出来事ですか？」

- **fc**: N2a の fc = [[P1]]
- **判定**: OK
- **理由**: 過去の体験があると分かったとき、「どんな状況だったのか？」と状況を問うのは自然な第一問。

### Q4a「食用動物の肉でしたか？」

- **fc**: N4a の fc = [[P3]]
- **判定**: OK（想起としては）
- **理由**: 「偽って飲ませた」と知れば「では実際の中身は何だったのか？」は自然に浮かぶ。材料を問う質問としての想起は問題ない。
- **注意**: この質問には想起とは別の問題がある（後述「想起の例外」参照）。

### Q4b「人間の肉でしたか？」

- **fc**: P4 の fc = [[N4a, N3b]]（食用動物の肉でない＋仲間が死んだ）
- **判定**: OK
- **理由**: 「食用動物の肉ではない」＋「遭難中に仲間が死んだ」が既知なら、「人間の肉か？」は自然に浮かぶ。

修正前は P4 fc = [[P3]] で跳躍があった（P3 単体から人肉への連想は不自然）。N3b を連鎖内の分岐として追加し、P4 の fc に合流させることで解消。詳細は `戦術連鎖の構造の判例.md` を参照。
