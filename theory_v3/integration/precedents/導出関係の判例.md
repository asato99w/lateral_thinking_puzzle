# 導出関係の判例

導出関係分析（Phase 2 Step 3）における共通具象元の発見と誤りパターンの判例集。

## 原則

導出関係は T からの演繹構造に基づく。演繹は具象→抽象（情報を落とす方向）であり、中間命題は複数の分割命題より具象的（情報が多い）でなければならない。

中間命題の二重要件:
1. **演繹的分岐**: 異なる情報を落とすことで複数の分割命題が得られる
2. **モデル非自明分割**: モデル集合を非自明に分割する（true_models ≠ {T} のみ）

---

## 参照元: desert_man 02（砂漠の男）

### 問題文（S）

砂漠の真ん中で裸の男が死んでいた。手には折れたマッチ棒を握りしめている。

### 真相（T）

男は数人で熱気球に乗っていたが、高度が下がり始めた。荷物を全て捨て、服も脱いで軽量化したが足りず、くじ引き（マッチの長さ）で一人が飛び降りることになった。短いマッチを引いた男が飛び降りて死亡した。

### 分割命題

| ID | ラベル | true_models |
|----|--------|-------------|
| P1 | 男は乗り物に乗っていた | T, M3, M7 |
| P2 | 男は空中にいた | T, M7 |
| P3 | マッチ棒は物事を決めるために使われた | T, M1, M5, M7, M8, M9 |
| P4 | 男は複数人のグループの中にいた | T, M1, G1, M5, M6, M7, M8, M9 |
| P5 | グループの中で誰かが命を落とす決定がなされた | T, M5 |
| P6 | 乗り物の重量を減らす必要があった | T |

---

## 共通具象元の発見

### OK: P3 と P5 の共通具象元

P3（マッチで決め事）と P5（命の決定）に対して、T には「マッチのくじ引きで犠牲者を決めた」という一つの事実がある。この事実は P3 と P5 の両方より具象的（両方の情報を含む）であり、異なる情報を落とすことで各分割命題が演繹される。

```
「マッチのくじ引きで犠牲者を決めた」
  →（「犠牲者」を落とす）→ P3: マッチで決め事をした
  →（「マッチ」を落とす）→ P5: 命を落とす決定がなされた
```

この中間命題は:
- 演繹的分岐: P3 と P5 の両方を演繹する ✓
- モデル分割: true_models = {T, M5}（M5 も決闘でマッチくじ引きにより命に関わる決定を行う）。{T} のみではない ✓

### 教訓

- 共通具象元は T の事実の中に既に存在している。T の事実を個別の分割命題と照合するのではなく、「複数の分割命題に跨る事実」として捉えることで発見できる
- 情報を落とす方向が 2 つ以上あるかどうかが、分岐の存在の判定基準となる

---

## 誤った方向性

### NG: 因果と演繹の混同

P2（空中）→ P6（重量削減）→ P5（命の決定）→ P3（マッチで決め事）という因果の連鎖を導出関係として扱おうとした。

**問題**: 因果は演繹ではない。「空中にいた」から「重量削減が必要」は因果的には自然だが、「空中にいた」という事実から「重量削減」は論理的に導かれない（空中にいても重量削減が不要な場合は多い）。

**教訓**: 導出関係は「A を知れば B が論理的に導かれる」という演繹であり、「A が原因で B が起きた」という因果とは区別すること。

### NG: 情報の欠落

「くじ引きで犠牲者を決めた」から P3（マッチ棒は物事を決めるために使われた）を導出しようとした。

**問題**: 「くじ引き」は「マッチ」を含まない。P3 にある「マッチ棒」という具象性が中間命題で失われており、中間命題から P3 を演繹できない。

**教訓**: 演繹は具象→抽象（情報を落とす方向）。中間命題は分割命題より情報が多くなければならない。中間命題に分割命題の情報が含まれていなければ、演繹関係は成立しない。

---

## 抽象化による発見

### 分割命題の抽象化から共通構造を見出す

P6（重量削減）と P5（命の決定）を個別に見ると無関係に見えるが、抽象化すると共通構造が浮かぶ:
- P6 → 「困難への対処」
- P5 → 「困難への対処（最終手段）」

この共通の抽象構造「困難への段階的対処」が、「軽量化では足りず犠牲者選択に至った」という共通具象元の存在を示唆する。

**注意**: この操作は発見的であり、抽象化の仕方によって異なる構造が見える。判例のパターンを参考に、適用者が自らの判断で試みること。
