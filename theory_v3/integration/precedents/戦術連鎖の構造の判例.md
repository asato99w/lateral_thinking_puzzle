# 戦術連鎖の構造の判例

戦術連鎖の構造に関する判例集。連鎖は一本鎖（直列）である必要はなく、分岐・合流を持つ構造として表現できる。

## 原則

戦術連鎖の構造は fc（formation_conditions）が自然な想起を反映するように決定する。ある命題の fc が連鎖内の異なる位置にある複数の先行命題を必要とする場合、連鎖を分割して「連鎖間参照」とするのではなく、分岐・合流を持つ1つの連鎖として表現する。

連鎖を一本鎖と捉えるのは不要な制約であり、この制約によって以下の問題が生じる:
- 命題の自然な出現位置と連鎖の境界が合わなくなる
- 連鎖間の参照が生まれ、構造の見通しが悪くなる
- 質問の順序が不自然になる

## 参照元: turtle_soup 05（ウミガメのスープ）

### 問題文（S）

ある男がレストランに入り、「ウミガメのスープ」を注文した。一口飲んだ男は店員に「これは本当にウミガメのスープか？」と尋ねた。店員は「はい、間違いなくウミガメのスープです」と答えた。男はその夜、自殺した。なぜだろう？

### 真相（T）

男は船乗りで、かつて仲間と船で遭難し、食料が尽きた。仲間の一人が死に、残った仲間は男に「ウミガメのスープだ」と言ってスープを飲ませた。実際にはそれは死んだ仲間の肉で作ったスープだった。レストランで本物のウミガメのスープを飲んだ男は、味が違うことに気づき、あのとき飲まされたものの正体を悟り、絶望して自殺した。

### 関連する命題

| ID | 内容 | fc |
|----|------|-----|
| N3a | 遭難時に仲間がいた | [[P2]] |
| N3b | 遭難中に仲間の一人が死んだ | [[N3a]] |
| P3 | 偽りのスープは他者が意図的に嘘をついて飲ませた | [[N3a, P1]] |
| N4a | 偽りのスープの材料は一般的な食用動物の肉ではなかった | [[P3]] |
| P4 | 偽りのスープの原材料は人間の肉だった | [[N4a, N3b]] |

## 判例: 分岐合流による跳躍の解消

### 修正前の構成（2連鎖・直列）

```
C3: P2 → N3a(仲間がいた) → P3(嘘をついた)
C4: P3 → N4a(食用動物でない) → P4(人肉)
```

P4 の fc = [[N4a]] または [[P3, N4a]] としても、「食用動物でない → 人肉？」は跳躍が残る。食用動物以外の候補は多い（珍獣、昆虫など）。人肉に至るには「仲間が死んだ」という情報が必要。

しかし「仲間が死んだ」(N3b) の自然な出現位置は遭難の掘り下げ（N3a の直後）であり、C4（肉の正体の絞り込み）に置くと質問順序が不自然になる。

**問題の根本**: 連鎖を一本鎖と捉えたために、N3b の自然な配置場所（C3 内）と使用場所（P4 の fc）が連鎖境界をまたいでしまう。

### 修正後の構成（1連鎖・分岐あり）

```
C3: P2 → N3a(仲間がいた) → N3b(仲間が死んだ) → P3(嘘をついた) → N4a(食用動物でない) ─┐
                              │                                                          ├→ P4(人肉)
                              └──────────────────────────────────────────────────────────┘
```

- N3b は C3 内で自然な位置（N3a の直後）に出現する
- N3b は P4 の fc にも合流する: P4 fc = [[N4a, N3b]]
- 「食用動物でない」＋「仲間が死んだ」→「人肉？」は自然に想起される
- 連鎖間参照は不要。1つの連鎖内の分岐として完結する

### 教訓

- 連鎖は一本鎖（直列）である必要はない
- 命題の fc が連鎖内の異なる位置にある先行命題を必要とする場合、分岐・合流として表現する
- 分岐は連鎖の自然な構造であり、「連鎖間参照」という不自然な構造を回避できる
