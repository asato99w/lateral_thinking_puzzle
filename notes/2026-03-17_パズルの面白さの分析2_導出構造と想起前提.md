# パズルの面白さの分析2：導出構造と想起前提

前ノート（2026-03-17_パズルの面白さの分類軸_登場人物の推論.md）の続き。

## 導出構造の詳細分析

各パズルの真相を「前提 → 導出」の流れとして記述した。

### 砂漠の男（前提：熱気球で高度が下がっていた）

- 高度低下 → 軽量化が必要 → 荷物を捨てる → 服も脱ぐ → **裸**
- まだ足りない → 一人降りるしかない → くじ引き → **折れたマッチ棒**
- 飛び降りた → **砂漠の真ん中で死んでいた**

### 毒キノコ（前提：男は山で遭難していた）

- 遭難中 → 食料を探す → キノコ狩りの場所にいる
- **毒キノコしかない** → 食用は誰かが採った → 人が来ている → 助かる → **安堵**

### ウミガメのスープ（前提：過去に遭難し、仲間の肉を「ウミガメのスープ」と偽って食べさせられた）

- レストランで**ウミガメのスープを注文** → **一口飲む** → 味が違う
- → 過去のは本物じゃなかった → 何だったか悟る → 絶望 → **自殺**

### カセットテープ（前提：第三者が殺して自殺に偽装した）

- 犯人が遺書を録音させた → 銃で殺した → **遺書のテープ**が残る
- 偽装のためテープを巻き戻した → テープが最初から再生できる状態
- → 本人は録音後に死んでいるので巻き戻せない → **殺人と断定**

### バーの男（前提：男はしゃっくりが止まらなかった）

- しゃっくり → 水で治したい → **「水をください」**
- バーテンダーが気づく → 驚かせれば治る → **銃を向けた**
- 止まった → 水は不要 → **「ありがとう」と帰った**

### 禁じられた地下室（前提：女性は監禁されていた）

- 監禁者の命令 → **開けてはいけないドア**
- 開けた → 普通の外の世界 → **何も異常なものはなかった**
- 自分が監禁されていたと認識 → **警察に駆け込んだ**

## 発見：想起されるべき前提

導出の流れを並べた結果、各パズルの「前提」の性質に重要な差があることが判明した。

### 想起前提があるパズル

以下のパズルでは、問題文に書かれていない隠れた状況（前提）があり、
それを思いつくことで導出が始まる。

| パズル | 想起前提 |
|---|---|
| 砂漠の男 | 熱気球 |
| 毒キノコ | 遭難していた |
| ウミガメのスープ | 過去に人肉を食べさせられた |
| バーの男 | しゃっくり |

### 想起前提がないパズル

**カセットテープ**: 「第三者が殺して偽装した」は導出の結論そのものであり、
想起すべき隠れた前提ではない。推論は問題文の物理的事実から直接行われる。
論理パズルに近い性質を持つ。質が相対的に低いと感じる理由はここにある。

**禁じられた地下室**: 「監禁されていた」は前提ではあるが、
そこからの導出が貧弱で、問題文の要素を豊かに説明しない。

## 暫定的な整理

面白さに関わる構造として、以下が見えてきた。

1. **急激な緊張の緩和** — 面白さの核心メカニズム（前ノート）
2. **想起前提の存在** — 水平思考パズルらしさに関わる。問題文にない隠れた状況を
   思いつくことで導出が始まる
3. **導出の豊かさ** — 一つの前提から問題文の多くの要素が説明される
   （科学理論の評価に類似：単純な前提から多くを説明できる理論が良い理論）
4. **登場人物の推論** — 緊張の緩和を実現する便利な実装（前ノート）。
   必須ではないが多用される
