# 逆算連鎖と探索領域の分析

## 出発点

経路やパラダイムの分析品質が低いまま積み重ねても構造がおかしくなる。理論を一旦忘れ、T に至る論理展開を逆算的に考えることから再出発する。

## T からの逆算連鎖

S を忘れ、T の論理構造を「なぜ？」の連鎖で逆算する：

```
男は自殺した
  ← なぜ？ 自分が人肉を食べていたと悟ったから
人肉を食べていたと悟った
  ← なぜ？ 過去に「ウミガメのスープ」として飲んだものが本物ではなかったと判明したから
本物ではなかったと判明した
  ← なぜ？ 本物のウミガメのスープを飲んだら、記憶の味と違ったから
記憶の味がある
  ← なぜ？ 過去に「ウミガメのスープ」と称されたものを飲んだから
「ウミガメのスープ」と偽られた
  ← なぜ？ 遭難中に仲間の肉で作ったスープを、そう偽って提供されたから
仲間の肉でスープを作った
  ← なぜ？ 食料が尽き、仲間が死んだから
```

各ステップが「なぜ？」に対する必然的な回答であり、恣意性が少ない。この連鎖に含まれる事実のグループがパズルのピースとなる。

## パズルのピース

逆算連鎖から抽出した事実群：

- **a**: 食料が尽き、仲間が死んだ
- **b**: 仲間の肉でスープを作った
- **c**: 「ウミガメのスープ」と偽って提供された
- **d**: 男はそれを飲んだ（味の記憶がある）
- **e**: 本物を飲んだら味が違った
- **f**: 過去のものは本物ではなかったと判明した
- **g**: 正体を悟り自殺した

## ピースの依存関係

**独立に発見可能なピース**（前提なしで戦略から直接到達しうる）：
- **a**（遭難、仲間の死）: 「男は何者か」「状況の絞り込み」から
- **c**（「ウミガメのスープ」と偽られた）: 「過去にスープと関わりがあるか」から
- **e**（本物を飲んだら味が違った）: S の確認行為から比較的近い

**依存するピース**：
- **d**（過去に飲んだ）: c を含む過去の知識が前提
- **f**（過去のものは本物ではなかった）: d + e の論理的帰結
- **b**（仲間の肉でスープを作った）: a + c からの推論
- **g**（悟り→自殺）: 全体の理解から

## 回収順序の自由度と制約

制約は論理的依存関係（f は d + e なしに成立しない等）。自由度は独立ピースの発見順序にある。

回収順序の例：
```
順序 1: e → c → d → f → a → b → g
（味の違和感から入り、過去を辿り、背景に至る）

順序 2: a → c → b → e → d → f → g
（背景から入り、偽りを発見し、現在と接続する）

順序 3: e → a → c → d → f → b → g
（味と背景を別々に進め、途中で結合する）
```

異なる回収順序が異なるプレイ体験を生む。

## 探索領域

独立に発見可能なピース（a, c, e）はそれぞれ単一の事実ではなく、探索可能な領域を開く：

- **探索領域 e**（味の違い）: なぜ違う？何との比較？記憶の対象は？
- **探索領域 c**（偽りのラベル）: なぜ偽った？誰が？正体は？
- **探索領域 a**（遭難、仲間の死）: 何が起きた？どう生き延びた？食料は？

各探索領域の特徴：
- 独立に発見・探索できる
- 内部で深掘りが可能
- 引力圏を持つ（その領域内の探索に留まりうる）
- T への到達には探索領域間の接続・統合が必要

依存ピース（d, f, b, g）は探索領域間の接続点に相当する。例えば b（仲間の肉でスープを作った）は a と c の統合から生まれる。

## パラダイム概念への疑問

この構造は「支配的な解釈枠が別の解釈枠に置き換わる」というクーン的パラダイムシフトとは異なる。置換ではなく統合のメカニズムである。

一方、S の自然な読み（現在の出来事としての解釈）はクーン的パラダイムに近い性質を持ち、探索の出発点・引力圏として機能する。

パラダイムという概念がこのパズル理論に適切かどうかは再検討が必要。一旦、独立に発見可能なピースの集合を「探索領域」と定義し、パラダイムとは区別して扱う。

## 探索のメカニズム：直接観測と推論

探索領域への入り方を「戦略駆動 vs アノマリー駆動」と分類するのは本質的でない。アノマリーも戦略的な探索によって発見される。戦略は常に探索の手段であり、入り方の分類ではなく、探索が生み出すものの構造が本質。

### 事実の獲得と推論

探索が生み出すものは大きく二つに分かれる：

**1. 直接観測**: 戦略による質問で事実を得る
- 例：「船乗りだった」「遭難した」「仲間が死んだ」「食料が尽きた」

**2. 推論による跳躍**: 獲得した事実から新しい理解を導く
- **リンク**: 複数の事実を結合して新しい理解を生む（構成的）
  - 例：「仲間が死んだ」+「食料が尽きた後にスープがあった」→ 仲間の肉がスープになった
- **アノマリー**: 事実間の矛盾を認識し、再解釈を迫る（変容的）
  - 例：「ウミガメのスープを飲んだことがない」+「味の記憶がある」→ 飲んだものはウミガメのスープではなかった

### リンクとアノマリーの違い

- **リンク**: 事実を足し合わせて新しい事実を導く。既存の理解と矛盾しない
- **アノマリー**: 事実間の矛盾が既存の理解の変容を迫る。再解釈が必要

どちらも推論だが、リンクは構成的（新しい理解を積み上げる）、アノマリーは変容的（既存の理解を書き換える）という点で働き方が異なる。

## 今後の検討事項

- リンクとアノマリーの構造的分析（ウミガメの全ピースに対する適用）
- 探索領域内でのリンク・アノマリーの発生パターン
- 探索領域間の接続がリンクとアノマリーのどちらで起きるか
- S の役割の再定義（最初の解釈枠としての S と、探索領域との関係）
- パラダイム概念の再検討（必要か、別の概念で置き換えるか）
- 他のパズルでの検証
