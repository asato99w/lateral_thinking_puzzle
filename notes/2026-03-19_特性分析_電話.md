# 特性分析：電話

## Step 1: テンプレート展開

### 1. 状態
- 通話中 / 待機中 / 電源オフ
- 鳴っている / 鳴っていない
- 繋がっている / 繋がらない（圏外、電源オフ、着信拒否）
- マナーモード / 通常モード / サイレント
- 受話器を上げている / 下りている（固定電話）
- スピーカーモード / 通常モード
- バッテリーあり / 切れている（携帯）
- 保留中
- 留守番電話が作動中

### 2. ルーティン
- 着信 → 着信音 → 出る → 通話 → 切る
- 発信 → 相手が出る → 通話 → 切る
- 相手が出ない → 呼び出し続く → 諦める / 留守電にメッセージ
- 就寝時にマナーモード → 起床時に戻す
- 不在時 → 着信履歴 / 留守電が録音される

### 3. 帰結
- 電話が鳴る → 誰かが連絡しようとしている
- 出ない → 不在 / 出られない / 意図的に出ない / 聞こえない
- 繋がらない → 圏外 / 電源オフ / 電池切れ
- 通話中の声 → 相手を特定 / 感情がわかる
- 通話後の行動変化 → 通話で何かを知らされた
- 留守電にメッセージ → 連絡を試みた証拠
- 着信履歴 → いつ、誰から、何回の記録
- 通話後に泣く/笑う/走り出す → 内容が感情を動かした

### 4. 遷移条件
- 鳴る：外部から着信（相手の意思）
- 出る/出ない：本人の意思または状況的制約
- 繋がる/繋がらない：通信環境、相手の状態
- マナーモード切替：意図的行為
- 電池切れ：時間経過（自然）
- 留守電録音：自動（設定依存）

### 5. 推論
- 何度かけても出ない → 何かあった / 無視 / 聞こえない
- 電話後に行動が急変 → 重要な情報を受け取った
- 電源が切られている → 連絡を断ちたい / 電池切れ / 故障
- 深夜の着信 → 緊急の用事
- 留守電を聞いて顔色変化 → 衝撃的内容
- かけようとしてやめる → 躊躇（伝えにくい内容）
- 通話中に声が変わる → 感情の動き / 別の人物
- 登録外番号から着信 → 知らない人 / 未登録の知り合い

### 6. 関係性
- 聴覚を通じた遠隔リアルタイムコミュニケーション
- 離れた人と繋がる手段
- 声による相手の識別
- リアルタイム性（即時性）
- 一方的アクセス（かける側が主導権）
- 社会的義務（出るべきという規範）
- プライバシー（通話内容は当事者のみ）
- 不在の証拠（出ないこと自体が情報）

## Step 2: 本質的機能の特定

**本質的機能：** 聴覚を通じた遠隔リアルタイムコミュニケーション

他の対象との違い：
- コンタクト/耳栓/手袋は「自分の感覚を変える」道具
- 電話は「離れた人と声で繋がる」道具
- 聴覚に関わるが、遮断ではなく**接続**

**非本質的特性（ミスリード素材）：**
- 電話機の種類（固定/携帯）
- 電話番号
- 着信履歴の詳細
- バッテリー残量

## Step 3: 演繹構造の導出

### 演繹1（因果：着信→存在の証明）
- 前提群：電話が鳴った ＋ 電話は相手がかけなければ鳴らない
- 演繹：誰かがこの番号にかけた → その人はその時点で電話を使える状態だった
- 反証：相手はかけられる状態でなかったはず → 別の人がかけた / 予約発信 / 転送

### 演繹2（因果：出ないことの含意）
- 前提群：電話をかけた ＋ 相手の電話は正常 ＋ 相手は手元にある
- 演繹：出るはず
- 反証：出ない → 不在 / 出られない / 意図的 / 聞こえない / 何かが起きた

### 演繹3（因果：通話後の行動変化）
- 前提群：通話前は平静 ＋ 通話した ＋ 通話後に行動が急変
- 演繹：通話内容が原因
- 反証：通話内容は平凡だったと言う → 嘘 / 通話以外の原因 / 背景音が原因

### 演繹4（同一性：声による人物特定）
- 前提群：電話の声はAの声 ＋ 声でAだと判断した
- 演繹：通話相手はA
- 反証：実はAではなかった → 声が似た別人 / 録音 / 声を変えている

### 演繹5（能力：電源オフの含意）
- 前提群：相手の電話が電源オフ ＋ 相手は普段電源を切らない
- 演繹：通常ではない状態にある
- 反証：本人は普通だと言う → 電池切れ / 意図的 / 圏外

### 演繹6（保存・持続：着信履歴）
- 前提群：着信があった ＋ 履歴は自動記録 ＋ 誰も操作していない
- 演繹：着信履歴に記録があるはず
- 反証：記録がない → 履歴が消された → 誰が、なぜ

### 演繹7（時間変化：不在着信の蓄積）
- 前提群：相手がいなくなってN時間 ＋ 親しい人が心配してかける
- 演繹：不在着信がN時間分蓄積しているはず
- 反証：0件 → 誰も心配していない / 電話が別の場所 / 誰かが対応した

### 演繹8（共起：携帯と場所）
- 前提群：携帯電話は持ち主が常に携帯 ＋ 携帯がここにある
- 演繹：持ち主はここにいる / ここに来た
- 反証：持ち主はいない → 忘れた / 意図的に残した → なぜ

### 演繹9（因果：圏外の含意）
- 前提群：相手に電話した ＋ 「電波の届かない場所に…」
- 演繹：相手は圏外にいる
- 反証：相手がいるはずの場所は圏内 → 別の場所 / 電源オフ / 電話を持っていない

### 演繹10（同一性：電話番号の一致）
- 前提群：この番号はAの番号 ＋ この番号から着信
- 演繹：Aからの着信
- 反証：Aはかけていないと言う → 他の人がAの電話を使った / 番号偽装

## Step 4: 所感

- **演繹の豊かさが非常に高い**。これまでの分析対象の中で最も多方向。
  接続/非接続、声の同一性、通話後の行動変化、着信履歴、場所の推定、
  不在着信の蓄積など多層的
- **最も面白い演繹構造**：演繹3（通話後の行動変化）と演繹4（声による人物特定）。
  通話後に急に行動が変わるのは強い緊張を生み、
  声が実は別人だったという展開は隠れアノマリーパターンに発展しうる
- **逆説構造**：「繋がりたいのに繋がらない」「繋がりたくないのに繋がる」を内在。
  演繹2に耳栓との接続がある（電話に出ない→聞こえない→耳栓）
- **既存パズルとの類似**：声の同一性（演繹4）はウミガメのスープの味覚比較と同型。
  通話後の行動変化（演繹3）は「食後の行動変化」と同型
- **独自の強み**：双方向性、リアルタイム性、不在そのものが情報になること。
  「出ないこと」が多様な推論を生む点は他の対象にない特徴
- **前提群の豊かさ**：非常に高い。聴覚との直接的関係に加え、
  遠隔通信が対人関係・時間・場所の推論を全て含む
