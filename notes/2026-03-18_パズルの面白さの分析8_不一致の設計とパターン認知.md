# パズルの面白さの分析8：不一致の設計とパターン認知

前ノート（2026-03-18_パズルの面白さの分析7_2つのパターンとアノマリー.md）の続き。

## 設計の出発点

隠れた緊張パターンのパズルを設計するにあたり、出発点を検討した。

### 当初のステップ案（却下）

1. 登場人物の隠れた緊張を設定する
2. アノマリーとなる観測を設計する
3. 推論の連鎖を構成する
4. 問題文を構成する

→ ステップ1が抽象的すぎて設計として機能しない。

### アノマリーの構造分析

隠れた緊張パターンにおけるアノマリーの観測には、
登場人物がある種のパラダイムを持っていることが必要。

構造：
- 登場人物がパラダイム（信念・経験的知識）を持つ
- パラダイムから具体的な予測が**導出**される
- 観測がその予測と一致しない → アノマリー

各パズルでの具体例：

**ウミガメのスープ:**
- パラダイム：かつてウミガメのスープを食べた
- 導出：ウミガメのスープはあの味のはず
- 観測：味が違う → アノマリー

**毒キノコ:**
- パラダイム：キノコが生える環境にいる（毒キノコが多いことから）
- 導出：食用キノコもある程度存在するはず
- 観測：食用キノコが一つもない → アノマリー

ポイント：パラダイム自体がパズルの鍵（隠れた前提）になる。
不一致の設計から始めることで、鍵・アノマリー・推論の連鎖が
一貫して導出される。

### 改訂した設計の出発点

「パラダイム → 導出された予測 → 観測との不一致」の設計から始める。

その先のステップ（問題文の構成等）については、
不一致の設計ができた上で判断する。

## 不一致のチャネルの列挙

不一致がどのような感覚・認知チャネルを通じて成立するかを列挙した。

**味覚・嗅覚:** 食べ物の味、場所の匂い、人の匂い、食材の匂いから成分判別、
水の味から水源判別、酒の味から産地判別、料理の味から作り手判別

**触覚・体感:** 物の重さ、手触り・質感、温度、布の素材感、地面の感触、
空気の湿度・気圧、風の強さ・方向、水流・水温、振動パターン

**聴覚:** 人の声、足音、機械音のリズム、空間の反響、動物の鳴き声、
静寂（あるはずの音がない）、楽器の音程・音色、風の音から地形判別、雨音

**視覚:** 自分の顔・姿、筆跡、物の色（熟し具合等）、影の方向から時刻判別、
星の位置から方角判別、植物の成長、水位変化、摩耗・劣化、埃の量、
煙の動き、雪の状態、光の色合い、潮の干満、人の表情、道の景色

**身体感覚:** 体重・体力、心拍・脈拍、呼吸のリズム、痛みから原因判別、
移動時間、平衡感覚、空腹度から経過時間判別

**動物・自然の反応:** 動物の行動パターン、吠える・吠えない、
鳥の行動から天候判別、虫の鳴き方から気温判別

**社会的・知識的:** 人の習慣・行動パターン、人数と騒音の関係、
泣き方から要求判別、言葉遣いから出身判別、服装から職業判別、
貨幣の手触りから真贋判別

**量・数:** 在庫・残量の変化、物の数の不一致、規則的パターンの乱れ、
消費速度から人数推測

## 列挙から得られた知見

列挙した項目のほとんどに共通して「パターン認知」が基盤にある。
パラダイムから予測を導出すること自体が、過去の経験からパターンを抽出し
現在に適用する行為であり、認知の基本と言える。

ただし、パターン認知の性質の違いはパズルの重要な要素である。
ウミガメのスープ（味覚・身体的）とカセットテープ（論理的・概念的）では
認知の性質が大きく異なり、パズルの体験の質に直結する（分析7の軸と対応）。
チャネルの違いを単なる表面的差異と片付けるべきではない。

## 有望なチャネルの選定

列挙の中から、身体的・経験的で設計の取りかかりになりそうなものを選定した。

- **物の重さ** — 手に取った瞬間にわかる。即座で身体的
- **匂い** — 記憶と感情に強く結びつく。無意識に作動する
- **静寂（あるはずの音がない）** — 不在の認識。毒キノコと似た構造を持ちうる
- **平衡感覚** — 傾きや加速。ごまかしが効かない身体感覚
- **移動にかかる時間** — 慣れた道の身体的な時間感覚
- **水の味** — 微妙だが経験者には確実にわかる

## 「重さ」での不一致の具体的列挙

「重さ」を起点に、「誰が・何の重さを・なぜ違うか」の組み合わせを列挙した。

### 日常生活
- 親が子供を抱き上げる → 予想より重い/軽い
- 毎朝持つ通勤鞄 → いつもより軽い/重い
- ペットを抱き上げる → 体重が変わっている
- 財布を手に取る → 異様に軽い/重い
- スーパーの買い物袋 → 入れた量に対して重さが合わない
- 冷蔵庫から飲み物を取り出す → 空のはずが重い、満タンのはずが軽い

### 職業的経験
- パン職人が生地を持つ → 発酵の具合が違う
- 漁師が網を引く → 異様に重い/軽い
- 郵便配達員が小包を持つ → 表示と重さが合わない
- バーテンダーが瓶を持つ → 中身の量が想定と違う
- 宝石商が石を手に取る → 見た目に対して重さが合わない（偽物）
- 農家が収穫物を持つ → 異様に重い/軽い
- 薬剤師が薬の容器を持つ → 中身の量が違う
- 引越し業者が家具を持つ → 中に何か入っている
- 看護師が患者を支える → 急激な体重変化
- 硬貨を扱う人 → 手触りと重さで偽造に気づく

### 特殊な状況
- 棺を運ぶ → 人が入っているはずなのに軽い/重い
- 銃を持つ → 弾が入っている/入っていない
- 楽器ケースを持つ → 楽器以外のものが入っている
- スーツケースを受け取る → 自分のものと重さが違う
- 封筒を受け取る → 書類にしては重い
- 本を手に取る → 中がくり抜かれている
- 贈り物の箱 → 大きさに対して軽すぎる/重すぎる
- 金庫を動かす → 空のはずが重い
- 水筒を振る → 入れていないはずの水が入っている
- リュックを背負う → 中身を入れた覚えがないのに重い

### 自然・環境
- 石を拾う → 見た目の大きさに対して異様に軽い/重い（中空、鉱物）
- 果物を持つ → 熟し具合と重さが合わない
- 卵を持つ → 普通より重い/軽い
- 薪を持つ → 異様に重い（水を含んでいる）/軽い（中が虫食い）
- 釣り上げた魚 → サイズに対して重い（何かを飲み込んでいる）

### 重さが違う理由の類型
- 中身が抜かれた/追加された
- 別の物にすり替えられた
- 物が改造されている（くり抜き、充填）
- 水分の吸収/蒸発
- 何かが隠されている
- そもそも違う物（外見が同じ別物）
- 認知する側の変化（体調、環境）

## 列挙から得られた観察

- **棺**の重さの不一致は、それ自体がドラマチックで含意が大きい。
  軽ければ中身がない、重ければ何かが追加されている
- 列挙の大半は「物が変わった」ケースだが、理由の類型の最後に挙げた
  **「認知する側の変化」は構造が異質**で面白い。
  物は同じなのに持つ側の身体が変わっている場合
- 重さの不一致は方向が「重い/軽い」の2種類しかない。
  味や匂いに比べるとバリエーションが限られる可能性がある
