# recall 認知的妥当性の検討

## 発端

砂漠の男 Q-1「男は地上からこの場所に来ましたか？」の recall_conditions が `[[D-1]]`（砂漠の真ん中で発見された）。砂漠にいるという事実だけでは到着手段を問う認知的動機にならない。

## 試行 1: Pattern 5（認知的動機の補強）— 失敗

### やったこと

1. 検査.md にレイヤー3「想起の認知的妥当性」を追加
2. 調整適用.md にパターン5「認知的動機の補強」を追加
   - 「動機となる中間的な認識（異常の検知、不可解さの認識など）を記述素として追加し、recall_conditions を更新する」
3. D-18「男がこの場所にいること自体が不可解」を追加（formation_conditions: [[D-1, D-2]]）
4. Q-1 の recall を `[[D-1]]` → `[[D-18]]` に変更

### なぜ無効だったか

D-1, D-2 は initial_confirmed なので D-18 はゲーム開始時に即時導出される。プレイヤーの認知状態は変更前と同一。「不可解だ」というラベルを貼っただけで、「不可解さ」→「到着手段」への認知的飛躍はそのまま残っている。

### 見落としていたこと

形式的検証（eval ALL PASS、シミュレーション正常クリア）を通過したことで実質的な改善がなされたと錯覚した。

## 試行 2: recall の非初期記述素への付け替え — 提案段階で却下

### 提案内容

recall_conditions が initial_confirmed だけで充足される質問を検出し、recall を非初期記述素（ゲーム中に reveals で得られるもの）に付け替えることで質問を遅延させる。

### なぜ的外れか

「初期か非初期か」という構造的性質に着目していた。しかし問題の本質は recall の内容から質問内容への認知的接続の質であり、初期/非初期は無関係。

- D-3（裸・初期）→ Q-2「裸の理由は？」: 自然
- D-16（裸の理由あり・非初期）→ Q-1「地上から来た？」: 不自然

データ構造の操作（タイミング操作、中間記述素の挿入）で認知的接続の質を改善しようとしていたこと自体が誤り。

## 現時点の理解

検査すべきは「recall の**内容**を知っているプレイヤーが、その質問の**内容**を思いつく認知的経路が存在するか」。これは構造的性質（初期/非初期、導出/基礎）とは独立した問題。

D-18「不可解だ」が失敗したのは初期完結だからではなく、「不可解」という認識から「到着手段」への認知的経路が成立していないから。
