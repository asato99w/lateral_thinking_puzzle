# 抽象ツリー: ウミガメのスープ

## 遭難:「仲間と海で遭難した」

具象の構成要素: 船乗り、仲間、海、船、遭難、食料が尽きた、仲間が死んだ

各枝は具象の特定の要素を落とし/一般化して抽象化する:

- **海**: 海で遭難した → 海での出来事（「仲間」「食料不足」「死」が落ちる）
- **危機**: 遭難して食料が尽きた → 危機的状況（「海」「船」「仲間」が落ちる）
- **仲間**: 仲間と遭難した → 仲間との困難な体験（「海」「食料不足」が落ちる）
- **生存**: 食料が尽きる状況を生き延びた → サバイバル（「海」「仲間」が落ちる）

## 人肉食:「死んだ仲間の肉でスープを作った」

具象の構成要素: 死んだ仲間、肉、スープ、作った、食料不足が動機

各枝は具象の特定の要素を落とし/一般化して抽象化する:

- **禁忌**: 人肉を食べた → 禁忌の食（「仲間」「スープ」「動機」が落ちる）
- **極限**: 食料不足で人肉を食べた → 極限状態での行為（「仲間」「スープ」が落ちる）
- **死者**: 死んだ仲間の体を利用した → 死者の利用（「食」「スープ」が落ちる）
- **食**: 人肉でスープを作った → 特殊な食材の調理（「仲間」「死」が落ちる）

## 欺瞞:「ウミガメのスープだと偽って飲ませた」

具象の構成要素: ウミガメのスープというラベル、偽り、仲間が主体、飲ませた、男は信じた

各枝は具象の特定の要素を落とし/一般化して抽象化する:

- **ラベル**: 食べ物の名前を偽った → 偽りのラベル（「仲間」「飲ませた」が落ちる）
- **情報**: 男は正体を知らなかった → 情報の非対称（「ラベル」「仲間」が落ちる）
- **関係**: 仲間が偽った → 仲間による欺瞞（「ラベル」「食」が落ちる）
- **意図**: 生存のために偽った → 善意の嘘（「ラベル」「食」が落ちる）

## パターン探索の目的と方針

### 目的

なるべく具象の度合いが高く、かつ独立した分け方を探索すること。

理由: そのような分割では、分割の双方が自由度を保ち、可能なT'が豊富である。二つの具体的な事実が得られているにも関わらず、それらを組み合わせて得られる答えが自明ではないと言える。

### 具象概念の葉集合による表現

具象概念は、葉（フラットな最上位抽象）の集合として捉える。

- 全葉の組み合わせ = 具象（共通して落ちる要素がなくなり、具象に戻る）
- 葉の部分集合 = 抽象化（部分集合が小さいほど抽象的）

各概念について、葉の部分集合の選び方で抽象の度合いと方向を調整できる。

### 2分割と3ケース

分割の仕方は多様にあるが、単純化のため2分割とする。3概念に対しては3ケース:

- (遭難+人肉食) / 欺瞞
- (遭難+欺瞞) / 人肉食
- (人肉食+欺瞞) / 遭難

グルーピングされた側は、2概念の葉を組み合わせてより具象的に表現でき、内部の重複を許容する。残る1概念側は独立性を確保する。

### 評価の指標

- **抽象度**: 選択した葉の数（多いほど具象的）
- **独立性**: 2つの分割間の独立の度合い（定量化は今後の課題）

目標: 独立性を確保しつつ抽象度を最大化（具象に近づける）分割を見つけること。

### パターン探索の成果物

良いバランス（高い具象度かつ十分な独立性）を持つ中間状態は複数存在しうる。パターン探索の成果物はそれらの列挙であり、列挙されたパターン群がプレイヤーの多様な解き筋に対応する。

## パターン探索の結果

### ケース1: (遭難+人肉食) / 欺瞞

**A. グループ側が具体的**
- グループ(8葉):「仲間と海で遭難し、食料が尽き、仲間が死に、死んだ仲間の肉でスープを作った」
- 独立(1葉:情報):「何かの正体を知らなかった」

**B. グループ側が抽象的**
- グループ(2葉:遭難-海, 人肉食-禁忌):「海での出来事があった」「禁忌の食があった」
- 独立(4葉):「ウミガメのスープだと偽って仲間が飲ませ、男は信じた」

**C. 同程度**
- グループ(4葉:遭難-海/危機, 人肉食-禁忌/極限):「海で危機的遭難があり食料が尽き人が死んだ。食料不足から禁忌の食に至った」
- 独立(2葉:情報/意図):「何かを偽り正体を知らせなかった。善意からだった」

### ケース2: (遭難+欺瞞) / 人肉食

**A. グループ側が具体的**
- グループ(8葉):「仲間と海で遭難し食料が尽き仲間が死んだ。仲間がウミガメのスープだと偽って飲ませ、男は信じた」
- 独立(1葉:禁忌):「禁忌の食があった」

**B. グループ側が抽象的**
- グループ(2葉:遭難-仲間, 欺瞞-意図):「仲間との困難な体験があった」「善意の嘘があった」
- 独立(4葉):「死んだ仲間の肉でスープを作った」

**C. 同程度**
- グループ(4葉:遭難-海/仲間, 欺瞞-情報/意図):「仲間と海で遭難し仲間が死んだ。何かの正体を善意で偽った」
- 独立(2葉:禁忌/食):「人肉という禁忌の食材が調理された」

### ケース3: (人肉食+欺瞞) / 遭難

**A. グループ側が具体的**
- グループ(8葉):「死んだ仲間の肉でスープを作り、ウミガメのスープだと偽って飲ませた」
- 独立(1葉:海):「海での出来事があった」

**B. グループ側が抽象的**
- グループ(2葉:人肉食-食, 欺瞞-ラベル):「特殊な食材の調理があった」「食べ物に偽りのラベルがあった」
- 独立(4葉):「仲間と海で遭難し、食料が尽き、仲間が死んだ」

**C. 同程度**
- グループ(4葉:人肉食-禁忌/死者, 欺瞞-ラベル/情報):「死んだ仲間の肉という禁忌の食があった。食べ物の正体が偽られていた」
- 独立(2葉:海/危機):「海で危機的な遭難があり、食料が尽き人が死んだ」
