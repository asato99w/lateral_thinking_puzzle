# 特性分析：耳栓

## Step 1: テンプレート展開

### 1. 状態
- つけている / 外している（二値的）
- 正しく装着されている / ずれている / 片方だけ
- 新品 / 劣化している（連続的、素材が硬化・縮小）
- 清潔 / 汚れている

### 2. ルーティン
- 就寝前につける → 睡眠中つけている → 起床時に外す（睡眠用）
- 作業開始時につける → 作業中つけている → 作業終了時に外す（騒音環境）
- 集中したい時につける → 集中が終わったら外す（自発的使用）
- 飛行機の離着陸時につける（気圧調整用）

### 3. 帰結
- つけている → 音が聞こえない / 聞こえにくい
- 外している → 音が聞こえる
- つけている → 話しかけられても気づかない
- つけている → 自分の声が大きくなる（自覚しにくい）
- つけている → 環境音の変化に気づけない
- つけたまま寝た → 目覚ましが聞こえない可能性
- 片方だけ → 方向によって聞こえ方が違う

### 4. 遷移条件
- つける / 外す：意図的行為
- ずれる / 外れる：睡眠中の体動、素材の劣化（自然に起きうる）
- 劣化：時間経過と使用（自然に進行）

### 5. 推論
- 話しかけたのに反応しない → 耳栓をしている / 聞こえていない
- 大きな音がしたのに反応しない → 耳栓をしている / 別の理由
- 目覚ましが鳴ったのに起きない → 耳栓で聞こえない
- 耳栓をしているのに音に反応した → 外れている / 音が大きすぎた / 振動で気づいた
- 本人が「何も聞こえなかった」→ 耳栓 / 本当に音がなかった / 嘘

### 6. 関係性
- 身体に直接接触（耳に挿入）
- 聴覚を遮断・減衰させる
- 他者からは見えにくい（コンタクトレンズと類似）
- 社会的コミュニケーションが制限される
- 安全に関わる（警報、呼びかけを聞き逃す）
- 睡眠の質に関わる

## Step 2: 本質的機能の特定

**本質的機能：** 聴覚の遮断（音を聞こえなくする）

コンタクトレンズが「見えるようにする」のに対し、耳栓は「聞こえなくする」。
方向が逆（感覚の付与 vs 感覚の遮断）だが、感覚の状態を変える点は同じ。

**非本質的特性（ミスリード素材）：**
- 素材、外見
- 劣化の度合い
- 清潔さ

## Step 3: 演繹構造の導出

### 演繹1（保存・持続：聴覚遮断の持続）
- 前提群：耳栓をつけて寝た ＋ 外す行為をしていない
- 演繹：今もつけているはず → 音は聞こえていないはず
- 反証：音に反応した → 外れた / 振動で気づいた / つけていなかった

### 演繹2（因果：聞こえないことの帰結）
- 前提群：耳栓をしている ＋ 目覚ましが鳴った
- 演繹：目覚ましに気づかないはず → 寝過ごすはず
- 反証：起きた → 耳栓が外れていた / 別のもの（光、振動、人）で起きた

### 演繹3（因果：聞こえないことによる不知）
- 前提群：耳栓をしている ＋ その間に音を伴うイベントが起きた
- 演繹：イベントに気づいていないはず
- 反証：気づいている → 耳栓をしていなかった / 音以外の手がかりで気づいた

### 演繹4（因果：反応の不在→遮断の推定）
- 前提群：この人の聴覚は正常 ＋ 大きな音がした
- 演繹：反応するはず
- 反証：反応しない → 聴覚が遮断されている → 耳栓 / イヤホン / 別の原因

### 演繹5（共起：耳栓と環境の整合性）
- 前提群：耳栓をして寝ている ＋ 騒音環境にいる
- 演繹：騒音が理由で耳栓をしている
- 反証：静かな環境なのに耳栓 → 騒音以外の理由（習慣、何かを聞きたくない、別の目的）

### 演繹6（同一性：聞こえの一貫性）
- 前提群：この人は耳栓をしていない ＋ 聴覚は正常
- 演繹：周囲の音はすべて聞こえているはず
- 反証：特定の音を聞いていないと主張 → 本当に鳴らなかった / 耳栓をしていた / 嘘

### 演繹7（能力：聴覚遮断の限界）
- 前提群：耳栓は完全な遮音ではない ＋ 非常に大きな音がした
- 演繹：聞こえるはず
- 反証：聞こえなかった → 耳栓以上の遮音 / 音が実際にはそれほど大きくなかった / 意識がなかった

### 演繹8（時間変化：装着の影響）
- 前提群：長時間耳栓をつけていると耳に違和感が残る
- 演繹：長時間使用後は耳に感覚がある
- 反証：長時間つけていたはずなのに違和感がない → つけていなかった / 途中で外した

## Step 4: 所感

- **コンタクトレンズとの構造的対称性**が際立つ。
  コンタクトは「見えるようにする」、耳栓は「聞こえなくする」。
  方向は逆だが、感覚の状態を二値的に切り替える点、
  他者から見えにくい点、つけたまま寝る可能性がある点が共通
- **最も豊かな演繹構造**：演繹3（聞こえないことによる不知）。
  「耳栓をしていた → イベントに気づかなかった」は
  コンタクトの逆方向の構造。「気づかなかった」ことが真相に関わるパズルの可能性
- **演繹5（静かな環境での耳栓）**も有望。
  なぜ静かなのに耳栓をしているのか → 隠された理由の示唆
- **パズルの種**：「大きな音がしたのに反応しなかった」→ 耳栓だけでは弱い。
  「なぜ耳栓をしていたか」が重大な事実に繋がれば成立しうる
- **前提群の豊かさ**：コンタクトレンズと同程度に豊か。
  五感に直結する対象の強みが確認できた
