# パズルの面白さの分析14：前提群と反証の構造

前ノート（分析13）の続き。

## パラダイムの主要機能：前提群の提供

パラダイムとは、ある状況について自然に持たれる**前提群**である。
その前提群から演繹が行われ、予測が導出される。

前提群の中には明示的なもの（意識されている）と
暗黙のもの（当然視されている）がある。
観察が予測を反証した時、前提群のどれかが誤りだったことになり、
**どの前提が誤りで、なぜか**がパズルの真相になる。

## 構造の定式化

```
前提群（パラダイム）→ 演繹 → 予測
                              ↕
                         観察（反証）
                              ↓
                    前提群のいずれかが誤り
                              ↓
                      どれが、なぜ → 真相
```

## 既存パズルへの適用

### 毒キノコ

- 前提群：
  - キノコが生える環境では食用と毒が共存する
  - この環境に人為的な変更は加えられていない
- 演繹：食用キノコもあるはず
- 観察（反証）：食用キノコがない
- 覆される前提：「人為的な変更がない」→ 誰かが食用キノコを採った

### ウミガメのスープ

- 前提群：
  - 同じ食べ物は同じ味がする
  - 過去に食べたのはウミガメのスープだった
- 演繹：同じ味のはず
- 観察（反証）：味が違う
- 覆される前提：「過去に食べたのはウミガメのスープだった」→ 別のものだった

### コンタクトレンズ

- 前提群：
  - 視力矯正なしではぼやける
  - 寝る前にコンタクトを外した（日常のルーティン）
- 演繹：起きたらぼやけるはず
- 観察（反証）：はっきり見える
- 覆される前提：「コンタクトを外した」→ 外していない → 寝落ちした

### カセットテープ

- 前提群：
  - 死者は物理的行為ができない
  - その場に本人しかいなかった（自殺の想定）
- 演繹：テープは録音後の状態のままのはず
- 観察（反証）：巻き戻されている
- 覆される前提：「本人しかいなかった」→ 第三者がいた

## 前提群の構造に関する補足

### 「一般原理＋隠れた前提」は便宜的な区分

先の分析で「一般原理」と「隠れた前提」に分けたが、
これは前提群の中の役割の違いに過ぎない。
本質は前提群全体から演繹が行われるということであり、
一般原理と隠れた前提を厳密に分離する必要はない。

### 前提の堅さと脆さ

前提群の中には堅い前提（覆りにくいもの）と
脆い前提（覆りうるもの）がある。
反証された時に脆い方が崩れる。

パズルにおいては、どの前提が崩れるかの意外性が面白さに関わる。

## 体系的手法への示唆

この構造をパズル設計に応用するには：

1. ある状況における前提群を詳細に展開する
2. 前提群から明確な演繹を行い、反証可能な予測を導出する
3. 反証条件（予測の否定が観察される状況）を特定する
4. どの前提が覆されるかを設計する
5. 覆される前提の意外性と、そこから導かれる真相の面白さを評価する

### 演繹構造を見出す原理（分析13より）

前提群を構成する際に活用できる原理：
- 保存・持続
- 因果（行為→痕跡）
- 共起
- 同一性
- 能力
- 時間変化

## 設計試行の例

この構造で生成した例：

**雪の足跡：**
- 前提群：雪の上を歩けば足跡が残る ＋ 雪が降ってから誰も来ていない
- 演繹：足跡はないはず
- 反証：足跡がある（入る方向のみ）
- 覆される前提：「誰も来ていない」→ 誰かが来て、まだ中にいる

**犬が吠えない：**
- 前提群：犬は見知らぬ人に吠える ＋ 侵入者は犬にとって見知らぬ人
- 演繹：犬は吠えるはず
- 反証：犬が吠えなかった
- 覆される前提：「見知らぬ人」→ 犬の知り合いだった

**植物の傾き：**
- 前提群：植物は光源に向かって成長する ＋ ずっとこの窓際にあった
- 演繹：窓の方に傾いているはず
- 反証：反対方向に傾いている
- 覆される前提：「ずっとここにあった」→ 最近移された
