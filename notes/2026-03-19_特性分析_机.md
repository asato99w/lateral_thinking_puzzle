# 特性分析：机

## Step 1: テンプレート展開

### 1. 状態
- 物が置かれている / 何もない（空）
- 整頓されている / 散らかっている
- 引き出しが開いている / 閉じている
- 引き出しに鍵がかかっている / かかっていない
- 定位置にある / 移動されている
- 傷がある / きれい
- 埃がある / 拭かれている
- 濡れている / 乾いている

### 2. ルーティン
- 朝：席について作業を始める → 物を出す
- 日中：作業 → 物の出し入れ、配置変更
- 退勤/終了：片付ける → 離席
- 定期的：掃除、整理
- 長期不在時：物が減る、埃が溜まる

### 3. 帰結
- 物が置かれている → 誰かが使っている/使っていた
- 特定の物の配置 → 使用者の作業内容がわかる
- 埃がない → 最近使われた/拭かれた
- 埃が溜まっている → しばらく使われていない
- コーヒーの輪染み → 飲み物を置いた
- 引き出しが開いている → 何かを取り出した/探している途中
- 傷やへこみ → 何かをぶつけた/重い物を置いた
- 物の配置が変わっている → 誰かが触った

### 4. 遷移条件
- 物を置く/片付ける：意図的行為
- 埃が溜まる：時間経過（自然）
- 傷がつく：衝撃（偶発的）
- 移動する：意図的（重いので一人では困難な場合あり）
- 引き出しの開閉：意図的行為
- 濡れる：こぼす（偶発的）

### 5. 推論
- 自分が片付けた状態と違う → 誰かが使った/触った
- 知らない物がある → 誰かが置いた
- いつもある物がない → 誰かが持ち出した
- 引き出しが開いている → 急いでいた/探し物の途中
- 埃のムラ → 一部だけ最近触られた
- 椅子が引かれている → 誰かが座った
- 机が微妙に移動 → 誰かが動かした → なぜ
- 鍵付き引き出しの中身が変わっている → 合鍵/別の方法
- 2つのカップ → 2人いた

### 6. 関係性
- 個人の作業空間（パーソナルスペース）
- 使用者の習慣・性格を反映
- 職業・活動と結びつく
- 引き出し＝プライベートな収納
- 椅子とセット
- 場所に固定されている
- 表面が作業の「舞台」として機能

## Step 2: 本質的機能の特定

**本質的機能：** 作業のための平面を提供する（物を置く・作業する場所）

五感との直接的関係がない。机は「場の提供」であり、
痕跡を保持する**記録媒体**としての性質を持つ。

**非本質的特性（ミスリード素材）：**
- 材質、サイズ、デザイン
- 場所（オフィス、自宅、学校）
- 引き出しの数や構造

## Step 3: 演繹構造の導出

### 演繹1（保存・持続：物の保存）
- 前提群：昨日Xを置いた ＋ 片付けていない ＋ 誰も触っていない
- 演繹：Xはまだそこにあるはず
- 反証：ない → 誰かが持ち出した

### 演繹2（保存・持続：配置の保存）
- 前提群：物をこの配置にした ＋ 変更していない
- 演繹：同じ配置のはず
- 反証：変わっている → 誰かが触った → 何のために

### 演繹3（因果：埃のパターン→使用頻度）
- 前提群：使われない表面には埃が均一に積もる ＋ しばらく使われていない
- 演繹：均一に埃が積もっているはず
- 反証：一部だけ埃がない → その部分で物が動かされた → 最近使われた

### 演繹4（因果：痕跡→行為）
- 前提群：机にインク跡がある ＋ インク跡は書く行為から生じる
- 演繹：ここで何かが書かれた
- 反証：使用者は書いていないと言う → 別の人 / 別の原因

### 演繹5（共起：物の組み合わせ→活動）
- 前提群：PCと書類とペンがある ＋ これは仕事の道具
- 演繹：仕事をしている/していた
- 反証：PCは閉じたまま、書類は白紙 → ふりをしていた / 別の目的

### 演繹6（因果：椅子の位置→着席）
- 前提群：椅子は通常机の下に収まっている ＋ 引き出されている
- 演繹：誰かが座った/座っている
- 反証：誰も座っていないはず → 誰かが来た

### 演繹7（時間変化：飲み物の温度→時間）
- 前提群：机上のコーヒーは熱く淹れられた ＋ 時間とともに冷める
- 演繹：温度から経過時間が推定できる
- 反証：「さっき淹れた」と言うが冷めきっている → 時間認識が違う / 嘘

### 演繹8（能力：引き出しの鍵）
- 前提群：引き出しに鍵がかかっている ＋ 鍵は自分だけが持っている
- 演繹：中身は変わっていないはず
- 反証：中身が変わっている → 合鍵 / 別の方法

### 演繹9（因果：机の移動→目的）
- 前提群：机は定位置 ＋ 重くて動かしにくい
- 演繹：動いていないはず
- 反証：微妙に動いている → 誰かが動かした → 机の下に何かを隠す/取る

### 演繹10（共起：複数人の痕跡）
- 前提群：この机は一人で使う ＋ 物は一人分
- 演繹：一人分の痕跡のはず
- 反証：2つのカップ、異なる筆跡 → 別の人も使った

## Step 4: 所感

- **五感との直接的関係がない**。電話やコンタクトレンズとの決定的な差。
  机は感覚を変える道具ではなく、痕跡を残す「舞台」
- **最も豊かな演繹構造**：演繹3（埃のパターン）。
  埃の有無のムラから物の移動を推定する構造は毒キノコの「不在」の推論と類似
- **逆説構造は見当たらない**。机の機能に内在的な矛盾がない
- **既存パズルとの類似**：演繹8（鍵付き引き出し）はカセットテープ型。
  演繹3（埃のパターン）は毒キノコ型
- **前提群の豊かさ**：中程度。痕跡ベースの演繹は多いが、
  ほぼ「誰かが触った/使った」に収束。多方向性は限定的
