# パズルの面白さの分析15：前提群テンプレートと適用

前ノート（分析14）の続き。

## コンタクトレンズの導出力の源泉

「コンタクトをつけたまま寝た」が高い導出力を持つ理由は、
コンタクトレンズという物体が多様な前提と多様な演繹を自然に備えているため。

コンタクト使用者なら誰でも持つ既存の前提群があり、
真相はその中の一つの前提の否定を指している。
前提群を構築する必要がなく、指すだけで全体が活性化される。

## コンタクトレンズの特性分析

- **状態**: 入っている / 外れている（二値的）
- **ルーティン**: 朝入れる → 日中つけている → 夜外す
- **各状態の帰結**:
  - 入っている → はっきり見える
  - 外れている → ぼやける
- **遷移条件**: 意識的な動作が必要（自然には起きない）
- **状態からの推論**:
  - 朝入っている → 自分で入れた → 準備した
  - 夜入っている → 外していない → 通常のルーティンを経ていない
- **関係性**: 身体に直接接触、知覚に影響、他者からは見えない

## 前提群を展開するテンプレート

任意の日常の対象について以下を展開し、前提群と演繹構造を見出す。

1. **状態**: どのような状態を取りうるか。二値的か連続的か
2. **ルーティン**: 状態遷移の時間的パターン。日常の中でどう使われるか
3. **帰結**: 各状態から観察可能な結果は何か
4. **遷移条件**: 状態が変わるには何が必要か（自然に起きるか、意図的行為が必要か）
5. **推論**: 予想外の状態が観察された場合、何が推論できるか
6. **関係性**: 人・身体・環境とどう関わるか

## 適用例

### 腕時計

1. **状態**: 着けている/外している。動いている/止まっている
2. **ルーティン**: 朝着ける、寝る前に外す。電池式なら常時動作、機械式なら巻きが必要
3. **帰結**: 着けている→時刻がわかる。止まっている→時刻が止まった時点で固定
4. **遷移条件**: 着脱は意図的行為。停止は電池切れ・衝撃・水没など
5. **推論**: 止まった時刻→何かが起きた時刻を示す。左腕でなく右腕→利き手が違う/別人の時計
6. **関係性**: 身体に装着、社会的シグナル、他者から見える

### 結婚指輪

1. **状態**: 着けている/外している（二値的）
2. **ルーティン**: 常時着用が規範。外す場面は限定的（水仕事、特定の職業）
3. **帰結**: 着けている→既婚と推定。指に日焼け跡→普段は着けている
4. **遷移条件**: 意図的行為。常時着用が規範なので、外していること自体が有標
5. **推論**: 日焼け跡はあるが指輪がない→最近外した→なぜ？
   指輪がないが日焼け跡もない→最近まで着けていなかった
6. **関係性**: 身体に装着、社会的意味が強い、他者から見える

### 家の鍵

1. **状態**: 持っている/持っていない。施錠/解錠
2. **ルーティン**: 外出時に持ち施錠する。帰宅時に解錠する
3. **帰結**: 施錠→中には鍵を持つ人しか入れない。解錠→誰かが開けた
4. **遷移条件**: 鍵がなければ施錠・解錠できない（原則として）
5. **推論**: 施錠したはずなのに解錠→別の鍵が存在/別の侵入手段。
   鍵を持っていないのに中にいる→鍵を使わずに入った/誰かに入れてもらった
6. **関係性**: 所有者を限定、アクセス制御の機能

### 処方薬

1. **状態**: 服用前の量/服用後の量（連続的に減少）
2. **ルーティン**: 決まった時間に決まった量を服用。日・週単位の周期
3. **帰結**: 減り具合→指示通り服用しているか推定可能。
   減りが早い→多く飲んでいる。減っていない→飲んでいない
4. **遷移条件**: 意図的な服用行為
5. **推論**: 3日分が1日で減っている→過剰摂取/他の人も飲んでいる。
   1週間経っても減っていない→飲んでいない→なぜ？
6. **関係性**: 身体に直接影響、健康に関わる、他者には見えにくい

## 追加適用例：腕時計の演繹構造

### 演繹A（持続：動作の継続）
- 前提群：この時計は昨日動いていた ＋ 何も起きていない
- 演繹：今日も動いているはず
- 反証：止まっている → 何かが起きた（衝撃、水没、電池切れ）

### 演繹B（因果：停止時刻の証拠性）
- 前提群：時計は動いていた ＋ ある事象で停止した
- 演繹：表示時刻＝事象が起きた時刻
- 反証：事象の時刻と表示時刻が違う → 事象以前に止まっていた/操作された

### 演繹C（持続：機械式時計の巻き）
- 前提群：機械式時計は巻かなければ約40時間で止まる ＋ 所有者は1週間不在
- 演繹：止まっているはず
- 反証：動いている → 不在の間に誰かが巻いた → 誰かがこの時計に触れた

### 演繹D（同一性：装着痕）
- 前提群：常に腕時計をする人は手首に日焼け跡がある ＋ この人は常に時計をしている
- 演繹：日焼け跡があるはず
- 反証：ない → 最近まで時計をしていなかった

### 演繹E（同一性：バンドの調整）
- 前提群：バンドは所有者の手首に合わせて調整される ＋ これは本人の時計
- 演繹：本人の手首にぴったり合うはず
- 反証：合わない → 本人の時計ではない/体型が変化した

### 演繹F（時間変化：時刻のズレ）
- 前提群：3日前に正確に合わせた ＋ 1日5秒進む
- 演繹：約15秒進んでいるはず
- 反証：3時間ずれている → 途中で止まった/誰かが変えた

### 演繹C（機械式時計の巻き）に関する所感
「1週間不在なのに動いている → 誰かが巻いた」は、
カセットテープの「死者にはテープを巻き戻せない → 第三者がいた」と
同型の構造（能力の原理）。

## 有望でない対象への適用テスト

テンプレートの力を検証するため、直感的に有望でない対象にも適用した。

### 傘

1. **状態**: 開/閉、濡れ/乾き、ある/ない
2. **ルーティン**: 雨の予報→持ち出す、雨→開く、帰宅→乾かす→定位置に戻す
3. **帰結**: 濡れた傘→雨の中から来た人がいる。定位置にない→持ち出された
4. **遷移条件**: 開閉は意図的。濡れるのは環境。壊れるのは強風
5. **推論**: 晴れなのに濡れている→別の場所では雨だった。見覚えのない傘→来客
6. **関係性**: 取り違えやすい、共有されることがある、天候に依存

**演繹構造：**
- 前提群：今日この地域は終日晴れ ＋ 傘は外に持ち出されていない
- 演繹：傘は乾いているはず
- 反証：濡れている → 別の地域から来た人が使った/雨以外で濡れた

- 前提群：家には傘が3本あった ＋ 誰も外出していない
- 演繹：3本あるはず
- 反証：2本しかない → 誰かが外出した

### タオル

1. **状態**: 乾/湿/濡れ、清潔/汚れ、畳まれ/掛かっている/床
2. **ルーティン**: 入浴後に使う→掛けて乾かす→定期的に洗濯
3. **帰結**: 濡れている→最近使われた。湿っている→数時間前
4. **遷移条件**: 濡れるのは使用（意図的）。乾くのは時間経過（受動的）
5. **推論**: 一人暮らしで2枚濡れている→もう一人いた。塩素の匂い→プールに行った
6. **関係性**: 個人に割り当てられる、活動の痕跡を残す

**演繹構造：**
- 前提群：家に一人しかいない ＋ 一度の入浴でタオル1枚
- 演繹：濡れたタオルは1枚以下のはず
- 反証：2枚濡れている → もう一人いた/二度入浴した

- 前提群：今日はプールに行っていない ＋ このタオルはこの人専用
- 演繹：塩素の匂いはしないはず
- 反証：塩素の匂いがする → プールに行った

### 有望でない対象への適用から得られた知見

傘とタオルは、コンタクトレンズや腕時計に比べて演繹の豊かさが明らかに劣る。
チェーンが短く、推論が直接的。
ただしタオルの「2枚濡れている」「塩素の匂い」にはわずかに可能性がある。

## 適用から得られた所感

- 結婚指輪の「日焼け跡はあるが指輪がない」はコンタクトレンズに近い構造。
  常時装着が前提で、不在自体がアノマリーになる
- 処方薬の減り方は演繹が豊かで、複数の推論経路が開かれる
- 家の鍵は「能力」の原理と直結し、カセットテープ型の推論に近い
- 対象の選び方には作為性があった（有望そうなものを直感的に選んでいた）
- 対象選定の指針として**ルーティンの強さ**と**状態変化の観察可能性**が
  重要な要因に見える
