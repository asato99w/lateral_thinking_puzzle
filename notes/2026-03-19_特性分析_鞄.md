# 特性分析：鞄

## Step 1: テンプレート展開

### 1. 状態
- 開いている / 閉じている（ファスナー、バックル等）
- 中身がある / 空
- 重い / 軽い（中身に依存、連続的）
- 持っている / 持っていない / どこかに置いてある
- 自分の鞄 / 他人の鞄
- 新しい / 使い古されている（連続的）
- 整理されている / 乱雑
- 濡れている / 乾いている
- 破損している / 無傷

### 2. ルーティン
- 外出前に必要なものを入れる → 持って出る → 外出先で出し入れ → 帰宅して下ろす → 中身を出す/出さない
- 通勤・通学鞄は毎日同じものを使い、中身も概ね固定
- 旅行鞄は出発前に詰めて、到着後に開ける
- 置き場所が決まっている（玄関、自室、職場の机など）

### 3. 帰結
- 重い → 中身が多い / 重いものが入っている
- 軽い → 中身が少ない / 空に近い
- 持っている → 外出中 / 外出の準備ができている
- 特定の場所にある → 持ち主はそこにいる / いた
- 中に特定のものがある → 持ち主の行動・目的がわかる
- 外側に汚れ・傷 → どこに行ったか、何に接触したかの痕跡
- 匂い → 中身や行き先の手がかり

### 4. 遷移条件
- 空→中身あり：物を入れる（意図的）
- 中身あり→空：物を出す（意図的）/ 盗まれる（他者の意図）
- 持つ / 置く：意図的行為
- 新品→使用感：時間経過と使用（自然に進行）
- 乾→濡れ：雨、水場（環境的）
- 無傷→破損：衝撃、摩耗（偶発的 / 時間経過）

### 5. 推論
- いつもの鞄がない → 外出した / 持ち出された
- いつもと違う場所に鞄がある → いつもと違う行動をした
- 鞄が異様に重い/軽い → 普段と違うものが入っている / 抜かれている
- 知らないものが入っている → 誰かが入れた
- あるはずのものがない → 出した / 盗まれた / 別の鞄に入れた
- 鞄が濡れている → 雨の中を歩いた / 水場に行った
- 他人の鞄が自分の手元にある → 取り違えた / 渡された
- 同じ外見の鞄が複数ある → 取り違えの可能性
- 鞄の底に砂がある → 砂浜やグラウンドに行った

### 6. 関係性
- 所有者との強い結びつき（個人の持ち物）
- 中身が所有者の生活・目的・職業を反映する
- 外見が社会的シグナル（ブランド、サイズ、種類）
- 移動と結びつく（鞄を持つ＝どこかに行く）
- 他の持ち物の容器（財布、鍵、書類などを包含する）
- プライベートな空間（他人が中を見るのは非日常的）

## Step 2: 本質的機能の特定

**本質的機能：** 物を入れて運ぶ（収容と携行）

本質的な演繹はこの機能に由来する：
- 中身に関すること（何が入っているか、入っていないか、重さ）
- 携行に関すること（どこに持っていくか、どこにあるか）

**非本質的特性（ミスリード素材）：**
- 外見（ブランド、色、デザイン）
- 汚れや傷の痕跡
- 匂い
- 新旧の度合い

## Step 3: 演繹構造の導出

### 演繹1（保存・持続：中身の保存）
- 前提群：昨日この鞄にXを入れた ＋ 出していない ＋ 誰も触っていない
- 演繹：Xは今も鞄の中にあるはず
- 反証：Xがない → 誰かが出した / 自分が出したことを忘れている / 別の鞄に入れた

### 演繹2（保存・持続：中身の不変）
- 前提群：入れたものはこれだけ ＋ 誰も追加していない
- 演繹：入れたもの以外は入っていないはず
- 反証：知らないものが入っている → 誰かが入れた → 誰が、なぜ

### 演繹3（因果：重さ→中身）
- 前提群：この鞄にはいつも同じものを入れている ＋ 中身を変えていない
- 演繹：いつもと同じ重さのはず
- 反証：重い → 何か追加された。軽い → 何か抜かれた

### 演繹4（因果：場所→人の所在）
- 前提群：この鞄は持ち主が外出時に持つ ＋ 鞄がここにある
- 演繹：持ち主はここにいる / ここに来た
- 反証：持ち主はいないはず → 鞄だけが運ばれた / 忘れた / 誰かに預けた

### 演繹5（因果：痕跡→行き先）
- 前提群：鞄の外側に付着物がある ＋ 付着物は特定の環境に由来する
- 演繹：その環境に行った
- 反証：本人はそこに行っていないと言う → 行った / 別の経路で付着した

### 演繹6（共起：中身の整合性）
- 前提群：仕事に行く人はPCと書類を入れる ＋ この人は仕事に行くと言っている
- 演繹：PCと書類が入っているはず
- 反証：着替えとパスポートが入っている → 仕事ではなく旅行 / 仕事と言って別のことをしている

### 演繹7（同一性：自分の鞄）
- 前提群：これは自分の鞄 ＋ 自分の鞄には見覚えのある中身が入っている
- 演繹：開けたら見覚えのある中身のはず
- 反証：見知らぬ中身 → 自分の鞄ではない → 取り違えた

### 演繹8（能力：開閉の制限）
- 前提群：鞄にはダイヤルロックがかかっている ＋ 番号を知るのは自分だけ
- 演繹：自分以外は開けられないはず
- 反証：中身が変わっている → 番号を知る人がいる / ロックが破られた

### 演繹9（時間変化：中身の経年変化）
- 前提群：食べ物を入れた ＋ 時間が経過した
- 演繹：傷んでいるはず
- 反証：まだ新鮮 → 最近入れ替えられた / 保冷されていた

## Step 4: 所感

- **最も豊かな演繹構造**：演繹6（共起：中身の整合性）が面白い。
  中身が持ち主の申告する行動と矛盾する場合、隠された意図や行動が浮かび上がる。
  連鎖が長く、分岐が多い
- **既存パズルとの類似**：演繹3（重さの変化）は毒キノコの「あるべきものがない」に
  近い構造。演繹8（ロック）は鍵/カセットテープ型の能力原理
- **パズル設計の種として有望**：演繹6（仕事に行くと言ったのに旅行の荷物）と
  演繹2（知らないものが入っている）が最も有望。
  行動の真意と鞄の中身の不一致から、隠された物語が展開しうる
- **前提群の豊かさ**：鞄は「中身」を軸に多方向の演繹が可能で、豊かさは高い。
  ただし演繹の多くが「中身が違う→誰かが触った」に収束しやすく、
  方向の多様性ではコンセントや腕時計にやや劣る
