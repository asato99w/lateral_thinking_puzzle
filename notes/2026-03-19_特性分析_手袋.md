# 特性分析：手袋

## Step 1: テンプレート展開

### 1. 状態
- はめている / 外している（二値的）
- 左右一対 / 片方だけ
- 乾いている / 濡れている / 汚れている
- 温かい / 冷たい（使用直後は体温で温かい）
- 新品 / 使い古されている
- 素材の種類（毛糸、革、ゴム、ラテックス、軍手など）
- サイズが合っている / 合っていない

### 2. ルーティン
- 寒い日：外出時にはめる → 室内で外す → 定位置に置く
- 作業用：作業開始時にはめる → 作業後に外す
- 医療用：処置前にはめる → 処置後に廃棄
- 季節性：冬に使い、夏にしまう

### 3. 帰結
- はめている → 手が温かい / 保護されている / 触覚が鈍る
- はめている → 指紋がつかない
- はめている → 細かい作業がしにくい
- 外している → 手が直接環境に触れる → 指紋が残る / 手が汚れる / 手が冷える
- 片方だけ → もう片方を落とした / 片手だけ必要な理由がある
- 手袋が温かい → 直前まで誰かがはめていた
- 手袋の内側が湿っている → 長時間はめていた（汗）

### 4. 遷移条件
- はめる / 外す：意図的行為
- 落とす：偶発的（気づかずに落ちる）
- 汚れる：作業や環境による
- 破れる：摩耗や外力
- 濡れる：雨、雪、水仕事

### 5. 推論
- 寒くないのに手袋をしている → 寒さ以外の理由（指紋隠し、怪我隠し、手を見せたくない）
- 寒いのに手袋をしていない → 持っていない / 手袋では困る作業がある
- 片方だけ落ちている → 誰かが通った / 急いでいた
- 手袋の内側が温かい → 直前まではめていた人がいる
- ゴム手袋が日常の場面にある → 防寒ではない目的
- 手袋のサイズが合わない → 本人のものではない
- 手袋をしているのに手が汚れている → 途中で外した / 穴がある

### 6. 関係性
- 身体に直接装着（手を覆う）
- 触覚を遮断/変化させる（五感との直接的関係）
- 指紋という個人識別情報を隠す
- 季節・気温と結びつく
- 職業・活動を示唆する
- 左右一対であること
- 社会的シグナル（素材やブランド）

## Step 2: 本質的機能の特定

**本質的機能：** 手の保護と環境からの遮断（温度、汚れ、怪我、接触から手を守る）

五感との関係では**触覚の遮断/変化**が本質。
副作用として**指紋の隠蔽**が随伴する。

**非本質的特性（ミスリード素材）：**
- 素材の種類やブランド
- 色やデザイン
- 左右の対称性

## Step 3: 演繹構造の導出

### 演繹1（因果：触覚の遮断）
- 前提群：手袋をはめている ＋ 手で物に触れた
- 演繹：物の温度や質感を直接感じられないはず
- 反証：質感を正確に報告した → 手袋をしていなかった / 外した / 非常に薄い手袋

### 演繹2（因果：指紋の有無）
- 前提群：手袋をはめている ＋ 物に触れた
- 演繹：指紋は残らないはず
- 反証：指紋が残っている → 手袋をしていなかった / 穴 / 別の人

### 演繹3（因果：指紋がないことの含意）
- 前提群：物に指紋がない ＋ 誰かが触ったはず
- 演繹：手袋を介して触れた
- 反証：触った人は素手だったと主張 → 指紋が拭かれた / 別の人

### 演繹4（共起：季節と手袋）
- 前提群：冬である ＋ 外出している
- 演繹：手袋をしている可能性が高い
- 反証：真冬に素手 → 持っていない / 急いで出た / 手袋では困る理由

### 演繹5（共起：季節と不整合）
- 前提群：夏である ＋ この人は手袋をしている
- 演繹：防寒目的ではないはず
- → なぜ夏に手袋を？ → 指紋隠し / 怪我 / 手を見せたくない / 職業的理由

### 演繹6（保存・持続：体温の痕跡）
- 前提群：手袋をはめていた ＋ 直前に外した
- 演繹：手袋の内側は体温で温かいはず
- 反証：冷たい → かなり前に外された / はめられていなかった

### 演繹7（同一性：サイズの一致）
- 前提群：この手袋はこの人のもの ＋ 手袋は持ち主のサイズに合う
- 演繹：この人の手にぴったりのはず
- 反証：合わない → この人のものではない

### 演繹8（保存・持続：一対の完全性）
- 前提群：手袋は左右一対 ＋ 一緒に保管/携帯している
- 演繹：2つ揃っているはず
- 反証：片方しかない → 落とした / 片方だけ使った → なぜ

### 演繹9（因果：手袋をしていても手が汚れる）
- 前提群：手袋をはめていた ＋ 手袋は手を保護する
- 演繹：手は汚れていないはず
- 反証：手が汚れている → 途中で外した / 穴 / 手袋をする前に汚れた

### 演繹10（能力：手袋と細かい作業）
- 前提群：厚い手袋をはめている ＋ 細かい作業を行った
- 演繹：うまくできないはず
- 反証：精密に行われている → 手袋を外した / 非常に薄い手袋

## Step 4: 所感

- **最も豊かな演繹構造**：演繹2・3（指紋の有無）。
  手袋→指紋なし、指紋なし→手袋という双方向の推論が成立。
  ただし犯罪パズルに偏りやすい
- **五感との関係**：触覚遮断は耳栓（聴覚）やコンタクト（視覚）と構造的に対称。
  ただし日常で触覚に強く依存する場面が限られ、遮断の帰結が耳栓ほど劇的でない
- **逆説構造の可能性**：演繹10（保護するが細かい作業を阻害）に逆説構造あり。
  ただし耳栓ほど鋭い矛盾ではない
- **独自の強み**：指紋の隠蔽、左右一対、季節との結びつき。
  演繹5（夏に手袋）は強い緊張を持つ
- **前提群の豊かさ**：中程度。触覚遮断自体は帰結が限定的だが、
  指紋・季節・一対性という別軸が補っている
