# 質問生成

## 概要

統合アルゴリズム Phase 6 の規則。戦術連鎖の各戦術を具体的な質問に置き換える。

## 入力

- 戦術連鎖（Phase 5 の出力）

## 出力

- 質問群（各質問は ID、テキスト、回答、想起条件、回答効果、メカニズムを持つ）

## 手順

1. 戦術連鎖の各戦術を、具体的な質問に置き換える
2. 各質問に以下を定義する:
   - テキスト（質問文）
   - 回答（Yes/No）
   - 想起条件: 仮説の族（OR of AND）
   - 回答効果（reveals する事実）
   - メカニズム: observation / link / anomaly

## 規則

- 全ての戦術に対応する質問が存在すること
- 想起条件の形式: 仮説 ID のリストの族（OR of AND）。事実 ID を含めない
- 想起条件は想起スコープ制約（ピース抽出規則）を満たすこと
- ゲーム開始時点でオープンとなる質問が少なくとも 1 つ存在すること
