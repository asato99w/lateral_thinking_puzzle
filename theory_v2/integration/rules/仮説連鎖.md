# 仮説連鎖

## 概要

統合アルゴリズム Phase 4 の規則。トリガーに含まれる仮説を起点に、各仮説を想起させる仮説群を再帰的に定義する。

## 入力

- 各ピースのトリガー（Phase 3 の出力）

## 出力

- 仮説群（各仮説は ID、ラベル、形成条件を持つ）

## 手順

1. トリガーに登場する仮説を起点とする
2. その仮説を想起させるのに十分な仮説の集合族（OR of AND）を定義する
3. 新たに登場した仮説に対して 2 を繰り返す
4. 再帰の深さ上限に達したら終了

※ 想起が直接的で自然な仮説については、無理に再帰を適用しなくてよい。

## 規則

- 形成条件の形式: 仮説 ID のリストの族（OR of AND）。事実 ID を含めない
- 形成条件に循環がないこと
- 再帰の深さ上限: 3
