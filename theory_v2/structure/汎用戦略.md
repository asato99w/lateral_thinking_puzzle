# 汎用戦略

任意のパズルに適用可能な探索方針。仮説連鎖の末端要素に対して適用し、S の事実へと辿る鎖を構成する。

## 戦略一覧

### 1. 常識的仮説の排除

「ありがちな説明」を否定することで、非日常的な方向へ探索を進める。

- 適用例: 自殺した → 病気？ → No → 日常的原因ではない

### 2. 観測事実の深掘り

S の事実の一つに注目し、その背景や詳細を問う。

- 適用例: 店員に尋ねた → なぜ確認した？ → 過去に飲んだことがある

### 3. 時間軸の拡張

「今」の出来事に対して「過去」を問う。S が現在の断面であることを利用する。

- 適用例: スープを飲んだ → 以前にも飲んだ？ → 過去の経験がある

### 4. 登場人物の探索

S に明示されていない人物の存在を問う。

- 適用例: 男が一人で行動している → 他に関係者がいる？ → 仲間がいた

### 5. 因果の逆転

S の出来事の因果関係を逆に辿る。結果から原因を問う。

- 適用例: 自殺した → 何がきっかけ？ → スープの味が違った

### 6. 矛盾の指摘

S の事実間、または S の事実と常識の間の不整合を問う。

- 適用例: 本物のスープなのに確認した → 確認する理由がある → 記憶と違う

### 7. ラベルと実体の乖離

名前や説明と実際の中身が異なる可能性を問う。

- 適用例: 「ウミガメのスープ」と言われた → 本当にウミガメだった？ → 偽りだった

### 8. 動機の探索

行動の理由・動機を問う。特に S で説明されていない行動に対して。

- 適用例: 自殺した → なぜ？ → 正体を悟った絶望
