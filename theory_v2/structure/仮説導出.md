# 仮説導出

観測事実集合から導出可能な仮説集合を決定する不動点計算。

## 入力

- 観測事実集合
- 仮説群（各仮説の形成条件）

## 出力

- 導出済み集合（事実 + 導出された仮説）

## 手順

1. 導出済み集合を観測事実集合で初期化する
2. 各仮説について以下を判定する:
   - 仮説が導出済み集合内の事実と一致する → 導出
   - 形成条件のいずれかのグループが全て導出済み → 導出
3. 新たに導出された仮説を導出済み集合に追加する
4. 変化がなくなるまで 2〜3 を繰り返す

## ゲーム中の適用

事実が reveals されるたびにこの計算を再実行する。これにより「事実の発見 → 仮説の導出 → 新しい質問のオープン」のサイクルが駆動される。
