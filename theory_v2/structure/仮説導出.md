# 仮説導出

確認済み記述素集合から導出可能な記述素集合を決定する不動点計算。

## 入力

- 確認済み記述素集合
- 記述素群（各記述素の formation_conditions）

## 出力

- 導出済み集合（確認済み記述素 + 導出された記述素）

## 手順

1. 導出済み集合を確認済み記述素集合で初期化する
2. 各記述素について以下を判定する:
   - formation_conditions のいずれかのグループが全て導出済み → 導出
3. 新たに導出された記述素を導出済み集合に追加する
4. 変化がなくなるまで 2〜3 を繰り返す

## ゲーム中の適用

記述素が reveals されるたびにこの計算を再実行する。これにより「記述素の確認 → 導出 → 新しい質問のオープン」のサイクルが駆動される。
