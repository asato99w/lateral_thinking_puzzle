# 展開規則一覧

## 共通展開規則

Fs, Ft いずれの展開にも適用可能な規則。

| 規則 | 方向 | 説明 |
|------|------|------|
| 前提抽出 | 下に潜る | 記述素が成立するために必要な前提を抽出する |
| 5W1H展開 | 周囲に広げる | Who / What / When / Where / Why / How を問い、記述素を生成する |
| 含意導出 | 先に進む | 記述素から論理的に導かれる帰結を抽出する |
| 因果逆推 | 上に遡る | 結果から原因を推論する。含意導出の逆方向 |
| 状況仮説 | 背景を生成 | 記述素が自然に成立するような背景状況を仮説として生成する |
| 感情推論 | 内面を推論 | 記述素から関与する人物の感情状態を推論する。行動の動機に接近する |
| 対比生成 | 反転を生成 | 記述素の反転や対比を生成する。暗黙の前提を浮き彫りにする |
| 否定代替仮説生成 | 代替を列挙 | 記述素が記述する事実に対して、T のもとで否定される代替的な説明を体系的に生成する |
| 上位概念化 | 抽象度を変える | 具体的な記述素から上位カテゴリを段階的に生成し、抽象度の階層を構築する |
| 合成 | 組み合わせる | 複数の記述素を組み合わせて新たな記述素を導出する |

## 否定代替仮説生成

記述素が記述する事実に対して、T のもとでは否定される代替的な説明を体系的に生成する。状況仮説規則が T と整合する背景を生成するのに対し、否定代替仮説生成は T で否定される仮説を意図的に生成する。「いいえ」「関係ない」の正答を持つ質問の基盤となる。

体系的な生成カテゴリ:

- **動機の代替**: 行為の理由として一般的に考えうる動機を列挙する
  - 例: Ps-6「男は自殺した」→ 恋愛問題 / 経済問題 / 精神疾患 / 犯罪への関与（いずれも T で否定）
- **手段の代替**: 結果に至る手段・経路の代替を列挙する
  - 例: 「死亡した」→ 殺された / 事故死 / 病死（T で否定される場合）
- **因果の代替**: 事象間の因果関係の代替的説明を列挙する
  - 例: 「スープを飲んで自殺した」→ 毒入り / 品質問題 / アレルギー（いずれも T で否定）
- **関与者の代替**: 事象に関与しうる人物の代替を列挙する
  - 例: 店員が関与 / 他の客が関与 / 家族が関与（T で否定される場合）
- **背景の代替**: 過去の出来事の代替を列挙する
  - 例: 「過去に重大な出来事があった」→ 戦争参加 / 犯罪関与 / 災害（T で否定される場合）

## 上位概念化

具体的な記述素に対して、複数の抽象度レベルにおける上位概念を生成する。プレイヤーが「はい/いいえ」の二値で段階的に対象を絞り込む体験を支援する。

手順:

1. 具体的な記述素を起点として、直近の上位カテゴリを特定する
2. 必要に応じてさらに上位のカテゴリを特定する
3. 各レベルを独立した記述素として生成する

例:
- Ft-29「人肉である」→「動物の肉である」→「食用の生物素材である」
- Pt-1「遭難した」→「命に関わる事故に遭った」→「過去に重大な出来事があった」
- Ft-22「隠す意図があった」→「真実を伝えなかった」→「情報の非対称性があった」

## 5W1H 展開の適用ガイドライン

5W1H 展開規則の適用時に以下のガイドラインに従う。

1. **全軸検討の義務**: 各入力記述素に対して Who / What / When / Where / Why / How の全6軸を検討し、意識的に省略しない
2. **属性への展開**: Who は人物そのものだけでなく、職業・年齢・経歴・専門性等の属性を含む
3. **文脈横断適用**: 現在の文脈だけでなく、過去・未来の時間軸、現場以外の場所にも適用する
4. **「関係ない」記述素の意図的生成**: T において当該軸が無関係である場合、その無関係性を明示する記述素を生成する（例:「レストランの場所は T の成立に関係しない」）

## Fs 用展開規則

Ps 起点の展開に特化した規則。大区分 I の豊かさに寄与する。

S は意図的に異常な要素や不整合を含むよう設計されている。Fs 用規則はこの性質を活用し、プレイヤーが構成する仮説の多様性を生む。

- **異常性検出**: 記述素が通常の文脈では異常であることを明示化し、その異常性自体を記述素として生成する。因果逆推の前段階として機能する。因果逆推が「なぜ」の原因を推論するのに対し、異常性検出は「これは通常ではない」という事実を生成する。
  - 例: 「一口飲んで真偽を確認した」→「通常の食事行動ではない。確認する特別な理由がある」

- **緊張関係抽出**: 2 つの記述素間の不整合そのものを記述素として生成する。合成が新たな事実を導出するのに対し、緊張関係抽出は不整合の存在自体を明示化する。パズルの推進力となる記述素を生成する。
  - 例: Ps-5「本物と確認された」+ Ps-6「自殺した」→「本物であることと自殺が結びつくのは異常な結びつきであり、追加の説明が必要」

## Ft 用展開規則

Pt 起点の展開に特化した規則。

Pt は具体的な情報を持つため共通規則でカバーできる部分が大きいが、T に頻出する欺瞞の構造を展開するために以下が有用。

- **隠蔽動機推論**: 欺瞞や隠蔽の行為から、隠されている内容の性質を推論する。因果逆推が「なぜ嘘をついたか」の原因を推論するのに対し、隠蔽動機推論は「隠蔽の対象がどのような性質のものか」まで踏み込む。
  - 例: 「ウミガメのスープだと嘘をついた」→「本当の材料は、知られると社会的に許されないものである」

## E2 用展開規則

E1 の出力を入力とし、パラダイム間の接続性に必要な記述素を補強する規則。

- **架け橋合成**: Fs と Ft の意味的に関連するペアを重点的に合成し、パラダイム間の共有記述素・架け橋候補を生成する。共通規則の「合成」が全ペアを無差別に対象とするのに対し、Fs×Ft に焦点を当てる。
  - 例: 「食事のために来店した」×「過去の体験を確認しに来た」→「来店には食事以外の目的もあった」

- **部分的理解生成**: Fs と Ft の間にある「一歩近づいた理解」を生成する。中間パラダイムの固有領域の素材となる。真相そのものではないが、表層よりも真相に近い理解を表す記述素を生成する。
  - 例: 「人肉を食べた」→「食材に通常でないものが使われた」

## 展開規則の分類

| 分類 | 適用対象 | 役割 |
|------|---------|------|
| 共通規則 | Fs, Ft | 記述素の一般的な性質に基づく展開 |
| Fs 用規則 | Fs | S の構造的特性（異常性・不整合）を活用し、大区分 I の探索体験を豊かにする |
| Ft 用規則 | Ft | T に頻出する構造（欺瞞・隠蔽）を展開し、核心への到達を支援する |
| E2 用規則 | E1 出力 | パラダイム間の接続性と中間パラダイムの固有領域を補強する |
