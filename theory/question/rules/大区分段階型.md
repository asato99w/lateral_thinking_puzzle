# 大区分段階型

## 入力

記述素空間 F、パラダイム群の大区分構造

## 手順

1. 記述素を大区分（I / II / III）に分類する

2. 各大区分から質問到達候補を選定する
   - 大区分 I（Fs 固有）: 初期オープン候補
   - 大区分 II（Fs ∩ Ft）: 橋渡し候補
   - 大区分 III（Ft 固有）: パラダイムシフト後のオープン候補

3. 各大区分内での質問の生成には、記述素直接変換型 / パラダイム駆動型 / 緊張解消型のいずれかを適用する

4. ans 写像を定義する

## 出力

質問群 Q（ans 写像付き、大区分ごとの段階情報付き）

## 特徴

- アプリの質問オープン判定の構造と直接対応する
- 大区分ごとの段階構造が明確
- 段階内の質問設計は他の規則に委ねる
- 他の規則との組み合わせが前提
