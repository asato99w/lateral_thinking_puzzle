# B: パラダイム駆動型

## 入力

記述素空間 F、パラダイム群

## 手順

1. 各パラダイムの「鍵記述素」を特定する。鍵記述素とは、そのパラダイムの説明力を決定づける記述素であり、他のパラダイムとの区別を生む記述素である

2. 鍵記述素に対応する質問を生成する

3. パラダイム間の遷移を駆動する記述素を特定し、対応する質問を生成する。特に大区分間の遷移（II → III）を可能にする質問が重要

4. ans 写像を定義する

## 出力

質問群 Q（ans 写像付き、パラダイムとの対応情報付き）

## 特徴

- パラダイムシフトの体験と連動する
- 質問数が絞られる
- パラダイム分析が前提となる
- 質問のオープン判定に直接利用可能（どのパラダイムに属する質問かが明確）
