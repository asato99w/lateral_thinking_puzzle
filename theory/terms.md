# 用語定義

## 基本要素

- **記述素**: 世界の一側面を表す最小の意味単位。他の記述素との論理演算を前提としない。H によって確信度が与えられる。
- **S**: 問題文。出題者が提示するテキスト。
- **T**: 真理論。正解のストーリーに対応する記述素の集合。T に含まれる記述素は真、T が明示的に否定する記述素は偽であり、それ以外は T のもとでは不定。P(O | T) = 1 が構造的に保証される。

## 記述素群

- **Ps**: S の記述素群。問題文から直接読み取れる事実の記述の集合。
- **Pt**: T の記述素群。真理論から直接読み取れる事実の記述の集合。
- **O**: 観測記述素群。回答者がある時点で真偽が確定している記述素の集合（0 または 1）。ゲーム開始時は O = Ps であり、質問への回答を通じて拡大していく。
- **R**: 不関与記述素群。質問への回答で「関係ない」と確定した記述素の集合。T のもとで不定な記述素に対応する。
- **完全確定**: 記述素空間の全記述素が O または R に振り分けられた状態。T が各記述素に対して真偽を持つか不定かを決定するため、完全確定の内容は T によって一意に定まる。質問の経路に依存しない理論的な概念である。

## 構造

- **記述素空間**: 記述素展開 E によって生成される全記述素の集合。分析上の世界そのものを構成する。記述素空間の外に記述素は存在しない。
- **Fs**: Ps を起点として生成される記述素空間。
- **Ft**: Pt を起点として生成される記述素空間。
- **H**: 暫定理論。記述素空間の全ての記述素に対する確率値 [0, 1] の割り当て。O に含まれる記述素は 0 か 1 の確定値を取り、それ以外の記述素は中間値を取る。質問を通じて更新される。
- **パラダイム**: 予測関数・想起可能集合・関係構造を含むもの。同じ記述素のペアでも、パラダイムによって関係の強さや意味が異なる。H のうち確率値が極端な（0.5 から離れた）記述素の集合として回答者に現れる。H がどのパラダイムに集中しているかによって、回答者が「見ている」関係構造自体が異なる。
- **パラダイム群**: 記述素空間に内在する複数のパラダイムの集合。記述素空間には複数の関係構造が重畳して内在しており、各パラダイムはそのうちの一つに対応する。T もいずれかのパラダイムに属する。
- **予測関数 P_pred(P)**: パラダイム P の世界観から論理的に導出される予測の全体。{d → 0 | 1} の写像。P_pred にキーがない記述素は P にとって不定（真偽どちらも導出できない）。アノマリー・tension・resolve・attention・同化のスコープとして使用される。
- **想起可能集合 Conceivable(P)**: P_pred(P) のキーの部分集合で、パラダイム P の概念体系において自然に想起される記述素の集合。パラダイムの「概念的な可視範囲」に対応する。データモデルに保持されるが、実行時のスコープ判定には使用しない（P_pred がスコープを担う）。R(P) のバリデーション（関係の src/tgt が想起可能であること）に使用する。
- **共有記述素**: 複数のパラダイムの P_pred に共通してキーを持つ記述素。P_pred(P₁) ∩ P_pred(P₂)。両方のパラダイムが予測を持つため、どちらの枠組みでも解釈可能である。
- **固有記述素**: あるパラダイムの P_pred にのみキーを持ち、他のどのパラダイムの P_pred にもキーがない記述素。P_pred(P) \ ∪_{P'≠P} P_pred(P')。そのパラダイムでのみ予測される事実に対応する。
- **説明力 explained_o(O, P)**: O の記述素のうち P_pred(P) が正しく予測するものの数。|{d ∈ O | P_pred(d) ≠ None かつ P_pred(d) = O(d)}|。P_pred 全体を対象とする。診断指標および depth の導出に使用する。
- **深度 (depth)**: パラダイムの説明力の大小関係から自動導出される順序。Explained(P) = {d | P_pred(d) = O*(d)} の包含関係に基づく。Explained(P) ⊂ Explained(P') ならば depth(P) < depth(P')。手動での割り当ては不要。

## 質問

- **ans(q, はい/いいえ/関係ない)**: 質問 q に対する各回答が O または R に追加する記述素と値の組。質問の一般的構造を定義する。
- **effect(q)**: 質問 q が実際に回答されたときの効果。T が各 q に対して正答 a_T(q) を決定し、effect(q) = ans(q, a_T(q)) として得られる。
- **関連記述素**: 質問テキストが概念的に前提とするが、回答によって真偽が更新されない記述素。効果記述素とともに質問の「参照する記述素」を構成する。
- **所属パラダイム (Question.paradigms)**: 質問が所属するパラダイムの集合。データで明示的に定義する。所属パラダイムに現パラダイムが含まれない質問はオープンしない。
- **open(H)**: H のもとでオープンされる質問の集合。以下の全条件を満たす質問がオープンされる: (1) 所属制約: 質問の所属パラダイムに P_current が含まれること（所属パラダイムが未定義の場合はこの制約を適用しない）、(2) 前提充足: q の prerequisites が全て O に含まれること、(3) H 値条件: ∃(d, v) ∈ effect(q) : H(d) ≈ v。所属制約は他の条件に先立って適用される。
- **Q(P)**: パラダイム P の質問集合。所属パラダイムに P を含む質問の集合。データで明示的に定義される所属に基づく。同一の質問が複数パラダイムの Q(P) に含まれることがある。
- **認知的前提 (prerequisites)**: 質問がオープンするために、プレイヤーが事前に認知していなければならない記述素の集合。アノマリーへの認知的到達経路を構成する条件であり、論理的成立条件ではなく認知的条件として設計される。
- **ゲート辺**: R(P) における浅い層の記述素から深い層の記述素への関係。同化により深い層の記述素の H が閾値に近づき、次の層の質問がオープンする。R(P) の形式定義（P_pred × P_pred × W）は変えず、構成方法上の概念として区別される。横辺（同一層内の因果・含意関係）と対をなす。
- ~~共有質問~~: 廃止。Q(P) による独立した質問集合の構成に置換。シフト方向の制御はシフト方向検証（シミュレーション）が担う。
- ~~架橋質問~~: 廃止。同上。

## 関係

- **アノマリー**: 観測 O の中で、パラダイム P の予測と矛盾する記述素。d ∈ P_pred(P) かつ d ∈ O かつ P_pred(d) ≠ O(d) であるもの。
- **アノマリー集合 Anomaly(O, P)**: {d ∈ P_pred(P) | d ∈ O かつ P_pred(d) ≠ O(d)}。tension(O, P) = |Anomaly(O, P)|。
- **緊張 (tension)**: 現在のパラダイムに対するアノマリーの蓄積度。tension(O, P) = |Anomaly(O, P)|。単調非減少。シフト候補の条件（tension strict `<`）に用いる。
- **resolve**: P_current のアノマリーのうち、P_target が正しく予測するものの数。resolve(O, P_current, P_target) = |{d ∈ Anomaly(O, P_current) | P_target.P_pred(d) = O(d)}|。シフト候補の条件（resolve >= N）および候補の優先順位に用いる。
- **attention**: P_current のアノマリーのうち、P_target の予測範囲に含まれるものの数。attention(O, P_current, P_target) = |{d ∈ Anomaly(O, P_current) | d ∈ P_pred(P_target)}|。候補の優先順位（resolve の同点時）に用いる。
- **近傍 (neighbors)**: パラダイム間の構造的な近接関係。完全確定 O* のもとで Remaining 集合の包含関係から導出される。シフト候補の前提条件として用いる。詳細は「近傍」規則を参照。
- **閾値 N(P_target)**: 遷移先パラダイムの resolve 閾値。N(P_target) = min over neighbor sources of resolve(O*, P_i, P_target)。詳細は「パラダイムシフトの機構」を参照。
- **整合性 (alignment)**: 候補パラダイムが暫定理論 H とどれだけ整合するかの指標。alignment(H, P) = (Σ_{d, P_pred(d)=1} H(d) + Σ_{d, P_pred(d)=0} (1 - H(d))) / |P_pred(P)|。P_pred 全体を対象とする。診断指標として使用し、シフト先選択には使用しない。

## 操作とアルゴリズム

- **記述素展開 (E)**: 記述素群から記述素空間を生成する操作。E(Ps) = Fs、E(Pt) = Ft。E の選択が記述素空間の広さと形を決定する。複数のアルゴリズムを比較する場合は E1, E2... と表記する。
- **展開規則**: 記述素展開を構成する個々のルール。記述素に対して適用し、新たな記述素を生成する。例：前提抽出、5W1H展開、含意導出。
- **パラダイム発見**: 記述素空間からパラダイム群を見出す操作。複数のアルゴリズムを比較する場合は A, B... と表記する。

### アルゴリズムの性質

本フレームワークにおけるアルゴリズムは、完全に機械的な手続きではなく、適用者の解釈や想像力に依存する部分を含む構造化された方法論である。異なる適用者が同じ入力に適用した場合、細部は異なりうるが、大まかな方向性（主要な構造や分類）は一致することが期待される。

この性質は記述素展開とパラダイム発見に共通する。質問生成は記述素からの変換であり、より機械的な手続きとして定義しうる可能性がある。

## 更新メカニズム

O に記述素が追加されたときの H の更新は、以下の二つの要因の相互作用で決まる。

### 空間構造の寄与

- 各パラダイムが固有の記述素間の関係構造を持つ
- 確定した記述素がどのパラダイムの関係構造のもとで解釈されるかによって、他の記述素への波及が異なる

### H の寄与（主観的能動性）

- **同化**: H は新たな観測を現在集中しているパラダイム内で解釈しようとする。同化のスコープは P_pred のキー集合（O との共通部分で予測が一致する記述素から伝播）。
- **抵抗**: H の集中が強いほど、パラダイムの切り替えに対する抵抗が大きい

### 更新の力学

1. O に記述素が追加されると、H はまず現在のパラダイム内で吸収を試みる（同化・抵抗）
2. 追加された観測が現在のパラダイムの予測と矛盾する場合、アノマリーとして緊張が蓄積する
3. シフト候補（近傍 + tension strict `<` + resolve >= N）が出現すると、パラダイムシフトが発生する

- **微調整**: 観測が現在のパラダイム内で吸収され、パラダイムの構成が維持される更新。
- **パラダイムシフト**: アノマリーの蓄積により、近傍のパラダイムがシフト候補条件を満たし、H がそのパラダイムへ移行する更新。詳細は「パラダイムシフトの機構」を参照。
