# パラダイムの近傍

## 概要

パラダイム間の構造的な近接関係を定義する。完全確定 O* のもとでのアノマリー集合の射影に基づき、遷移時の跳躍（中間パラダイムを飛ばした遷移）を防止する。

## 定義

### Remaining 集合

P_current のアノマリー集合を基準に、各候補パラダイム P' がどれだけ「残課題」を持つかを定量化する。

**Remaining(P', P_current) = Anomaly(O*, P_current) ∩ Anomaly(O*, P')**

P' が P_current のアノマリーをどれだけ共有しているか。Remaining が小さいほど、P' は P_current の問題をより多く解決している。

### Hasse 図による近傍

Remaining 集合の包含関係で Hasse 図を構築する。

Remaining(P_a) ⊂ Remaining(P_b) ⊂ Anomaly(P_current) のとき、P_b は P_a と P_current の中間にある。この場合 P_a は P_current の近傍ではない（P_b が中間に存在するため）。

**P' が P_current の近傍 ⟺ Remaining(P') は Anomaly(P_current) の真部分集合であり、かつ Remaining(P') ⊂ Remaining(P'') ⊂ Anomaly(P_current) を満たす中間パラダイム P'' が存在しない**

すなわち Hasse 図における直接の後続のみが近傍である。

## 計算

1. 各パラダイムについて Anomaly(O*, P) を計算する
2. 各 P_current について、全候補 P' の Remaining(P', P_current) を計算する
3. Remaining 集合の包含関係で Hasse 図を構築する
4. 直接後続を P_current の neighbors に設定する

前提条件として tension strict `<` を組み込む: tension(O*, P') >= tension(O*, P_current) の場合、P' は近傍にしない。

## 性質

- **静的**: O* から一意に計算される。質問の経路に依存しない。
- **非対称**: P_a が P_b の近傍であっても、P_b が P_a の近傍とは限らない。
- **跳躍防止**: 中間パラダイムが存在する遷移を構造的に排除する。
- **ループ非依存**: ループ防止は tension strict `<` が担当する。近傍自体にループ耐性は不要。

## 引っ掛けパラダイムの扱い

引っ掛け（トリック）パラダイム S は、メインパスのパラダイムに導入アノマリー（P_current にはないが S にはある新たなアノマリー）を持ちうる。

- tension strict `<` により、S の total アノマリーが P_current より少なければ遷移可能
- 導入アノマリーがあっても、ネットで tension が減少していれば許容
- ループの安全性: S → P_current は tension strict `<` により成立しない
