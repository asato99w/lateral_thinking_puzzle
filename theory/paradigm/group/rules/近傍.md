# パラダイムの近傍

## 定義

パラダイム P の近傍 N(P) を、記述素の重なりによって定義する。

**N(P) = { P' ∈ Paradigms | D(P) ∩ D(P') ≠ ∅, P' ≠ P }**

D(P) を共有するパラダイム同士は、同じ記述素について予測を持つため、互いの観測が影響しうる関係にある。重なりがなければ、一方の観測は他方の緊張にも整合性にも寄与しない。

## 共有記述素の内部構造

近傍 P, P' の共有記述素 D(P) ∩ D(P') は、予測の関係によって二つに分かれる。

- **予測一致**: 両パラダイムが同じ値を予測する記述素。d ∈ D⁺(P) ∩ D⁺(P') または d ∈ D⁻(P) ∩ D⁻(P')。
- **予測衝突**: 両パラダイムが異なる値を予測する記述素。d ∈ D⁺(P) ∩ D⁻(P') または d ∈ D⁻(P) ∩ D⁺(P')。

予測一致の記述素は、その観測がどちらのパラダイムにとってもアノマリーにならない（またはどちらにとってもなる）。予測衝突の記述素は、必ず一方にアノマリーを、他方に確認を与える。

## 閾値の導出

完全確定 O* のもとで、各パラダイムのアノマリー数 anomalies(O*, P) が確定する。

パラダイム P の緊張閾値 threshold(P) を以下で定義する。

**threshold(P) = max { anomalies(O*, P') | P' ∈ N(P), anomalies(O*, P') < anomalies(O*, P) }**

すなわち、P の近傍のうち P より少ないアノマリーを持つパラダイムの中で、最もアノマリーが多いもの（P に最も近い「より良い」近傍）のアノマリー数を閾値とする。

### 根拠

- tension(O, P) は観測の蓄積とともに anomalies(O*, P) に単調に近づく
- tension が threshold(P) を超えた時点で、少なくとも一つの近傍 P' が P よりアノマリーが少ない状態にある蓋然性が高い
- alignment の比較で P' がシフト先として選ばれる条件と整合する
- 完全確定は T によって一意に定まるため、質問の経路に依存しない

### 性質

- T（真理論を含むパラダイム）は anomalies(O*, T) = 0 であり、threshold は不要（シフト元にならない）
- 近傍に P より少ないアノマリーを持つパラダイムが存在しない場合、P はその近傍の中で最も安定であり、シフト元にならない
- 健全性条件（アノマリーの単調減少）が満たされていれば、メインパスの各パラダイムに対して閾値が定まることが保証される
