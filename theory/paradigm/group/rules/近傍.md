# パラダイムの近傍（廃止）

> **この規則は廃止された。** 近傍が担っていた二つの機能は、いずれも新枠組みで代替される。

## 廃止の経緯

旧枠組みでは近傍 N(P) = { P' | D(P) ∩ D(P') ≠ ∅ } が以下の二つの目的で使用されていた:

1. **シフト候補の絞り込み**: 近傍のうちアノマリーがより少ないパラダイムに限定
2. **閾値の導出**: 近傍のうち最もアノマリーが多い「より良い」パラダイムのアノマリー数を閾値とする

新枠組みではいずれも代替手段がある:

1. **シフト候補の絞り込み**: explained_o(O, P') > explained_o(O, P_current) で代替。P_pred 全体を使った説明力の比較であり、近傍条件は不要。
2. **閾値の導出**: 固有想起アノマリー + min 共有想起アノマリーで代替。Conceivable の固有・共有構造に基づく導出であり、近傍を経由しない。
