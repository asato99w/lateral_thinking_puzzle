# 関係構造の構成

パラダイムの所属記述素（D⁺, D⁻）が決定された後、R(P) を構成するルール。

## 前提

R(P) ⊆ D(P) × D(P) × W であり、(d₁, d₂, w) は「P の説明において d₁ が d₂ の理由・原因・前提である」ことを表す。

## Rule 1: D⁺ 内の関係

D⁺(P) 内の記述素間の因果・含意関係。P の肯定的説明の中の論理構造。

例: P1 において Fs-01（食事目的で来店） → Fs-05（特定の目的）は、P1 の説明の中で前者が後者を含意する。

## Rule 2: D⁺ から D⁻ への関係

D⁺ の事実が成り立つことが、D⁻ の事実が成り立たないことの根拠となる関係。

例: P1 において Fs-01（食事目的で来店） → Ft-03（命の危険にさらされた）=0 は、日常の食事目的が極限体験を排除する関係。

この関係は同化において D⁻ 記述素の H を 0 方向に押す効果を持つ。

## Rule 3: 共有記述素への接続

D⁺ から共有記述素（隣接パラダイムの D(P) との交差部分）への関係。段階的開示の連鎖を可能にするために重要。

- 固有記述素 → 共有記述素 の関係が存在すること
- 同化により共有記述素の H が ε 圏内に到達できる w であること

## Rule 4: 強さの設定

w の値は以下の基準で設定する:

- **強い関係（w > 0.6）**: 1ホップの同化で open(H) の ε 圏内に到達する。確認すれば次が自然に問われる関係。
- **中程度の関係（0.3 < w ≤ 0.6）**: 単独では ε 圏内に到達しないが、複数ソースの合流で到達しうる。複数の裏付けが揃って見えてくる関係。
- **弱い関係（w ≤ 0.3）**: 直接的な同化効果は小さい。背景的な関連性。
