# 暫定理論とゲーム構造

## 暫定理論

暫定理論とは、プレイヤーが現時点で持っている「何が起きたのか」の説明である。理論側のパラダイムとは異なり、プレイヤーの内的な仮説であり、パラダイムと近似的に対応するが完全には一致しない。

## 暫定理論と記述素の関係

暫定理論を持つプレイヤーの H は、記述素空間の各要素に対して3つのグループを形成する。

- **H(d) ≈ 1**: 暫定理論が「真」と予測する記述素群
- **H(d) ≈ 0**: 暫定理論が「偽」と予測する記述素群
- **不定**: 暫定理論が予測を持たない記述素群

H(d) ≈ 1 と H(d) ≈ 0 は対称な概念である。いずれもプレイヤーの暫定理論の積極的な構成要素であり、「何であるか」と「何でないか」の両方が説明を形成する。

パラダイムはこの3分類のパターンとして特徴づけられる。パラダイムシフトとは、このパターンが組み変わることである。パターンの組み換えには、肯定の変化だけでなく否定の変化も含まれる。

## 暫定理論と T のギャップ

各記述素について、暫定理論における H(d) と T における d の値との差がギャップである。ギャップには二つの方向がある:

- **H(d) ≈ 0, T(d) = 1**: プレイヤーが偽と考えているが実際は真。否定の誤り。
- **H(d) ≈ 1, T(d) = 0**: プレイヤーが真と考えているが実際は偽。肯定の誤り。

いずれの方向のギャップも大きい場合、その記述素に関する質問は提示すべきでない。プレイヤーの暫定理論から遠すぎる情報は、理解の文脈を欠き、ゲーム体験を損なう。

## 質問のオープン条件

オープンされる質問の集合は H の関数として定義される。

**open(H) = { q | ∃(d, v) ∈ effect(q) : H(d) ≈ v }**

質問 q がオープンされるのは、effect(q) の中に、プレイヤーの信念 H(d) と整合する記述素 d が存在するときである。プレイヤーが強い信念を持ち、かつその信念と一致する効果を持つ質問のみがオープンされる。H(d) と v が矛盾する質問はオープンされない。

これにより、プレイヤーの現在のパラダイムに接地された質問のみが提示される。遠いパラダイムに属する記述素の質問は、H がそこに極端な値を持たないため自然にオープンされない。

## フェーズ構造

パラダイムがフェーズを構成する。各フェーズの質問群には、決定的な影響力を持つ質問が含まれる。

- **決定的な質問を引き当てた場合**: 暫定理論が大きく T 方向へ偏り、早期に次のフェーズへ進む（最短経路）
- **全質問を消化した場合**: O の蓄積が次のフェーズのオープン条件を必ず満たす（完全経路の保証）

パラダイム間の遷移は一方通行ではなく、並行・収束・後退がありうる。ただしフェーズの質問群を消化していくことで、大区分の構造に沿って最終的に T に収束する。

## ゲーム要件

1. **最短経路の存在**: 答えを知るプレイヤーが、最小の質問数で自然に T へ到達できる
2. **完全経路の保証**: 全質問消化後に、誰でも T に至る構成である

後者は閾値設定の制約となる。各フェーズの質問群が全て回答された状態での O の蓄積が、次のフェーズのオープン条件を満たすことを保証しなければならない。この要件とゲームの面白さの両立が設計上の核心的な難しさである。
