# A: 記述素直接変換型

## 入力

記述素空間 F

## 手順

1. F から以下を除外し、質問到達候補を得る
   - Ps（既に O に含まれる）
   - 合成により生成された記述素
   - 異常性検出により生成された記述素
   - 緊張関係抽出により生成された記述素

2. 含意関係にある記述素をグルーピングする。d₁ が確定すれば d₂ も自明に確定する場合、d₁ を代表とし d₂ をグループに含める

3. 各代表記述素を質問形式に変換し、ans 写像を定義する

## 出力

質問群 Q（ans 写像付き）

## 特徴

- シンプルで網羅的
- 質問数が多くなりやすい
- 体験設計の視点が欠ける（どの質問が重要か、どの順で提示すべきかの情報がない）
