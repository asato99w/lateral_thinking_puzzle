# 04 仮説事実連鎖の展開

## 方針

探索領域のオープントリガーに含まれる仮説について、形成条件（事実集合の族）を再帰的に展開する。S の事実（初期観測）以外は仮説として扱い、質問の回答で Yes を得た時点で事実に確定する。

暫定の展開深さ: 5（トリガーの仮説から数えて 5 層まで）

## 探索領域 a のトリガー仮説の展開

### 仮説「遭難した」

```
仮説「遭難した」
  ← {船乗りである, 危機的な状況を経験した}
  ← {船乗りである, 特殊な過去がある}
  ← {過去に海上にいた, 危機的な状況を経験した}
```

### 仮説「船乗りである」

```
仮説「船乗りである」
  ← {男の職業が関係する, 海に関わる職業である}
  ← {過去に海上にいた}
```

### 仮説「過去に海上にいた」

```
仮説「過去に海上にいた」
  ← {男の過去が関係する, 海が関係する}
```

### 仮説「海が関係する」

```
仮説「海が関係する」
  ← {ウミガメが海の生物である, スープの由来が重要}
  ← {Ps-2(ウミガメのスープを注文した)} ← S の事実（末端）
```

### 仮説「スープの由来が重要」

```
仮説「スープの由来が重要」
  ← {Ps-4(確認行為), Ps-2(ウミガメのスープを注文した)} ← S の事実（末端）
```

### 仮説「危機的な状況を経験した」

```
仮説「危機的な状況を経験した」
  ← {特殊な過去がある, 生死に関わる出来事があった}
  ← {男の過去が関係する, 自殺するほどの体験がある}
```

### 仮説「特殊な過去がある」

```
仮説「特殊な過去がある」
  ← {自殺の原因は日常的なものではない}
  ← {男の過去が関係する}
```

### 仮説「自殺の原因は日常的なものではない」

```
仮説「自殺の原因は日常的なものではない」
  ← {スープに問題はない, 病気ではない}
  ← {スープに問題はない, 店員は正直}
  ← {病気ではない, 経済的問題ではない}
```

### 仮説「スープに問題はない」

```
仮説「スープに問題はない」
  ← {Ps-3(一口飲んだ), Ps-5(店員は本物と答えた)} ← S の事実（末端）
```

### 仮説「病気ではない」

```
仮説「病気ではない」
  ← {Ps-6(自殺した)} ← S の事実（末端）
  ← {Ps-1(レストランに入った)} ← S の事実（末端）
```

### 仮説「店員は正直」

```
仮説「店員は正直」
  ← {Ps-5(店員は本物と答えた)} ← S の事実（末端）
```

### 仮説「男の過去が関係する」

```
仮説「男の過去が関係する」
  ← {Ps-4(確認行為)} ← S の事実（末端）
  ← {自殺の原因は日常的なものではない, Ps-6(自殺した)}
```

### 仮説「仲間が死んだ」

```
仮説「仲間が死んだ」
  ← {仲間がいた, 危機的な状況を経験した}
```

### 仮説「仲間がいた」

```
仮説「仲間がいた」
  ← {男の過去が関係する, 他の人物が関与する}
  ← {遭難した} ← 上位の仮説に接続
```

### 仮説「他の人物が関与する」

```
仮説「他の人物が関与する」
  ← {自殺の原因は男一人の問題ではない}
  ← {Ps-4(確認行為)} ← S の事実（末端）
```
