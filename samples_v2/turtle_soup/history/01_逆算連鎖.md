# 01 逆算連鎖

## 入力

- S: ある男がレストランに入り、ウミガメのスープを注文した。一口飲んで店員に「これは本当にウミガメのスープか？」と尋ねた。店員は「はい、間違いなくウミガメのスープです」と答えた。男はその夜、自殺した。なぜか？
- T: 男はかつて仲間と船で遭難し、食料が尽きた。仲間の一人が死に、残った者はその肉でスープを作り「ウミガメのスープだ」と偽って男に飲ませた。レストランで本物のウミガメのスープを飲み、味が違うことに気づいた男は、あのとき飲まされたものの正体を悟り、絶望して自殺した。

## 実施内容

T を「なぜ？」の連鎖で逆算した。

```
男は自殺した
  ← なぜ？ 自分が人肉を食べていたと悟ったから
    ← なぜ悟った？ 過去に飲んだ「ウミガメのスープ」が本物ではなかったと判明したから
      ← なぜ判明した？ 本物のウミガメのスープを飲んだら、記憶の味と違ったから
        ← なぜ記憶がある？ 過去に「ウミガメのスープだ」と言われてスープを飲んだから
          ← なぜ偽った？ 遭難中に仲間の肉で作ったスープを、そう偽って提供したから
            ← なぜそうなった？ 食料が尽き、仲間が死んだから
```

各ステップが「なぜ？」に対する必然的な回答であり、恣意性が少ない。
