# 02 ピース抽出

## 実施内容

逆算連鎖の各ステップからピースを抽出し、依存関係を特定した。

依存関係は「このピースの成立に他のピースの知識が前提か」で判断する。逆算連鎖の矢印（←）は因果の方向だが、依存はそれと一致するとは限らない。

### ピース一覧

| ID | ラベル | 依存 | 根拠 |
|----|--------|------|------|
| a | 食料が尽き、仲間が死んだ | なし | 他のピースなしに成立する |
| b | 仲間の肉でスープを作った | a, c | 仲間が死んだこと(a)と偽りのラベル(c)の両方がないと成立しない |
| c | 「ウミガメのスープ」と偽って提供された | なし | 他のピースなしに成立する |
| d | 男はそれを飲んだ（味の記憶がある） | c | 偽りのスープを飲まされた文脈(c)が前提 |
| e | 本物を飲んだら味が違った | なし | レストランでの体験として独立に成立する |
| f | 過去のものは本物ではなかったと判明した | d, e | 過去に飲んだこと(d)と本物との比較(e)の両方が必要 |
| g | 正体を悟り自殺した | a, b, c, d, e, f | 全体の理解が前提 |

独立ピース: a, c, e
依存ピース: b, d, f, g
