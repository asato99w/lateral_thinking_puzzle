# 10 戦略連鎖の構成

## 実施内容

仮説連鎖の末端のうち、S の事実に直接到達しないものに対して汎用戦略を適用し、S への鎖を構成した。

### S に直接到達しない末端要素

04 の展開から、以下の末端が S に到達していない。

| 末端仮説 | 適用戦略 | 接続先の S 事実 |
|----------|----------|----------------|
| 男の職業が関係する | 登場人物の探索 | Ps-1（レストランに入った） |
| 海に関わる職業である | 観測事実の深掘り | Ps-2（ウミガメのスープ → 海） |
| 生死に関わる出来事があった | 因果の逆転 | Ps-6（自殺した） |
| 自殺するほどの体験がある | 因果の逆転 | Ps-6（自殺した） |
| 経済的問題ではない | 常識的仮説の排除 | Ps-1（レストランに行ける） |
| 自殺の原因は男一人の問題ではない | 観測事実の深掘り | Ps-4（確認行為 → 他者の関与） |

### 構成された戦略連鎖の例

```
Ps-6(自殺した) → 因果の逆転 → 「生死に関わる出来事があった」
  → ... → 「危機的な状況を経験した」 → 「遭難した」 → ピース a のトリガー
```

```
Ps-1(レストランに入った) → 登場人物の探索 → 「男の職業が関係する」
  → 「船乗りである」 → 「過去に海上にいた」 → ピース a のトリガー
```

```
Ps-4(確認行為) → 観測事実の深掘り → 「スープの由来が重要」
  → ピース c, e のトリガー
```

### 備考

ピース c, e のトリガー仮説は S の事実から直接到達するため、戦略連鎖の構成は不要。戦略連鎖が必要なのは主にピース a のトリガーに至る経路。
