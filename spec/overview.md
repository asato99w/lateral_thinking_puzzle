# アプリ仕様概要

## コンセプト

水平思考クイズのモバイルアプリ。AI を使用せず、理論的枠組みに基づいた事前設計のデータ構造でゲーム進行を制御する。

## 基本情報

- **プラットフォーム**: モバイルアプリ（iOS / Android）
- **対象**: プレイヤー向け
- **AI**: 不使用

## ゲームの基本構造

- プレイヤーは選択式の質問を通じて水平思考クイズを解く
- 質問は徐々にオープンされていく（ハイブリッド型）
  - **観測連動**: 特定の記述素が O に含まれると、それを前提とする新たな質問がオープンされる（微調整に対応）
  - **パラダイムシフト連動**: パラダイムシフトの条件を満たすと、質問群がまとまってオープンされる（革命に対応）

## 理論との対応

| 理論的概念 | アプリでの実現 |
|-----------|-------------|
| 記述素 | 質問と回答のペアとして表現 |
| O（観測記述素群） | プレイヤーが選択した質問の回答結果 |
| R（不関与記述素群） | 「関係ない」と回答された質問 |
| パラダイム | 質問群のグループ |
| パラダイムシフト | 新たな質問群のまとまったオープン |
| 微調整 | 個別の質問のオープン |
| 説明力 | 進行度・ヒントの判定基準 |
